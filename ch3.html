第3章 功和能

3.1 功

功的定义

恒力的功：$W = \vv{F}\cdot\Delta\vv{r} = F\cos\theta\cdot s$

变力的功（第二类曲线积分）：

$$W = \int_A^B \vv{F}\cdot\dd\vv{r} = \int_A^B (F_x\dd x + F_y\dd y + F_z\dd z)$$

功率：$P = \frac{\dd W}{\dd t} = \vv{F}\cdot\vv{v}$

注意：功是标量，有正负。$0°<\theta<90°$ 做正功；$90°<\theta<180°$ 做负功；$\theta=90°$ 不做功。功与参考系有关！
例1：弹簧力 $F=-kx$ 从 $x_1$ 到 $x_2$ 做的功。
解
$W = \int_{x_1}^{x_2}(-kx)\dd x = -\frac{1}{2}k(x_2^2-x_1^2) = \frac{1}{2}kx_1^2 - \frac{1}{2}kx_2^2$
例2：万有引力 $F = -G\frac{Mm}{r^2}$（引力方向沿 $r$ 减小方向），从 $r_1$ 到 $r_2$ 做的功。
解
$$W = \int_{r_1}^{r_2}\left(-\frac{GMm}{r^2}\right)\dd r = GMm\left(\frac{1}{r_2}-\frac{1}{r_1}\right)$$

3.2 动能定理

质点动能定理

动能：$E_k = \frac{1}{2}mv^2$

$$W_{合} = \Delta E_k = \frac{1}{2}mv_2^2 - \frac{1}{2}mv_1^2$$

合外力做的功 = 动能的增量

质点系动能定理
$$W_{外} + W_{内} = \Delta E_k$$

所有外力做的功 + 所有内力做的功 = 系统总动能的增量

易错：质点系中内力可以做功（如弹簧弹开两物体），不能忽略！

3.3 保守力与势能

保守力

力 $\vv{F}$ 是保守力，当且仅当做功只与始末位置有关，与路径无关。等价条件：$\oint\vv{F}\cdot\dd\vv{r}=0$

常见保守力：重力、弹力、万有引力、库仑力

非保守力：摩擦力、空气阻力

势能

保守力做功 = 势能减少量：$W_{保} = -(E_{p2}-E_{p1}) = -\Delta E_p$

常见势能：

类型势能零点
重力势能$E_p = mgh$自选参考面
弹性势能$E_p = \frac{1}{2}kx^2$自然长度处
万有引力势能$E_p = -\frac{GMm}{r}$$r\to\infty$ 处为零
势能与力的关系
$$F_x = -\frac{\partial E_p}{\partial x},\quad F_y = -\frac{\partial E_p}{\partial y},\quad F_z = -\frac{\partial E_p}{\partial z}$$

即 $\vv{F} = -\nabla E_p$（力指向势能减小最快的方向）

势能曲线分析

在 $E_p(x)$ 图上：

  • $E_p'(x_0)=0$ → 平衡点。$E_p''(x_0)>0$ 稳定平衡；$E_p''(x_0)<0$ 不稳定平衡
  • 动能 $E_k = E - E_p(x)$，粒子只能在 $E_p(x)\leq E$ 的区域运动
  • $E_p(x)=E$ 的点为回转点（turning point）

3.4 功能原理与机械能守恒

功能原理

系统机械能 $E = E_k + E_p$

$$W_{外} + W_{非保内} = \Delta E = \Delta E_k + \Delta E_p$$

外力做的功 + 非保守内力做的功 = 机械能的增量

机械能守恒定律

条件：$W_{外}=0$ 且 $W_{非保内}=0$（只有保守内力做功）

$$E_k + E_p = \text{const}$$ $$\frac{1}{2}mv_1^2 + E_{p1} = \frac{1}{2}mv_2^2 + E_{p2}$$
例3（真题高频：圆弧物块+小球）：光滑水平面上质量 $M$ 的圆弧物块静止，质量 $m$ 的小球从顶端滑到底端（高度 $R$），求底端两者速度。
解

动量守恒（水平方向无外力）：$0 = mv_1 - Mv_2$，即 $mv_1 = Mv_2$

机械能守恒：$mgR = \frac{1}{2}mv_1^2 + \frac{1}{2}Mv_2^2$

由 $v_2 = \frac{m}{M}v_1$ 代入：$mgR = \frac{1}{2}mv_1^2(1+\frac{m}{M})$

$$v_1 = \sqrt{\frac{2gRM}{m+M}},\quad v_2 = \frac{m}{M}\sqrt{\frac{2gRM}{m+M}}$$

3.5 碰撞

一维弹性碰撞公式

$m_1$ 以 $v_1$ 碰静止的 $m_2$（动量+动能都守恒）：

$$v_1' = \frac{m_1-m_2}{m_1+m_2}v_1,\quad v_2' = \frac{2m_1}{m_1+m_2}v_1$$

特殊情况：$m_1=m_2$ → 速度交换；$m_1\gg m_2$ → $v_1'\approx v_1,\;v_2'\approx 2v_1$

完全非弹性碰撞

碰后两者粘合（$v'$相同），动量守恒：$m_1v_1 = (m_1+m_2)v'$

能量损失：$\Delta E_k = \frac{1}{2}\frac{m_1m_2}{m_1+m_2}v_1^2$（折合质量 $\mu = \frac{m_1m_2}{m_1+m_2}$）

3.6 变力做功典型计算（PPT重点）

变力做功的三种常见类型
  1. 力是位置的函数 $\vv{F}=\vv{F}(x,y)$：直接线积分 $W=\int_A^B\vv{F}\cdot\dd\vv{r}$
  2. 力是速度的函数 $F=F(v)$：先由牛二求 $v(t)$ 或 $v(x)$，再积分
  3. 力是时间的函数 $F=F(t)$：先求 $v(t)$，再求 $x(t)$，最后积分
例（二维变力做功）：力 $\vv{F}=(3x^2+6xy)\ii+(3x^2+2y)\jj$（SI），质点从 $O(0,0)$ 沿抛物线 $y=x^2$ 移动到 $A(1,1)$，求做功。
解

沿曲线 $y=x^2$，$\dd y=2x\dd x$：

$$W=\int_0^1[(3x^2+6x\cdot x^2)\dd x+(3x^2+2x^2)\cdot 2x\dd x]$$ $$=\int_0^1(3x^2+6x^3+6x^3+4x^3)\dd x=\int_0^1(3x^2+16x^3)\dd x$$ $$=\left[x^3+4x^4\right]_0^1=1+4=5\,\text{J}$$

可以验证 $\frac{\partial F_x}{\partial y}=6x=\frac{\partial F_y}{\partial x}$，该力是保守力，功与路径无关。

例（功能原理综合：弹簧+滑块）：劲度系数 $k$ 的弹簧水平放置，一端固定。质量 $m$ 的滑块以 $v_0$ 冲向弹簧，动摩擦系数 $\mu$。求弹簧最大压缩量 $x_m$。
解

从接触到最大压缩（$v=0$）：

动能定理：$0-\frac{1}{2}mv_0^2=-\frac{1}{2}kx_m^2-\mu mg\cdot x_m$

$$\frac{1}{2}kx_m^2+\mu mg\cdot x_m-\frac{1}{2}mv_0^2=0$$

解二次方程：$x_m=\frac{-\mu mg+\sqrt{\mu^2m^2g^2+kmv_0^2}}{k}$（取正根）

3.7 势能曲线与平衡分析

势能曲线的物理信息
  • 力：$F=-\dd E_p/\dd x$，曲线下降→力为正方向
  • 平衡位置：$\dd E_p/\dd x=0$。极小→稳定平衡；极大→不稳定平衡
  • 运动范围：画 $E=E_k+E_p$ 水平线，$E_p\leq E$ 的区域才能到达
  • 转折点：$E_p=E$ 处 $v=0$

完全弹性非完全弹性完全非弹性
动量守恒守恒守恒
动能守恒部分损失最大损失
恢复系数 $e$$1$$(0,1)$$0$

解题策略：碰撞瞬间→动量守恒；碰前/碰后→可用能量方法。弹道摆必须分两阶段！

3.9 势能曲线分析专题（PPT重点）

势能曲线读图要点

给定 $E_p(x)$ 图和总能量 $E$ 水平线：

  1. 可达区域：$E\geq E_p(x)$ 的 $x$ 范围
  2. 动能：$E_k=E-E_p(x)$（竖直距离）
  3. 转折点：$E=E_p(x)$ 处，$v=0$
  4. 力的方向：$F=-\dd E_p/\dd x$，$E_p$ 下降方向为力的方向
  5. 稳定平衡：$E_p$ 极小点；不稳定平衡：$E_p$ 极大点
例（选择题常考）：Lennard-Jones势能 $E_p(r)=4\varepsilon[(\sigma/r)^{12}-(\sigma/r)^6]$ 描述分子间作用力。
分析

$r\to\infty$：$E_p\to 0$（无作用力）

$r=r_0=2^{1/6}\sigma$：$E_p$ 取极小 $-\varepsilon$（稳定平衡位置）

$r0$（斥力）

$r>r_0$：$F<0$（引力）

分子在 $r_0$ 附近做小振动（近似简谐）

选择题陷阱：
  • $E_p$ 为负不代表粒子被"束缚"——关键看总能量 $E$ 是否大于势阱边界
  • 平衡点处力为零但加速度为零，不代表速度为零
  • 力最大的位置是 $E_p$ 曲线斜率最大处，不是 $E_p$ 极值处

3.10 功和能解题策略总结

何时用动能定理 vs 机械能守恒 vs 功能原理
方法适用条件优势
动能定理 $W_{合}=\Delta E_k$任何情况万能，但需算所有力的功
机械能守恒 $E_{k1}+E_{p1}=E_{k2}+E_{p2}$只有保守内力做功不需要知道过程细节
功能原理 $W_{外}+W_{非保}=\Delta E$有非保守力做功可以处理摩擦等问题
解题建议：
  • 只涉及始末状态、不关心中间过程 → 优先用能量法
  • 需要求力或加速度 → 用牛顿定律
  • 涉及碰撞 → 碰撞瞬间用动量，碰前/碰后用能量
  • 涉及弹簧/重力/万有引力 → 定义势能，用机械能守恒

3.11 功的计算典型例题专题

例（引力做功）：质点受引力 $F = -A/x^2$（$A>0$，力沿 $x$ 轴负方向），求质点从 $x_1$ 运动到 $x_2$（$x_2>x_1>0$）过程中引力做的功。
解

力沿 $x$ 方向，直接积分：

$$W = \int_{x_1}^{x_2} F\dd x = \int_{x_1}^{x_2}\left(-\frac{A}{x^2}\right)\dd x = \left[\frac{A}{x}\right]_{x_1}^{x_2} = A\left(\frac{1}{x_2}-\frac{1}{x_1}\right)$$

物理分析：因为 $x_2>x_1>0$，故 $\frac{1}{x_2}<\frac{1}{x_1}$，所以 $W<0$。这说明质点远离力心时引力做负功，符合直觉。

顺便可以定义势能：$E_p(x) = -A/x$（取 $x\to\infty$ 为零势能点），可以验证 $W = E_p(x_1)-E_p(x_2) = -\Delta E_p$。

例（二维变力做功+保守力判断）：力 $\vv{F} = xy\ii + y^2\jj$（SI），质点沿直线 $y=x$ 从原点 $O(0,0)$ 移动到点 $A(1,1)$。求功，并判断 $\vv{F}$ 是否为保守力。
解

第一步：计算功

沿 $y=x$，有 $\dd y = \dd x$，且 $y=x$。代入线积分：

$$W = \int_O^A (F_x\dd x + F_y\dd y) = \int_O^A (xy\dd x + y^2\dd y)$$

用 $y=x$，$\dd y=\dd x$ 参数化：

$$W = \int_0^1 (x\cdot x\,\dd x + x^2\,\dd x) = \int_0^1 2x^2\dd x = \left[\frac{2x^3}{3}\right]_0^1 = \frac{2}{3}\,\text{J}$$

第二步：判断是否为保守力

保守力的充要条件（二维）：$\frac{\partial F_x}{\partial y} = \frac{\partial F_y}{\partial x}$

$$\frac{\partial F_x}{\partial y} = \frac{\partial (xy)}{\partial y} = x$$ $$\frac{\partial F_y}{\partial x} = \frac{\partial (y^2)}{\partial x} = 0$$

因为 $x \neq 0$（一般情况下），所以 $\frac{\partial F_x}{\partial y} \neq \frac{\partial F_y}{\partial x}$，该力不是保守力。

验证思路：若该力是保守力，则沿任意路径从 $O$ 到 $A$ 的功都相同。可以另选路径（先沿 $x$ 轴到 $(1,0)$，再沿 $y$ 轴到 $(1,1)$）计算验证： $$W' = \int_0^1 x\cdot 0\,\dd x + \int_0^1 y^2\dd y = 0 + \frac{1}{3} = \frac{1}{3}\,\text{J} \neq \frac{2}{3}\,\text{J}$$ 两条路径功不同，确认不是保守力。
例（链条问题——PPT经典）：质量为 $m$、长为 $L$ 的均匀链条放在光滑桌面上，有 $L/4$ 长度垂在桌边。求将链条全部拉上桌面所需做的功。
解

设链条线密度 $\lambda = m/L$。以桌面为零势能面。

方法一：直接积分（变力做功）

设某时刻垂下部分长度为 $x$（$0 \leq x \leq L/4$），则拉力需要克服垂下部分的重力：

$$F(x) = \lambda x g = \frac{mg}{L}x$$

将链条从垂下 $L/4$ 拉到全部上桌（$x$ 从 $L/4$ 变到 $0$），拉力做正功：

$$W = \int_{L/4}^{0} F(x)(-\dd x) = \int_0^{L/4} \frac{mg}{L}x\,\dd x = \frac{mg}{L}\cdot\frac{x^2}{2}\bigg|_0^{L/4} = \frac{mg}{L}\cdot\frac{L^2}{32} = \frac{mgL}{32}$$

方法二：用势能变化（更简洁）

初态：垂下部分长 $L/4$，其质心在桌面下方 $L/8$ 处

末态：链条全部在桌面上，势能为零

拉力做的功 = 系统势能增量（桌面光滑，无摩擦）：

$$W = \Delta E_p = 0 - \left(-\lambda\cdot\frac{L}{4}\cdot g\cdot\frac{L}{8}\right) = \frac{m}{L}\cdot\frac{L}{4}\cdot g\cdot\frac{L}{8} = \frac{mgL}{32}$$
易错点：
  • 链条是变质量问题，不能直接用 $W=Fs$，必须积分或用势能法
  • 垂下部分的质心在其中点处（均匀链条），即桌面下方 $x/2$ 处
  • 如果桌面有摩擦，还需加上克服摩擦力做的功

3.12 能量守恒综合应用（PPT大题）

例（光滑轨道+粗糙水平面）：质量为 $m$ 的滑块从高 $h$ 的光滑弧形轨道顶端由静止滑下，到达底端后进入动摩擦系数为 $\mu$ 的粗糙水平面。求：(1) 滑块到达底端时的速度；(2) 滑块在水平面上滑行的距离。
解

(1) 光滑轨道段——机械能守恒

轨道光滑，只有重力做功：

$$mgh = \frac{1}{2}mv_1^2$$ $$v_1 = \sqrt{2gh}$$

(2) 粗糙水平面——功能原理

水平面上，摩擦力做功使动能减为零：

$$0 - \frac{1}{2}mv_1^2 = -\mu mg\cdot s$$ $$s = \frac{v_1^2}{2\mu g} = \frac{2gh}{2\mu g} = \frac{h}{\mu}$$

或者一步到位：整个过程用功能原理

$$W_{非保} = \Delta E = (0+0) - (0+mgh) = -mgh$$ $$-\mu mg\cdot s = -mgh \implies s = h/\mu$$
解题技巧：如果中间过程有光滑段也有粗糙段，可以直接对整个过程列功能原理，不需要分段求解。重力做功等价于势能变化，只需关注非保守力（摩擦力）做的功。
例（弹簧枪+子弹打木块——动量与能量两阶段）：弹簧枪中弹簧被压缩 $x_0$，劲度系数为 $k$。弹簧释放后将质量为 $m$ 的子弹发射出去，子弹水平射入静止在光滑水平面上质量为 $M$ 的木块并嵌入其中。求：(1) 子弹出枪口速度；(2) 子弹嵌入木块后两者共同速度；(3) 碰撞中损失的动能。
解

第一阶段：弹簧释放——机械能守恒

$$\frac{1}{2}kx_0^2 = \frac{1}{2}mv_0^2$$ $$v_0 = x_0\sqrt{k/m}$$

第二阶段：子弹打木块——完全非弹性碰撞，动量守恒

碰撞瞬间内力远大于外力，动量守恒：

$$mv_0 = (m+M)v$$ $$v = \frac{mv_0}{m+M} = \frac{m}{m+M}\cdot x_0\sqrt{\frac{k}{m}} = \frac{x_0\sqrt{km}}{m+M}$$

第三阶段：计算碰撞中的动能损失

$$\Delta E_k = \frac{1}{2}mv_0^2 - \frac{1}{2}(m+M)v^2$$ $$= \frac{1}{2}mv_0^2 - \frac{1}{2}(m+M)\cdot\frac{m^2v_0^2}{(m+M)^2}$$ $$= \frac{1}{2}mv_0^2\left(1-\frac{m}{m+M}\right) = \frac{1}{2}\cdot\frac{mM}{m+M}\cdot v_0^2 = \frac{1}{2}\mu v_0^2$$

其中约化质量 $\mu = mM/(m+M)$。代入 $v_0$：

$$\Delta E_k = \frac{Mkx_0^2}{2(m+M)}$$

碰撞中损失的动能转化为热能和形变能。

两阶段法——解题核心思路：
  • 碰撞瞬间：时间极短，位移近似为零，只能用动量守恒（不能用能量守恒，除非完全弹性碰撞）
  • 碰撞前/后的运动过程：时间较长，可以用能量方法（机械能守恒/功能原理）
  • 绝对不能把碰撞和碰前/碰后的运动混在一个方程里！
功和能综合题解题流程
  1. 画图：标出初末位置、速度方向、各个力
  2. 分阶段：识别碰撞/爆炸等不可逆过程，将问题分为若干阶段
  3. 选方法：
    • 碰撞/爆炸瞬间 → 动量守恒
    • 只有保守力做功 → 机械能守恒
    • 有摩擦等非保守力 → 功能原理
    • 需要求力/加速度 → 牛顿定律
  4. 列方程：对每个阶段分别列出对应的守恒/定理方程
  5. 联立求解：各阶段通过公共速度或能量衔接

3.13 保守力判定方法详解（PPT补充）

保守力的三个等价判据
  1. 做功与路径无关：$\int_A^B\vv{F}\cdot\dd\vv{r}$ 仅取决于 $A$、$B$ 的位置
  2. 环路积分为零：$\oint\vv{F}\cdot\dd\vv{r}=0$
  3. 偏导数条件（二维）：$\frac{\partial F_x}{\partial y} = \frac{\partial F_y}{\partial x}$；（三维）：$\nabla\times\vv{F}=0$

三者完全等价，考试中用偏导数条件最快。

例（判断保守力并求势能）：力 $\vv{F}=(2xy+3)\ii+(x^2-4y)\jj$，判断是否为保守力，若是，求势能函数。
解

Step 1：判断

$$\frac{\partial F_x}{\partial y}=\frac{\partial(2xy+3)}{\partial y}=2x,\quad \frac{\partial F_y}{\partial x}=\frac{\partial(x^2-4y)}{\partial x}=2x$$

二者相等，故 $\vv{F}$ 是保守力。

Step 2：求势能函数

由 $F_x=-\frac{\partial E_p}{\partial x}$：

$$E_p=-\int F_x\dd x=-\int(2xy+3)\dd x=-(x^2y+3x)+g(y)$$

由 $F_y=-\frac{\partial E_p}{\partial y}$：

$$-\frac{\partial E_p}{\partial y}=x^2-g'(y)=x^2-4y \implies g'(y)=4y \implies g(y)=2y^2+C$$ $$\boxed{E_p=-(x^2y+3x)+2y^2+C}$$
求势能的一般方法：
  • 先对 $F_x$ 关于 $x$ 积分，得到含 $g(y)$ 的中间结果
  • 再对 $y$ 求偏导，与 $F_y$ 比较确定 $g(y)$
  • 常数 $C$ 由势能零点条件确定

3.14 广义动能定理（质点系）（PPT补充）

质点系的功能关系总览
定理公式含义
质点动能定理$W_{合}=\Delta E_k$合力功 = 单个质点动能变化
质点系动能定理$W_{外}+W_{内}=\Delta E_k$所有力（含内力）的总功 = 系统总动能变化
功能原理$W_{外}+W_{非保内}=\Delta E$非保守力做功 = 机械能变化
机械能守恒$\Delta E=0$无外力做功且无非保守内力做功
关键区别：
  • 质点动能定理中的"功"包括所有力（含内力），适用于单个质点
  • 功能原理中把保守内力的功"折算"为势能变化，只剩外力和非保守内力
  • 质点系动能定理→功能原理的推导关键：$W_{保内}=-\Delta E_p$

3.15 能量图与束缚态（PPT补充）

粒子在势能场中的运动状态分类
  • 束缚态：总能量 $E < $ 势阱边界高度，粒子被限制在有限区域内做往复运动
  • 非束缚态：$E$ 大于势阱边界，粒子可以逃逸到无穷远
  • 临界态：$E$ 恰好等于势阱边界，粒子可以到达边界但速度为零
例（逃逸速度推导）：地球表面一物体需要多大初速度才能脱离地球引力（不再返回）？
解

势能：$E_p(r) = -\frac{GMm}{r}$，$E_p(\infty)=0$

临界条件：$r\to\infty$ 时 $v\to 0$，即 $E=0$

$$\frac{1}{2}mv_{esc}^2 + \left(-\frac{GMm}{R}\right) = 0$$ $$v_{esc} = \sqrt{\frac{2GM}{R}} = \sqrt{2gR} \approx 11.2\,\text{km/s}$$

这就是第二宇宙速度（逃逸速度）。$E>0$ 即可逃逸，$E<0$ 则被束缚（做椭圆轨道运动）。

宇宙速度汇总：
  • 第一宇宙速度 $v_1=\sqrt{gR}\approx 7.9\,\text{km/s}$（环绕速度，贴地圆轨道）
  • 第二宇宙速度 $v_2=\sqrt{2gR}\approx 11.2\,\text{km/s}$（逃逸速度）
  • 关系：$v_2=\sqrt{2}\,v_1$

3.16 变力做功的高级技巧（MOOC/真题）

非线性弹力做功

当弹力不满足胡克定律，例如 $F = -kx^3$（非线性弹簧），做功必须通过积分计算：

$$W = \int_0^{x_m}(-kx^3)\dd x = -\frac{k x_m^4}{4}$$

若质量为 $m$ 的物体以初速 $v_0$ 压缩该弹簧，求最大压缩量 $x_m$：

由动能定理 $0 - \frac{1}{2}mv_0^2 = -\frac{k x_m^4}{4}$，解得：

$$x_m = \left(\frac{2mv_0^2}{k}\right)^{1/4}$$

注意指数为 $1/4$，与线性弹簧的 $1/2$ 次方不同。

万有引力做功

质量为 $m$ 的物体在万有引力场中从 $r_1$ 运动到 $r_2$，万有引力做的功：

$$W = \int_{r_1}^{r_2}\left(-\frac{GMm}{r^2}\right)\dd r = GMm\left(\frac{1}{r_2}-\frac{1}{r_1}\right)$$

这是保守力，功只与始末位置有关，与路径无关。

例（重物在液体中的运动——浮力+阻力变力问题，PPT）：一个底面积为 $S$、高度为 $h$、质量为 $m$ 的均匀长方体从水面上方高度 $y_0$（$y_0 > h$）处自由落入液体（密度 $\rho$）。忽略空气阻力，液体阻力与速度成正比。分析物体进入液体过程中的受力和速度变化。
解

阶段一：自由下落至接触液面

下落高度 $y_0$，到达液面时速度：$v_0 = \sqrt{2g y_0}$

阶段二：部分浸没（$0 < y < h$，$y$ 为浸入深度）

受力分析：重力 $mg$（向下）、浮力 $\rho g S y$（向上）、阻力 $-bv$（向上）

运动方程：$m\frac{\dd v}{\dd t} = mg - \rho g S y - bv$

浮力随浸入深度线性增加，需要分段处理。

阶段三：完全浸没（$y > h$）

浮力恒定为 $\rho g S h$，运动方程：$m\frac{\dd v}{\dd t} = mg - \rho g S h - bv$

若忽略液体阻力（$b=0$），用能量方法从液面到完全浸没的过程：

$$\frac{1}{2}mv^2 - \frac{1}{2}mv_0^2 = mgh - \int_0^h \rho g S y\,\dd y = mgh - \frac{1}{2}\rho g S h^2$$

完全浸没时速度：$v = \sqrt{v_0^2 + 2gh - \rho g S h^2/m}$

若 $v_0 = \sqrt{2g y_0}$ 且物体密度远大于液体密度，则 $v \approx \sqrt{2g(y_0 + h)}$。

例（变密度弹簧做功）：一根弹簧的弹力为 $F = -k_1 x - k_2 x^2$（$k_1, k_2 > 0$），$x$ 为弹簧的形变量。求弹簧从自然长度压缩到 $x_0$ 所需外力做的功。
解

外力 $F_{ext} = -F = k_1 x + k_2 x^2$（缓慢压缩，外力等于弹力大小）。

外力做的功（$x$ 从 $0$ 到 $x_0$，压缩取正方向）：

$$W_{ext} = \int_0^{x_0}(k_1 x + k_2 x^2)\dd x = \frac{1}{2}k_1 x_0^2 + \frac{1}{3}k_2 x_0^3$$

弹簧储存的弹性势能：$E_p = \frac{1}{2}k_1 x_0^2 + \frac{1}{3}k_2 x_0^3$

可以验证：$F = -\frac{\dd E_p}{\dd x} = -(k_1 x + k_2 x^2)$，与弹力一致。

解题技巧：当力是位置的复杂函数时，能量方法（动能定理或势能法）通常比直接用牛顿第二定律积分更简便。特别是只需要求始末状态的速度时，能量法可以绕过中间过程的复杂微分方程。

3.17 万有引力势能与轨道能量（MOOC/真题高频）

万有引力势能与轨道能量公式

引力势能（以 $r\to\infty$ 为零点）：

$$E_p = -\frac{GMm}{r}$$

圆轨道上的能量关系：

由向心力条件 $\frac{GMm}{r^2} = \frac{mv^2}{r}$，得动能：

$$E_k = \frac{1}{2}mv^2 = \frac{GMm}{2r}$$

势能：$E_p = -\frac{GMm}{r}$

总机械能：

$$E = E_k + E_p = \frac{GMm}{2r} - \frac{GMm}{r} = -\frac{GMm}{2r}$$
Virial定理（圆轨道）

圆轨道上的能量满足简洁的比例关系：

$$E_k = -\frac{E}{2} = -\frac{E_p}{2}$$

即动能等于总能量的相反数的一半，等于势能的相反数的一半。

这一关系在天体力学和量子力学（氢原子）中都成立。

椭圆轨道能量

椭圆轨道的总机械能仅由半长轴 $a$ 决定：

$$E = -\frac{GMm}{2a}$$

圆轨道是椭圆的特殊情况（$a = r$），公式一致。

例（2022期中真题：椭圆轨道的条件）：质量为 $m$ 的航天器在距地心 $r_1$ 处以速度 $u_1 = \sqrt{2\alpha GM/r_1}$ 沿垂直于径向的方向发射（$M$ 为地球质量）。求 $\alpha$ 的范围使轨道为椭圆。
解

Step 1：计算总能量

初始时速度垂直于径向，动能：

$$E_k = \frac{1}{2}mu_1^2 = \frac{1}{2}m\cdot\frac{2\alpha GM}{r_1} = \frac{\alpha GMm}{r_1}$$

势能：$E_p = -\frac{GMm}{r_1}$

总能量：

$$E = E_k + E_p = \frac{\alpha GMm}{r_1} - \frac{GMm}{r_1} = \frac{(\alpha - 1)GMm}{r_1}$$

Step 2：椭圆条件

椭圆轨道要求 $E < 0$（束缚态），即：

$$\alpha - 1 < 0 \implies \alpha < 1$$

Step 3：不撞击地面的条件

发射点在近地点（速度垂直于径向），角动量 $L = mu_1 r_1$。

远地点处同样 $L = mv_2 r_2$，且需要 $r_2 > R$（$R$ 为地球半径，此处假设 $r_1 > R$）。

利用椭圆轨道公式，半长轴 $a = -GMm/(2E) = r_1/[2(1-\alpha)]$。

近日点距离 $r_1 = a(1-e)$，由此解得偏心率和轨道参数。

为使近日点不低于地表（假设 $r_1$ 即为近日点），需要轨道是合理的椭圆。

另一方面，若 $\alpha$ 过小，速度不足以维持轨道，物体会落回地面。

圆轨道条件：$\alpha = 1/2$（此时 $u_1 = \sqrt{GM/r_1}$，恰好是圆轨道速度）。

$\alpha < 1/2$ 时，速度小于圆轨道速度，发射点变为远地点而非近地点，轨道会更低，可能撞击地面。

结论：

$$\boxed{\frac{1}{2} < \alpha < 1}$$

$\alpha = 1/2$：圆轨道；$\alpha = 1$：抛物线轨道（逃逸）。椭圆轨道要求 $1/2 < \alpha < 1$。

例（从高 $h$ 处落到地面——考虑重力变化）：质量为 $m$ 的物体从距地面高度 $h$ 处自由下落（忽略空气阻力），考虑万有引力随高度的变化，求落地速度。
解

用能量守恒（万有引力势能）：

$$\frac{1}{2}mv^2 + \left(-\frac{GMm}{R}\right) = 0 + \left(-\frac{GMm}{R+h}\right)$$ $$\frac{1}{2}mv^2 = GMm\left(\frac{1}{R} - \frac{1}{R+h}\right) = \frac{GMmh}{R(R+h)}$$ $$v = \sqrt{\frac{2GMh}{R(R+h)}}$$

近地面近似（$h \ll R$）：

$R + h \approx R$，且 $GM/R^2 = g$，故：

$$v \approx \sqrt{\frac{2GMh}{R^2}} = \sqrt{2gh}$$

退化为我们熟悉的自由落体公式。

数值感受：$h = 100\,\text{km}$ 时，精确值与近似值的相对误差约 $h/(2R) \approx 0.8\%$，近似非常好。但 $h \sim R$ 时误差不可忽略。

注意：近地轨道（$h \ll R$）才能用 $E_p = mgh$；远离地面时必须用 $E_p = -GMm/r$。判断标准：$h/R$ 是否远小于 1。对于航天问题，一般都需要用万有引力势能。

3.18 功能原理在流体中的应用（PPT补充）

粗糙面+弧形轨道综合问题的一般思路
  1. 识别各段轨道的性质（光滑/粗糙、直线/曲线）
  2. 全程用功能原理：$W_{摩擦} = \Delta E_k + \Delta E_p$
  3. 摩擦力做的功：$W_f = -\mu mg \cos\theta \cdot s$（$s$ 为接触面长度，$\theta$ 为接触面倾角）
  4. 弧形轨道上若光滑，只需考虑首末高度差
例（环形轨道最低点所需最小速度）：质量为 $m$ 的小球沿光滑竖直圆形轨道内侧运动，轨道半径为 $R$。求小球在最低点的最小速度，使其能通过最高点而不脱离轨道。
解

Step 1：最高点的临界条件

在最高点，重力提供向心力（法向力 $N = 0$ 为临界）：

$$mg = \frac{mv_{top}^2}{R} \implies v_{top} = \sqrt{gR}$$

Step 2：最低点到最高点的能量守恒

高度差 $= 2R$（最低点到最高点），轨道光滑：

$$\frac{1}{2}mv_{bottom}^2 = \frac{1}{2}mv_{top}^2 + mg(2R)$$ $$\frac{1}{2}mv_{bottom}^2 = \frac{1}{2}m(gR) + 2mgR = \frac{5}{2}mgR$$ $$\boxed{v_{bottom} = \sqrt{5gR}}$$

物理理解：最低点速度不是 $\sqrt{4gR}$（那只是克服高度差所需），还需要额外的 $\sqrt{gR}$ 对应最高点维持圆周运动的最低速度。

例（弹簧+竖直圆弧轨道）：弹簧（劲度系数 $k$）竖直放置，上端连接质量为 $m$ 的物体，弹簧被压缩 $x_0$ 后释放，物体沿光滑竖直圆弧轨道（半径 $R$）运动。求使物体到达轨道顶部刚好不脱离的 $x_0$。
解

设弹簧底端为轨道最低点，最高点在轨道顶部（高度 $2R$）。

最高点临界条件同上：$v_{top} = \sqrt{gR}$

从弹簧释放到最高点，机械能守恒（弹性势能→动能+重力势能）：

$$\frac{1}{2}kx_0^2 = \frac{1}{2}mv_{top}^2 + mg(2R)$$ $$\frac{1}{2}kx_0^2 = \frac{1}{2}m(gR) + 2mgR = \frac{5}{2}mgR$$ $$x_0^2 = \frac{5mgR}{k}$$ $$\boxed{x_0 = \sqrt{\frac{5mgR}{k}}}$$
例（弹簧弹射器+重力场问题）：水平弹簧（劲度系数 $k$）被压缩 $d$，弹射质量 $m$ 的小球。小球离开弹簧后沿光滑斜面（倾角 $\theta$）上滑。求小球沿斜面上滑的最大距离 $s$。
解

弹簧释放弹性势能，小球沿斜面上升高度 $h = s\sin\theta$。

机械能守恒：

$$\frac{1}{2}kd^2 = mgs\sin\theta$$ $$\boxed{s = \frac{kd^2}{2mg\sin\theta}}$$

若斜面有摩擦（动摩擦系数 $\mu$），则：

$$\frac{1}{2}kd^2 = mgs\sin\theta + \mu mg\cos\theta \cdot s$$ $$s = \frac{kd^2}{2mg(\sin\theta + \mu\cos\theta)}$$
解题建议：弧形轨道和弹簧问题中，关键是识别"临界条件"——通常是最高点法向力为零（$N=0$），这给出最高点最小速度，再用能量守恒反推初始条件。

3.19 碰撞中的能量损失详解（真题重点）

完全非弹性碰撞的能量损失

碰撞中损失的动能等于质心系中的全部动能：

$$\Delta E_k = \frac{1}{2}\mu v_{rel}^2$$

其中：

  • $\mu = \frac{m_1 m_2}{m_1 + m_2}$ 为约化质量（reduced mass）
  • $v_{rel} = v_1 - v_2$ 为碰前相对速度

这是完全非弹性碰撞（$e=0$）的情况，损失最大。

一般碰撞的能量损失（用恢复系数 $e$）

对于恢复系数为 $e$ 的碰撞：

$$\boxed{\Delta E_k = \frac{1}{2}\mu v_{rel}^2(1-e^2)}$$

特殊情况：

$e$ 值$\Delta E_k$碰撞类型
$e = 1$$0$完全弹性——无能量损失
$0 < e < 1$$\frac{1}{2}\mu v_{rel}^2(1-e^2)$非完全弹性——部分损失
$e = 0$$\frac{1}{2}\mu v_{rel}^2$完全非弹性——最大损失
例（弹道摆详细分析——真题高频）：质量为 $m$ 的子弹以水平速度 $v_0$ 射入悬挂的质量为 $M$ 的木块中并嵌入其中（完全非弹性碰撞），木块摆起后上升最大高度 $h$。
解

Phase 1：碰撞瞬间——动量守恒

碰撞时间极短，重力冲量可忽略：

$$mv_0 = (m + M)v$$ $$v = \frac{mv_0}{m + M}$$

Phase 2：碰后上升——机械能守恒

碰后木块+子弹整体以速度 $v$ 上升，到最高点速度为零：

$$\frac{1}{2}(m + M)v^2 = (m + M)gh$$ $$h = \frac{v^2}{2g} = \frac{m^2 v_0^2}{2g(m + M)^2}$$

Phase 3：碰撞中的动能损失

$$\Delta E_k = \frac{1}{2}mv_0^2 - \frac{1}{2}(m+M)v^2 = \frac{1}{2}mv_0^2\left(1 - \frac{m}{m+M}\right) = \frac{M}{m+M}\cdot\frac{1}{2}mv_0^2$$

即损失了初始动能的 $\frac{M}{m+M}$ 倍。由于 $M \gg m$（木块远重于子弹），几乎全部动能都在碰撞中损失了。

弹道摆的测量原理：通过测量 $h$ 反推子弹速度：

$$v_0 = \frac{m + M}{m}\sqrt{2gh}$$

这是 18 世纪测量子弹速度的经典方法。

例（部分穿透问题）：若子弹穿过木块后以速度 $v_1$ 飞出，木块速度 $v_2$，则：
解

动量守恒：$mv_0 = mv_1 + Mv_2$

动能损失：$\Delta E_k = \frac{1}{2}mv_0^2 - \frac{1}{2}mv_1^2 - \frac{1}{2}Mv_2^2$

这个动能损失转化为子弹和木块的形变能（热能）。注意此时 $\Delta E_k$ 小于完全非弹性碰撞的损失（因为穿透时两者仍有相对运动）。

碰撞中动能损失的能量去向：
  • 形变能：物体发生永久形变（塑性变形）所消耗的能量
  • 热能：碰撞面摩擦生热、内部分子运动加剧
  • 声能：碰撞产生声波
  • 在宏观碰撞中，形变能和热能占主要部分；声能通常很小
  • 完全弹性碰撞中，形变是可恢复的（弹性形变），无能量损失
