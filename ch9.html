第9章 热力学基础

大纲7学时——全课程学时最多的章节！期末重中之重！

9.1 热力学第一定律

热力学第一定律
$$Q = \Delta E + W$$

系统吸收的热量 = 内能增加 + 系统对外做的功

符号约定：吸热 $Q>0$，放热 $Q<0$；系统膨胀做功 $W>0$，被压缩 $W<0$

微分形式：$\delta Q = \dd E + \delta W = \dd E + p\dd V$

核心区分：$E$（内能）是状态量（只与始末状态有关）；$Q$ 和 $W$ 是过程量（与过程路径有关）。
理想气体内能

$E = \nu\frac{i}{2}RT$，只是温度的函数。

对任意过程：$\Delta E = \nu C_V\Delta T = \nu\frac{i}{2}R\Delta T$

$\Delta E$ 的万能公式：无论什么过程，理想气体内能变化只取决于温度变化：$\Delta E = \nu C_V(T_2-T_1)$。这是因为理想气体内能只是 $T$ 的函数。

9.2 四种等值过程（核心中的核心！）

等体过程（$V=\text{const}$）

等体过程

$W = \int p\dd V = 0$（不膨胀不做功）

$Q = \Delta E = \nu C_V\Delta T$

过程方程：$\frac{p_1}{T_1} = \frac{p_2}{T_2}$

等压过程（$p=\text{const}$）

等压过程

$W = p\Delta V = p(V_2-V_1) = \nu R\Delta T$

$Q = \nu C_p\Delta T$

$\Delta E = \nu C_V\Delta T$

验证：$Q = \Delta E + W$ → $\nu C_p\Delta T = \nu C_V\Delta T + \nu R\Delta T$ → $C_p = C_V + R$ ✓

过程方程：$\frac{V_1}{T_1} = \frac{V_2}{T_2}$

等温过程（$T=\text{const}$）

等温过程

$\Delta E = 0$（温度不变→内能不变）

$W = Q = \int_{V_1}^{V_2}p\dd V = \int_{V_1}^{V_2}\frac{\nu RT}{V}\dd V = \nu RT\ln\frac{V_2}{V_1} = \nu RT\ln\frac{p_1}{p_2}$

过程方程：$p_1V_1 = p_2V_2$（等温线是 $p$-$V$ 图上的双曲线）

绝热过程（$Q=0$）

绝热过程

$\Delta E = -W$（不吸热，做功靠消耗内能）

绝热方程（三种等价形式）：

$$pV^\gamma = \text{const},\quad TV^{\gamma-1} = \text{const},\quad T^\gamma p^{1-\gamma} = \text{const}$$

绝热功：$W = -\Delta E = \nu C_V(T_1-T_2) = \frac{p_1V_1-p_2V_2}{\gamma-1} = \frac{\nu R(T_1-T_2)}{\gamma-1}$

绝热线比等温线更陡！因为 $pV^\gamma = \text{const}$ 中 $\gamma > 1$，所以绝热线下降更快。

四种过程总结对比表（必背！）

过程过程方程$W$$Q$$\Delta E$
等体$p/T=C$$0$$\nu C_V\Delta T$$\nu C_V\Delta T$
等压$V/T=C$$\nu R\Delta T$$\nu C_p\Delta T$$\nu C_V\Delta T$
等温$pV=C$$\nu RT\ln\frac{V_2}{V_1}$$W$$0$
绝热$pV^\gamma=C$$\frac{\nu R(T_1-T_2)}{\gamma-1}$$0$$-W$

9.3 循环过程与热机效率

循环过程

系统回到初态 → $\Delta E = 0$ → $W_{净} = Q_{吸} - |Q_{放}|$

$p$-$V$ 图上：顺时针→正循环（热机），$W>0$；逆时针→逆循环（制冷机），$W<0$

$W_{净}$ = $p$-$V$ 图上封闭曲线围成的面积

热机效率
$$\eta = \frac{W}{Q_1} = \frac{Q_1-|Q_2|}{Q_1} = 1 - \frac{|Q_2|}{Q_1}$$

$Q_1$=每循环吸收的总热量，$|Q_2|$=每循环放出的总热量

制冷系数
$$\varepsilon = \frac{|Q_2|}{W} = \frac{|Q_2|}{Q_1-|Q_2|}$$

9.4 卡诺循环与卡诺定理

卡诺循环

两个等温过程 + 两个绝热过程组成的可逆循环

工作在高温热源 $T_1$ 和低温热源 $T_2$ 之间：

$$\eta_C = 1 - \frac{T_2}{T_1}$$

卡诺制冷机：$\varepsilon_C = \frac{T_2}{T_1-T_2}$

卡诺定理
  1. 在相同高温源和低温源之间，可逆热机效率 > 不可逆热机效率
  2. 所有可逆热机效率相同，只取决于两个热源温度
例1：热机工作在 $T_1=600\,\text{K}$ 和 $T_2=300\,\text{K}$ 之间，每循环吸热 $Q_1=2000\,\text{J}$，实际效率 $\eta=40\%$，求做功和放热。
解

最大效率：$\eta_C = 1-300/600 = 50\%$，实际 $40\% < 50\%$ ✓（合理）

做功：$W = \eta Q_1 = 0.4\times2000 = 800\,\text{J}$

放热：$|Q_2| = Q_1 - W = 1200\,\text{J}$

9.5 热力学第二定律

两种等价表述

克劳修斯表述：热量不能自发地从低温物体传到高温物体（不引起其他变化）

开尔文表述：不可能从单一热源吸热全部转化为功而不引起其他变化（第二类永动机不可能）

两种表述等价——违反一个必违反另一个。

9.6 熵

熵的定义

$\dd S = \frac{\delta Q_{rev}}{T}$（可逆过程中，微小热量/温度）

熵是状态量，与路径无关。

理想气体熵变公式（万能公式！）
$$\Delta S = \nu C_V\ln\frac{T_2}{T_1} + \nu R\ln\frac{V_2}{V_1}$$ $$\Delta S = \nu C_p\ln\frac{T_2}{T_1} - \nu R\ln\frac{p_2}{p_1}$$

（对任意过程都可用，因为 $S$ 是状态量）

不可逆过程熵变计算方法

三步走：

  1. 确定始末状态（$T_1,V_1$ → $T_2,V_2$）
  2. 设计一个连接始末状态的可逆过程
  3. 沿可逆过程计算 $\Delta S = \int\frac{\delta Q_{rev}}{T}$
熵增原理
$$\Delta S_{孤立} \geq 0$$

孤立系统的熵永不减少。等号对应可逆过程，大于号对应不可逆过程。

熵增原理是热力学第二定律的数学表达。

例2（真题高频：自由膨胀的熵变）：理想气体从 $V_1$ 自由膨胀到 $V_2$（$Q=0,W=0$），求熵变。
解

自由膨胀：$Q=0,W=0$ → $\Delta E=0$ → $T_2=T_1$（理想气体）

设计可逆过程（等温膨胀）：

$$\Delta S = \nu C_V\ln\frac{T_2}{T_1} + \nu R\ln\frac{V_2}{V_1} = 0 + \nu R\ln\frac{V_2}{V_1} = \nu R\ln\frac{V_2}{V_1} > 0$$

$\Delta S > 0$：不可逆过程→熵增。

例3（热传导的熵变）：温度 $T_1$ 的物体与 $T_2$（$T_1>T_2$）的物体接触，传递热量 $Q$，两物体温度不变（热容很大），求总熵变。
解

高温物体：$\Delta S_1 = -Q/T_1$（放热）

低温物体：$\Delta S_2 = +Q/T_2$（吸热）

$$\Delta S_{总} = \frac{Q}{T_2} - \frac{Q}{T_1} = Q\left(\frac{1}{T_2}-\frac{1}{T_1}\right) > 0$$

T-S图

T-S图中，曲线下面积 = $\int T\dd S = Q_{rev}$（可逆过程吸收的热量）。等温线为水平线，绝热可逆为竖直线，卡诺循环为矩形。

9.8 循环过程分析方法

循环过程解题步骤
  1. 画p-V图，标明各状态点的 $(p,V,T)$（用状态方程互相关联）
  2. 逐段分析每个过程的 $W$、$Q$、$\Delta E$
  3. 整个循环：$\Delta E_{循环}=0$，$W_{净}=Q_{吸}-|Q_{放}|$
  4. 效率 $\eta=W_{净}/Q_{吸}=1-|Q_{放}|/Q_{吸}$
例（PPT高频题型：三角形循环）：1 mol 单原子理想气体经历循环 $A(p_0,V_0)\to B(2p_0,V_0)\to C(p_0,2V_0)\to A$。AB等体，BC直线，CA等压。求效率。
解

温度：$T_A=p_0V_0/R$，$T_B=2p_0V_0/R$，$T_C=2p_0V_0/R$（注意 $T_B=T_C$！）

AB（等体）：$W_{AB}=0$，$Q_{AB}=C_V(T_B-T_A)=\frac{3}{2}R\cdot\frac{p_0V_0}{R}=\frac{3}{2}p_0V_0>0$（吸热）

BC（直线过程）：$W_{BC}=\frac{(2p_0+p_0)(2V_0-V_0)}{2}=\frac{3}{2}p_0V_0$

$\Delta E_{BC}=C_V(T_C-T_B)=0$，$Q_{BC}=W_{BC}=\frac{3}{2}p_0V_0>0$（吸热）

CA（等压）：$W_{CA}=p_0(V_0-2V_0)=-p_0V_0$

$Q_{CA}=C_p(T_A-T_C)=\frac{5}{2}R\cdot(-\frac{p_0V_0}{R})=-\frac{5}{2}p_0V_0<0$（放热）

$W_{净}=0+\frac{3}{2}p_0V_0-p_0V_0=\frac{1}{2}p_0V_0$

$Q_{吸}=\frac{3}{2}+\frac{3}{2}=3p_0V_0$

$$\eta=\frac{W_{净}}{Q_{吸}}=\frac{p_0V_0/2}{3p_0V_0}=\frac{1}{6}\approx16.7\%$$

9.9 熵变计算技巧总结

理想气体熵变万能公式
$$\Delta S=\nu C_V\ln\frac{T_2}{T_1}+\nu R\ln\frac{V_2}{V_1}=\nu C_p\ln\frac{T_2}{T_1}-\nu R\ln\frac{p_2}{p_1}$$

适用于任何过程（可逆或不可逆），因为熵是状态量。

不可逆过程熵变计算法

找一条连接始末态的可逆路径，沿可逆路径计算 $\Delta S=\int\delta Q_{rev}/T$。

常用可逆路径：等体+等温，等压+等温，等温+绝热等。

例（自由膨胀熵变）：1 mol 理想气体从 $V_1$ 自由膨胀到 $2V_1$。求熵变。
解

自由膨胀：$Q=0$，$W=0$，$\Delta E=0$，$T$ 不变。

用可逆等温过程替代：$\Delta S=\nu R\ln(V_2/V_1)=R\ln2=5.76\,\text{J/K}$

虽然 $Q=0$，但 $\Delta S>0$！这是不可逆过程，不能用 $\Delta S=Q/T=0$。

致命错误：不可逆过程中 $\Delta S\neq Q/T$！$\dd S=\delta Q/T$ 只对可逆过程成立。熵变要用可逆路径算。

9.10 多段过程综合计算（PPT高频大题）

例（等温+绝热组合）：2 mol 双原子理想气体从状态 $A(p_0,V_0)$ 先等温膨胀到 $B(p_B,2V_0)$，再绝热膨胀到 $C(p_C,4V_0)$。求 $T_C$ 和整个过程的 $\Delta E$、$W$、$Q$。（$\gamma=7/5$）
解

AB等温：$T_A=T_B=p_0V_0/(2R)$，$p_B=p_0V_0/(2V_0)=p_0/2$

BC绝热：$T_BV_B^{\gamma-1}=T_CV_C^{\gamma-1}$

$T_C=T_B(V_B/V_C)^{\gamma-1}=T_B(2V_0/4V_0)^{0.4}=T_B\cdot(1/2)^{0.4}=T_B/2^{0.4}$

$2^{0.4}\approx1.32$，$T_C=T_B/1.32=0.758T_B$

AB过程：$\Delta E_{AB}=0$，$W_{AB}=Q_{AB}=2RT_A\ln2$

BC过程：$Q_{BC}=0$，$\Delta E_{BC}=2C_V(T_C-T_B)=2\times\frac{5}{2}R(0.758T_B-T_B)=-2.42RT_B$

$W_{BC}=-\Delta E_{BC}=2.42RT_B$

总计：$\Delta E=\Delta E_{AB}+\Delta E_{BC}=-2.42RT_B$

$W=W_{AB}+W_{BC}=2RT_B\ln2+2.42RT_B=(1.386+2.42)RT_B=3.81RT_B$

$Q=Q_{AB}+Q_{BC}=2RT_B\ln2=1.386RT_B$

验证：$Q=\Delta E+W=-2.42RT_B+3.81RT_B=1.39RT_B\approx1.386RT_B$ ✓

例（卡诺循环计算）：卡诺热机工作在 $T_1=500\,\text{K}$ 和 $T_2=300\,\text{K}$ 之间，每循环从高温热源吸热 $1000\,\text{J}$。求效率、做功、放热。
解

$\eta=1-T_2/T_1=1-300/500=0.4=40\%$

$W=\eta Q_1=0.4\times1000=400\,\text{J}$

$|Q_2|=Q_1-W=600\,\text{J}$

制冷系数（若逆转）：$\varepsilon=T_2/(T_1-T_2)=300/200=1.5$

  • 等温过程 → 水平线
  • 绝热可逆过程（等熵） → 竖直线

9.12 热力学解题易错总结

1. $\Delta E=\nu C_V\Delta T$ 适用于任何过程
不仅仅是等容过程！因为理想气体内能只是温度函数。很多同学只在等容时用这个公式，其实所有过程都适用。
2. 等温过程 $\Delta E=0$，但 $Q\neq0$
等温不是"没有热量交换"，而是吸的热全部转化为功。
3. 绝热自由膨胀不能用绝热方程
$pV^\gamma=C$ 只对准静态绝热过程成立。自由膨胀是不可逆过程，不满足准静态条件。正确结论：$\Delta T=0$，$\Delta S>0$。
4. 循环效率中 $Q_{吸}$ 只取正值部分
$\eta=W/Q_{吸}$，$Q_{吸}$ 只包括 $Q>0$ 的过程的热量之和。不能直接用 $Q_{净}$ 代替！
5. 熵变可以为负（对非孤立系统）
$\Delta S\geq0$ 是对孤立系统说的。一个子系统的熵可以减少（比如被冷却），只要总系统熵不减即可。
6. $p$-$V$ 图上绝热线比等温线更陡
等温线：$p\propto 1/V$；绝热线：$p\propto 1/V^\gamma$（$\gamma>1$），所以绝热线更陡。在p-V图上绝热膨胀温度降低更多。

9.13 多体热传导熵变计算

有限温差热传导的熵变

当两个有限热容的物体接触达到热平衡时，过程不可逆，须用可逆路径计算各自熵变再相加。

核心方法：对每个物体，设计一个可逆过程（无穷多个无穷小温差的热源依次加热/冷却），沿可逆路径积分 $\Delta S = \int_{T_i}^{T_f}\frac{\delta Q_{rev}}{T}=mc\ln\frac{T_f}{T_i}$。

例1（两杯水混合）：温度 $T_1$ 的 $m_1$ 克水与温度 $T_2$ 的 $m_2$ 克水混合（$T_1 > T_2$），比热容为 $c$。求平衡温度和总熵变。
解

第一步：求平衡温度 $T_f$

热量守恒（忽略散热）：$m_1 c(T_1 - T_f) = m_2 c(T_f - T_2)$

$$T_f = \frac{m_1 T_1 + m_2 T_2}{m_1 + m_2}$$

第二步：分别计算熵变

热水（从 $T_1$ 降到 $T_f$）：$\Delta S_1 = m_1 c\ln\frac{T_f}{T_1} < 0$（熵减少）

冷水（从 $T_2$ 升到 $T_f$）：$\Delta S_2 = m_2 c\ln\frac{T_f}{T_2} > 0$（熵增加）

第三步：总熵变

$$\Delta S_{总} = m_1 c\ln\frac{T_f}{T_1} + m_2 c\ln\frac{T_f}{T_2}$$

当 $m_1 = m_2 = m$ 时，$T_f = (T_1+T_2)/2$，

$$\Delta S_{总} = mc\ln\frac{T_f^2}{T_1 T_2} = mc\ln\frac{(T_1+T_2)^2}{4T_1 T_2}$$

由均值不等式 $(T_1+T_2)^2 > 4T_1 T_2$（当 $T_1\neq T_2$），故 $\Delta S_{总} > 0$ ✓

例2（金属块投入大水池）：热容为 $C$ 的金属块（温度 $T_1$）投入温度为 $T_2$ 的大水池（$T_1 > T_2$），水池足够大使其温度不变。求金属块和水池各自的熵变及总熵变。
解

金属块最终温度 = $T_2$（水池温度不变）

金属块的熵变（设计可逆冷却过程）：

$$\Delta S_{金属} = \int_{T_1}^{T_2}\frac{C\dd T}{T} = C\ln\frac{T_2}{T_1} < 0$$

水池的熵变（水池温度恒为 $T_2$，相当于等温过程，本身就是可逆的）：

水池吸收的热量：$Q = C(T_1 - T_2)$

$$\Delta S_{水池} = \frac{Q}{T_2} = \frac{C(T_1-T_2)}{T_2}$$

总熵变：

$$\Delta S_{总} = C\ln\frac{T_2}{T_1} + \frac{C(T_1-T_2)}{T_2} = C\left[\frac{T_1-T_2}{T_2} - \ln\frac{T_1}{T_2}\right]$$

令 $x = T_1/T_2 > 1$，则 $\Delta S_{总} = C[(x-1)-\ln x]$。由于 $x-1 > \ln x$（对 $x>1$），故 $\Delta S_{总} > 0$ ✓

多体熵变计算口诀：分别算，再相加。每个物体用可逆路径单独算熵变，总熵变 = 所有子系统熵变之和。对于恒温热源（热容无穷大），$\Delta S = Q/T$；对于有限热容物体，$\Delta S = mc\ln(T_f/T_i)$ 或 $C\ln(T_f/T_i)$。

9.14 特殊过程分析

多方过程

多方过程 $pV^n = C$

多方过程是四种基本过程的统一推广，多方指数 $n$ 取不同值对应不同过程：

多方指数 $n$对应过程过程方程
$n=0$等压过程$p = C$
$n=1$等温过程$pV = C$
$n=\gamma$绝热过程$pV^\gamma = C$
$n=\infty$等容过程$V = C$（因为 $p^{1/n}V=C\to V=C$）

多方过程的功：

$$W = \int_{V_1}^{V_2}p\dd V = \frac{p_1V_1-p_2V_2}{n-1} = \frac{\nu R(T_1-T_2)}{n-1}\quad(n\neq1)$$

多方过程的摩尔热容：

$$C_n = C_V\frac{n-\gamma}{n-1}$$

热量：$Q = \nu C_n \Delta T = \nu C_V\frac{n-\gamma}{n-1}\Delta T$

验证特殊情况：
$n=0$（等压）：$C_0 = C_V\frac{0-\gamma}{0-1}=\gamma C_V = C_p$ ✓
$n=1$（等温）：$C_1 = C_V\frac{1-\gamma}{1-1}\to\infty$（等温过程热容无穷大，符合"温度不变但吸热"）✓
$n=\gamma$（绝热）：$C_\gamma = C_V\frac{\gamma-\gamma}{\gamma-1}=0$（绝热过程不吸热）✓
$n=\infty$：$C_\infty = C_V\frac{\infty}{\infty} = C_V$ ✓
例（多方过程的功和热量）：1 mol 单原子理想气体（$\gamma=5/3$）经历 $pV^{1.2}=C$ 的过程，从状态 $(p_1, V_1, T_1)$ 变化到 $(p_2, V_2, T_2)$。已知 $T_1=600\,\text{K}$，$T_2=300\,\text{K}$。求功 $W$、热量 $Q$ 和内能变化 $\Delta E$。
解

$n=1.2$

功：$W = \frac{\nu R(T_1-T_2)}{n-1} = \frac{1\times8.314\times(600-300)}{1.2-1} = \frac{2494.2}{0.2} = 12471\,\text{J} \approx 12.5\,\text{kJ}$

内能变化：$\Delta E = \nu C_V(T_2-T_1) = 1\times\frac{3}{2}\times8.314\times(-300) = -3741\,\text{J} \approx -3.74\,\text{kJ}$

热量（第一定律）：$Q = \Delta E + W = -3741 + 12471 = 8730\,\text{J} \approx 8.73\,\text{kJ}$

验证（用多方热容公式）：

$C_n = C_V\frac{n-\gamma}{n-1} = \frac{3}{2}R\times\frac{1.2-5/3}{1.2-1} = \frac{3}{2}R\times\frac{-7/15}{0.2} = \frac{3}{2}R\times(-7/3) = -\frac{7}{2}R$

$Q = \nu C_n\Delta T = 1\times(-\frac{7}{2}\times8.314)\times(-300) = 8730\,\text{J}$ ✓

注意 $C_n$ 可以为负！当 $1 < n < \gamma$ 时，$C_n < 0$，意味着系统吸热但温度反而下降（或放热但温度升高）。这看似矛盾，但因为功的交换，能量收支是自洽的。

节流过程（焦耳-汤姆孙效应）

节流过程

气体通过多孔塞（或节流阀）从高压侧缓慢流向低压侧的不可逆绝热过程。

基本特征：

  • 绝热：$Q = 0$（管壁绝热）
  • 焓不变：$H_1 = H_2$（即 $E_1 + p_1V_1 = E_2 + p_2V_2$）
  • 理想气体：$H = \nu C_p T$，焓只是温度函数 → $T_1 = T_2$（节流前后温度不变）
  • 实际气体：节流后温度可升可降，取决于焦耳-汤姆孙系数 $\mu_J = (\partial T/\partial p)_H$

$\mu_J > 0$：节流后温度降低（致冷效应，大多数气体在室温附近）

$\mu_J < 0$：节流后温度升高（如氢气、氦气在常温下）

$\mu_J = 0$：转变温度（致冷与致热的分界线）

考试提示：节流过程最常考的结论是"焓不变"。如果题目说"理想气体节流"，则温度不变、内能不变。

  • $DA$（绝热压缩）：沿 $S=S_A$ 的竖直线向上（$T$ 升高）
  • 矩形面积 = $(T_1-T_2)(S_B-S_A) = W_{净}$

等容加热：$\Delta S = \nu C_V\ln(T_2/T_1)$，在T-S图上为向右上方弯曲的曲线（$T = T_0 e^{S/(\nu C_V)}$，指数函数）。

等压加热：$\Delta S = \nu C_p\ln(T_2/T_1)$，形状类似等容线但更平缓（因 $C_p > C_V$，同样的温度变化产生更大的熵变）。

等温过程：水平线（$T$ 不变，$S$ 变化）。

绝热可逆过程（等熵）：竖直线（$S$ 不变，$T$ 变化）。

T-S图的核心用途：
(1) 曲线下面积 = $\int T\dd S = Q_{rev}$（可逆过程吸放热量）
(2) 封闭曲线面积 = $W_{净}$（净功）
(3) 顺时针→正循环（热机），逆时针→逆循环（制冷机）
(4) 卡诺循环的矩形面积最大化解释了为什么卡诺效率是最高效率
易错：T-S图上不可逆绝热过程不是竖直线！不可逆绝热过程 $Q=0$ 但 $\Delta S > 0$，所以是向右倾斜的线。只有可逆绝热（等熵）过程才是竖直线。

9.16 熵的统计意义（PPT补充）

Boltzmann熵公式
$$\boxed{S = k\ln\Omega}$$

$\Omega$ = 宏观状态对应的微观状态数（热力学概率）

$k = 1.38\times10^{-23}\,\text{J/K}$（Boltzmann常量）

这个公式刻在Boltzmann的墓碑上，是连接宏观热力学与微观统计力学的桥梁。

熵的微观含义
  • 熵是系统"无序度"的量度：$\Omega$ 越大（微观状态越多），系统越"混乱"，熵越大
  • 熵增原理的统计解释：孤立系统自发地从 $\Omega$ 小的状态向 $\Omega$ 大的状态演化，因为后者出现的概率压倒性地大
  • 热力学第二定律的统计本质：不是"不可能"，而是"极其不可能"——违反熵增的概率在宏观系统中小到可以忽略
例（自由膨胀的微观解释）：$N$ 个分子从容器的左半部分自由膨胀到整个容器。
分析

初态：$N$ 个分子都在左半边，$\Omega_1=1$（只有一种排列方式——都在左边）

末态：每个分子可以在左或右，$\Omega_2=2^N$

熵变：$\Delta S=k\ln\Omega_2-k\ln\Omega_1=k\ln 2^N=Nk\ln 2$

对 1 mol 气体：$\Delta S=N_Ak\ln 2=R\ln 2=5.76\,\text{J/K}$

与宏观计算结果 $\Delta S=\nu R\ln(V_2/V_1)=R\ln 2$ 完全一致！

为什么不会自发收缩？：$N$ 个分子全部回到左半边的概率 $=1/2^N$。对 $N=N_A\approx 6\times10^{23}$，这个概率是 $1/2^{6\times10^{23}}\approx 10^{-1.8\times10^{23}}$——比宇宙年龄内任何事件的概率都小得多。"不是不可能，是等不到"。

9.18 多方过程（PPT补充）

多方过程的一般形式

多方过程：$pV^n = \text{const}$，其中 $n$ 称为多方指数。

四种基本过程都是多方过程的特例：

$n$ 值对应过程过程方程说明
$n=0$等压过程$p = \text{const}$$pV^0 = p = C$
$n=1$等温过程$pV = \text{const}$Boyle定律
$n=\gamma$绝热过程$pV^\gamma = \text{const}$Poisson方程
$n\to\infty$等容过程$V = \text{const}$$p^{1/n}V = C \to V = C$
多方过程的基本公式

过程方程的等价形式：

$$pV^n = \text{const},\quad TV^{n-1} = \text{const},\quad T^n p^{1-n} = \text{const}$$

功（$n\neq 1$ 时）：

$$W = \int_{V_1}^{V_2}p\dd V = \frac{p_1V_1 - p_2V_2}{n-1} = \frac{\nu R(T_1 - T_2)}{n-1}$$

多方热容：

$$C_n = C_V\frac{n-\gamma}{n-1}$$

热量：$Q = \nu C_n\Delta T = \nu C_V\frac{n-\gamma}{n-1}(T_2 - T_1)$

p-V图上多方过程曲线的陡峭程度

在 $p$-$V$ 图上，过同一点的多方过程曲线斜率为 $\frac{\dd p}{\dd V} = -n\frac{p}{V}$，因此：

  • $n=0$（等压）：水平线——最平
  • $n=1$（等温）：双曲线
  • $n=\gamma\approx1.4$（绝热）：比等温线更陡
  • $n\to\infty$（等容）：竖直线——最陡

结论：$n$ 越大，曲线越陡。

例：$n=2$ 过程中的功和热量：1 mol 双原子理想气体（$\gamma = 7/5$）经历 $pV^2 = C$ 的过程，从 $T_1 = 400\,\text{K}$ 变到 $T_2 = 300\,\text{K}$。求 $W$、$Q$、$\Delta E$。
解

功：$W = \frac{\nu R(T_1 - T_2)}{n-1} = \frac{1\times8.314\times(400-300)}{2-1} = 831.4\,\text{J}$

内能变化：$\Delta E = \nu C_V(T_2-T_1) = 1\times\frac{5}{2}\times8.314\times(-100) = -2078.5\,\text{J}$

热量：$Q = \Delta E + W = -2078.5 + 831.4 = -1247.1\,\text{J}$（放热）

验证：$C_n = C_V\frac{n-\gamma}{n-1} = \frac{5}{2}R\times\frac{2-7/5}{2-1} = \frac{5}{2}R\times\frac{3}{5} = \frac{3}{2}R$

$Q = \nu C_n\Delta T = 1\times\frac{3}{2}\times8.314\times(-100) = -1247.1\,\text{J}$ ✓

$1 < n < \gamma$ 时 $C_n < 0$（负热容）：当 $n-\gamma < 0$ 且 $n-1 > 0$ 时，$C_n = C_V\frac{n-\gamma}{n-1} < 0$。物理含义：系统吸热但温度反而下降（或放热但温度升高）。这并不违反热力学定律——因为气体膨胀做的功大于吸收的热量，多出的能量来自内能的减少，导致温度下降。例如 $n=1.2$（在 $1$ 和 $\gamma=1.4$ 之间）：$C_{1.2} = \frac{5}{2}R\times\frac{1.2-1.4}{1.2-1} = \frac{5}{2}R\times(-1) = -\frac{5}{2}R < 0$。

9.19 熵变计算的系统方法（真题重点）

核心原则

熵是状态函数 → 熵变只取决于始末状态，与过程路径无关。

因此，计算任何过程（包括不可逆过程）的熵变时，都可以选择任意可逆路径来计算：

$$\Delta S = \int_{\text{可逆路径}}\frac{\dd Q}{T}$$
理想气体熵变的通用公式

由可逆过程 $\dd Q = \nu C_V\dd T + p\dd V = \nu C_V\dd T + \frac{\nu RT}{V}\dd V$ 积分得：

$$\boxed{\Delta S = \nu C_V\ln\frac{T_2}{T_1} + \nu R\ln\frac{V_2}{V_1}}$$

等价形式（消去 $V$ 或 $T$）：

$$\boxed{\Delta S = \nu C_p\ln\frac{T_2}{T_1} - \nu R\ln\frac{p_2}{p_1}}$$
各特殊过程的熵变
过程条件熵变公式
等温过程$T_1 = T_2$$\Delta S = \nu R\ln\dfrac{V_2}{V_1} = \nu R\ln\dfrac{p_1}{p_2}$
等压过程$p_1 = p_2$$\Delta S = \nu C_p\ln\dfrac{T_2}{T_1}$
等容过程$V_1 = V_2$$\Delta S = \nu C_V\ln\dfrac{T_2}{T_1}$
可逆绝热$Q = 0$，可逆$\Delta S = 0$（等熵过程）
不可逆绝热（自由膨胀）$Q=0$，$W=0$，$\Delta T=0$$\Delta S = \nu R\ln\dfrac{V_2}{V_1} > 0$
例1：两物体接触达到热平衡的熵变

物体A（质量$m_A$，比热$c_A$，初温$T_A$）与物体B（质量$m_B$，比热$c_B$，初温$T_B$），$T_A > T_B$，接触后达到热平衡。

解

Step 1：求平衡温度

$$T_f = \frac{m_Ac_AT_A + m_Bc_BT_B}{m_Ac_A + m_Bc_B}$$

Step 2：各物体的熵变（用可逆路径：无穷小温差逐步加热/冷却）

热物体A（降温）：$\Delta S_A = m_Ac_A\ln\dfrac{T_f}{T_A} < 0$

冷物体B（升温）：$\Delta S_B = m_Bc_B\ln\dfrac{T_f}{T_B} > 0$

Step 3：总熵变

$$\Delta S_{total} = m_Ac_A\ln\frac{T_f}{T_A} + m_Bc_B\ln\frac{T_f}{T_B}$$

可以证明 $\Delta S_{total} > 0$（由 $\ln$ 函数的凹性/Jensen不等式）。

物理含义：热量从高温传到低温是不可逆过程，总熵必增。

例2：不可逆等压膨胀的熵变

理想气体从 $(p, V)$ 不可逆地等压膨胀到 $(p, 2V)$，求熵变。

解

始态：$(p, V, T_1)$；末态：$(p, 2V, T_2)$

由等压过程：$T_2/T_1 = V_2/V_1 = 2$，故 $T_2 = 2T_1$

因为熵是状态函数，不可逆等压过程的熵变 = 可逆等压过程的熵变：

$$\Delta S = \nu C_p\ln\frac{T_2}{T_1} = \nu C_p\ln 2$$

关键理解：无论过程是否可逆，只要始末状态相同，$\Delta S$ 就相同！

计算不可逆过程的熵变，绝不能直接用 $\int\dd Q/T$：
  • $\dd S = \delta Q/T$ 只对可逆过程成立
  • 对不可逆过程：$\dd S > \delta Q/T$（Clausius不等式）
  • 正确方法：找一条连接相同始末态的可逆路径，沿可逆路径算 $\Delta S$
  • 典型错误：自由膨胀 $Q=0$，误算 $\Delta S = Q/T = 0$。实际上 $\Delta S = \nu R\ln(V_2/V_1) > 0$
熵变计算的万能策略：
1. 确定始态 $(T_1, V_1, p_1)$ 和末态 $(T_2, V_2, p_2)$
2. 直接代入万能公式 $\Delta S = \nu C_V\ln(T_2/T_1) + \nu R\ln(V_2/V_1)$
3. 不需要关心中间过程是否可逆——因为公式本身就是沿可逆路径推导的，而结果只依赖始末状态

9.20 循环过程解题方法（真题高频大题）

循环过程解题的标准步骤
  1. 识别循环中每个过程的类型：等温、等压、等容、绝热或多方过程
  2. 确定各状态点参数：用状态方程 $pV=\nu RT$ 求出每个状态点的 $(p, V, T)$
  3. 逐段计算 $Q$、$W$、$\Delta E$：利用各过程的公式
  4. 利用循环条件验证：$\sum\Delta E = 0$，$W_{net} = Q_{吸} - |Q_{放}|$
  5. 计算效率：$\eta = \dfrac{W_{net}}{Q_{吸}} = 1 - \dfrac{|Q_{放}|}{Q_{吸}}$
例1：矩形p-V循环（两个等压 + 两个等容）

1 mol 单原子理想气体（$C_V = \frac{3}{2}R$，$C_p = \frac{5}{2}R$）经历循环：

$A(p_1,V_1) \xrightarrow{\text{等压}} B(p_1,V_2) \xrightarrow{\text{等容}} C(p_2,V_2) \xrightarrow{\text{等压}} D(p_2,V_1) \xrightarrow{\text{等容}} A$

其中 $p_1 > p_2$，$V_2 > V_1$。求循环效率。

解

各状态温度：$T_A = p_1V_1/R$，$T_B = p_1V_2/R$，$T_C = p_2V_2/R$，$T_D = p_2V_1/R$

AB（等压膨胀，$p_1$）：

$W_{AB} = p_1(V_2 - V_1)$，$Q_{AB} = C_p(T_B - T_A) = \frac{5}{2}(p_1V_2 - p_1V_1) = \frac{5}{2}p_1(V_2-V_1) > 0$（吸热）

BC（等容冷却，$V_2$）：

$W_{BC} = 0$，$Q_{BC} = C_V(T_C - T_B) = \frac{3}{2}(p_2 - p_1)V_2 < 0$（放热）

CD（等压压缩，$p_2$）：

$W_{CD} = p_2(V_1 - V_2) < 0$，$Q_{CD} = C_p(T_D - T_C) = \frac{5}{2}p_2(V_1-V_2) < 0$（放热）

DA（等容加热，$V_1$）：

$W_{DA} = 0$，$Q_{DA} = C_V(T_A - T_D) = \frac{3}{2}(p_1 - p_2)V_1 > 0$（吸热）

净功：$W_{net} = p_1(V_2-V_1) + p_2(V_1-V_2) = (p_1-p_2)(V_2-V_1)$（矩形面积 ✓）

吸热：$Q_{吸} = Q_{AB} + Q_{DA} = \frac{5}{2}p_1(V_2-V_1) + \frac{3}{2}(p_1-p_2)V_1$

$$\eta = \frac{(p_1-p_2)(V_2-V_1)}{\frac{5}{2}p_1(V_2-V_1) + \frac{3}{2}(p_1-p_2)V_1}$$
例2：等温 + 等容 + 绝热组成的循环

1 mol 双原子理想气体（$\gamma = 7/5$）经历：$A(p_0, V_0) \xrightarrow{\text{等温}} B(p_0/2, 2V_0) \xrightarrow{\text{等容}} C(p_C, 2V_0) \xrightarrow{\text{绝热}} A$。求 $p_C$ 和效率。

解

温度：$T_A = p_0V_0/R$，$T_B = T_A$（等温），$T_C = p_C\cdot2V_0/R$

求 $p_C$：CA 为绝热过程，$p_CV_C^\gamma = p_AV_A^\gamma$

$$p_C(2V_0)^{7/5} = p_0 V_0^{7/5} \quad\Rightarrow\quad p_C = p_0\left(\frac{1}{2}\right)^{7/5} = \frac{p_0}{2^{7/5}} = \frac{p_0}{2.639} \approx 0.379p_0$$

AB（等温膨胀）：$\Delta E_{AB} = 0$，$W_{AB} = Q_{AB} = RT_A\ln 2 = p_0V_0\ln 2$（吸热）

BC（等容冷却）：$W_{BC} = 0$

$T_C = p_C\cdot2V_0/R = 0.379p_0\cdot2V_0/R = 0.758T_A$

$Q_{BC} = C_V(T_C - T_B) = \frac{5}{2}R(0.758T_A - T_A) = -0.605RT_A = -0.605p_0V_0$（放热）

CA（绝热压缩）：$Q_{CA} = 0$，$W_{CA} = -\Delta E_{CA} = -C_V(T_A-T_C) = -\frac{5}{2}R\cdot0.242T_A = -0.605p_0V_0$

效率：$W_{net} = W_{AB} + 0 + W_{CA} = 0.693p_0V_0 - 0.605p_0V_0 = 0.088p_0V_0$

$Q_{吸} = Q_{AB} = 0.693p_0V_0$

$$\eta = \frac{0.088}{0.693} \approx 12.7\%$$
Carnot循环在p-V图和T-S图上的表现

p-V图：两条等温线 + 两条绝热线，形成一个"鼓形"封闭曲线。

  • $A\to B$：等温膨胀（$T_1$），沿双曲线向右
  • $B\to C$：绝热膨胀，沿更陡的曲线向右下
  • $C\to D$：等温压缩（$T_2$），沿双曲线向左
  • $D\to A$：绝热压缩，沿陡曲线向左上

T-S图：矩形！这是Carnot循环最简洁的表示。

  • $A\to B$：等温（$T=T_1$，水平线向右，$S$ 增大）
  • $B\to C$：绝热可逆（$S=S_B$，竖直线向下，$T$ 降低）
  • $C\to D$：等温（$T=T_2$，水平线向左，$S$ 减小）
  • $D\to A$：绝热可逆（$S=S_A$，竖直线向上，$T$ 升高）

矩形面积 $= (T_1 - T_2)(S_B - S_A) = W_{net} = Q_{net}$

T-S图的妙用：T-S图上封闭曲线的面积 = 循环做的净功 = 吸收的净热量。卡诺循环是矩形，面积最容易计算。任何工作在 $T_1$ 和 $T_2$ 之间的循环，在T-S图上都被卡诺矩形"包围"，因此卡诺循环效率最高。

9.21 制冷机与热泵（PPT补充）

逆循环的两种应用

逆循环（逆时针）：外界对系统做功 $W$，从低温热源（$T_2$）吸热 $Q_2$，向高温热源（$T_1$）放热 $Q_1 = Q_2 + W$。

制冷机

目的：从低温热源吸热（制冷）

制冷系数：

$$\omega = \frac{Q_2}{W} = \frac{Q_2}{Q_1 - Q_2}$$

Carnot制冷机（最大制冷系数）：

$$\omega_{Carnot} = \frac{T_2}{T_1 - T_2}$$
热泵

目的：向高温热源放热（供暖）

供暖系数 (COP)：

$$\text{COP} = \frac{Q_1}{W} = \frac{Q_1}{Q_1 - Q_2}$$

Carnot热泵：

$$\text{COP}_{Carnot} = \frac{T_1}{T_1 - T_2}$$
制冷系数与COP的关系
$$\boxed{\text{COP} = \omega + 1}$$

证明：$\text{COP} = Q_1/W = (Q_2 + W)/W = Q_2/W + 1 = \omega + 1$

例：冬季供暖问题

室外温度 $T_2 = -10°\text{C} = 263\,\text{K}$，室内温度 $T_1 = 20°\text{C} = 293\,\text{K}$。求 Carnot 制冷系数和热泵 COP。

解

制冷系数：$\omega = \frac{T_2}{T_1 - T_2} = \frac{263}{293-263} = \frac{263}{30} = 8.77$

热泵COP：$\text{COP} = \frac{T_1}{T_1 - T_2} = \frac{293}{30} = 9.77$

验证：$\text{COP} = \omega + 1 = 8.77 + 1 = 9.77$ ✓

物理意义：COP = 9.77 意味着消耗 1 J 的电功，可以向室内输送 9.77 J 的热量。这比直接用电加热（1 J 电 → 1 J 热）高效得多！这就是热泵供暖比电暖器节能的原因。

制冷系数可以大于1：制冷系数 $\omega$ 不是效率（效率 $\leq 1$），而是"性能系数"，完全可以大于1，甚至远大于1。这不违反热力学定律，因为制冷机并不是把功"转化"为热——而是利用功把热量从低温搬运到高温。上例中 $\omega = 8.77$，意味着消耗 1 J 功就能从室外"搬运" 8.77 J 热量到室内。
温差越小，制冷/供暖越容易：
  • $T_1 - T_2 \to 0$ 时，$\omega \to \infty$，$\text{COP} \to \infty$（温差小，很容易搬运热量）
  • $T_1 - T_2$ 很大时，$\omega$ 和 COP 都很小（温差大，搬运热量代价高）
  • $T_2 \to 0$ 时，$\omega \to 0$（无法从绝对零度吸热——热力学第三定律的影子）

9.22 Clausius不等式与可逆性判据（PPT补充）

Clausius不等式

对任意循环过程：

$$\boxed{\oint\frac{\dd Q}{T} \leq 0}$$
  • 等号成立 ↔ 循环可逆
  • 严格小于零 ↔ 循环不可逆

这是热力学第二定律最一般的数学表达。

任意过程的熵变与热量的关系

对于从状态 A 到状态 B 的任意过程：

$$\Delta S = S_B - S_A \geq \int_A^B\frac{\dd Q}{T}$$
  • 等号：可逆过程（$\Delta S = \int\dd Q/T$）
  • 大于号：不可逆过程（$\Delta S > \int\dd Q/T$，熵的增加量大于热量传递带来的熵流）
熵增原理（孤立系统）

孤立系统中 $\dd Q = 0$，因此：

$$\Delta S_{isolated} \geq 0$$
  • $= 0$：过程可逆（平衡态之间的准静态过程）
  • $> 0$：过程不可逆（一切自发过程）
  • $< 0$：不可能发生
判断过程方向的方法

计算 $\Delta S_{universe}$（宇宙 = 系统 + 环境的总熵变）：

  • $\Delta S_{universe} > 0$：过程不可逆，可以自发进行
  • $\Delta S_{universe} = 0$：过程可逆（理想极限）
  • $\Delta S_{universe} < 0$：过程不可能自发发生
例1：热量自发传导的方向

温度 $T_{hot}$ 的物体与温度 $T_{cold}$（$T_{hot} > T_{cold}$）的物体接触，传递热量 $Q$（两物体热容足够大，温度不变）。

解

正向传热（热 → 冷）：

$\Delta S = -\frac{Q}{T_{hot}} + \frac{Q}{T_{cold}} = Q\left(\frac{1}{T_{cold}} - \frac{1}{T_{hot}}\right) > 0$

$\Delta S > 0$ → 过程自发发生 ✓

反向传热（冷 → 热）：

$\Delta S = \frac{Q}{T_{hot}} - \frac{Q}{T_{cold}} = Q\left(\frac{1}{T_{hot}} - \frac{1}{T_{cold}}\right) < 0$

$\Delta S < 0$ → 过程不能自发发生 ✓（符合Clausius表述）

例2：判断一个热机是否可能

某热机工作在 $T_1 = 600\,\text{K}$ 和 $T_2 = 300\,\text{K}$ 之间，每循环吸热 $Q_1 = 1000\,\text{J}$，做功 $W = 600\,\text{J}$。判断是否可能。

解

放热：$Q_2 = Q_1 - W = 400\,\text{J}$

宇宙总熵变（一个循环）：

$$\Delta S = -\frac{Q_1}{T_1} + \frac{Q_2}{T_2} = -\frac{1000}{600} + \frac{400}{300} = -1.667 + 1.333 = -0.333\,\text{J/K}$$

$\Delta S < 0$ → 不可能！

验证：该热机效率 $\eta = 600/1000 = 60\%$，而 Carnot 效率 $= 1 - 300/600 = 50\%$。效率超过 Carnot 极限，违反热力学第二定律。

Clausius不等式的应用总结：
1. 判断循环可逆性：算 $\oint\dd Q/T$，等于零则可逆，小于零则不可逆
2. 判断过程方向：算 $\Delta S_{universe}$，大于零则自发，小于零则不可能
3. 判断热机可行性：效率不得超过 Carnot 效率，即 $\eta \leq 1 - T_2/T_1$
4. 计算不可逆过程熵变：不能用 $\int\dd Q/T$（会偏小），必须用可逆路径
常见误区：
  • "熵总是增加的" → 错！只有孤立系统的熵才不减少。非孤立系统（如被冷却的物体）的熵可以减少
  • "不可逆过程的 $\Delta S$ 一定大于零" → 错！非孤立系统的不可逆过程中，系统的 $\Delta S$ 可以小于零，但系统 + 环境的总 $\Delta S > 0$
  • "$Q = 0$ 则 $\Delta S = 0$" → 错！绝热自由膨胀 $Q = 0$ 但 $\Delta S > 0$（不可逆过程）

9.23 PPT补充例题精选（热力学综合）

例1：绝热容器突然停止——机械能转内能

绝热容器内贮有质量为 $m$、摩尔质量为 $M$ 的理想气体（自由度 $i$），容器以速度 $u$ 作定向运动。使容器突然停止，求分子速度平方平均值的增量 $\Delta\overline{v^2}$。

解

定向运动动能全部转化为气体内能：

$$\Delta E = \frac{1}{2}mu^2$$

由内能与温度关系 $E = \frac{i}{2}\frac{m}{M}RT$，内能增量对应温度升高 $\Delta T$：

$$\frac{1}{2}mu^2 = \frac{i}{2}\frac{m}{M}R\Delta T$$

又 $\frac{1}{2}m_0\overline{v^2} = \frac{i}{2}k_BT$（能量均分，$i$ 个平方项中 3 个平动），注意对单原子分子 $\frac{1}{2}m_0\overline{v^2} = \frac{3}{2}k_BT$，所以：

$$\frac{1}{2}m_0\Delta\overline{v^2} = \frac{3}{2}k_B\Delta T \implies \Delta\overline{v^2} = \frac{3k_B\Delta T}{m_0} = \frac{3R\Delta T}{M}$$

代入 $\Delta T = \frac{Mu^2}{iR}$：

$$\boxed{\Delta\overline{v^2} = \frac{3u^2}{i}}$$

单原子分子 $i=3$：$\Delta\overline{v^2}=u^2$；双原子分子 $i=5$：$\Delta\overline{v^2}=\frac{3}{5}u^2$。

例2：两种气体绝热混合

绝热容器体积为 $2V_0$，由绝热板分为相等两半。A 半有 1 mol 单原子分子气体，B 半有 2 mol 双原子分子气体，两半压强均为 $p_0$。抽去隔板后，求混合平衡温度 $T$ 和压强 $p$。

解

混合前各自初温（由 $pV=\nu RT$）：$T_A = p_0V_0/R$，$T_B = p_0V_0/(2R)$。

混合前总内能：

$$E_A = \frac{3}{2}RT_A = \frac{3}{2}p_0V_0,\quad E_B = 2\times\frac{5}{2}RT_B = \frac{5}{2}p_0V_0$$ $$E_{total} = 4p_0V_0$$

混合后内能不变（绝热、刚性容器无功）：

$$E_{total} = \left(1\times\frac{3}{2} + 2\times\frac{5}{2}\right)RT = \frac{13}{2}RT$$ $$T = \frac{8p_0V_0}{13R}$$

混合后压强：$p = \frac{(\nu_A+\nu_B)RT}{2V_0} = \frac{3RT}{2V_0} = \frac{12p_0}{13}$。

例3：导热活塞分隔两种气体的平衡温度

刚性绝热容器中有无摩擦的导热活塞，左边 1 mol 氖气（$i=3$）温度 $T_1$，右边 1 mol 氧气（$i=5$）温度 $T_2$。求平衡温度 $T$。

解

导热活塞→力学平衡（两边压强相等）+ 热平衡（温度相同）。整体绝热且活塞无摩擦 → 系统对外不做功。

由 $\Delta E_{total}=0$：

$$\frac{3}{2}R(T-T_1) + \frac{5}{2}R(T-T_2) = 0$$ $$\boxed{T = \frac{3T_1 + 5T_2}{8}}$$
例4：证明 p-V 图上两绝热线不相交

试证明在 $p$-$V$ 图上，两条绝热线不可能相交。

证明（反证法）

假设两条绝热线交于点 A。过 A 作一条等温线，与两绝热线分别交于 B、C。

沿循环 A→B（绝热）→B→C（等温）→C→A（绝热）构成一个循环。其中：

  • 两段绝热过程 $Q=0$
  • 等温过程从单一热源吸热 $Q$

整个循环的净效果：从单一热源吸热，全部转化为功（循环回到原态，$\Delta E=0$），无其他变化。

这违反热力学第二定律（开尔文表述），故假设不成立。两条绝热线不相交。

例5：核电厂效率计算

核电厂热功率 1590 MW，输电功率 540 MW。水蒸气温度 556 K，冷凝器温度 313 K。求理想效率和实际效率。

解

理想（Carnot）效率：

$$\eta_C = 1 - \frac{T_2}{T_1} = 1 - \frac{313}{556} = 43.7\%$$

实际效率：

$$\eta = \frac{W}{Q_1} = \frac{540}{1590} = 34.0\%$$

实际效率小于 Carnot 效率，符合卡诺定理。冷凝器需要带走的热量：

$$Q_2 = Q_1 - W = 1590 - 540 = 1050\;\text{MW}$$
例6：有限热源的最大输出功

物体热容量 $C$（常数），初温 $T_0$，环境恒温 $T'$（$T' < T_0$）。可逆热机工作于物体与环境之间，直到物体降温至 $T'$，求最大输出功。

解

设某时刻物体温度为 $T$，热机从物体吸热 $\delta Q_1 = -C\dd T$（物体降温，$\dd T < 0$），Carnot 效率 $\eta = 1 - T'/T$。

微元做功：

$$\dd W = \eta\,\delta Q_1 = \left(1 - \frac{T'}{T}\right)(-C\dd T)$$

从 $T_0$ 积分到 $T'$：

$$W = \int_{T_0}^{T'}\left(1 - \frac{T'}{T}\right)(-C)\dd T = C\int_{T'}^{T_0}\left(1 - \frac{T'}{T}\right)\dd T$$ $$= C\left[(T_0 - T') - T'\ln\frac{T_0}{T'}\right]$$ $$\boxed{W_{max} = C\left(T_0 - T' - T'\ln\frac{T_0}{T'}\right)}$$
例7：用熵增原理证明热机不可能实现

某热机从 $T_1=440\;\text{K}$ 吸热 1000 kJ，向 $T_2=280\;\text{K}$ 放热 568 kJ，输出功 432 kJ。证明此热机不可能制成。

证明

取热机 + 热源 + 冷源为孤立系统，一个循环中热机熵不变。

$$\Delta S_{热源} = -\frac{1000}{440} = -2.273\;\text{kJ/K}$$ $$\Delta S_{冷源} = +\frac{568}{280} = +2.029\;\text{kJ/K}$$ $$\Delta S_{iso} = -2.273 + 2.029 = -0.244\;\text{kJ/K} < 0$$

总熵减少，违反熵增原理，故该热机不可能实现。

验证：设计效率 $\eta = 432/1000 = 43.2\%$，而 Carnot 效率 $= 1 - 280/440 = 36.4\%$。效率超过 Carnot 极限。

本节要点：
1. 绝热容器中的非准静态过程（突然停止、自由混合）：用 $\Delta E_{total}=0$（无功无热）
2. 两绝热线不相交：经典反证法题，用热二定律
3. 有限热源做功：Carnot 效率随温度变化，需要积分
4. 判断热机可行性：用 $\Delta S_{iso}\geq 0$ 或 $\eta\leq\eta_C$ 均可
