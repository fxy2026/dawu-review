第9章 热力学基础

大纲7学时——全课程学时最多的章节！期末重中之重！

9.1 热力学第一定律

热力学第一定律
$$Q = \Delta E + W$$

系统吸收的热量 = 内能增加 + 系统对外做的功

符号约定：吸热 $Q>0$，放热 $Q<0$；系统膨胀做功 $W>0$，被压缩 $W<0$

微分形式：$\delta Q = \dd E + \delta W = \dd E + p\dd V$

核心区分：$E$（内能）是状态量（只与始末状态有关）；$Q$ 和 $W$ 是过程量（与过程路径有关）。
理想气体内能

$E = \nu\frac{i}{2}RT$，只是温度的函数。

对任意过程：$\Delta E = \nu C_V\Delta T = \nu\frac{i}{2}R\Delta T$

$\Delta E$ 的万能公式：无论什么过程，理想气体内能变化只取决于温度变化：$\Delta E = \nu C_V(T_2-T_1)$。这是因为理想气体内能只是 $T$ 的函数。

9.2 四种等值过程（核心中的核心！）

等体过程（$V=\text{const}$）

等体过程

$W = \int p\dd V = 0$（不膨胀不做功）

$Q = \Delta E = \nu C_V\Delta T$

过程方程：$\frac{p_1}{T_1} = \frac{p_2}{T_2}$

等压过程（$p=\text{const}$）

等压过程

$W = p\Delta V = p(V_2-V_1) = \nu R\Delta T$

$Q = \nu C_p\Delta T$

$\Delta E = \nu C_V\Delta T$

验证：$Q = \Delta E + W$ → $\nu C_p\Delta T = \nu C_V\Delta T + \nu R\Delta T$ → $C_p = C_V + R$ ✓

过程方程：$\frac{V_1}{T_1} = \frac{V_2}{T_2}$

等温过程（$T=\text{const}$）

等温过程

$\Delta E = 0$（温度不变→内能不变）

$W = Q = \int_{V_1}^{V_2}p\dd V = \int_{V_1}^{V_2}\frac{\nu RT}{V}\dd V = \nu RT\ln\frac{V_2}{V_1} = \nu RT\ln\frac{p_1}{p_2}$

过程方程：$p_1V_1 = p_2V_2$（等温线是 $p$-$V$ 图上的双曲线）

绝热过程（$Q=0$）

绝热过程

$\Delta E = -W$（不吸热，做功靠消耗内能）

绝热方程（三种等价形式）：

$$pV^\gamma = \text{const},\quad TV^{\gamma-1} = \text{const},\quad T^\gamma p^{1-\gamma} = \text{const}$$

绝热功：$W = -\Delta E = \nu C_V(T_1-T_2) = \frac{p_1V_1-p_2V_2}{\gamma-1} = \frac{\nu R(T_1-T_2)}{\gamma-1}$

绝热线比等温线更陡！因为 $pV^\gamma = \text{const}$ 中 $\gamma > 1$，所以绝热线下降更快。

四种过程总结对比表（必背！）

过程过程方程$W$$Q$$\Delta E$
等体$p/T=C$$0$$\nu C_V\Delta T$$\nu C_V\Delta T$
等压$V/T=C$$\nu R\Delta T$$\nu C_p\Delta T$$\nu C_V\Delta T$
等温$pV=C$$\nu RT\ln\frac{V_2}{V_1}$$W$$0$
绝热$pV^\gamma=C$$\frac{\nu R(T_1-T_2)}{\gamma-1}$$0$$-W$

9.3 循环过程与热机效率

循环过程

系统回到初态 → $\Delta E = 0$ → $W_{净} = Q_{吸} - |Q_{放}|$

$p$-$V$ 图上：顺时针→正循环（热机），$W>0$；逆时针→逆循环（制冷机），$W<0$

$W_{净}$ = $p$-$V$ 图上封闭曲线围成的面积

热机效率
$$\eta = \frac{W}{Q_1} = \frac{Q_1-|Q_2|}{Q_1} = 1 - \frac{|Q_2|}{Q_1}$$

$Q_1$=每循环吸收的总热量，$|Q_2|$=每循环放出的总热量

制冷系数
$$\varepsilon = \frac{|Q_2|}{W} = \frac{|Q_2|}{Q_1-|Q_2|}$$

9.4 卡诺循环与卡诺定理

卡诺循环

两个等温过程 + 两个绝热过程组成的可逆循环

工作在高温热源 $T_1$ 和低温热源 $T_2$ 之间：

$$\eta_C = 1 - \frac{T_2}{T_1}$$

卡诺制冷机：$\varepsilon_C = \frac{T_2}{T_1-T_2}$

卡诺定理
  1. 在相同高温源和低温源之间，可逆热机效率 > 不可逆热机效率
  2. 所有可逆热机效率相同，只取决于两个热源温度
例1：热机工作在 $T_1=600\,\text{K}$ 和 $T_2=300\,\text{K}$ 之间，每循环吸热 $Q_1=2000\,\text{J}$，实际效率 $\eta=40\%$，求做功和放热。
解

最大效率：$\eta_C = 1-300/600 = 50\%$，实际 $40\% < 50\%$ ✓（合理）

做功：$W = \eta Q_1 = 0.4\times2000 = 800\,\text{J}$

放热：$|Q_2| = Q_1 - W = 1200\,\text{J}$

9.5 热力学第二定律

两种等价表述

克劳修斯表述：热量不能自发地从低温物体传到高温物体（不引起其他变化）

开尔文表述：不可能从单一热源吸热全部转化为功而不引起其他变化（第二类永动机不可能）

两种表述等价——违反一个必违反另一个。

9.6 熵

熵的定义

$\dd S = \frac{\delta Q_{rev}}{T}$（可逆过程中，微小热量/温度）

熵是状态量，与路径无关。

理想气体熵变公式（万能公式！）
$$\Delta S = \nu C_V\ln\frac{T_2}{T_1} + \nu R\ln\frac{V_2}{V_1}$$ $$\Delta S = \nu C_p\ln\frac{T_2}{T_1} - \nu R\ln\frac{p_2}{p_1}$$

（对任意过程都可用，因为 $S$ 是状态量）

不可逆过程熵变计算方法

三步走：

  1. 确定始末状态（$T_1,V_1$ → $T_2,V_2$）
  2. 设计一个连接始末状态的可逆过程
  3. 沿可逆过程计算 $\Delta S = \int\frac{\delta Q_{rev}}{T}$
熵增原理
$$\Delta S_{孤立} \geq 0$$

孤立系统的熵永不减少。等号对应可逆过程，大于号对应不可逆过程。

熵增原理是热力学第二定律的数学表达。

例2（真题高频：自由膨胀的熵变）：理想气体从 $V_1$ 自由膨胀到 $V_2$（$Q=0,W=0$），求熵变。
解

自由膨胀：$Q=0,W=0$ → $\Delta E=0$ → $T_2=T_1$（理想气体）

设计可逆过程（等温膨胀）：

$$\Delta S = \nu C_V\ln\frac{T_2}{T_1} + \nu R\ln\frac{V_2}{V_1} = 0 + \nu R\ln\frac{V_2}{V_1} = \nu R\ln\frac{V_2}{V_1} > 0$$

$\Delta S > 0$：不可逆过程→熵增。

例3（热传导的熵变）：温度 $T_1$ 的物体与 $T_2$（$T_1>T_2$）的物体接触，传递热量 $Q$，两物体温度不变（热容很大），求总熵变。
解

高温物体：$\Delta S_1 = -Q/T_1$（放热）

低温物体：$\Delta S_2 = +Q/T_2$（吸热）

$$\Delta S_{总} = \frac{Q}{T_2} - \frac{Q}{T_1} = Q\left(\frac{1}{T_2}-\frac{1}{T_1}\right) > 0$$

T-S图

T-S图中，曲线下面积 = $\int T\dd S = Q_{rev}$（可逆过程吸收的热量）。等温线为水平线，绝热可逆为竖直线，卡诺循环为矩形。
