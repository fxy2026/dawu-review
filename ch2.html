第2章 牛顿运动定律

2.1 牛顿三大定律

牛顿第一定律（惯性定律）

任何物体在不受外力作用时（或合力为零），保持静止或匀速直线运动状态。

意义：(1) 定义了惯性参考系——牛顿定律成立的参考系；(2) 定义了力——改变运动状态的原因。

牛顿第二定律（核心）
$$\vv{F}_{合} = m\vv{a} = m\frac{\dd\vv{v}}{\dd t} = \frac{\dd\vv{p}}{\dd t}$$

分量形式：$F_x = ma_x,\quad F_y = ma_y,\quad F_z = ma_z$

自然坐标系形式：$F_\tau = ma_\tau = m\frac{\dd v}{\dd t},\quad F_n = ma_n = m\frac{v^2}{\rho}$

牛顿第三定律（作用-反作用）
$$\vv{F}_{12} = -\vv{F}_{21}$$

同时、等大、反向、共线、分别作用在两个不同物体上。

注意：牛顿第三定律在惯性系和非惯性系中都成立，但惯性力没有反作用力。

2.2 常见力

力公式说明
万有引力$F = G\frac{m_1m_2}{r^2}$$G=6.67\times10^{-11}\,\text{N·m}^2/\text{kg}^2$
重力$\vv{P} = m\vv{g}$$g\approx9.8\,\text{m/s}^2$（地球表面）

线性阻力$f = -bv$低速流体阻力

2.3 牛顿定律的应用

解题步骤

  1. 选研究对象（隔离体）
  2. 受力分析（画力图，不遗漏不多画）
  3. 建坐标系（选取使分解最简单的方向）
  4. 列方程（$F_x=ma_x$, $F_y=ma_y$）
  5. 解方程（联立约束条件）
例1（滑轮系统）：Atwood机——轻绳跨过不计质量的定滑轮连接 $m_1$ 和 $m_2$（$m_1>m_2$），求加速度和绳中张力。
解

对 $m_1$（取向下为正）：$m_1 g - T = m_1 a$

对 $m_2$（取向上为正）：$T - m_2 g = m_2 a$

两式相加：$(m_1-m_2)g = (m_1+m_2)a$

$$a = \frac{m_1-m_2}{m_1+m_2}g,\quad T = \frac{2m_1 m_2}{m_1+m_2}g$$
例2（含阻力的运动方程）：物体从 $v_0$ 开始在水平面上受阻力 $f=-bv$ 运动，求 $v(t)$。
解

$m\frac{\dd v}{\dd t} = -bv \implies \frac{\dd v}{v} = -\frac{b}{m}\dd t$

积分：$\ln v - \ln v_0 = -\frac{b}{m}t$

$$v = v_0 e^{-bt/m}$$

终端速度为 $0$（指数衰减），$\tau = m/b$ 为时间常数。

例3（竖直圆周运动）：小球在竖直面内做圆周运动（半径 $R$），求最高点不脱离的最小速度。
解

最高点：$mg + N = m\frac{v^2}{R}$

临界条件 $N=0$：$mg = m\frac{v_{min}^2}{R}$

$$v_{min} = \sqrt{gR}$$

2.4 非惯性系与惯性力

惯性力

在加速度为 $\vv{a}_0$ 的非惯性系中，为使牛顿定律形式不变，引入惯性力：

$$\vv{F}^* = -m\vv{a}_0$$

修正后的运动方程：$\vv{F} + \vv{F}^* = m\vv{a}'$（$\vv{a}'$ 为相对非惯性系的加速度）

常见惯性力

直线加速系统

若非惯性系以 $\vv{a}_0$ 做平动加速，惯性力 $\vv{F}^* = -m\vv{a}_0$，方向与 $\vv{a}_0$ 相反。

旋转系统：惯性离心力

在以角速度 $\omega$ 转动的参考系中：

$$\vv{F}_{cf} = m\omega^2 r\,\hat{\vv{r}}$$

方向沿径向向外（离心），大小 $F_{cf} = m\omega^2 r$。

Coriolis力（科里奥利力）$\vv{F}_{Cor} = -2m\vv{\omega}\times\vv{v}'$：地球自转效应（台风旋转方向、傅科摆等），本课程仅定性了解。
例4（电梯问题）：质量 $m$ 的人站在加速上升的电梯中（$a_0$ 向上），求地板对人的支撑力。
解法1（惯性系）

以地面为参考系：$N - mg = ma_0$，故 $N = m(g+a_0)$

解法2（非惯性系）

在电梯参考系中人静止，惯性力 $F^* = ma_0$（向下）：

$N - mg - ma_0 = 0$，故 $N = m(g+a_0)$

2.5 重要注意事项（PPT强调）

牛顿第二定律使用要点
  1. 瞬时性：加速度与力同时存在、同时消失、同时变化
  2. 仅适用于质点：广延物体需分解为无穷多质点分别应用
  3. 静摩擦力不正比于支持力：静摩擦力由牛顿第二定律确定，$f_s\leq\mu_s N$ 只是上限
  4. 仅适用于惯性系：非惯性系需引入惯性力
约束条件汇总
约束类型条件
不可伸长绳两端加速度相等
轻绳绳中张力处处相等 $T=\text{const}$
轻滑轮+轻绳两侧张力相等
滑动摩擦$f=\mu N$
纯滚动$v_C=R\omega$，$a_C=R\alpha$
例5（PPT课堂练习：斜面+绳+滑轮）：倾角 $\theta$ 的光滑斜面上的物块A（质量 $M+m_0$）用绳连接一挂在斜面顶端光滑滑轮上的物体B（质量 $m_0$），A在斜面底端。A先加速后减速，求加速度的转换条件。
解

A沿斜面上滑加速阶段（B下降）：

对B：$m_0 g-T=m_0 a$

对A：$T-(M+m_0)g\sin\theta=(M+m_0)a$

$a=\dfrac{m_0 g-(M+m_0)g\sin\theta}{m_0+M+m_0}=\dfrac{m_0-(M+m_0)\sin\theta}{M+2m_0}g$

当 $m_0 > (M+m_0)\sin\theta$ 时 $a>0$，即 A 加速上滑。当B落地后只剩A在斜面上减速。

例6（PPT重点：变力运动——终端速度推导）：质量 $m$ 的物体从静止下落，空气阻力 $f=bv^2$。求终端速度和 $v(t)$。
解

运动方程：$mg-bv^2=m\dfrac{\dd v}{\dd t}$

终端速度（$a=0$）：$v_t=\sqrt{mg/b}$

分离变量：$\dfrac{m\dd v}{mg-bv^2}=\dd t$，即 $\dfrac{\dd v}{v_t^2-v^2}=\dfrac{g}{v_t^2}\dd t$

利用部分分式或双曲正切：

$$v(t)=v_t\tanh\left(\frac{gt}{v_t}\right)$$

当 $t\to\infty$ 时 $v\to v_t$；当 $t\to0$ 时 $v\approx gt$（自由落体）。

例7（PPT重点：微元法——旋转绳张力）：均匀绳子（总质量 $m$、长 $L$）绕一端匀速转动，角速度 $\omega$。求距轴 $r$ 处张力。
解

取 $r$ 到 $r+\dd r$ 的微元 $\dd m=(m/L)\dd r$

向心力由张力差提供：$T(r)-T(r+\dd r)=\dd m\cdot\omega^2 r$

即 $-\dd T=(m/L)\omega^2 r\dd r$

边界条件：$T(L)=0$（绳末端自由端）

$$T(r)=\frac{m\omega^2}{2L}(L^2-r^2)$$

最大张力在 $r=0$（轴端）：$T_{\max}=m\omega^2 L/2$

2.6 典型力学模型（PPT总结）

模型1：斜面上的物体

倾角 $\theta$，质量 $m$，动摩擦系数 $\mu$

不滑动条件：$\tan\theta\leq\mu_s$

匀速下滑：$\mu_k=\tan\theta$

加速下滑：$a=g(\sin\theta-\mu_k\cos\theta)$

匀速推上去的最小力：$F=mg(\sin\theta+\mu_k\cos\theta)$（沿斜面方向推）

模型2：圆锥摆

质量 $m$ 的小球用长 $L$ 的绳在水平面上做圆周运动，绳与竖直方向夹角 $\theta$：

$T\cos\theta=mg$，$T\sin\theta=m\omega^2 L\sin\theta$

$\omega=\sqrt{g/(L\cos\theta)}$，$T=2\pi\sqrt{L\cos\theta/g}$

模型3：汽车过拱桥/凹桥

拱桥顶端（向心力向下）：$mg-N=mv^2/R$ → $N=m(g-v^2/R)$

$v=\sqrt{gR}$ 时 $N=0$（飞离地面）

凹桥底部（向心力向上）：$N-mg=mv^2/R$ → $N=m(g+v^2/R)$

例8（弹簧秤+升降机组合）：升降机中弹簧秤挂质量 $m$ 的物体。求各种运动状态下弹簧秤示数。
解

在非惯性系中：$T-mg-ma_0=0$（人为引入惯性力 $-ma_0$）

  • 匀速/$a_0=0$：$T=mg$
  • 加速上升/$a_0$ 向上：$T=m(g+a_0)$（超重）
  • 加速下降/$a_0$ 向下：$T=m(g-a_0)$（失重）
  • 自由下落/$a_0=g$：$T=0$（完全失重）

2.7 牛顿定律解题陷阱清单

1. 受力图画错：只画外力，不画内力（选单个物体时）。不要画"运动的力"——力来自相互作用，不来自运动本身。
2. 坐标轴选择不当：斜面问题通常沿斜面/垂直斜面建系最方便。圆周运动用切向/法向分解。
3. 加速度方向搞反：先设加速度方向，若解出负值说明方向与假设相反，结果仍正确。
4. 静摩擦力大小不确定：静摩擦力由运动方程决定，$f_s\leq\mu_s N$ 只是约束条件，不是等号。
5. 圆周运动合力≠向心力：向心加速度 $a_n=v^2/R$ 是法向分量，切向还可能有 $a_t$。

2.8 连接体问题专题

整体法 vs 隔离法

整体法与隔离法的选择策略

整体法：将多个物体视为一个系统，分析系统所受外力的合力。

  • 适用场景：求系统的加速度，或者不需要求内力（绳中张力、接触面之间的力）时
  • 优点：不需要考虑内力，方程少
  • 注意：整体法中内力不出现在方程中（牛顿第三定律抵消）

隔离法：将单个物体从系统中隔离出来，分析它所受的全部力（含内力）。

  • 适用场景：需要求内力（绳中张力、接触力、摩擦力等）时
  • 优点：可以求出任意物体之间的相互作用力

最常用策略：先整体后隔离——先用整体法求加速度，再隔离某个物体求内力。

使用整体法的前提：系统中所有物体的加速度必须相同（或有确定的运动学约束关系）。若各物体加速度不同且无简单约束，必须逐个隔离。
例9（叠放体+斜面：求临界摩擦系数）：如图，质量分别为 $m_1$ 和 $m_2$ 的两个物体叠放在倾角为 $\theta$ 的固定斜面上，$m_1$ 在上、$m_2$ 在下。$m_2$ 与斜面之间的动摩擦系数为 $\mu_2$，$m_1$ 与 $m_2$ 之间的静摩擦系数为 $\mu_1$。求两物体恰好一起匀速下滑时 $\mu_2$ 的值，以及此时 $\mu_1$ 至少为多少才能保证 $m_1$ 不在 $m_2$ 上滑动。
解

第一步：整体法求 $\mu_2$

将 $m_1 + m_2$ 视为一个整体，沿斜面方向（取下滑为正）：

$$（m_1+m_2)g\sin\theta - \mu_2(m_1+m_2)g\cos\theta = 0$$

匀速下滑，$a=0$。解得：

$$\mu_2 = \tan\theta$$

第二步：隔离 $m_1$ 求 $\mu_1$ 的最小值

对 $m_1$，它做匀速运动（$a=0$）。沿斜面方向：

$$m_1 g\sin\theta - f_1 = 0$$

其中 $f_1$ 是 $m_2$ 对 $m_1$ 的静摩擦力（沿斜面向上）。

垂直斜面方向：$N_1 = m_1 g\cos\theta$

$f_1 = m_1 g\sin\theta$，需满足 $f_1 \leq \mu_1 N_1$：

$$m_1 g\sin\theta \leq \mu_1 m_1 g\cos\theta$$ $$\mu_1 \geq \tan\theta$$

所以 $\mu_1$ 至少为 $\tan\theta$ 才能保证两者一起运动。

例10（三体连接系统：滑轮+斜面）：如图，光滑定滑轮通过轻绳连接两个物体。$m_1$ 悬挂在滑轮左侧自由下落，$m_2$ 放在倾角 $\theta$ 的光滑斜面上沿斜面上滑。轻绳不可伸长。求系统加速度和绳中张力。
解

设加速度为 $a$（$m_1$ 向下，$m_2$ 沿斜面向上），绳中张力为 $T$。

对 $m_1$（隔离法）：取向下为正

$$m_1 g - T = m_1 a \quad \cdots(1)$$

对 $m_2$（隔离法）：取沿斜面向上为正

$$T - m_2 g\sin\theta = m_2 a \quad \cdots(2)$$

约束条件：绳不可伸长 → 两端加速度大小相同。

(1)+(2) 消去 $T$：

$$m_1 g - m_2 g\sin\theta = (m_1 + m_2)a$$ $$\boxed{a = \frac{m_1 - m_2\sin\theta}{m_1 + m_2}\,g}$$

代回 (1) 得张力：

$$T = m_1(g - a) = m_1 g\left(1 - \frac{m_1 - m_2\sin\theta}{m_1+m_2}\right) = \frac{m_1 m_2(1+\sin\theta)}{m_1+m_2}\,g$$ $$\boxed{T = \frac{m_1 m_2(1+\sin\theta)}{m_1+m_2}\,g}$$

讨论：当 $m_1 = m_2\sin\theta$ 时 $a=0$（平衡）；当 $m_1 < m_2\sin\theta$ 时 $a<0$（$m_2$ 实际沿斜面下滑）。

例11（带摩擦的三体问题）：水平桌面上质量为 $M$ 的物体两侧分别用轻绳经桌边缘的两个光滑定滑轮连接质量为 $m_1$ 和 $m_2$ 的悬挂物体（$m_1 > m_2$）。$M$ 与桌面之间的动摩擦系数为 $\mu$。求加速度。
解

设 $m_1$ 侧下降，$m_2$ 侧上升，加速度为 $a$。设左绳张力 $T_1$，右绳张力 $T_2$。

对 $m_1$：$m_1 g - T_1 = m_1 a$

对 $m_2$：$T_2 - m_2 g = m_2 a$

对 $M$（水平方向）：$T_1 - T_2 - \mu Mg = Ma$

三式相加：

$$(m_1 - m_2)g - \mu Mg = (m_1 + m_2 + M)a$$ $$\boxed{a = \frac{(m_1 - m_2 - \mu M)g}{m_1 + m_2 + M}}$$

当 $m_1 - m_2 \leq \mu M$ 时系统静止不动（$a \leq 0$，实际为静摩擦）。

连接体问题常见陷阱：
  • 忘记分析运动学约束：定滑轮两侧加速度大小相同；动滑轮处加速度有 $2:1$ 关系
  • 整体法中摩擦力不能遗漏：叠放体中上面物体的重力会影响下面物体的正压力
  • 先判断运动方向：若不确定，可以先假设一个方向，解出负值说明实际方向相反

2.9 弹簧问题专题

弹簧的基本性质

弹簧力的特点

胡克定律：$F = -kx$，其中 $k$ 为劲度系数（弹簧常数），$x$ 为形变量（伸长/压缩量）。

弹簧力的关键特性：

  • 弹簧力不能突变：弹簧的形变量 $x$ 是连续变化的（需要时间改变长度），因此弹簧力也是连续变化的
  • 但弹簧可以瞬间被剪断，此时弹簧力突然消失（变为零）

弹簧突然剪断/释放的瞬间分析

核心原理：加速度可以突变

力改变 → 加速度立即改变（牛顿第二定律的瞬时性）。

但位移和速度不能突变（需要有限时间累积）。

分析步骤：

  1. 画出剪断前的状态，确定弹簧形变量和各处的力
  2. 剪断瞬间：弹簧力突然消失（或发生变化），其他力（重力、绳张力等）不变
  3. 用牛顿第二定律求剪断瞬间各物体的加速度
关键区分：
  • 剪断弹簧：弹簧力瞬间消失（弹簧被破坏）
  • 剪断绳子：绳中张力瞬间消失（但与之相连的弹簧力不会瞬间改变，因为弹簧长度来不及变化）
  • 核心判据：弹簧力由形变量决定，形变量不能突变 → 弹簧力不突变；绳的张力由运动状态决定，可以突变
例12（竖直弹簧上端剪断）：弹簧竖直悬挂，上端固定于天花板，下端挂质量为 $m$ 的物体，系统静止。现剪断弹簧上端，求剪断瞬间物体的加速度。
解

剪断前：弹簧伸长量 $x_0 = mg/k$，弹簧拉力 $F = kx_0 = mg$（向上），与重力平衡。

剪断瞬间：弹簧被剪断 → 弹簧力立即消失 → 物体只受重力。

$$a = g \quad \text{（向下，自由落体）}$$

注意：整个弹簧+物体系统自由下落，弹簧上端和下端的加速度都是 $g$。

例13（竖直弹簧下端剪断——经典考题）：竖直弹簧上端固定于天花板，下端悬挂质量为 $m$ 的物体A，A下面再用一根轻绳悬挂质量为 $m$ 的物体B，系统静止。现剪断A与B之间的绳子，求剪断瞬间A和B的加速度。
解

剪断前的分析：

对B：$T = mg$（绳子张力等于B的重力）

对A：$kx_0 = mg + T = 2mg$（弹簧拉力 = A的重力 + 绳的拉力）

所以弹簧伸长量 $x_0 = 2mg/k$，弹簧力 $F_{弹} = 2mg$（向上）。

剪断瞬间：

绳子张力突然消失（$T = 0$），但弹簧力不变（弹簧形变量来不及改变，仍为 $x_0$，弹簧力仍为 $2mg$）。

对B：只受重力 $mg$（绳断了，弹簧不与B直接相连）

$$\boxed{a_B = g \quad \text{（向下）}}$$

对A：受力 = 弹簧力（$2mg$ 向上）$-$ 重力（$mg$ 向下）$= mg$（向上）

$$\boxed{a_A = g \quad \text{（向上）}}$$

弹簧的串联与并联

串联弹簧（首尾相连）

两弹簧 $k_1,k_2$ 串联，等效劲度系数 $k_{eq}$：

串联时两弹簧受力相同 $F$，总伸长量 $= x_1 + x_2 = F/k_1 + F/k_2$

$$\frac{1}{k_{eq}} = \frac{1}{k_1} + \frac{1}{k_2} \implies k_{eq} = \frac{k_1 k_2}{k_1 + k_2}$$

串联变软：$k_{eq} < \min(k_1, k_2)$

并联弹簧（并排连接）

两弹簧 $k_1,k_2$ 并联，等效劲度系数 $k_{eq}$：

并联时两弹簧形变量相同 $x$，总力 $= k_1 x + k_2 x$

$$k_{eq} = k_1 + k_2$$

并联变硬：$k_{eq} > \max(k_1, k_2)$

类比电阻：弹簧串联公式类似电阻并联（$1/k_{eq} = \sum 1/k_i$），弹簧并联公式类似电阻串联（$k_{eq} = \sum k_i$）。与电阻恰好"反过来"。
$n$ 个弹簧的推广
连接方式等效劲度系数
$n$ 个弹簧串联$\dfrac{1}{k_{eq}} = \displaystyle\sum_{i=1}^{n}\frac{1}{k_i}$
$n$ 个相同弹簧（$k_0$）串联$k_{eq} = k_0/n$
$n$ 个弹簧并联$k_{eq} = \displaystyle\sum_{i=1}^{n}k_i$
$n$ 个相同弹簧（$k_0$）并联$k_{eq} = nk_0$
例14（弹簧串联应用）：两根弹簧 $k_1 = 200\,\text{N/m}$，$k_2 = 300\,\text{N/m}$ 串联后悬挂质量 $m = 6\,\text{kg}$ 的物体。求各弹簧的伸长量和系统总伸长量。
解

串联弹簧中两根弹簧受力相同，均为 $F = mg = 60\,\text{N}$。

$k_1$ 的伸长量：$x_1 = F/k_1 = 60/200 = 0.3\,\text{m}$

$k_2$ 的伸长量：$x_2 = F/k_2 = 60/300 = 0.2\,\text{m}$

总伸长量：$x = x_1 + x_2 = 0.5\,\text{m}$

验证：$k_{eq} = \frac{200\times300}{200+300} = 120\,\text{N/m}$，$x = F/k_{eq} = 60/120 = 0.5\,\text{m}$ $\checkmark$

例15（弹簧系统剪断综合题）：如图，天花板上用弹簧（劲度系数 $k$）悬挂物体A（质量 $2m$），A下方用另一弹簧（劲度系数 $2k$）连接物体B（质量 $m$）。系统平衡后，剪断上方弹簧，求剪断瞬间A和B的加速度。
解

剪断前的平衡分析：

对B：下方弹簧拉力 $F_2 = mg$（向上），即下方弹簧伸长 $x_2 = mg/(2k)$

对A：上方弹簧拉力 $F_1$（向上）$= 2mg +$ 下方弹簧对A的拉力（向下）$= 2mg + mg = 3mg$

上方弹簧伸长 $x_1 = 3mg/k$

剪断上方弹簧瞬间：

上方弹簧力消失（弹簧被剪断）；下方弹簧力不变（仍为 $mg$，因为下方弹簧形变量来不及变化）。

对A：受力 = 重力 $2mg$（向下）$+$ 下方弹簧拉力 $mg$（向下）$= 3mg$（向下）

$$a_A = \frac{3mg}{2m} = \frac{3g}{2} \quad \text{（向下）}$$

对B：受力 = 重力 $mg$（向下）$+$ 下方弹簧拉力 $mg$（向上）$= 0$

$$a_B = 0$$

物理图像：剪断瞬间B处于平衡状态（弹簧力恰好平衡重力），加速度为零；而A因为失去了上方弹簧的支撑，受到重力和下方弹簧向下拉力的合力而加速下落。

弹簧问题常见陷阱：
  • 弹簧力不能突变（形变量连续），但绳中张力可以突变（绳无形变量）
  • 剪断弹簧 vs 剪断绳子的效果完全不同，必须区分清楚
  • 弹簧既可以拉伸也可以压缩，分析时注意弹簧处于拉伸还是压缩状态
  • 串联弹簧的等效系数取"倒数之和的倒数"，不要与并联搞混

2.12 Coriolis力与旋转参考系（PPT补充）

旋转参考系中的惯性力

在以角速度 $\vv{\omega}$ 旋转的非惯性系中，物体除了受真实力外，还需加入两个惯性力：

  1. 离心力：$\vv{F}_{cf} = -m\vv{\omega}\times(\vv{\omega}\times\vv{r}) = m\omega^2\vv{r}_\perp$（方向沿径向指向外侧）
  2. Coriolis力：$\vv{F}_{Cor} = -2m\vv{\omega}\times\vv{v}'$（$\vv{v}'$ 为旋转系中的速度）
Coriolis力的特点
  • 大小：$F_{Cor} = 2m\omega v'\sin\theta$（$\theta$ 为 $\vv{\omega}$ 与 $\vv{v}'$ 的夹角）
  • 方向：垂直于 $\vv{v}'$（类似洛伦兹力），不做功
  • 只在物体相对旋转系有运动（$\vv{v}'\neq 0$）时才存在
例（地球上的Coriolis效应）：地球自转 $\omega\approx 7.3\times10^{-5}\,\text{rad/s}$。在北半球：
分析
  • 水平运动的物体偏向右方（北半球），河流冲刷右岸更严重
  • 自由落体：不严格沿铅垂线下落，略偏东（因地球自转赤道线速度随高度增加）
  • Foucault摆：摆平面缓慢旋转，$T_{旋转}=24/\sin\phi$ 小时（$\phi$ 为纬度）

在赤道（$\phi=0$）Coriolis力水平分量为零，Foucault摆不转动。在两极（$\phi=90°$）旋转周期恰好24小时。

判断Coriolis力方向的快速方法（北半球）：面朝运动方向，Coriolis力指向你的右手边。南半球相反。

2.13 非惯性系中Newton定律的修正形式（PPT补充）

非惯性系中的运动方程

线性加速参考系（加速度 $\vv{a}_0$）：

$$m\vv{a}' = \vv{F} + (-m\vv{a}_0)$$

只需添加一个惯性力 $\vv{F}^* = -m\vv{a}_0$。

旋转参考系（完整形式）：

$$m\vv{a}' = \vv{F} + m\omega^2\vv{r}_\perp - 2m\vv{\omega}\times\vv{v}'$$

真实力 + 离心力 + Coriolis力

例（加速电梯中的弹簧测力计）：电梯以 $a$ 向上加速，弹簧秤挂一质量 $m$ 的物体。求示数。
解

在电梯（非惯性系）中，物体静止。平衡方程：

$$T - mg - ma = 0 \implies T = m(g+a)$$

"示数增大"——等效重力加速度 $g'=g+a$。

若电梯向下加速 $a$（或自由落体 $a=g$），则 $T=m(g-a)$。$a=g$ 时 $T=0$（完全失重）。

惯性力的本质：
  • 惯性力不是真实的力——没有施力物体，不满足Newton第三定律
  • 惯性力是非惯性系中的"数学修正项"，使Newton定律在形式上仍成立
  • 所有在非惯性系中感受到的"额外力"（电梯中变重/变轻、转弯时向外甩、旋转木马上的离心感）都是惯性力

2.14 空气阻力问题（PPT/MOOC高频）

线性阻力 $f = -bv$

线性阻力模型（低速情况）

物体在流体中低速运动时，阻力与速度成正比：$f = -bv$（$b$ 为阻力系数）。

末速度（终端速度）：当合力为零时 $mg = bv_T$，故：

$$v_T = \frac{mg}{b}$$

从静止下落的速度：运动方程 $m\dfrac{\dd v}{\dd t} = mg - bv$

分离变量并积分：

$$v(t) = v_T\left(1 - e^{-bt/m}\right) = v_T\left(1 - e^{-t/\tau}\right)$$

其中 $\tau = m/b$ 为时间常数，物理意义：

  • $t = \tau$ 时：$v = 0.632\,v_T$（达到63.2%的末速度）
  • $t = 2\tau$ 时：$v = 0.865\,v_T$（达到86.5%）
  • $t = 3\tau$ 时：$v = 0.950\,v_T$（达到95%，工程上近似认为已达末速度）

二次阻力 $f = -cv^2$

二次阻力模型（高速情况）

高速运动时阻力与速度平方成正比：$f = cv^2$（方向与运动方向相反）。

末速度：$mg = cv_T^2$，故：

$$v_T = \sqrt{\frac{mg}{c}}$$

从静止下落的速度：$m\dfrac{\dd v}{\dd t} = mg - cv^2$

$$v(t) = v_T\tanh\left(\frac{gt}{v_T}\right)$$

当 $t \to 0$ 时 $v \approx gt$（自由落体）；当 $t \to \infty$ 时 $v \to v_T$。

例16（雨滴末速度）：球形雨滴半径 $r = 1\,\text{mm}$，密度 $\rho = 1000\,\text{kg/m}^3$，空气阻力系数 $c = 0.5\rho_{air}\cdot C_D\cdot\pi r^2$（$\rho_{air} = 1.2\,\text{kg/m}^3$，$C_D \approx 0.45$）。估算末速度。
解

雨滴质量：$m = \frac{4}{3}\pi r^3\rho = \frac{4}{3}\pi(10^{-3})^3\times1000 \approx 4.19\times10^{-6}\,\text{kg}$

阻力系数：$c = 0.5\times1.2\times0.45\times\pi\times(10^{-3})^2 \approx 8.48\times10^{-7}\,\text{kg/m}$

$$v_T = \sqrt{\frac{mg}{c}} = \sqrt{\frac{4.19\times10^{-6}\times9.8}{8.48\times10^{-7}}} \approx 6.9\,\text{m/s}$$

这解释了为什么雨滴不会以很高速度砸到地面——空气阻力限制了下落速度。

例17（线性阻力下的水平运动）：质量 $m$ 的物体以初速度 $v_0$ 在光滑水平面上运动，受线性阻力 $f = -bv$。求物体最终走过的总距离。
解

$v(t) = v_0 e^{-bt/m}$

$$x = \int_0^\infty v_0 e^{-bt/m}\,\dd t = v_0\cdot\frac{m}{b} = \frac{mv_0}{b}$$

虽然速度永远不精确为零，但总路程是有限值 $x_{max} = mv_0/b$。

求末速度的快速方法：末速度时合力为零，直接令 $mg = bv_T$（线性阻力）或 $mg = cv_T^2$（二次阻力），不必求解微分方程。
易错：阻力公式中 $f = -bv$ 的负号表示方向与速度相反。列运动方程时若已取正方向，要注意阻力的符号。下落问题中取向下为正时，阻力向上为负：$ma = mg - bv$。

2.15 约束条件与连接体问题（真题重点）

约束条件的分类

三种基本约束
  1. 绳长约束：不可伸长的绳子总长度不变 → 对位移关系求导得速度约束 → 再求导得加速度约束
  2. 接触面法线约束：两物体接触时，沿接触面法线方向的速度/加速度分量相同（不穿透也不分离）
  3. 刚性连接约束：刚性杆连接的两点之间距离不变，沿杆方向的速度分量相同

经典模型：Atwood机

定滑轮两侧不等质量

轻绳跨过无质量的光滑定滑轮，两端分别悬挂质量 $m_1 > m_2$ 的物体。

约束：绳长不变 → 两侧加速度大小相等、方向相反。

$$a = \frac{(m_1 - m_2)g}{m_1 + m_2}$$ $$T = \frac{2m_1 m_2 g}{m_1 + m_2}$$

重要性质：

  • $T$ 总介于 $m_1 g$ 和 $m_2 g$ 之间：$m_2 g < T < m_1 g$
  • 当 $m_1 = m_2$ 时，$a = 0$，$T = m_1 g = m_2 g$（平衡）
  • 当 $m_1 \gg m_2$ 时，$a \to g$，$T \to 2m_2 g$
例18（斜面上的连接体）：光滑斜面（倾角 $\theta$），物体A（质量 $m_A$）在斜面上，通过轻绳经斜面顶端的光滑定滑轮连接悬挂物体B（质量 $m_B$）。$m_B > m_A\sin\theta$，求加速度和绳中张力。
解

设加速度 $a$（B下降，A沿斜面上升），绳张力 $T$。

对B（取向下为正）：$m_B g - T = m_B a \quad\cdots(1)$

对A（取沿斜面向上为正）：$T - m_A g\sin\theta = m_A a \quad\cdots(2)$

(1)+(2)：

$$a = \frac{m_B - m_A\sin\theta}{m_A + m_B}\,g$$ $$T = \frac{m_A m_B(1+\sin\theta)}{m_A + m_B}\,g$$
例19（绳绕钉子的几何约束）：一根绳绕过固定在天花板的光滑小钉，一端悬挂物体（质量 $m$），另一端以恒定速度 $u$ 水平拉动。当绳与竖直方向成 $\theta$ 角时，求物体的速度。
解

设绳从钉子到物体的长度为 $l$，拉出端的长度增加速率为 $u$。绳总长不变：

$$\frac{\dd l}{\dd t} = -u \quad\text{（物体端绳变短）}$$

物体的竖直位置 $y = l\cos\theta$，水平位置 $x = l\sin\theta$。

对于钉子位置固定的情况，物体上升速度（沿绳方向）为 $v_{绳} = u$。

物体竖直方向速度分量：$v_y = u\cos\theta$（当 $\theta$ 变化较慢时的近似）。

精确分析需要考虑 $\theta$ 的变化，但对于绳绕过光滑钉子的简单情况：沿绳方向速度等于拉绳速度 $u$，物体竖直速度 $v = u/\cos\theta$。

有质量滑轮两侧绳张力不等：
  • 无质量滑轮：两侧张力相等（$T_1 = T_2$），因为滑轮无转动惯量
  • 有质量滑轮：两侧张力之差提供滑轮的角加速度，$(T_1 - T_2)R = I\alpha$
  • 考试中若题目说"轻滑轮"则两侧张力相等，否则需考虑滑轮的转动
约束关系的建立技巧：
  • 写出绳长表达式（用各物体坐标表示），对时间求导
  • 动滑轮约束：动滑轮两侧绳的速度之和等于动滑轮速度的2倍
  • 遇到复杂约束，先画运动学关系图，再列微分约束

2.16 物体脱离曲面问题（PPT经典）

脱离条件

脱离曲面的判据

物体沿曲面运动时，曲面对物体施加法向支持力 $N$。脱离的条件是：

$$N = 0$$

当计算得到 $N < 0$ 时，说明物体已经脱离曲面（曲面不能提供拉力）。

脱离后物体做抛体运动，不再沿曲面运动。

例20（经典题：从光滑球面顶端滑下）：一个小物体从光滑球面（半径 $R$）的最高点由静止开始滑下。求物体脱离球面时的角度和速度。
解

设物体滑到与竖直方向成 $\theta$ 角的位置。取沿径向（指向球心）为正方向。

法向方程（向心方向）：

$$mg\cos\theta - N = \frac{mv^2}{R}$$

能量守恒（从顶端到角度 $\theta$，下降高度 $h = R - R\cos\theta = R(1-\cos\theta)$）：

$$\frac{1}{2}mv^2 = mgR(1-\cos\theta)$$ $$v^2 = 2gR(1-\cos\theta)$$

代入法向方程：

$$mg\cos\theta - N = \frac{m\cdot 2gR(1-\cos\theta)}{R} = 2mg(1-\cos\theta)$$ $$N = mg\cos\theta - 2mg(1-\cos\theta) = mg(3\cos\theta - 2)$$

脱离条件 $N = 0$：

$$3\cos\theta - 2 = 0 \implies \cos\theta = \frac{2}{3}$$ $$\boxed{\theta = \arccos\frac{2}{3} \approx 48.2°}$$

脱离速度：

$$v^2 = 2gR\left(1 - \frac{2}{3}\right) = \frac{2gR}{3}$$ $$\boxed{v = \sqrt{\frac{2gR}{3}}}$$

脱离高度：距球心高度 $h = R\cos\theta = 2R/3$，即在球顶以下 $R/3$ 处脱离。

例21（竖直圆弧内侧最低点的正压力）：质量 $m$ 的小球从光滑竖直圆弧轨道（半径 $R$）的最高点由静止释放，沿内侧滑到最低点。求最低点处轨道对小球的支持力。
解

从最高点到最低点，下降高度 $h = 2R$（直径）。

能量守恒：

$$\frac{1}{2}mv^2 = mg\cdot 2R \implies v^2 = 4gR$$

最低点法向方程（向心方向向上）：

$$N - mg = \frac{mv^2}{R} = \frac{m\cdot 4gR}{R} = 4mg$$ $$\boxed{N = 5mg}$$

更一般地，若从高度 $h$ 处释放：$N = mg + 2mgh/R$

脱离后的运动：
  • 物体脱离曲面后做抛体运动（仅受重力），不再沿曲面运动
  • 脱离瞬间的速度方向沿曲面的切线方向（不是沿径向）
  • 判断物体是否脱离时，先假设未脱离列方程，若解出 $N < 0$ 则说明实际已脱离
球面外侧 vs 内侧：
  • 球面外侧运动：$mg\cos\theta - N = mv^2/R$，可能脱离（$N$ 减至0）
  • 球面内侧运动：$N - mg\cos\theta = mv^2/R$，$N$ 总为正，不会脱离
  • 竖直圆周运动内侧最高点：$N + mg = mv^2/R$，临界条件 $N = 0$ 时 $v_{min} = \sqrt{gR}$

2.17 弹簧串并联与弹簧问题汇总（PPT补充）

弹簧连接的等效系数回顾

串联与并联公式
连接方式等效劲度系数特点
串联$\dfrac{1}{k_{eq}} = \dfrac{1}{k_1} + \dfrac{1}{k_2}$变软，$k_{eq} < \min(k_1,k_2)$
并联$k_{eq} = k_1 + k_2$变硬，$k_{eq} > \max(k_1,k_2)$

记忆口诀：弹簧串联像电阻并联，弹簧并联像电阻串联——"反着来"。

弹簧力 vs 绳张力的突变性

力的突变规则
  • 弹簧力不能突变：弹簧的形变量是连续的（需要时间改变长度），因此弹簧力也是连续变化的
  • 绳张力可以突变：绳没有弹性形变量，张力由运动状态决定，切断或松弛时可以瞬间变为零
  • 核心判据：有弹性形变的元件（弹簧）→ 力不能突变；无弹性形变的元件（绳、刚性杆）→ 力可以突变
例22（竖直弹簧系统突然剪断）：天花板上用弹簧1（劲度系数 $k_1$）悬挂物体A（质量 $m$），A下方再用弹簧2（劲度系数 $k_2$）连接物体B（质量 $m$）。系统平衡后，剪断弹簧1。求剪断瞬间A和B的加速度。
解

剪断前平衡分析：

对B：弹簧2拉力 $F_2 = mg$（向上），伸长量 $x_2 = mg/k_2$

对A：弹簧1拉力 $F_1 = mg + F_2 = 2mg$（向上），伸长量 $x_1 = 2mg/k_1$

剪断弹簧1的瞬间：

弹簧1被剪断 → 弹簧1的力立即消失

弹簧2未被剪断 → 弹簧2的形变量不能突变，弹簧2的力仍为 $mg$

对A：受力 = 重力 $mg$（向下）+ 弹簧2对A的拉力 $mg$（向下）= $2mg$（向下）

$$a_A = \frac{2mg}{m} = 2g \quad\text{（向下）}$$

对B：受力 = 重力 $mg$（向下）+ 弹簧2对B的拉力 $mg$（向上）= $0$

$$a_B = 0$$

B在剪断瞬间加速度为零——弹簧2的拉力恰好平衡B的重力。

例23（弹簧+绳组合：剪绳）：天花板固定弹簧（劲度系数 $k$），弹簧下端连物体A（质量 $m$），A下方用绳悬挂物体B（质量 $m$），系统平衡。现剪断A与B之间的绳。求剪断瞬间A和B的加速度。
解

剪断前：弹簧伸长量 $x_0 = 2mg/k$，弹簧力 $= 2mg$（向上），绳张力 $T = mg$

剪断绳的瞬间：

绳的张力立即消失（$T = 0$），但弹簧力不变（仍为 $2mg$，因形变量来不及变化）。

对A：弹簧力 $2mg$（向上）$-$ 重力 $mg$（向下）$= mg$（向上）

$$a_A = g \quad\text{（向上）}$$

对B：只受重力

$$a_B = g \quad\text{（向下，自由落体）}$$
简谐运动中的等效弹性系数

物体在弹簧系统中做微小振动时，振动周期 $T = 2\pi\sqrt{m/k_{eq}}$。

确定 $k_{eq}$ 的方法：将物体偏离平衡位置 $x$，求回复力 $F = -k_{eq}x$。

  • 两弹簧串联夹物体：$k_{eq} = k_1 k_2/(k_1+k_2)$ → 周期较长（振动较慢）
  • 两弹簧并联夹物体：$k_{eq} = k_1 + k_2$ → 周期较短（振动较快）
  • 物体两侧各接一根弹簧（两端固定）：等效为并联 $k_{eq} = k_1 + k_2$
弹簧问题常见陷阱补充：
  • 竖直弹簧悬挂物体：平衡位置不在弹簧原长处，但以平衡位置为原点时回复力仍为 $F = -kx$（重力被平衡位置的弹力抵消）
  • 弹簧可以被压缩也可以被拉伸：分析时注意判断弹簧处于压缩还是拉伸状态
  • 弹簧的弹性势能 $E_p = \frac{1}{2}kx^2$，其中 $x$ 是相对于原长的形变量

2.18 摩擦力分析专题（真题易错）

摩擦力基本规则

静摩擦力 vs 动摩擦力
  • 静摩擦力：$f_s \leq \mu_s N$，大小和方向由平衡方程或运动方程确定，$\mu_s N$ 只是上限
  • 动摩擦力：$f_k = \mu_k N$，方向与相对滑动方向相反，大小确定
  • 通常 $\mu_s > \mu_k$（静摩擦系数略大于动摩擦系数）

传送带问题

传送带上物体的运动分析

水平传送带以恒定速度 $v_b$ 运动，物体（质量 $m$）从静止放上传送带，动摩擦系数 $\mu_k$。

阶段1（$v_{物} < v_b$）：物体速度小于传送带速度

  • 物体相对传送带向后滑动 → 摩擦力向前（与传送带运动方向相同）
  • 加速度 $a = \mu_k g$（匀加速）
  • 持续到 $v_{物} = v_b$ 为止，历时 $t_1 = v_b/(\mu_k g)$

阶段2（$v_{物} = v_b$）：物体与传送带速度相同

  • 无相对滑动 → 摩擦力为静摩擦力 → 可以为零
  • 若无其他外力，物体以 $v_b$ 匀速运动

阶段3（若物体被额外加速至 $v_{物} > v_b$）：

  • 物体相对传送带向前滑动 → 摩擦力向后（减速）
  • 加速度 $a = -\mu_k g$（匀减速）
例24（传送带综合题）：水平传送带长 $L = 4\,\text{m}$，速度 $v_b = 2\,\text{m/s}$。物体从传送带左端以 $v_0 = 6\,\text{m/s}$ 的速度滑上传送带（与传送带运动方向相同），$\mu_k = 0.4$。物体到达右端时的速度是多少？
解

物体初速 $v_0 = 6 > v_b = 2$，物体相对传送带向前滑动，摩擦力向后。

阶段1（$v > v_b = 2$）：$a = -\mu_k g = -0.4\times9.8 = -3.92\,\text{m/s}^2$

减速到 $v_b$ 的时间：$t_1 = (v_0 - v_b)/|a| = (6-2)/3.92 \approx 1.02\,\text{s}$

阶段1走过的距离：$x_1 = v_0 t_1 + \frac{1}{2}a t_1^2 = 6(1.02) + \frac{1}{2}(-3.92)(1.02)^2 \approx 4.08\,\text{m}$

$x_1 \approx 4.08 > L = 4\,\text{m}$，说明物体在减速到 $v_b$ 之前就到达了右端。

实际在右端时：$v^2 = v_0^2 + 2aL = 36 + 2(-3.92)(4) = 36 - 31.36 = 4.64$

$$v = \sqrt{4.64} \approx 2.15\,\text{m/s}$$
例25（叠块问题）：物体A（质量 $m_A$）放在物体B（质量 $m_B$）上面，B放在光滑水平面上。A与B之间的静摩擦系数为 $\mu_s$。对B施加水平力 $F$，求A和B刚好不发生相对滑动的最大力 $F_{crit}$。
解

假设A和B一起运动（无相对滑动），系统加速度：

$$a = \frac{F}{m_A + m_B}$$

隔离A：A的加速度由B对A的静摩擦力提供：

$$f_s = m_A a = \frac{m_A F}{m_A + m_B}$$

不滑动条件：$f_s \leq \mu_s m_A g$

$$\frac{m_A F}{m_A + m_B} \leq \mu_s m_A g$$ $$\boxed{F_{crit} = \mu_s(m_A + m_B)g}$$

当 $F > F_{crit}$ 时，A和B发生相对滑动：B加速更快，A被甩在后面。此时需分别列方程：

  • 对A：$f_k = \mu_k m_A g = m_A a_A$ → $a_A = \mu_k g$
  • 对B：$F - \mu_k m_A g = m_B a_B$ → $a_B = (F - \mu_k m_A g)/m_B$
摩擦力分析的正确流程：
  1. 先假设不滑动：假设两接触面之间无相对滑动（整体法求加速度）
  2. 求出所需摩擦力：隔离一个物体，求维持同步运动所需的摩擦力大小
  3. 检验是否超限：比较所需摩擦力与最大静摩擦力 $\mu_s N$
  4. 若未超限 → 假设成立，摩擦力就是计算值（可能远小于 $\mu_s N$）
  5. 若超限 → 假设不成立，两物体相对滑动，摩擦力改用 $\mu_k N$，需重新列方程
静摩擦力方向的判断：静摩擦力的方向阻止相对滑动趋势。若不确定方向，可以假设一个方向，列方程后由正负号判断。

2.19 绳索张力分布（PPT补充）

Capstan方程：绳绕过粗糙圆柱体

Capstan（绞盘）方程

一根绳子绕过粗糙圆柱面（摩擦系数 $\mu$），一端张力 $T_0$（松端），另一端张力 $T$（紧端）。绳子绕过的角度为 $\theta$（弧度），则：

$$T(\theta) = T_0\,e^{\mu\theta}$$

张力随绕过角度指数增长。这就是绳子绕柱子几圈就能固定重物的原理。

Capstan方程的推导

取绳上微元（角度 $\dd\theta$），分析微元受力：

两端张力差提供微元沿切线方向的平衡，法向力由张力合力提供：

  • 法向：$\dd N = T\,\dd\theta$（绳段两端张力的法向分量之和）
  • 切向：$\dd T = \mu\,\dd N = \mu T\,\dd\theta$（摩擦力等于张力差）
$$\frac{\dd T}{T} = \mu\,\dd\theta \implies \ln\frac{T}{T_0} = \mu\theta \implies T = T_0 e^{\mu\theta}$$
例26（绳绕柱的张力比）：摩擦系数 $\mu = 0.3$。
计算不同绕法的张力比
绕法角度 $\theta$张力比 $T/T_0 = e^{\mu\theta}$
绕半圈$\pi \approx 3.14$$e^{0.3\pi} \approx 2.57$
绕1圈$2\pi \approx 6.28$$e^{0.6\pi} \approx 6.59$
绕2圈$4\pi \approx 12.57$$e^{1.2\pi} \approx 43.4$
绕3圈$6\pi \approx 18.85$$e^{1.8\pi} \approx 280$

绕3圈时，只需 $T_0 = 1\,\text{N}$ 的力就能抵住 $T = 280\,\text{N}$ 的拉力！

应用：船只系泊时绳子在系船柱上绕几圈，水手只需很小的力就能固定大船。登山保护、钓鱼线绕轮也利用此原理。

均匀绳在重力下的张力分布

竖直悬挂的均匀绳

总质量 $m$、总长 $L$ 的均匀绳竖直悬挂，线密度 $\lambda = m/L$。距下端 $y$ 处的张力：

$$T(y) = \lambda g y = \frac{mg}{L}\,y$$

张力从下端 $T(0) = 0$ 线性增加到上端 $T(L) = mg$。

若下端悬挂质量 $M$ 的物体：$T(y) = Mg + \lambda g y$

旋转绳的张力分布

绕一端匀速旋转的绳

均匀绳（质量 $m$、长 $L$）绕一端以角速度 $\omega$ 在水平面内匀速转动。距轴心 $r$ 处的张力：

$$T(r) = \frac{m\omega^2}{2L}(L^2 - r^2) = \frac{\lambda\omega^2}{2}(L^2 - r^2)$$

其中 $\lambda = m/L$ 为线密度。

推导：取 $r$ 到 $r + \dd r$ 的微元，质量 $\dd m = \lambda\,\dd r$，需要向心力 $\dd m\cdot\omega^2 r$。此向心力由张力差提供：

$$T(r) - T(r+\dd r) = -\dd T = \lambda\omega^2 r\,\dd r$$

从 $r$ 积分到 $L$（边界条件 $T(L) = 0$）：

$$T(r) = \int_r^L \lambda\omega^2 r'\,\dd r' = \frac{\lambda\omega^2}{2}(L^2 - r^2)$$

最大张力在轴端 $r = 0$：$T_{max} = \frac{1}{2}m\omega^2 L$

例27（旋转绳的应用）：质量 $m = 0.2\,\text{kg}$、长 $L = 1\,\text{m}$ 的均匀绳绕一端以 $n = 60\,\text{r/min}$ 旋转。求绳中点处的张力和最大张力。
解

$\omega = 2\pi n/60 = 2\pi\,\text{rad/s}$

$\lambda = m/L = 0.2\,\text{kg/m}$

中点张力（$r = L/2 = 0.5\,\text{m}$）：

$$T(L/2) = \frac{\lambda\omega^2}{2}\left(L^2 - \frac{L^2}{4}\right) = \frac{3\lambda\omega^2 L^2}{8}$$ $$= \frac{3\times0.2\times(2\pi)^2\times1}{8} = \frac{3\times0.2\times39.48}{8} \approx 2.96\,\text{N}$$

最大张力（$r = 0$）：

$$T_{max} = \frac{\lambda\omega^2 L^2}{2} = \frac{0.2\times39.48\times1}{2} \approx 3.95\,\text{N}$$
绳索问题的共同思路：取微元 $\dd r$（或 $\dd\theta$），对微元列Newton第二定律，得到张力的微分方程，结合边界条件（自由端 $T = 0$）积分求解。这是微元法的典型应用。
