三、选择填空高频概念题

概念辨析类（几乎每年出）
  1. 温度的微观本质：平均平动动能的量度（不是总动能）
  2. 理想气体内能只是温度的函数
  3. $T$升高时Maxwell分布曲线变矮变胖变宽
  4. $v_p < \bar{v} < v_{rms}$
  5. 绝热自由膨胀：$W=0,Q=0,\Delta T=0$，但不可逆
  6. 同时性的相对性
  7. 固有时间最短、固有长度最长
  8. 纯滚动是静摩擦，不是滑动摩擦
  9. 平行轴定理只能从质心轴出发
  10. 角动量守恒条件：合力矩为零（不是合力为零）

四、典型计算题模板

模板1：有质量滑轮的Atwood机

解题步骤
  1. 对每个物体/滑轮分别列方程（牛二 or 转动定律）
  2. 写约束条件：$a = R\alpha$（绳不伸长+不打滑）
  3. 注意：两侧绳张力不相等（$T_1 \neq T_2$）
  4. 联立求解 $a$、$T_1$、$T_2$

$a = \frac{(m_1-m_2)g}{m_1+m_2+J/R^2}$

模板2：波动方程建立

解题步骤
  1. 从题目提取：$A$、$\omega=2\pi/T=2\pi f$、$u$、$\lambda=u/f$
  2. 确定已知点的振动方程 $y(x_0,t)$（通常是原点或某给定点）
  3. 用时间延迟法写出任意点 $x$ 的方程：
    • 波沿 $+x$：$y(x,t) = A\cos[\omega(t-\frac{x-x_0}{u})+\varphi_0]$
    • 波沿 $-x$：$y(x,t) = A\cos[\omega(t+\frac{x-x_0}{u})+\varphi_0]$
  4. 利用初始条件（$t=0$ 时某点的位移和速度方向）确定 $\varphi_0$

模板3：热力学过程计算

解题步骤
  1. 判断过程类型（等容/等压/等温/绝热）
  2. 用状态方程求末态参量（$pV=\nu RT$，绝热用 $pV^\gamma=C$）
  3. 按公式算 $W$、$\Delta E$、$Q$：
    • $\Delta E = \nu C_V\Delta T$（任何过程都成立）
    • $W$ 按过程类型选公式
    • $Q = \Delta E + W$（热一律）

模板4：循环效率

解题步骤
  1. 在 $p$-$V$ 图上标出各状态点的 $p$、$V$、$T$
  2. 分段计算每个过程的 $Q$
  3. 将吸热（$Q>0$）和放热（$Q<0$）分开
  4. 效率 $\eta = 1 - \frac{|Q_{放}|}{Q_{吸}}$
  5. 顺时针为热机，逆时针为制冷机

模板5：熵变计算

解题步骤
  1. 确定初末状态（$T_1,V_1$ → $T_2,V_2$）
  2. 直接用万能公式：$\Delta S = \nu C_V\ln\frac{T_2}{T_1} + \nu R\ln\frac{V_2}{V_1}$
  3. 不可逆过程也用此公式（熵是状态量！）
  4. 自由膨胀：$T$ 不变 → $\Delta S = \nu R\ln\frac{V_2}{V_1} > 0$

五、考场时间分配建议

120分钟试卷时间规划
阶段时间内容
第一遍0-40 min选择+填空（不纠结，不会的先跳）
第二遍40-100 min计算题（每题12-15分钟）
第三遍100-115 min回头做跳过的题+检查
收尾115-120 min检查姓名学号、答题卡
关键提醒：
  • 计算题一定要写公式和步骤，即使结果算错也能拿步骤分
  • 不要在某一道选择题上花超过3分钟
  • 如果计算题完全不会，也要画图+写已知量+列相关公式
