第6&7章 机械振动与机械波

大纲合计10学时——期末重点章节

6.1 简谐运动（SHM）

简谐运动方程
$$x = A\cos(\omega t + \varphi_0)$$

三要素：

  • 振幅 $A$：最大位移，$A>0$
  • 角频率 $\omega$：$\omega = 2\pi/T = 2\pi\nu$，由系统固有参数决定
  • 初相 $\varphi_0$：由初始条件 $(x_0, v_0)$ 决定
速度与加速度
$$v = \frac{\dd x}{\dd t} = -A\omega\sin(\omega t+\varphi_0)$$ $$a = \frac{\dd v}{\dd t} = -A\omega^2\cos(\omega t+\varphi_0) = -\omega^2 x$$

运动微分方程：$\ddot{x} + \omega^2 x = 0$（判断SHM的标志！）

常见简谐运动系统

系统角频率周期
弹簧振子$\omega = \sqrt{k/m}$$T = 2\pi\sqrt{m/k}$
单摆（小角度）$\omega = \sqrt{g/l}$$T = 2\pi\sqrt{l/g}$
复摆（物理摆）$\omega = \sqrt{mgh/J}$$T = 2\pi\sqrt{J/(mgh)}$

确定初相 $\varphi_0$ 的方法

由初始条件求 $A$ 和 $\varphi_0$

设 $t=0$ 时 $x_0 = A\cos\varphi_0$，$v_0 = -A\omega\sin\varphi_0$

$$A = \sqrt{x_0^2 + (v_0/\omega)^2}$$ $$\tan\varphi_0 = -\frac{v_0}{\omega x_0}$$

注意：$\tan$ 有两解（差 $\pi$），需结合 $x_0$ 和 $v_0$ 的符号确定象限。

例1：弹簧振子 $m=0.5\,\text{kg}$, $k=200\,\text{N/m}$，初始 $x_0=0.02\,\text{m}$, $v_0=-0.4\,\text{m/s}$，写出 $x(t)$。
解

$\omega = \sqrt{k/m} = \sqrt{400} = 20\,\text{rad/s}$

$A = \sqrt{0.02^2+(0.4/20)^2} = \sqrt{0.0004+0.0004} = 0.02\sqrt{2}\,\text{m}$

$\tan\varphi_0 = -\frac{-0.4}{20\times0.02} = 1$，$\varphi_0 = \pi/4$（$x_0>0,v_0<0$，第一象限）

$x = 0.02\sqrt{2}\cos(20t+\pi/4)\,\text{m}$

旋转矢量法

将简谐运动视为匀速圆周运动在直径上的投影。矢量 $\vv{A}$（长度 $A$）以角速度 $\omega$ 逆时针旋转，其在 $x$ 轴投影即 $x(t)$。

用途：直观判断初相、相位关系、合振动。

6.2 简谐运动的能量

SHM能量

动能：$E_k = \frac{1}{2}mv^2 = \frac{1}{2}kA^2\sin^2(\omega t+\varphi_0)$

势能：$E_p = \frac{1}{2}kx^2 = \frac{1}{2}kA^2\cos^2(\omega t+\varphi_0)$

总能量：$E = E_k + E_p = \frac{1}{2}kA^2 = \text{const}$

平均动能 = 平均势能 = $\frac{1}{2}E = \frac{1}{4}kA^2$

能量以 $2\omega$ 的频率在动能和势能之间交换（频率是位移频率的两倍）。

6.3 阻尼振动与受迫振动

阻尼振动

运动方程：$m\ddot{x} + b\dot{x} + kx = 0$（$b$ 为阻尼系数）

欠阻尼（$b < 2\sqrt{km}$）：$x = A_0 e^{-\beta t}\cos(\omega' t+\varphi_0)$

其中 $\beta = b/(2m)$（衰减系数），$\omega' = \sqrt{\omega_0^2-\beta^2}$（阻尼频率，略低于固有频率）

受迫振动与共振

驱动力 $F = F_0\cos\omega_d t$ 下的稳态解：

$$x = A(\omega_d)\cos(\omega_d t - \varphi)$$

稳态振幅 $A$ 在 $\omega_d \approx \omega_0$ 时最大 → 共振

共振振幅与阻尼成反比：阻尼越小，共振峰越尖锐。

6.4 振动的叠加

同方向同频率振动的合成

$x_1 = A_1\cos(\omega t+\varphi_1)$，$x_2 = A_2\cos(\omega t+\varphi_2)$

合振动仍为简谐运动：$x = A\cos(\omega t+\varphi)$

$$A = \sqrt{A_1^2+A_2^2+2A_1A_2\cos(\varphi_2-\varphi_1)}$$ $$\tan\varphi = \frac{A_1\sin\varphi_1+A_2\sin\varphi_2}{A_1\cos\varphi_1+A_2\cos\varphi_2}$$

特殊情况：

  • $\Delta\varphi = 2n\pi$：$A = A_1+A_2$（相长/加强）
  • $\Delta\varphi = (2n+1)\pi$：$A = |A_1-A_2|$（相消/减弱）
同方向不同频率：拍

$\omega_1\approx\omega_2$ 时产生拍：$\nu_{拍} = |\nu_1-\nu_2|$

例2：$x_1 = 3\cos(2t+\pi/6)\,\text{cm}$，$x_2 = 4\cos(2t+2\pi/3)\,\text{cm}$，求合振动。
解

$\Delta\varphi = 2\pi/3 - \pi/6 = \pi/2$，$\cos(\pi/2)=0$

$A = \sqrt{9+16+0} = 5\,\text{cm}$

$\tan\varphi = \frac{3\sin(\pi/6)+4\sin(2\pi/3)}{3\cos(\pi/6)+4\cos(2\pi/3)} = \frac{1.5+3.464}{2.598-2} = \frac{4.964}{0.598} = 8.3$

$\varphi \approx 1.45\,\text{rad} \approx 83°$

$x = 5\cos(2t+1.45)\,\text{cm}$

7.1 机械波的基本概念

波的基本量

波速 $u$、波长 $\lambda$、频率 $\nu$、周期 $T$ 的关系：

$$u = \lambda\nu = \frac{\lambda}{T}$$

波速由介质决定（绳波：$u=\sqrt{T/\mu}$，$T$ 为张力，$\mu$ 为线密度）

横波：振动方向⊥传播方向（如绳波）

纵波：振动方向∥传播方向（如声波）

7.2 简谐波的波动方程

平面简谐波方程（沿 $+x$ 方向传播）
$$y(x,t) = A\cos\left[\omega\left(t-\frac{x}{u}\right)+\varphi_0\right]$$

等价形式：

$$y = A\cos(\omega t - kx + \varphi_0) = A\cos\left[2\pi\left(\frac{t}{T}-\frac{x}{\lambda}\right)+\varphi_0\right]$$

其中波数 $k = \frac{2\pi}{\lambda} = \frac{\omega}{u}$

沿 $-x$ 方向传播：将 $-kx$ 改为 $+kx$（或 $-x/u$ 改为 $+x/u$）

由已知点振动方程写波动方程

核心方法：
  1. 已知原点 $x=0$ 处振动 $y(0,t) = A\cos(\omega t+\varphi_0)$
  2. 波沿 $+x$ 传播，$x$ 处的振动比原点延迟 $x/u$
  3. 将 $t$ 替换为 $t-x/u$：$y(x,t) = A\cos[\omega(t-x/u)+\varphi_0]$

若已知的是 $x=x_0$ 处的振动，将 $t$ 替换为 $t-(x-x_0)/u$（沿 $+x$）或 $t+(x-x_0)/u$（沿 $-x$）。

例3：原点振动 $y_O = 0.03\cos(4\pi t)\,\text{m}$，波沿 $+x$ 方向传播，波速 $u=2\,\text{m/s}$，写出波动方程。
解

$\omega = 4\pi\,\text{rad/s}$，$A = 0.03\,\text{m}$，$\varphi_0 = 0$

$y(x,t) = 0.03\cos\left[4\pi\left(t-\frac{x}{2}\right)\right] = 0.03\cos(4\pi t - 2\pi x)\,\text{m}$

波长：$\lambda = u/\nu = uT = 2\times(2\pi/4\pi\cdot2) = 2\times0.5 = 1\,\text{m}$

波形图 vs 振动图

波形图振动图
横轴$x$（固定 $t$）$t$（固定 $x$）
纵轴$y$$y$
周期性空间周期 $\lambda$时间周期 $T$
读取传播方向波形平移方向无法直接读取
由波形图判传播方向→判各点速度方向：波形沿传播方向"平移"，前方的点将到达当前位置。即：$+x$ 传播时，某点的速度方向看"后方"的波形将要到达的状态。

7.3 波的能量

波的能量密度与强度

瞬时能量密度：$w = \rho A^2\omega^2\sin^2(\omega t-kx+\varphi_0)$

（动能密度和势能密度同相——这与驻波不同！）

平均能量密度：$\bar{w} = \frac{1}{2}\rho A^2\omega^2$

平均能流密度（波的强度）：$I = \bar{w}\cdot u = \frac{1}{2}\rho u A^2\omega^2$

7.4 波的干涉

两列同频波的干涉条件

波程差 $\Delta r = r_2 - r_1$，初相差 $\Delta\varphi_0 = \varphi_{02}-\varphi_{01}$

总相位差 $\Delta\phi = \frac{2\pi}{\lambda}\Delta r - \Delta\varphi_0$

干涉加强：$\Delta\phi = 2n\pi$　→　$A_{max} = A_1+A_2$

干涉减弱：$\Delta\phi = (2n+1)\pi$　→　$A_{min} = |A_1-A_2|$

当 $\Delta\varphi_0=0$（同相波源）时简化为 $\Delta r = n\lambda$（加强）或 $\Delta r = (n+\frac{1}{2})\lambda$（减弱）

7.5 驻波

驻波方程

两列等幅、同频、反向传播的波叠加：

$$y = 2A\cos(kx)\cos(\omega t)$$

波腹（振幅最大 $2A$）：$\cos(kx) = \pm1$，即 $x = n\frac{\lambda}{2}$

波节（振幅为零）：$\cos(kx) = 0$，即 $x = (2n+1)\frac{\lambda}{4}$

相邻波腹或波节间距 = $\lambda/2$

半波损失

波从波疏介质射入波密介质（如绳波从轻绳到重绳），反射时有半波损失——相位突变 $\pi$（相当于 $\lambda/2$ 的额外路程差）。

从波密到波疏反射：无半波损失。

两端固定弦的本征频率

弦长 $L$，两端为波节：$L = n\frac{\lambda_n}{2}$，$n=1,2,3,...$

$$\nu_n = n\frac{u}{2L} = n\nu_1$$

$\nu_1$ 为基频（一次谐波），$\nu_n$ 为 $n$ 次谐波。

7.6 多普勒效应

多普勒效应公式
$$\nu' = \frac{u + v_o}{u - v_s}\nu_0$$

符号约定：以"观察者向波源接近"为正方向。

  • $v_o$：观察者速度，接近波源取 $+$
  • $v_s$：波源速度，接近观察者取 $+$
例4：火车以 $30\,\text{m/s}$ 驶近观察者，鸣笛频率 $800\,\text{Hz}$，声速 $340\,\text{m/s}$。求观察者听到的频率。
解

观察者静止 $v_o=0$，波源接近 $v_s=30\,\text{m/s}$

$\nu' = \frac{340+0}{340-30}\times800 = \frac{340}{310}\times800 \approx 877\,\text{Hz}$

常考变形：反射问题——波打到运动反射面再回来，等效于两次多普勒效应。先将反射面视为"观察者"求接收频率，再将反射面视为"波源"求反射频率。
