第7章 机械振动与机械波

大纲合计10学时——期末重点章节

7.1 简谐运动（SHM）

简谐运动方程
$$x = A\cos(\omega t + \varphi_0)$$

三要素：

  • 振幅 $A$：最大位移，$A>0$
  • 角频率 $\omega$：$\omega = 2\pi/T = 2\pi\nu$，由系统固有参数决定
  • 初相 $\varphi_0$：由初始条件 $(x_0, v_0)$ 决定
速度与加速度
$$v = \frac{\dd x}{\dd t} = -A\omega\sin(\omega t+\varphi_0)$$ $$a = \frac{\dd v}{\dd t} = -A\omega^2\cos(\omega t+\varphi_0) = -\omega^2 x$$

运动微分方程：$\ddot{x} + \omega^2 x = 0$（判断SHM的标志！）

常见简谐运动系统

系统角频率周期
弹簧振子$\omega = \sqrt{k/m}$$T = 2\pi\sqrt{m/k}$
单摆（小角度）$\omega = \sqrt{g/l}$$T = 2\pi\sqrt{l/g}$
复摆（物理摆）$\omega = \sqrt{mgh/J}$$T = 2\pi\sqrt{J/(mgh)}$

确定初相 $\varphi_0$ 的方法

由初始条件求 $A$ 和 $\varphi_0$

设 $t=0$ 时 $x_0 = A\cos\varphi_0$，$v_0 = -A\omega\sin\varphi_0$

$$A = \sqrt{x_0^2 + (v_0/\omega)^2}$$ $$\tan\varphi_0 = -\frac{v_0}{\omega x_0}$$

注意：$\tan$ 有两解（差 $\pi$），需结合 $x_0$ 和 $v_0$ 的符号确定象限。

例1：弹簧振子 $m=0.5\,\text{kg}$, $k=200\,\text{N/m}$，初始 $x_0=0.02\,\text{m}$, $v_0=-0.4\,\text{m/s}$，写出 $x(t)$。
解

$\omega = \sqrt{k/m} = \sqrt{400} = 20\,\text{rad/s}$

$A = \sqrt{0.02^2+(0.4/20)^2} = \sqrt{0.0004+0.0004} = 0.02\sqrt{2}\,\text{m}$

$\tan\varphi_0 = -\frac{-0.4}{20\times0.02} = 1$，$\varphi_0 = \pi/4$（$x_0>0,v_0<0$，第一象限）

$x = 0.02\sqrt{2}\cos(20t+\pi/4)\,\text{m}$

旋转矢量法

将简谐运动视为匀速圆周运动在直径上的投影。矢量 $\vv{A}$（长度 $A$）以角速度 $\omega$ 逆时针旋转，其在 $x$ 轴投影即 $x(t)$。

用途：直观判断初相、相位关系、合振动。

7.2 简谐运动的能量

SHM能量

动能：$E_k = \frac{1}{2}mv^2 = \frac{1}{2}kA^2\sin^2(\omega t+\varphi_0)$

势能：$E_p = \frac{1}{2}kx^2 = \frac{1}{2}kA^2\cos^2(\omega t+\varphi_0)$

总能量：$E = E_k + E_p = \frac{1}{2}kA^2 = \text{const}$

平均动能 = 平均势能 = $\frac{1}{2}E = \frac{1}{4}kA^2$

能量以 $2\omega$ 的频率在动能和势能之间交换（频率是位移频率的两倍）。

7.3 阻尼振动与受迫振动

阻尼振动

运动方程：$m\ddot{x} + b\dot{x} + kx = 0$（$b$ 为阻尼系数）

欠阻尼（$b < 2\sqrt{km}$）：$x = A_0 e^{-\beta t}\cos(\omega' t+\varphi_0)$

其中 $\beta = b/(2m)$（衰减系数），$\omega' = \sqrt{\omega_0^2-\beta^2}$（阻尼频率，略低于固有频率）

受迫振动与共振

驱动力 $F = F_0\cos\omega_d t$ 下的稳态解：

$$x = A(\omega_d)\cos(\omega_d t - \varphi)$$

稳态振幅 $A$ 在 $\omega_d \approx \omega_0$ 时最大 → 共振

共振振幅与阻尼成反比：阻尼越小，共振峰越尖锐。

7.4 振动的叠加

同方向同频率振动的合成

$x_1 = A_1\cos(\omega t+\varphi_1)$，$x_2 = A_2\cos(\omega t+\varphi_2)$

合振动仍为简谐运动：$x = A\cos(\omega t+\varphi)$

$$A = \sqrt{A_1^2+A_2^2+2A_1A_2\cos(\varphi_2-\varphi_1)}$$ $$\tan\varphi = \frac{A_1\sin\varphi_1+A_2\sin\varphi_2}{A_1\cos\varphi_1+A_2\cos\varphi_2}$$

特殊情况：

  • $\Delta\varphi = 2n\pi$：$A = A_1+A_2$（相长/加强）
  • $\Delta\varphi = (2n+1)\pi$：$A = |A_1-A_2|$（相消/减弱）
同方向不同频率：拍

$\omega_1\approx\omega_2$ 时产生拍：$\nu_{拍} = |\nu_1-\nu_2|$

例2：$x_1 = 3\cos(2t+\pi/6)\,\text{cm}$，$x_2 = 4\cos(2t+2\pi/3)\,\text{cm}$，求合振动。
解

$\Delta\varphi = 2\pi/3 - \pi/6 = \pi/2$，$\cos(\pi/2)=0$

$A = \sqrt{9+16+0} = 5\,\text{cm}$

$\tan\varphi = \frac{3\sin(\pi/6)+4\sin(2\pi/3)}{3\cos(\pi/6)+4\cos(2\pi/3)} = \frac{1.5+3.464}{2.598-2} = \frac{4.964}{0.598} = 8.3$

$\varphi \approx 1.45\,\text{rad} \approx 83°$

$x = 5\cos(2t+1.45)\,\text{cm}$

7.5 机械波的基本概念

波的基本量

波速 $u$、波长 $\lambda$、频率 $\nu$、周期 $T$ 的关系：

$$u = \lambda\nu = \frac{\lambda}{T}$$

波速由介质决定（绳波：$u=\sqrt{T/\mu}$，$T$ 为张力，$\mu$ 为线密度）

横波：振动方向⊥传播方向（如绳波）

纵波：振动方向∥传播方向（如声波）

7.6 简谐波的波动方程

平面简谐波方程（沿 $+x$ 方向传播）
$$y(x,t) = A\cos\left[\omega\left(t-\frac{x}{u}\right)+\varphi_0\right]$$

等价形式：

$$y = A\cos(\omega t - kx + \varphi_0) = A\cos\left[2\pi\left(\frac{t}{T}-\frac{x}{\lambda}\right)+\varphi_0\right]$$

其中波数 $k = \frac{2\pi}{\lambda} = \frac{\omega}{u}$

沿 $-x$ 方向传播：将 $-kx$ 改为 $+kx$（或 $-x/u$ 改为 $+x/u$）

由已知点振动方程写波动方程

核心方法：
  1. 已知原点 $x=0$ 处振动 $y(0,t) = A\cos(\omega t+\varphi_0)$
  2. 波沿 $+x$ 传播，$x$ 处的振动比原点延迟 $x/u$
  3. 将 $t$ 替换为 $t-x/u$：$y(x,t) = A\cos[\omega(t-x/u)+\varphi_0]$

若已知的是 $x=x_0$ 处的振动，将 $t$ 替换为 $t-(x-x_0)/u$（沿 $+x$）或 $t+(x-x_0)/u$（沿 $-x$）。

例3：原点振动 $y_O = 0.03\cos(4\pi t)\,\text{m}$，波沿 $+x$ 方向传播，波速 $u=2\,\text{m/s}$，写出波动方程。
解

$\omega = 4\pi\,\text{rad/s}$，$A = 0.03\,\text{m}$，$\varphi_0 = 0$

$y(x,t) = 0.03\cos\left[4\pi\left(t-\frac{x}{2}\right)\right] = 0.03\cos(4\pi t - 2\pi x)\,\text{m}$

波长：$\lambda = u/\nu = uT = 2\times(2\pi/4\pi\cdot2) = 2\times0.5 = 1\,\text{m}$

波形图 vs 振动图

波形图振动图
横轴$x$（固定 $t$）$t$（固定 $x$）
纵轴$y$$y$
周期性空间周期 $\lambda$时间周期 $T$
读取传播方向波形平移方向无法直接读取
由波形图判传播方向→判各点速度方向：波形沿传播方向"平移"，前方的点将到达当前位置。即：$+x$ 传播时，某点的速度方向看"后方"的波形将要到达的状态。

7.7 波的能量

波的能量密度与强度

瞬时能量密度：$w = \rho A^2\omega^2\sin^2(\omega t-kx+\varphi_0)$

（动能密度和势能密度同相——这与驻波不同！）

平均能量密度：$\bar{w} = \frac{1}{2}\rho A^2\omega^2$

平均能流密度（波的强度）：$I = \bar{w}\cdot u = \frac{1}{2}\rho u A^2\omega^2$

7.8 波的干涉

两列同频波的干涉条件

波程差 $\Delta r = r_2 - r_1$，初相差 $\Delta\varphi_0 = \varphi_{02}-\varphi_{01}$

总相位差 $\Delta\phi = \frac{2\pi}{\lambda}\Delta r - \Delta\varphi_0$

干涉加强：$\Delta\phi = 2n\pi$　→　$A_{max} = A_1+A_2$

干涉减弱：$\Delta\phi = (2n+1)\pi$　→　$A_{min} = |A_1-A_2|$

当 $\Delta\varphi_0=0$（同相波源）时简化为 $\Delta r = n\lambda$（加强）或 $\Delta r = (n+\frac{1}{2})\lambda$（减弱）

7.9 驻波

驻波方程

两列等幅、同频、反向传播的波叠加：

$$y = 2A\cos(kx)\cos(\omega t)$$

波腹（振幅最大 $2A$）：$\cos(kx) = \pm1$，即 $x = n\frac{\lambda}{2}$

波节（振幅为零）：$\cos(kx) = 0$，即 $x = (2n+1)\frac{\lambda}{4}$

相邻波腹或波节间距 = $\lambda/2$

半波损失

波从波疏介质射入波密介质（如绳波从轻绳到重绳），反射时有半波损失——相位突变 $\pi$（相当于 $\lambda/2$ 的额外路程差）。

从波密到波疏反射：无半波损失。

两端固定弦的本征频率

弦长 $L$，两端为波节：$L = n\frac{\lambda_n}{2}$，$n=1,2,3,...$

$$\nu_n = n\frac{u}{2L} = n\nu_1$$

$\nu_1$ 为基频（一次谐波），$\nu_n$ 为 $n$ 次谐波。

7.10 多普勒效应

多普勒效应公式
$$\nu' = \frac{u + v_o}{u - v_s}\nu_0$$

符号约定：以"观察者向波源接近"为正方向。

  • $v_o$：观察者速度，接近波源取 $+$
  • $v_s$：波源速度，接近观察者取 $+$
例4：火车以 $30\,\text{m/s}$ 驶近观察者，鸣笛频率 $800\,\text{Hz}$，声速 $340\,\text{m/s}$。求观察者听到的频率。
解

观察者静止 $v_o=0$，波源接近 $v_s=30\,\text{m/s}$

$\nu' = \frac{340+0}{340-30}\times800 = \frac{340}{310}\times800 \approx 877\,\text{Hz}$

常考变形：反射问题——波打到运动反射面再回来，等效于两次多普勒效应。先将反射面视为"观察者"求接收频率，再将反射面视为"波源"求反射频率。

7.11 SHM初相判定详解（PPT重点）

旋转矢量法确定初相

$x=A\cos(\omega t+\varphi_0)$，$v=-A\omega\sin(\omega t+\varphi_0)$

$t=0$ 时：$x_0=A\cos\varphi_0$，$v_0=-A\omega\sin\varphi_0$

解法：

  1. $A=\sqrt{x_0^2+(v_0/\omega)^2}$
  2. $\cos\varphi_0=x_0/A$，$\sin\varphi_0=-v_0/(\omega A)$
  3. 由 $\cos\varphi_0$ 和 $\sin\varphi_0$ 的正负确定 $\varphi_0$ 所在象限
最常见错误：用 $\tan\varphi_0=-v_0/(\omega x_0)$ 时忘记判断象限！$\tan$ 有两个解（差 $\pi$），必须结合 $\cos\varphi_0$ 或 $\sin\varphi_0$ 的符号来定。
例（初相判定）：弹簧振子 $\omega=2\,\text{rad/s}$，$t=0$ 时 $x_0=-0.03\,\text{m}$，$v_0=0.06\,\text{m/s}$。
解

$A=\sqrt{0.03^2+(0.06/2)^2}=\sqrt{0.0009+0.0009}=0.03\sqrt{2}\,\text{m}$

$\cos\varphi_0=x_0/A=-0.03/(0.03\sqrt{2})=-1/\sqrt{2}$

$\sin\varphi_0=-v_0/(\omega A)=-0.06/(2\times0.03\sqrt{2})=-1/\sqrt{2}$

$\cos\varphi_0<0$，$\sin\varphi_0<0$ → 第三象限 → $\varphi_0=-3\pi/4$（或 $5\pi/4$）

$$x=0.03\sqrt{2}\cos(2t-\frac{3\pi}{4})\,\text{m}$$

7.12 复摆（物理摆）

复摆周期公式

刚体绕水平轴 $O$ 做小角度摆动，质心到轴距 $h$：

$$T=2\pi\sqrt{\frac{J_O}{mgh}}$$

其中 $J_O$ 为绕轴 $O$ 的转动惯量（用平行轴定理：$J_O=J_C+mh^2$）

等效摆长：$l_{eq}=J_O/(mh)=J_C/(mh)+h$

7.13 波动方程建立步骤（PPT详解）

由原点振动写任意点波动方程

已知原点振动 $y(0,t)=A\cos(\omega t+\varphi_0)$，波速 $u$，沿 $+x$ 传播：

  1. $x$ 处质点的振动比原点落后 $\Delta t=x/u$
  2. 用 $t-x/u$ 替换原点方程中的 $t$：
$$y(x,t)=A\cos[\omega(t-x/u)+\varphi_0]=A\cos(\omega t-kx+\varphi_0)$$

其中 $k=\omega/u=2\pi/\lambda$（波数）

若沿 $-x$ 传播：$y=A\cos(\omega t+kx+\varphi_0)$

已知任意点振动写波动方程

若已知 $x=x_1$ 处的振动 $y(x_1,t)=A\cos(\omega t+\varphi_1)$：

先求原点初相：$\varphi_0=\varphi_1+kx_1$（$+x$ 方向传播）

再写波动方程：$y=A\cos(\omega t-kx+\varphi_0)=A\cos(\omega t-kx+\varphi_1+kx_1)$

符号陷阱：向 $+x$ 传播时，$x>0$ 的点相位比原点落后（$-kx$），$x<0$ 的点相位超前。

7.14 驻波详解

驻波方程与特征

两列同频率、同振幅、反向传播的波叠加：

$$y=y_1+y_2=2A\cos(kx)\cos(\omega t)$$

特征：

  • 波节：$\cos(kx)=0$，即 $x=(2n+1)\lambda/4$，始终静止
  • 波腹：$|\cos(kx)|=1$，即 $x=n\lambda/2$，振幅最大 $2A$
  • 相邻波节距 = 相邻波腹距 = $\lambda/2$
  • 波节两侧相位相反（$\pi$ 差）
  • 不传播能量（驻波不是行波！）
弦振动的本征频率

两端固定的弦（长 $L$）：$L=n\cdot\lambda_n/2$ → $\lambda_n=2L/n$

$$\nu_n=\frac{u}{\lambda_n}=\frac{nu}{2L}=n\nu_1,\quad n=1,2,3,...$$

$n=1$：基频（基波），$n=2$：二次谐波……

半波损失

波从波疏介质射向波密介质反射时，反射波有 $\pi$ 的相位跃变（半波损失）。

等效于反射点是波节（固定端反射）。

从波密到波疏：无半波损失，反射点是波腹（自由端反射）。

7.15 振动合成专题

两个垂直方向SHM的合成：李萨如图形

当质点同时参与两个互相垂直方向的简谐运动时，合成运动的轨迹称为李萨如（Lissajous）图形。

设：

$$x = A_1\cos(\omega_1 t + \varphi_1), \quad y = A_2\cos(\omega_2 t + \varphi_2)$$

轨迹形状取决于频率比 $\omega_1:\omega_2$ 和初相差 $\Delta\varphi = \varphi_2 - \varphi_1$。

  • $\omega_1:\omega_2 = 1:1$ → 椭圆族（含直线、圆作为特例）
  • $\omega_1:\omega_2 = 1:2$ → "8"字形族
  • $\omega_1:\omega_2 = 2:3$ → 更复杂的封闭曲线
  • 若频率比为无理数 → 轨迹不封闭，遍历整个矩形区域

利用李萨如图形测频率：用已知频率的振动与待测频率比较，通过图形与 $x$ 轴（或 $y$ 轴）的切点数之比确定频率比。若图形与 $x$ 轴切点数为 $n_x$，与 $y$ 轴切点数为 $n_y$，则 $\omega_x:\omega_y = n_y:n_x$。

同频率垂直振动合成（重点）

两个同频率、互相垂直的简谐运动：

$$x = A_1\cos(\omega t), \quad y = A_2\cos(\omega t + \Delta\varphi)$$

消去时间参数 $t$，得到轨迹方程：

$$\frac{x^2}{A_1^2} + \frac{y^2}{A_2^2} - \frac{2xy}{A_1 A_2}\cos\Delta\varphi = \sin^2\Delta\varphi$$

这是一个椭圆方程，椭圆内接于矩形 $|x|\le A_1$，$|y|\le A_2$。

不同相位差 $\Delta\varphi$ 对应的特殊情况：

相位差 $\Delta\varphi$轨迹形状说明
$0$直线 $y = \dfrac{A_2}{A_1}x$正斜率直线（同相）
$\pi/4$斜椭圆椭圆长轴在第1、3象限
$\pi/2$正椭圆 $\dfrac{x^2}{A_1^2}+\dfrac{y^2}{A_2^2}=1$轴沿坐标轴；若 $A_1=A_2$ 则为圆
$3\pi/4$斜椭圆椭圆长轴在第2、4象限
$\pi$直线 $y = -\dfrac{A_2}{A_1}x$负斜率直线（反相）

旋转方向判定：$0 < \Delta\varphi < \pi$ 时为顺时针；$\pi < \Delta\varphi < 2\pi$ 时为逆时针。

例（垂直振动合成）：$x = 3\cos\omega t\,\text{cm}$，$y = 4\cos(\omega t + \pi/2)\,\text{cm}$，求合成轨迹方程并描述运动。
解

已知 $A_1 = 3\,\text{cm}$，$A_2 = 4\,\text{cm}$，$\Delta\varphi = \pi/2$。

由 $\Delta\varphi = \pi/2$，$\cos(\pi/2) = 0$，$\sin^2(\pi/2) = 1$，代入通式：

$$\frac{x^2}{9} + \frac{y^2}{16} = 1$$

轨迹为正椭圆，半长轴 $b = 4\,\text{cm}$（沿 $y$ 轴），半短轴 $a = 3\,\text{cm}$（沿 $x$ 轴）。

旋转方向：$\Delta\varphi = \pi/2 \in (0, \pi)$，故为顺时针旋转。

验证：$t = 0$ 时 $x = 3$，$y = 0$（最右端）；$t$ 稍增，$x$ 减小，$y = 4\cos(\omega t+\pi/2) = -4\sin\omega t$ 变负（向下），确认顺时针。

考试技巧：判断旋转方向时，只需看 $t=0$ 附近质点往哪个方向移动即可。不必死记公式。

7.16 波的叠加与干涉计算

两同频波源干涉的系统分析

两个同频率、同振幅、同相位的点波源 $S_1$ 和 $S_2$ 相距 $d$，在介质中产生波长为 $\lambda$ 的波。

空间某点 $P$ 到两波源的距离分别为 $r_1$ 和 $r_2$，则 $P$ 点的波程差：

$$\Delta r = r_2 - r_1$$

干涉条件（同相波源 $\Delta\varphi_0 = 0$）：

  • 加强点：$\Delta r = n\lambda$，$n = 0, \pm1, \pm2, ...$
  • 减弱点：$\Delta r = (n + \frac{1}{2})\lambda$，$n = 0, \pm1, \pm2, ...$
例（连线上的干涉）：两个同相点波源 $S_1$、$S_2$ 相距 $d = 5\lambda$，求 $S_1S_2$ 连线上加强点和减弱点的个数及位置。
解

设 $S_1$ 在原点，$S_2$ 在 $x = d = 5\lambda$。连线上的点 $P$ 距 $S_1$ 为 $x$（$0 < x < 5\lambda$）。

$r_1 = x$，$r_2 = d - x = 5\lambda - x$

波程差：$\Delta r = r_2 - r_1 = 5\lambda - 2x$

加强条件：$5\lambda - 2x = n\lambda$ → $x = \frac{(5-n)}{2}\lambda$

要求 $0 < x < 5\lambda$，即 $0 < \frac{5-n}{2}\lambda < 5\lambda$ → $-5 < n < 5$

$n = -4, -3, -2, -1, 0, 1, 2, 3, 4$，共 9个加强点（不含两端波源位置）。

薄膜干涉简介（等厚干涉）

光波在薄膜上、下表面反射产生两列相干光，光程差决定干涉结果。

折射率为 $n$、厚度为 $e$ 的薄膜，近似垂直入射时光程差：

$$\delta = 2ne + \frac{\lambda}{2}$$

其中 $\lambda/2$ 是由半波损失引起的附加光程差（一次发生半波损失时）。

等厚干涉条件：

  • 明纹（加强）：$\delta = 2ne + \frac{\lambda}{2} = k\lambda$，即 $2ne = (k-\frac{1}{2})\lambda$，$k = 1, 2, 3, ...$
  • 暗纹（减弱）：$\delta = 2ne + \frac{\lambda}{2} = (k+\frac{1}{2})\lambda$，即 $2ne = k\lambda$，$k = 0, 1, 2, ...$

劈尖干涉：厚度线性变化的薄膜（空气劈尖），产生等间距的平行条纹。相邻明纹（或暗纹）对应的厚度差 $\Delta e = \lambda/(2n)$。

牛顿环：平凸透镜与平面玻璃之间的空气薄膜产生同心圆环条纹。暗环半径 $r_k = \sqrt{k R\lambda}$，$k = 0, 1, 2, ...$（$R$ 为曲率半径）。

半波损失判断：光从光疏介质（低折射率）射向光密介质（高折射率）界面反射时，有半波损失。透射无半波损失。两个反射面若都有或都没有半波损失，则附加光程差为 $0$ 而非 $\lambda/2$。

7.17 多普勒效应应用

反射问题的多普勒效应

运动物体反射声波——等效于两次多普勒效应：

  1. 第一步：将运动反射面视为"观察者"，求它接收到的频率 $\nu_1$
  2. 第二步：将运动反射面视为"波源"（以频率 $\nu_1$ 发出），求最终观察者接收的频率 $\nu'$

反射面以速度 $v$ 接近静止波源（频率 $\nu_0$，声速 $u$）：

第一步：$\nu_1 = \frac{u + v}{u}\nu_0$（反射面作为观察者接近波源）

第二步：$\nu' = \frac{u}{u - v}\nu_1 = \frac{u + v}{u - v}\nu_0$

频率变化量：$\Delta\nu = \nu' - \nu_0 = \frac{2v}{u - v}\nu_0$

当 $v \ll u$ 时近似为 $\Delta\nu \approx \frac{2v}{u}\nu_0$

例（警车追逐问题）：警车以 $v_1 = 40\,\text{m/s}$ 追逐以 $v_2 = 30\,\text{m/s}$ 同向行驶的汽车，警笛频率 $\nu_0 = 1000\,\text{Hz}$，声速 $u = 340\,\text{m/s}$。求汽车司机听到的频率。
解

警车为波源，接近汽车，$v_s = v_1 = 40\,\text{m/s}$（取正）；

汽车为观察者，远离波源，$v_o = -v_2 = -30\,\text{m/s}$（远离取负）。

注意：题目为同向追逐，波源接近观察者但观察者也在远离。

$$\nu' = \frac{u + v_o}{u - v_s}\nu_0 = \frac{340 + (-30)}{340 - 40}\times 1000 = \frac{310}{300}\times 1000$$ $$\nu' = 1033\,\text{Hz}$$

频率升高，因为警车比汽车快（追上），两者间距在缩小。

例（反射测速）：静止雷达向以速度 $v$ 驶来的汽车发射频率为 $\nu_0$ 的超声波，求反射回来的频率。
解

第一步：汽车作为"观察者"接近静止波源：$\nu_1 = \frac{u+v}{u}\nu_0$

第二步：汽车反射，作为频率 $\nu_1$ 的"波源"接近静止观察者（雷达）：

$$\nu' = \frac{u}{u-v}\nu_1 = \frac{u}{u-v}\cdot\frac{u+v}{u}\nu_0 = \frac{u+v}{u-v}\nu_0$$

拍频（反射波与发射波的频率差）：

$$\Delta\nu = \nu' - \nu_0 = \frac{2v}{u-v}\nu_0 \approx \frac{2v}{u}\nu_0 \quad (v \ll u)$$

测出拍频 $\Delta\nu$ 即可算出车速 $v$。

超声波测速原理

利用多普勒效应的测速技术广泛应用于交通测速、医学（血流测速）和工业检测。

基本原理：向运动目标发射已知频率 $\nu_0$ 的超声波（或电磁波），检测反射波的频率偏移 $\Delta\nu$，由此计算目标速度：

$$v = \frac{\Delta\nu \cdot u}{2\nu_0 - \Delta\nu} \approx \frac{\Delta\nu \cdot u}{2\nu_0} \quad (\Delta\nu \ll \nu_0)$$

医学超声多普勒：超声波照射血管，被运动的红细胞反射。若超声频率 $\nu_0 = 5\,\text{MHz}$，血流速度 $v \sim 0.5\,\text{m/s}$，声在软组织中速度 $u \approx 1540\,\text{m/s}$，则频移约 $\Delta\nu \approx 2v\nu_0/u \approx 3.2\,\text{kHz}$——恰好在可听范围内，可直接用音频信号监测。

电磁波的多普勒效应：电磁波无需介质传播，公式形式不同。当 $v \ll c$ 时：$\Delta\nu/\nu_0 \approx v/c$（非相对论近似）。雷达测速和天文红移均基于此原理。

7.18 振动能量与振幅关系（MOOC补充）

SHM能量与振幅的关系

总能量：$E = \frac{1}{2}kA^2 = \frac{1}{2}m\omega^2 A^2$

动能与势能以 $2\omega$ 的频率振荡交替；平均动能 = 平均势能 = $E/2$

特殊位置的能量分配
位置势能 $E_p$动能 $E_k$说明
$x = 0$$0$$E$平衡位置，全部为动能
$x = \pm A$$E$$0$端点，全部为势能
$x = A/2$$E/4$$3E/4$$E_p = \frac{1}{2}k(A/2)^2 = E/4$
$x = A/\sqrt{2}$$E/2$$E/2$能量等分点
能量变化与振幅变化
  • 能量加倍 $E' = 2E$ → $A' = \sqrt{2}\,A$（振幅变为 $\sqrt{2}$ 倍）
  • 振幅减半 $A' = A/2$ → $E' = E/4$（能量变为原来的 $1/4$）

7.19 波的反射规律（PPT补充）

固定端与自由端反射
  • 固定端反射：反射波相位突变 $\pi$（半波损失），反射点为波节
  • 自由端反射：反射波无相位突变，反射点为波腹

设入射波 $y_i = A\cos(\omega t - kx)$，在 $x = L$ 处固定端反射，反射波方程：

$$y_r = -A\cos(\omega t + kx - 2kL)$$

在 $x = L$ 处自由端反射，反射波方程：

$$y_r = A\cos(\omega t + kx - 2kL)$$
驻波的形成与性质

入射波与反射波叠加形成驻波：

$$y = y_i + y_r = 2A\cos(kx)\cos(\omega t)$$

驻波的关键性质：

  • 驻波不传输能量——能量在波节与波腹之间往返交换
  • 相邻波节之间的所有质点同相振动
  • 波节两侧的质点反相振动（相位差 $\pi$）
  • 固定端反射 → 反射端为波节；自由端反射 → 反射端为波腹
例（固定端反射波方程）：入射波 $y_i = A\cos(\omega t - kx)$ 在 $x = L$ 处遇固定端反射，写出反射波方程。
解

反射波沿 $-x$ 方向传播，一般形式为 $y_r = A'\cos(\omega t + kx + \varphi)$。

固定端条件：$x = L$ 处合位移恒为零，即 $y_i(L,t) + y_r(L,t) = 0$ 对所有 $t$ 成立。

$y_i(L,t) = A\cos(\omega t - kL)$

要求 $y_r(L,t) = -A\cos(\omega t - kL) = A\cos(\omega t - kL + \pi)$

而 $y_r(L,t) = A'\cos(\omega t + kL + \varphi)$，比较得 $A' = A$，$kL + \varphi = -kL + \pi$

所以 $\varphi = -2kL + \pi$，代入：

$$y_r = A\cos(\omega t + kx - 2kL + \pi) = -A\cos(\omega t + kx - 2kL)$$

7.20 多普勒效应计算（真题高频）

多普勒效应通用公式
$$\nu' = \nu_0 \frac{u \pm v_R}{u \mp v_S}$$

口诀：上方符号取"接近"，下方符号取"远离"。

  • $u$：波在介质中的传播速度
  • $v_R$：观测者（Receiver）速度
  • $v_S$：波源（Source）速度
特殊情况速查表
情形公式频率变化
波源静止，观测者接近$\nu' = \nu_0\dfrac{u + v_R}{u}$升高
波源静止，观测者远离$\nu' = \nu_0\dfrac{u - v_R}{u}$降低
观测者静止，波源接近$\nu' = \nu_0\dfrac{u}{u - v_S}$升高
观测者静止，波源远离$\nu' = \nu_0\dfrac{u}{u + v_S}$降低
波源与观测者同速同向$\nu' = \nu_0$无变化
例（火车接近）：火车以 $v = 60\,\text{m/s}$ 驶向静止观测者，鸣笛频率 $\nu_0 = 500\,\text{Hz}$，声速 $u = 340\,\text{m/s}$，求观测者听到的频率。
解

观测者静止 $v_R = 0$，波源接近 $v_S = 60\,\text{m/s}$：

$$\nu' = \nu_0 \frac{u}{u - v_S} = 500 \times \frac{340}{340 - 60} = 500 \times \frac{340}{280} \approx 607\,\text{Hz}$$
反射多普勒效应

运动物体以速度 $v$ 接近静止波源，反射回的频率：

$$\nu'' = \nu_0 \frac{u + v}{u - v}$$

广泛应用于雷达测速、医学超声多普勒血流检测。

当 $v \ll u$ 时频移近似：$\Delta\nu \approx \dfrac{2v}{u}\nu_0$

声波 vs 光波的区别：声波有介质，波源运动与观测者运动的效果不同（即使速度大小相同，频率变化不等）。而光波（电磁波）无介质，只取决于相对速度，两者等价（相对论多普勒效应）。

7.21 波动方程建立方法（PPT重点）

波动方程建立步骤
  1. 确定基本参数：由题意求出波速 $u$、角频率 $\omega = 2\pi\nu$、波数 $k = \omega/u = 2\pi/\lambda$
  2. 选定参考点：通常选原点或已知振动信息的点
  3. 写参考点的振动方程：$y(x_0, t) = A\cos(\omega t + \varphi_1)$
  4. 考虑传播延迟，写出完整波动方程

沿 $+x$ 方向传播：$y = A\cos(\omega t - kx + \varphi_0)$

沿 $-x$ 方向传播：$y = A\cos(\omega t + kx + \varphi_0)$

初相的确定方法

由 $y = A\cos(\omega t - kx + \varphi_0)$ 得速度：

$$v_y = \frac{\partial y}{\partial t} = -A\omega\sin(\omega t - kx + \varphi_0)$$

在参考点 $x = 0$，$t = 0$ 时：

$$y_0 = A\cos\varphi_0, \quad v_{y0} = -A\omega\sin\varphi_0$$

结合 $y_0$ 和 $v_{y0}$ 的正负号判断 $\varphi_0$ 所在象限：

$y_0$$v_{y0}$$\cos\varphi_0$$\sin\varphi_0$$\varphi_0$ 象限
$>0$$<0$$>0$$>0$第一象限 $(0, \pi/2)$
$>0$$>0$$>0$$<0$第四象限 $(-\pi/2, 0)$
$<0$$>0$$<0$$<0$第三象限 $(-\pi, -\pi/2)$
$<0$$<0$$<0$$>0$第二象限 $(\pi/2, \pi)$
例1（波源处初相确定）：波源在 $x = 0$，$t = 0$ 时 $y = 0$ 且向 $+y$ 方向运动，波沿 $+x$ 传播，求波动方程。
解

$t = 0$, $x = 0$: $y_0 = 0 = A\cos\varphi_0$, $v_{y0} > 0$

$\cos\varphi_0 = 0$ → $\varphi_0 = \pm\pi/2$

$v_{y0} = -A\omega\sin\varphi_0 > 0$ → $\sin\varphi_0 < 0$ → $\varphi_0 = -\pi/2$

$$y = A\cos(\omega t - kx - \pi/2) = A\sin(\omega t - kx)$$
例2（非原点初相确定）：波沿 $+x$ 方向传播，已知 $x = x_0$ 处在 $t = 0$ 时 $y = A/2$ 且 $v_y > 0$，求完整波动方程。
解

在 $x = x_0$, $t = 0$ 时的振动相位为 $\phi_0 = -kx_0 + \varphi_0$。

$y(x_0, 0) = A\cos\phi_0 = A/2$ → $\cos\phi_0 = 1/2$ → $\phi_0 = \pm\pi/3$

$v_y(x_0, 0) = -A\omega\sin\phi_0 > 0$ → $\sin\phi_0 < 0$ → $\phi_0 = -\pi/3$

由 $\phi_0 = -kx_0 + \varphi_0 = -\pi/3$，得 $\varphi_0 = kx_0 - \pi/3$

$$y = A\cos\!\left(\omega t - kx + kx_0 - \frac{\pi}{3}\right) = A\cos\!\left[\omega t - k(x - x_0) - \frac{\pi}{3}\right]$$
快速判断法：若 $t = 0$ 时某点 $y = 0$ 且 $v_y > 0$，该点处于向正方向穿越平衡位置的状态，对应 $\sin$ 函数的起点，初相为 $-\pi/2$。

7.22 PPT补充：特殊SHM系统（建模专题）

板在圆柱上的振动

例（PPT例6-1）：质量为 $m$ 的均匀板水平置于两个相同的转动圆柱上，圆柱转速相同、方向相向，板与圆柱间摩擦系数为 $\mu$，两转轴间距为 $2l$。证明板沿水平方向做简谐振动，并求周期。
解

设板质心在两转轴中点 $O$ 右方 $x_c$ 处。板受两个圆柱的支持力 $N_1$、$N_2$ 和摩擦力 $f_1$、$f_2$。

板不转动，对两端取力矩参考点列力矩平衡方程：

$$N_1 = \frac{mg(l+x_c)}{2l},\quad N_2 = \frac{mg(l-x_c)}{2l}$$

两圆柱对板的摩擦力方向相同（都向中间推），合力：

$$F = f_1 - f_2 = \mu N_1 - \mu N_2 = \mu\frac{mg\cdot 2x_c}{2l} = -\frac{\mu mg}{l}x_c$$

（取偏离中心为正方向时，合力指向中心）

这是 $F = -kx_c$ 的形式，$k = \mu mg/l$，因此为简谐运动。

$$\omega = \sqrt{\frac{k}{m}} = \sqrt{\frac{\mu g}{l}},\quad T = 2\pi\sqrt{\frac{l}{\mu g}}$$

比重计的振动

例（PPT补充例）：质量为 $m$ 的比重计，放在密度为 $\rho$ 的液体中，圆管直径为 $d$。证明比重计在竖直方向的运动为简谐振动，求周期。
解

平衡时浮力等于重力：$\rho g V_0 = mg$（$V_0$ 为排水体积）。

偏离平衡位置 $x$（向下为正）后，排水体积增加 $\Delta V = \frac{\pi d^2}{4}x$，

合力 $F = mg - \rho g(V_0 + \frac{\pi d^2}{4}x) = -\rho g\frac{\pi d^2}{4}x$

这是线性回复力，系统做简谐运动：

$$\omega = \sqrt{\frac{\rho g\pi d^2}{4m}},\quad T = 2\pi\sqrt{\frac{4m}{\rho g\pi d^2}}$$

U形管振荡

例（PPT补充例：经典题型）：竖直等截面光滑U形管内装水，密度为 $\rho$，截面积为 $S$，水柱总质量为 $m$。使水偏离平衡位置后自由振动，求周期。
解

设某时刻右管液面高出平衡位置 $H$（左管低 $H$），则两管液面差为 $2H$。

回复力为高出部分液体的重力：$F = -\rho g S\cdot 2H$

水不可压缩，各处速度大小相同，整体动能：$E_k = \frac{1}{2}mv^2 = \frac{1}{2}m\dot{H}^2$

由动能定理（或牛顿第二定律 $m\ddot{H} = -2\rho g S\cdot H$）：

$$\omega = \sqrt{\frac{2\rho g S}{m}} = \sqrt{\frac{2g}{L}},\quad T = 2\pi\sqrt{\frac{L}{2g}}$$

其中 $L = m/(\rho S)$ 为水柱总长度。

记忆技巧：U形管振荡周期 $T = 2\pi\sqrt{L/(2g)}$，类似单摆公式但分母多了个 $2$（因为两侧液柱都提供回复力）。

例（PPT：弯管振荡）：等截面细弯管中水柱长度为 $L$，弯管两臂与竖直方向分别成角 $\alpha$ 和 $\beta$，求水的振荡圆频率。
解

设右臂液面偏离平衡位置 $H_1$（沿管方向），左臂 $H_2$。不可压缩条件：$H_1 = H_2$。

右侧液面竖直升高 $H_1\sin\beta$，左侧竖直降低 $H_2\sin\alpha$。

回复力（沿管方向合力）为高出部分液体在两臂的重力分量之和：

$$F = -\rho g S(\sin\alpha + \sin\beta)H$$

总质量 $m = \rho S L$：

$$\omega = \sqrt{\frac{g(\sin\alpha + \sin\beta)}{L}}$$

当 $\alpha = \beta = 90°$（标准竖直U管），$\omega = \sqrt{2g/L}$，与上面结果一致。

弹簧-滑轮系统

例（PPT练习）：轻弹簧（劲度系数 $k$）一端固定在墙上，另一端连接质量为 $m$ 的物体，绳绕过转动惯量为 $J$、半径为 $R$ 的定滑轮。求系统振动频率。
解

设物体下移 $x$（从平衡位置算起），弹簧伸长 $x$，滑轮转角 $\theta = x/R$。

物体：$m\ddot{x} = mg - T_1$

滑轮：$(T_1 - T_2)R = J\beta = J\ddot{x}/R$

弹簧：$T_2 = k(l+x)$

平衡条件：$mg = k\cdot l$

消去各力得：

$$(m + J/R^2)\ddot{x} = -kx$$ $$\omega = \sqrt{\frac{k}{m+J/R^2}}$$

物理意义：滑轮的转动惯量等效地增加了系统的惯性，等效质量为 $m_{eff} = m + J/R^2$。

弹簧串联与并联

弹簧组合的等效劲度系数
连接方式等效劲度系数角频率
并联（两弹簧同时受力，伸长量相同）$k_{eq} = k_1 + k_2$$\omega = \sqrt{(k_1+k_2)/m}$
串联（两弹簧首尾相连，受力相同）$\dfrac{1}{k_{eq}} = \dfrac{1}{k_1}+\dfrac{1}{k_2}$$\omega = \sqrt{k_1k_2/[m(k_1+k_2)]}$

判断方法：如果两弹簧伸长量相同（都等于物体位移），就是并联；如果两弹簧受力相同（力依次传递），就是串联。

物体两侧各有一弹簧（如 $k_1$-$m$-$k_2$）：两弹簧变形方向相反但力方向相同，等效为并联：$k_{eq} = k_1+k_2$。

有质量弹簧的振动

例（PPT补充例）：劲度系数为 $k$、原长为 $l$、质量为 $m$ 的均匀弹簧，一端固定，另一端连接质量为 $M$ 的物体，在光滑水平面上振动。求角频率。
解

弹簧均匀变形，距固定端 $s$ 处的位移为 $sx/l$（$x$ 为自由端位移），速度为 $s\dot{x}/l$。

弹簧的动能：

$$E_{k,弹} = \int_0^l \frac{1}{2}\frac{m}{l}\!\left(\frac{s\dot{x}}{l}\right)^2 ds = \frac{1}{2}\frac{m}{3}\dot{x}^2$$

总动能：$E_k = \frac{1}{2}(M+m/3)\dot{x}^2$，势能：$E_p = \frac{1}{2}kx^2$

由机械能守恒，对时间求导令其为零：

$$\boxed{\omega = \sqrt{\frac{k}{M+m/3}}}$$

结论：有质量弹簧等效于在物体质量上加上弹簧质量的 $\frac{1}{3}$。

7.23 PPT补充：非惯性系中的单摆

等效重力场方法（PPT重点）

在非惯性系中，引入惯性力 $\vv{F}_i = -m\vv{a}_0$（$\vv{a}_0$ 为参照系加速度），将重力与惯性力的合力视为等效重力：

$$\vv{g}_{eff} = \vv{g} - \vv{a}_0$$

单摆周期变为：

$$T = 2\pi\sqrt{\frac{l}{|\vv{g}_{eff}|}} = 2\pi\sqrt{\frac{l}{|\vv{g}-\vv{a}_0|}}$$
例1（加速上升的电梯）：单摆悬于以 $a_0$ 匀加速上升的电梯内。
解

$\vv{g}_{eff} = g + a_0$（向下），$T = 2\pi\sqrt{l/(g+a_0)}$，周期变短（摆得更快）。

例2（减速下降的电梯，$a_0 < g$）：单摆悬于以 $a_0$ 匀加速下降的电梯内。
解

$g_{eff} = g - a_0$，$T = 2\pi\sqrt{l/(g-a_0)}$，周期变长。

特别地，当 $a_0 = g$（自由下落/失重）时，$g_{eff} = 0$，$T \to \infty$——单摆不振动。

例3（水平加速的车厢）：单摆悬于以 $a_0$ 水平加速的车厢内。
解

$\vv{g}_{eff} = \vv{g} - \vv{a}_0$，竖直分量 $g$，水平分量 $a_0$：

$$g_{eff} = \sqrt{g^2 + a_0^2},\quad T = 2\pi\sqrt{\frac{l}{\sqrt{g^2+a_0^2}}}$$

单摆平衡位置不再是竖直的，而是向后偏离一个角度 $\alpha = \arctan(a_0/g)$。

例4（PPT练习：沿倾角 $\alpha$ 斜面加速）：单摆悬于以 $a_0$ 沿倾角为 $\alpha$ 的斜面向上运动的车厢内。
解

$\vv{a}_0$ 沿斜面向上，分解为水平和竖直分量：

$a_{0,水平} = a_0\cos\alpha$，$a_{0,竖直} = a_0\sin\alpha$

等效重力分量：水平 $a_0\cos\alpha$（惯性力向后），竖直 $g + a_0\sin\alpha$（向下）

$$g_{eff} = \sqrt{(a_0\cos\alpha)^2 + (g+a_0\sin\alpha)^2} = \sqrt{a_0^2 + g^2 + 2ga_0\sin\alpha}$$ $$T = 2\pi\sqrt{\frac{l}{\sqrt{a_0^2+g^2+2ga_0\sin\alpha}}}$$
解题口诀：不管参照系怎么加速，都用 $\vv{g}_{eff} = \vv{g} - \vv{a}_0$ 求等效重力，然后代入 $T = 2\pi\sqrt{l/g_{eff}}$ 即可。先画矢量图确定 $\vv{g}_{eff}$ 的方向和大小。

7.24 PPT补充：复摆进阶与轻杆摆

例（PPT：凹球面上的小振动）：小球在半径为 $R$ 的光滑凹球面底部做小幅振动，求周期。
解

设偏角 $\theta$ 很小，切向运动方程：$mR\ddot{\theta} = -mg\sin\theta \approx -mg\theta$

$$\ddot{\theta} + \frac{g}{R}\theta = 0 \quad \Rightarrow \quad \omega = \sqrt{g/R},\quad T = 2\pi\sqrt{R/g}$$

形式上与单摆相同，$R$ 取代了摆长 $l$。

例（PPT：轻杆摆+弹簧）：轻杆长 $L$，下端挂质量 $m$ 的小球，杆竖直时弹簧（劲度系数 $k$）无伸长。当摆角很小时，求周期。
解

摆角 $\theta$ 很小时，弹簧伸长 $\approx L\theta$，弹力 $f = kL\theta$，方向近似为切向。

切向方程：$mL\ddot{\theta} = -(mg + kL)\theta$

$$\omega = \sqrt{\frac{mg+kL}{mL}} = \sqrt{\frac{g}{L}+\frac{k}{m}},\quad T = 2\pi\sqrt{\frac{mL}{mg+kL}}$$

讨论：当 $k=0$ 时退化为单摆 $T=2\pi\sqrt{L/g}$。弹簧提供额外的回复力矩，使频率增大、周期减小。

例（PPT练习：均质杆+弹簧+小球复摆）：均质杆长 $L$、质量 $M$，上端可绕水平轴转动。弹簧（$k$）连接在某处，小球质量 $m$ 在下端。求微振动周期。
解思路

用刚体转动定律 $J\ddot{\theta} = M_{合}$：

  1. 转动惯量 $J = \frac{1}{3}ML^2 + mL^2$
  2. 重力矩 $M_g = -(Mg\frac{L}{2} + mgL)\sin\theta \approx -(Mg\frac{L}{2}+mgL)\theta$
  3. 弹力矩 $M_e = -kl^2\theta$（$l$ 为弹簧连接点到轴的距离）
  4. 合力矩 $M_{合} = -\left(\frac{MgL}{2}+mgL+kl^2\right)\theta$
$$\omega = \sqrt{\frac{MgL/2+mgL+kl^2}{\frac{1}{3}ML^2+mL^2}}$$
例（PPT练习：挖孔圆盘复摆）：半径为 $R$ 的均质圆盘上挖出一个半径 $R/2$ 的圆孔（孔的边缘与盘的边缘相切），剩余部分质量为 $m$，以过盘心 $O$ 并与盘面垂直的轴做微振动。求角频率。
解思路

利用"整体 = 剩余 + 挖去"的方法：

  • 剩余部分质心到轴的距离 $r_c$：由质心公式求出
  • 剩余部分对轴 $O$ 的转动惯量 $J = J_{总} - J_{孔}$（平行轴定理）
  • $\omega = \sqrt{mgr_c/J}$

PPT给出结果：$\omega = \sqrt{g/R}$（具体数值取决于挖孔位置和大小）。

7.25 PPT补充：能量法求角频率

能量法（与牛顿定律法等价的另一种方法）

对于保守系统，机械能守恒：$E = E_k + E_p = \text{const}$

对时间求导：$\frac{dE}{dt} = 0$

将动能和势能分别用广义坐标 $x$ 及其导数 $\dot{x}$ 表示，求导后消去 $\dot{x}$（不为零），即得到 $\ddot{x} + \omega^2 x = 0$ 的形式。

优点：避免分析约束力（如绳子张力、滑轮支持力等），特别适合有滑轮和绳子的复杂系统。

例（PPT能量法：弹簧-滑轮系统）：弹簧（$k$）连物体（$m$），绕过均质圆柱滑轮（质量 $M$，半径 $R$），$J = \frac{1}{2}MR^2$。求角频率。
解

物体位移 $x$，速度 $v = \dot{x}$，滑轮角速度 $\Omega = v/R$。

总动能：$E_k = \frac{1}{2}mv^2 + \frac{1}{2}J\Omega^2 = \frac{1}{2}mv^2 + \frac{1}{4}Mv^2 = \frac{1}{2}(m+M/2)v^2$

势能（弹簧，以平衡位置为零）：$E_p = \frac{1}{2}kx^2$

机械能守恒对 $t$ 求导：

$$(m+M/2)v\dot{v} + kxv = 0$$

$v\neq 0$ 消去，得 $(m+M/2)\ddot{x} + kx = 0$：

$$\omega = \sqrt{\frac{k}{m+M/2}}$$
