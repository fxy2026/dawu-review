第8章 气体动理论

8.1 平衡态与理想气体状态方程

热力学第零定律

若 $A$ 与 $C$ 热平衡，$B$ 与 $C$ 热平衡，则 $A$ 与 $B$ 热平衡。→ 温度概念的基础。

理想气体状态方程
$$pV = \nu RT = NkT$$

$R = 8.314\,\text{J/(mol·K)}$（普适气体常量）

$k = R/N_A = 1.38\times10^{-23}\,\text{J/K}$（Boltzmann常量）

$N_A = 6.022\times10^{23}\,\text{mol}^{-1}$（Avogadro常量）

也可写为：$p = nkT$（$n = N/V$ 为分子数密度）

道尔顿分压定律

混合气体总压 = 各组分分压之和：$p = p_1 + p_2 + \cdots$

$p_i = n_i kT$（各组分分子数密度 × $kT$）

8.2 压强公式与温度的微观本质

理想气体压强公式
$$p = \frac{1}{3}nm\overline{v^2} = \frac{2}{3}n\bar{\varepsilon}_k$$

$n$ = 分子数密度，$m$ = 单个分子质量，$\overline{v^2}$ = 方均根速率的平方

$\bar{\varepsilon}_k = \frac{1}{2}m\overline{v^2}$ = 分子平均平动动能

温度的微观本质
$$\bar{\varepsilon}_k = \frac{3}{2}kT$$

温度是分子平均平动动能的量度。温度相同 → 不同气体分子的平均平动动能相同。

能量均分定理

在温度 $T$ 的平衡态中，每个自由度的平均能量为 $\frac{1}{2}kT$

分子平均总动能：$\bar{\varepsilon} = \frac{i}{2}kT$

理想气体内能：$E = \nu\frac{i}{2}RT = \frac{i}{2}NkT$（只是温度的函数！）

8.4 Maxwell速率分布律

Maxwell速率分布函数
$$f(v) = 4\pi\left(\frac{m}{2\pi kT}\right)^{3/2}v^2\exp\left(-\frac{mv^2}{2kT}\right)$$

含义：$f(v)\dd v$ = 速率在 $v \sim v+\dd v$ 范围内的分子占总数的比率

归一化：$\int_0^\infty f(v)\dd v = 1$

三种统计速率（必背！）
名称公式（用 $m$）公式（用摩尔质量 $M$）系数
最概然速率 $v_p$$\sqrt{\frac{2kT}{m}}$$\sqrt{\frac{2RT}{M}}$$\sqrt{2}$
平均速率 $\bar{v}$$\sqrt{\frac{8kT}{\pi m}}$$\sqrt{\frac{8RT}{\pi M}}$$\sqrt{8/\pi}\approx1.60$
方均根速率 $v_{rms}$$\sqrt{\frac{3kT}{m}}$$\sqrt{\frac{3RT}{M}}$$\sqrt{3}\approx1.73$

大小关系：$v_p < \bar{v} < v_{rms}$（$\sqrt{2} < \sqrt{8/\pi} < \sqrt{3}$）

记忆口诀：最概然 $\sqrt{2}$、平均 $\sqrt{8/\pi}$、方均根 $\sqrt{3}$，系数分别为 1.41、1.60、1.73，依次增大。
例：$T=300\,\text{K}$ 时 $N_2$（$M=0.028\,\text{kg/mol}$）的三种速率。
解

$v_p = \sqrt{\frac{2\times8.314\times300}{0.028}} = \sqrt{178157} \approx 422\,\text{m/s}$

$\bar{v} = \sqrt{\frac{8\times8.314\times300}{\pi\times0.028}} \approx 476\,\text{m/s}$

$v_{rms} = \sqrt{\frac{3\times8.314\times300}{0.028}} \approx 517\,\text{m/s}$

8.5 平均自由程

平均碰撞频率与平均自由程

平均碰撞频率：$\bar{Z} = \sqrt{2}\pi d^2 n\bar{v}$

平均自由程：$\bar{\lambda} = \frac{\bar{v}}{\bar{Z}} = \frac{1}{\sqrt{2}\pi d^2 n} = \frac{kT}{\sqrt{2}\pi d^2 p}$

（$d$ 为分子有效直径）

8.6 Maxwell分布曲线分析（PPT重点）

分布曲线的特征
  • $f(v)$ 从零出发，先升后降，在 $v_p$ 处取极大值
  • 曲线下总面积 = 1（归一化）
  • $v_1\sim v_2$ 区间内分子比率 = $\int_{v_1}^{v_2}f(v)\dd v$（曲线下面积）
温度对分布曲线的影响
  • $T$ 升高 → 曲线变矮胖（峰降低、展宽、右移）
  • $T$ 降低 → 曲线变高瘦（峰升高、收窄、左移）
  • 无论温度如何变化，曲线下面积始终 = 1
  • $v_p$ 随 $T$ 增大而增大：$v_p = \sqrt{2kT/m}$
$f(v_p)$ 的计算（PPT考题）

将 $v_p = \sqrt{2kT/m}$ 代入 $f(v)$ 得最大值：

$$f(v_p) = 4\pi\left(\frac{m}{2\pi kT}\right)^{3/2}\cdot\frac{2kT}{m}\cdot e^{-1} = \frac{4}{\sqrt{\pi}}\cdot\frac{1}{v_p}\cdot e^{-1}$$

即 $f(v_p) = \frac{4}{\sqrt{\pi}\,v_p\,e}$

例（PPT课堂练习）：某理想气体在温度 $T_1$ 和 $T_2$（$T_2>T_1$）时的Maxwell速率分布曲线，判断下列说法：
解

① $T_2$ 时 $v_p$ 更大 ✓（$v_p\propto\sqrt{T}$）

② $T_2$ 时 $f(v_p)$ 更小 ✓（$f(v_p)\propto 1/v_p \propto 1/\sqrt{T}$）

③ 两条曲线下面积相同 ✓（都等于1）

④ $T_2$ 时高速分子比例增大 ✓（曲线右移展宽）

注意：
  • $f(v)$ 不是概率，是概率密度！$f(v)\dd v$ 才是概率（比率）
  • $f(v_p)$ 的量纲是 $\text{s/m}$（速率的倒数）
  • 不同气体（不同$m$）在同一温度下曲线也不同——质量大的峰更高更窄

8.7 压强公式推导要点（PPT重点）

理想气体微观模型假设
  1. 分子是弹性质点，除碰撞外无相互作用
  2. 大量分子运动是完全无序的（各方向等概率）
  3. 各方向速度分量等价：$\overline{v_x^2} = \overline{v_y^2} = \overline{v_z^2} = \frac{1}{3}\overline{v^2}$
单分子对壁的冲量

分子以 $v_x$ 撞壁后弹回，动量变化 $\Delta p = 2mv_x$

单位时间内撞击面积 $A$ 的次数 = $\frac{1}{2}nv_x\cdot A$（只有正向运动的一半分子）

平均力 = 冲量之和/时间 → 除以面积得压强：

$$p = nm\overline{v_x^2} = \frac{1}{3}nm\overline{v^2} = \frac{2}{3}n\bar{\varepsilon}_k$$
例（分子撞壁频率）：$T = 300\,\text{K}$，$p = 1.0\times10^5\,\text{Pa}$ 的 $N_2$。求单位时间撞击容器壁单位面积的分子数 $\Gamma$。
解

$\Gamma = \frac{1}{4}n\bar{v}$

$n = p/(kT) = 1.0\times10^5/(1.38\times10^{-23}\times300) = 2.42\times10^{25}\,\text{m}^{-3}$

$\bar{v} = \sqrt{8RT/(\pi M)} = 476\,\text{m/s}$

$\Gamma = \frac{1}{4}\times2.42\times10^{25}\times476 = 2.88\times10^{27}\,\text{m}^{-2}\text{s}^{-1}$

8.8 混合气体问题（PPT重点）

混合气体的处理方法
  1. 分压定律：$p = p_1 + p_2 = n_1kT + n_2kT$
  2. 内能可加：$E = E_1 + E_2 = \nu_1\frac{i_1}{2}RT + \nu_2\frac{i_2}{2}RT$
  3. 等效自由度：混合气体可以定义等效 $C_V$ 和 $\gamma$
混合气体比热容
$$C_{V,\text{mix}} = \frac{\nu_1 C_{V1} + \nu_2 C_{V2}}{\nu_1+\nu_2},\quad C_{p,\text{mix}} = C_{V,\text{mix}} + R$$ $$\gamma_{\text{mix}} = \frac{C_{p,\text{mix}}}{C_{V,\text{mix}}}$$
例（PPT）：1 mol He 和 2 mol H₂ 混合，求 $C_{V,\text{mix}}$、$C_{p,\text{mix}}$ 和 $\gamma_{\text{mix}}$。
解

He: 单原子，$i_1=3$，$C_{V1}=\frac{3}{2}R$；H₂: 双原子，$i_2=5$，$C_{V2}=\frac{5}{2}R$

$C_{V,\text{mix}} = \frac{1\times\frac{3}{2}R + 2\times\frac{5}{2}R}{1+2} = \frac{\frac{3}{2}+5}{3}R = \frac{13}{6}R$

$C_{p,\text{mix}} = C_{V,\text{mix}} + R = \frac{13}{6}R + R = \frac{19}{6}R$

$\gamma_{\text{mix}} = \frac{19/6}{13/6} = \frac{19}{13} \approx 1.46$

8.9 Boltzmann分布与等温气压公式

Boltzmann分布律

在重力场中，温度为 $T$ 的等温大气，高度 $h$ 处的分子数密度：

$$n(h) = n_0\exp\left(-\frac{mgh}{kT}\right) = n_0\exp\left(-\frac{E_p}{kT}\right)$$

$n_0$ 为 $h=0$ 处的分子数密度。

等温气压公式
$$p(h) = p_0\exp\left(-\frac{Mgh}{RT}\right)$$

（由 $p=nkT$ 直接得出）

例：海平面气压 $p_0=1.013\times10^5\,\text{Pa}$，$T=300\,\text{K}$，求 $h=8848\,\text{m}$（珠峰）处大气压。（空气 $M=0.029\,\text{kg/mol}$）
解

$p = p_0 e^{-Mgh/(RT)} = 1.013\times10^5\times e^{-0.029\times9.8\times8848/(8.314\times300)}$

$= 1.013\times10^5\times e^{-1.013} = 1.013\times10^5\times0.363$

$\approx 3.68\times10^4\,\text{Pa} \approx 0.36\,\text{atm}$

Boltzmann分布的本质：粒子在能量高的状态出现的概率与 $e^{-E/(kT)}$ 成正比。温度越高，高能态粒子越多。

8.10 气体动理论解题要点（PPT总结）

常见易混辨析
概念正确理解常见错误
温度的意义分子平均平动动能的量度误认为是总动能的量度
内能$E=\nu\frac{i}{2}RT$，包含平动+转动只算平动部分
$\bar{\varepsilon}_k = \frac{3}{2}kT$只是平动动能，对所有气体成立误以为是总动能
速率分布$f(v)$ 是概率密度把 $f(v)$ 当成速率 $v$ 的分子数
$v_p$$f(v)$ 取极大值处的速率与 $\bar{v}$ 或 $v_{rms}$ 混淆
典型计算公式速查
  • 分子数密度：$n = p/(kT) = \nu N_A/V$
  • 分子质量：$m = M/N_A$
  • 压强：$p = nkT = \frac{1}{3}nm\overline{v^2} = \frac{2}{3}n\bar{\varepsilon}_k$
