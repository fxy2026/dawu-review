第5章 刚体力学

5.0 刚体运动的基本形式（PPT补充）

刚体的定义与运动分类

刚体：任意两质点之间的距离保持不变的物体（理想模型）。

运动形式特征举例
平动各点运动完全相同，任一连线保持平行直线运动的电梯
定轴转动所有质点绕同一固定直线做圆周运动门的开合、电机转子
定点转动一点固定，物体绕该点转动，转轴方向可变陀螺运动
平面运动物体各点始终在平行于某固定平面的平面内运动斜面上的滚动圆柱
一般运动平动 + 转动的组合飞行中翻滚的物体

关键：平面运动 = 质心平动 + 绕过质心轴的转动（Chasles定理）

5.1 转动惯量

转动惯量的定义

离散：$J = \sum m_i r_i^2$　　连续：$J = \int r^2\dd m$

其中 $r$ 是质量元到转轴的距离。

常见刚体转动惯量

刚体转轴转动惯量
细棒（长$L$，质量$M$）过中心，垂直$\frac{1}{12}ML^2$
细棒过端点，垂直$\frac{1}{3}ML^2$
圆环/薄壁圆筒（$R$）中心轴$MR^2$
圆盘/实心圆柱（$R$）中心轴$\frac{1}{2}MR^2$
实心球（$R$）直径$\frac{2}{5}MR^2$
球壳（$R$）直径$\frac{2}{3}MR^2$
平行轴定理
$$J = J_C + Md^2$$

$J_C$ 为过质心的轴的转动惯量，$d$ 为两平行轴间距。

例：验证细棒过端点 = 过中心 + 平行轴定理。
解
$J_{端} = J_C + M(L/2)^2 = \frac{1}{12}ML^2 + \frac{1}{4}ML^2 = \frac{1}{3}ML^2$ ✓

5.2 转动定律

定轴转动的基本方程
$$M = J\alpha$$

合力矩 = 转动惯量 × 角加速度（类比 $F=ma$）

力矩 $M = Fr\sin\theta = F\cdot d$（$d$ 为力臂）

例（Atwood机+有质量滑轮）：质量 $m_1,m_2$（$m_1>m_2$）用轻绳跨过半径 $R$、转动惯量 $J$ 的滑轮连接。求加速度。
解

对 $m_1$：$m_1g - T_1 = m_1a$

对 $m_2$：$T_2 - m_2g = m_2a$

对滑轮：$(T_1-T_2)R = J\alpha = J\frac{a}{R}$

联立：$a = \frac{(m_1-m_2)g}{m_1+m_2+J/R^2}$

5.3 转动中的角动量与能量

刚体角动量守恒

刚体角动量：$L = J\omega$

角动量守恒（合力矩为零时）：$J_1\omega_1 = J_2\omega_2$

转动动能与功

转动动能：$E_k = \frac{1}{2}J\omega^2$

力矩做功：$W = \int M\dd\theta$

功能定理：$W = \Delta(\frac{1}{2}J\omega^2)$

滚动问题

纯滚动条件

$v_C = R\omega$（质心平动速度=半径×角速度）

总动能 = 平动 + 转动：$E_k = \frac{1}{2}Mv_C^2 + \frac{1}{2}J\omega^2$

例（斜面滚动）：实心球从高 $h$ 处无滑动滚下斜面，求底端速度。
解

$Mgh = \frac{1}{2}Mv^2 + \frac{1}{2}\cdot\frac{2}{5}MR^2\cdot\frac{v^2}{R^2} = \frac{1}{2}Mv^2(1+\frac{2}{5}) = \frac{7}{10}Mv^2$

$$v = \sqrt{\frac{10gh}{7}}$$

（比纯滑动 $\sqrt{2gh}$ 小——部分能量转化为转动动能）

比较：同一斜面同一高度，纯滑动最快，圆环最慢（$J$最大），顺序：滑动 > 实心球 > 圆盘 > 球壳 > 圆环

5.6 转动惯量的积分计算（PPT重点）

转动惯量积分法详解

$J=\int r^2\dd m$，其中 $r$ 是质量微元到转轴的距离。

关键：根据物体形状选择合适的微元。

例（均匀圆盘绕中心轴）：质量 $M$、半径 $R$。
解

面密度 $\sigma=M/(\pi R^2)$。取同心圆环微元（半径 $r$，宽 $\dd r$）：

$$\dd m=\sigma\cdot2\pi r\dd r$$ $$J=\int_0^R r^2\cdot\sigma\cdot2\pi r\dd r=\frac{2M}{R^2}\int_0^R r^3\dd r=\frac{2M}{R^2}\cdot\frac{R^4}{4}=\frac{1}{2}MR^2$$
例（均匀球体绕直径轴）：质量 $M$、半径 $R$。
解

将球分割为垂直于轴的薄圆盘。距中心 $z$ 处盘半径 $r=\sqrt{R^2-z^2}$，厚 $\dd z$。

每个薄盘的转动惯量 $\dd J=\frac{1}{2}\dd m\cdot r^2$。

$\dd m=\rho\pi r^2\dd z=\rho\pi(R^2-z^2)\dd z$

$$J=\int_{-R}^{R}\frac{1}{2}\rho\pi(R^2-z^2)^2\dd z=\frac{2}{5}MR^2$$

5.7 刚体问题解题模板

平动+转动联立方程

对于涉及滑轮/绕绳/滚动的问题，通常需要联立：

  1. 各物体的牛顿第二定律（平动或转动）
  2. 约束条件（绳不伸长、纯滚动 $v=R\omega$）

有质量滑轮的Atwood机需要3个方程（两个物体+滑轮转动方程）。

例（PPT重点：杆的释放问题）：均匀杆（$M,L$）绕一端水平轴自由转动，从水平位置静止释放。求转过 $\theta$ 角时角速度。
解

用功能定理：重力做功 = 转动动能增量

重心下降 $h=\frac{L}{2}\sin\theta$：

$$Mg\frac{L}{2}\sin\theta=\frac{1}{2}J\omega^2=\frac{1}{2}\cdot\frac{ML^2}{3}\cdot\omega^2$$ $$\omega=\sqrt{\frac{3g\sin\theta}{L}}$$

在 $\theta=0$（水平）：$\alpha_0=3g/(2L)$，杆端加速度 $a=L\alpha_0=3g/2>g$！

常见错误：
  • 忘记有质量滑轮两侧绳张力不等
  • 滚动问题中混淆"滑动摩擦"和"静摩擦"——纯滚动用静摩擦
  • 平行轴定理只能从质心轴平移，不能从任意轴平移

5.8 进动（定性了解）

陀螺进动

快速旋转的陀螺绕竖直轴缓慢转动的现象称为进动。

进动角速度：$\Omega=\frac{Mgl}{J\omega}$

其中 $M$ 为陀螺质量，$l$ 为质心到支点距离，$J$ 为绕自转轴转动惯量，$\omega$ 为自转角速度。

自转越快 → 进动越慢。

5.9 刚体综合问题解题模板

例（滑轮+斜面综合）：质量 $M$、半径 $R$ 的实心圆柱（转动惯量 $J=\frac{1}{2}MR^2$）绕定轴转动，用轻绳连接在斜面上的物体（质量 $m$，倾角 $\theta$，无摩擦）。求系统加速度。
解

对斜面物体：$T-mg\sin\theta=ma$（沿斜面向上为正）

等等——重新判断运动方向。若 $m$ 下滑（$mg\sin\theta>T$）：

对 $m$：$mg\sin\theta-T=ma$

对圆柱：$TR=J\alpha=\frac{1}{2}MR^2\cdot\frac{a}{R}$，$T=\frac{1}{2}Ma$

代入：$mg\sin\theta-\frac{1}{2}Ma=ma$

$$a=\frac{mg\sin\theta}{m+M/2},\quad T=\frac{Mmg\sin\theta}{2m+M}$$
例（角动量守恒：人跳上转台）：半径 $R$、转动惯量 $J_0$ 的转台静止。质量 $m$ 的人以 $v_0$ 沿切线方向跳上转台边缘。求转台角速度。
解

跳上前系统总角动量（对转台轴）：$L=mv_0R$

跳上后：$L=(J_0+mR^2)\omega$

角动量守恒：$\omega=\frac{mv_0R}{J_0+mR^2}$

解题checklist：
  • 每个物体/转动体单独列方程
  • 约束条件写全（$a=R\alpha$、绳不伸长等）
  • 有质量滑轮两侧 $T_1\neq T_2$
  • 能量法可以验证结果（或直接用能量法求解）

5.10 功能法解刚体问题（PPT高频）

刚体总动能

刚体做一般运动（平动+转动）时，总动能为：

$$E_k = \frac{1}{2}Mv_C^2 + \frac{1}{2}J_C\omega^2$$

其中 $v_C$ 为质心速度，$J_C$ 为绕过质心轴的转动惯量。

功能定理：外力做功（包括重力、摩擦力等）= 系统总动能的增量

$$W_{外} = \Delta E_k = \frac{1}{2}Mv_C^2 + \frac{1}{2}J_C\omega^2 - \frac{1}{2}Mv_{C0}^2 - \frac{1}{2}J_C\omega_0^2$$
功能法 vs 牛顿法：功能法不需要求中间力（如绳张力、静摩擦力），直接从初末状态求解，对于"求末速度"类问题非常高效。
例（圆柱沿斜面无滑动滚下 vs 纯滑动）：质量 $M$、半径 $R$ 的均匀实心圆柱从高 $h$ 处沿斜面无滑动滚下，用功能定理求底部速度，并与纯滑动（无摩擦）情形对比。
解

纯滚动情形：

纯滚动条件：$v_C = R\omega$，圆柱转动惯量 $J_C = \frac{1}{2}MR^2$。

静摩擦力不做功（纯滚动接触点瞬时速度为零），只有重力做功：

$$Mgh = \frac{1}{2}Mv^2 + \frac{1}{2}\cdot\frac{1}{2}MR^2\cdot\frac{v^2}{R^2} = \frac{1}{2}Mv^2 + \frac{1}{4}Mv^2 = \frac{3}{4}Mv^2$$ $$\boxed{v = \sqrt{\frac{4gh}{3}} \approx 1.155\sqrt{gh}}$$

纯滑动情形（无摩擦斜面）：

圆柱不转动，全部势能转化为平动动能：

$$Mgh = \frac{1}{2}Mv^2 \implies v = \sqrt{2gh} \approx 1.414\sqrt{gh}$$

对比：纯滑动的底部速度更大，因为纯滚动时部分势能转化为转动动能。

例（均匀杆绕一端从水平释放）：均匀细杆（质量 $M$，长 $L$），一端固定在光滑铰链上可绕水平轴自由转动。从水平位置静止释放，求转到竖直位置时的角速度和自由端速度。
解

杆绕端点的转动惯量：$J = \frac{1}{3}ML^2$

从水平到竖直，质心下降高度 $h = \frac{L}{2}$。

由功能定理（只有重力做功，铰链力力矩为零）：

$$Mg\frac{L}{2} = \frac{1}{2}J\omega^2 = \frac{1}{2}\cdot\frac{ML^2}{3}\cdot\omega^2 = \frac{ML^2\omega^2}{6}$$ $$\omega = \sqrt{\frac{3g}{L}}$$

自由端速度（端点做圆周运动，半径为 $L$）：

$$v_{端} = L\omega = L\sqrt{\frac{3g}{L}} = \sqrt{3gL}$$

注意 $v_{端} = \sqrt{3gL} > \sqrt{2gL}$（自由落体从同高度 $L$ 落下的速度），这是因为杆的质心下降 $L/2$ 释放的势能全部集中在端点的动能上。

易错点：功能法中要注意纯滚动时静摩擦力不做功，不要把摩擦力做功项写进去！但如果有滑动摩擦（非纯滚动），则摩擦力做负功。

5.11 垂直轴定理（薄板）

垂直轴定理

对于薄板（质量分布在一个平面内的刚体），取薄板所在平面内两条互相垂直的轴 $x$ 和 $y$，以及垂直于薄板的轴 $z$（三轴交于一点），则：

$$\boxed{J_z = J_x + J_y}$$

注意：此定理只适用于薄板！不能用于三维物体（球体、圆柱体等）。

证明思路：对于薄板上的质量元 $\dd m$，到 $z$ 轴距离 $r^2 = x^2+y^2$，到 $x$ 轴距离 $y$，到 $y$ 轴距离 $x$。因此 $J_z = \int r^2\dd m = \int(x^2+y^2)\dd m = J_y + J_x$。
例（由垂直轴定理推导圆盘绕直径的转动惯量）：已知均匀圆盘绕中心轴（垂直盘面）转动惯量 $J_z = \frac{1}{2}MR^2$，求绕任意一条直径的转动惯量。
解

取圆盘所在平面内两条互相垂直的直径为 $x$ 轴和 $y$ 轴，$z$ 轴垂直于盘面过圆心。

由对称性：$J_x = J_y$（绕任何直径的转动惯量都相等）。

由垂直轴定理：

$$J_z = J_x + J_y = 2J_x$$ $$J_x = \frac{J_z}{2} = \frac{1}{2}\cdot\frac{1}{2}MR^2 = \boxed{\frac{1}{4}MR^2}$$

这是一个非常优雅的推导——无需做任何积分！

例（正方形薄板）：均匀正方形薄板，边长 $a$，质量 $M$。求绕过中心且垂直板面的轴的转动惯量。
解

先求绕过中心且平行于一边的轴的转动惯量。将薄板看作许多平行细棒，每根绕中心的转动惯量为 $\frac{1}{12}(\dd m)a^2$，积分后：

$$J_x = \frac{1}{12}Ma^2$$

由对称性 $J_y = J_x = \frac{1}{12}Ma^2$。

由垂直轴定理：$J_z = J_x + J_y = \frac{1}{6}Ma^2$。

5.12 角动量守恒在刚体中的应用

角动量守恒条件

当系统所受合外力矩为零时，系统的总角动量守恒：

$$L_{初} = L_{末} \implies J_1\omega_1 = J_2\omega_2$$

常见情形：内力改变了转动惯量（人收缩手臂、转盘对接等），但无外力矩。

例（子弹射入转盘边缘）：质量 $m$ 的子弹以速度 $v_0$ 沿转盘的切线方向射入质量 $M$、半径 $R$ 的静止均匀圆盘边缘并嵌入其中。转盘可绕过中心的光滑竖直轴自由转动。求射入后转盘的角速度和系统动能损失。
解

角动量守恒（对转盘中心轴，外力矩为零）：

初态：子弹对轴的角动量 $L_0 = mv_0R$，转盘静止。

末态：子弹嵌入边缘后，系统转动惯量为

$$J_{末} = J_{盘} + mR^2 = \frac{1}{2}MR^2 + mR^2 = \left(\frac{M}{2}+m\right)R^2$$

角动量守恒：$mv_0R = J_{末}\omega$

$$\boxed{\omega = \frac{mv_0}{(\frac{M}{2}+m)R} = \frac{2mv_0}{(M+2m)R}}$$

动能损失：

初动能：$E_{k0} = \frac{1}{2}mv_0^2$

末动能：$E_{k} = \frac{1}{2}J_{末}\omega^2 = \frac{1}{2}\left(\frac{M}{2}+m\right)R^2\cdot\frac{4m^2v_0^2}{(M+2m)^2R^2} = \frac{m^2v_0^2}{M+2m}$

动能损失：

$$\Delta E_k = \frac{1}{2}mv_0^2 - \frac{m^2v_0^2}{M+2m} = \frac{1}{2}mv_0^2\cdot\frac{M}{M+2m} = \frac{Mmv_0^2}{2(M+2m)}$$

这是一个完全非弹性碰撞，部分动能转化为热能和形变能。

例（两个同轴转盘对接）：两个同轴的均匀圆盘，转盘A的转动惯量为 $J_1$，角速度 $\omega_1$；转盘B的转动惯量为 $J_2$，角速度 $\omega_2$（方向与A相同）。两转盘对接后达到共同角速度。求末角速度和能量损失。
解

角动量守恒（对接过程中摩擦力为内力矩，无外力矩）：

$$J_1\omega_1 + J_2\omega_2 = (J_1+J_2)\omega$$ $$\boxed{\omega = \frac{J_1\omega_1+J_2\omega_2}{J_1+J_2}}$$

能量损失：

$$\Delta E = E_{k,初} - E_{k,末} = \frac{1}{2}J_1\omega_1^2 + \frac{1}{2}J_2\omega_2^2 - \frac{1}{2}(J_1+J_2)\omega^2$$

代入 $\omega$ 化简后：

$$\boxed{\Delta E = \frac{J_1J_2}{2(J_1+J_2)}(\omega_1-\omega_2)^2}$$

$\Delta E \geq 0$，等号当且仅当 $\omega_1 = \omega_2$ 时成立。

能量损失转化为摩擦热。这个公式的结构类似于一维完全非弹性碰撞的动能损失公式 $\Delta E_k = \frac{m_1m_2}{2(m_1+m_2)}(v_1-v_2)^2$。

特殊情形：若转盘B初始静止（$\omega_2=0$），则 $\omega = \frac{J_1\omega_1}{J_1+J_2}$，能量损失 $\Delta E = \frac{J_1J_2\omega_1^2}{2(J_1+J_2)}$。若 $J_1=J_2$，则角速度减半，损失初始动能的一半。
常见陷阱：
  • 两转盘转向相反时，$\omega_2$ 取负值！若 $J_1\omega_1 = J_2|\omega_2|$，对接后 $\omega=0$，全部动能损失。
  • 角动量守恒不代表动能守恒——对接过程中有摩擦力做功，属于非弹性碰撞。

5.14 滚动问题深入分析（PPT补充）

纯滚动 vs 边滑边滚
纯滚动（无滑动）边滑边滚
条件$v_C=R\omega$$v_C\neq R\omega$
摩擦力类型静摩擦 $f\leq\mu_s N$滑动摩擦 $f=\mu_k N$
摩擦力做功不做功（接触点瞬时速度为零）做负功（产生热量）
能量守恒机械能守恒（若无其他非保守力）机械能不守恒
例（从滑动到纯滚动的过渡）：保龄球（均匀实心球，$M$, $R$）以初速 $v_0$ 在水平粗糙面上（$\mu_k$）无旋转地滑出。求：(1) 何时开始纯滚动？(2) 纯滚动时的速度？
解

滑动阶段分析：滑动摩擦力 $f=\mu_k Mg$ 使质心减速、同时提供力矩使球加速旋转。

平动：$v=v_0-\mu_k g\,t$

转动（绕质心，$J=\frac{2}{5}MR^2$）：$\mu_k MgR=\frac{2}{5}MR^2\alpha$，$\alpha=\frac{5\mu_k g}{2R}$

角速度：$\omega=\alpha t=\frac{5\mu_k g}{2R}t$

纯滚动条件：$v=R\omega$

$$v_0-\mu_k g\,t = R\cdot\frac{5\mu_k g}{2R}t = \frac{5\mu_k g}{2}t$$ $$v_0 = \frac{7\mu_k g}{2}t \implies \boxed{t=\frac{2v_0}{7\mu_k g}}$$

纯滚动速度：

$$v = v_0-\mu_k g\cdot\frac{2v_0}{7\mu_k g} = v_0\left(1-\frac{2}{7}\right) = \boxed{\frac{5}{7}v_0}$$

损失的动能 $=\frac{1}{2}Mv_0^2-\frac{1}{2}M(\frac{5}{7}v_0)^2-\frac{1}{2}\cdot\frac{2}{5}MR^2\cdot(\frac{5v_0}{7R})^2=\frac{1}{2}Mv_0^2\cdot\frac{2}{7}$ 转化为摩擦热。

滚动问题易错汇总：
  • 纯滚动时静摩擦力不一定等于 $\mu_s N$，由运动方程确定
  • 斜面纯滚动需验证 $f\leq\mu_s N$，否则实际为边滑边滚
  • 滚动接触点速度为零——这是纯滚动的等价条件
  • 圆柱/球从斜面滚下到达底部速度与质量和半径无关，只取决于形状因子 $J/(MR^2)$

5.15 刚体平面运动分析（PPT补充）

刚体平面运动的动力学方程

平面运动 = 质心平动 + 绕质心转动，需要联立两组方程：

平动方程（质心运动定理）：

$$\vv{F}_{合} = M\vv{a}_C$$

转动方程（对质心轴）：

$$M_C = J_C\alpha$$

加上运动学约束（如纯滚动 $a_C=R\alpha$），方程组封闭。

例（圆柱沿斜面纯滚动——求摩擦力）：均匀实心圆柱（$M$, $R$）沿倾角 $\theta$ 的粗糙斜面无滑动滚下。求加速度和所需最小摩擦系数。
解

沿斜面方向（取下滑为正）：$Mg\sin\theta-f=Ma_C$ ...(1)

绕质心转动：$fR=\frac{1}{2}MR^2\alpha$ ...(2)

纯滚动约束：$a_C=R\alpha$ ...(3)

由 (2)(3)：$f=\frac{1}{2}Ma_C$。代入 (1)：

$$Mg\sin\theta=\frac{3}{2}Ma_C \implies \boxed{a_C=\frac{2g\sin\theta}{3}}$$ $$f=\frac{1}{2}Ma_C=\frac{1}{3}Mg\sin\theta$$

纯滚动要求 $f\leq\mu_s N=\mu_s Mg\cos\theta$：

$$\frac{1}{3}Mg\sin\theta\leq\mu_s Mg\cos\theta \implies \boxed{\mu_s\geq\frac{1}{3}\tan\theta}$$

若 $\mu_s<\frac{1}{3}\tan\theta$，圆柱将边滑边滚。

5.16 物理摆（复摆）（MOOC/真题）

复摆的周期公式

物理摆（复摆）是指刚体在重力作用下绕固定水平轴做微小角振动的系统。与简单摆不同，复摆的质量不集中在一个点上。

$$\boxed{T = 2\pi\sqrt{\frac{J}{mgl}}}$$

其中：

  • $J$ — 刚体对悬挂轴的转动惯量
  • $m$ — 刚体总质量
  • $l$ — 悬挂轴到质心的距离
  • $g$ — 重力加速度

推导：设刚体偏离平衡位置角度为 $\theta$（微小），重力力矩为 $M = -mgl\sin\theta \approx -mgl\theta$，由转动定律 $J\ddot{\theta} = -mgl\theta$，得到简谐运动方程 $\ddot{\theta} + \frac{mgl}{J}\theta = 0$，角频率 $\omega = \sqrt{mgl/J}$。

等效摆长

复摆可以用一个简单摆来等效，其摆长为：

$$l_{eq} = \frac{J}{ml}$$

使得 $T = 2\pi\sqrt{l_{eq}/g}$，与简单摆公式形式相同。

例（均匀杆绕一端的复摆）：均匀细杆（质量 $M$，长度 $L$）绕一端的水平轴做微小摆动，求周期。
解

杆绕端点的转动惯量：$J = \frac{1}{3}ML^2$

质心到悬挂轴的距离：$l = L/2$

代入复摆周期公式：

$$T = 2\pi\sqrt{\frac{J}{Mgl}} = 2\pi\sqrt{\frac{\frac{1}{3}ML^2}{Mg\cdot\frac{L}{2}}} = 2\pi\sqrt{\frac{2L}{3g}}$$

等效摆长：$l_{eq} = \frac{J}{Ml} = \frac{\frac{1}{3}ML^2}{M\cdot\frac{L}{2}} = \frac{2L}{3}$

对比相同长度的简单摆：$T_{simple} = 2\pi\sqrt{L/g}$

两者之比：$T_{复摆}/T_{简摆} = \sqrt{2/3} \approx 0.816$，复摆周期更短。

例（均匀圆盘绕边缘点的复摆）：均匀圆盘（质量 $M$，半径 $R$）绕过边缘的水平轴做微小摆动，求周期。
解

圆盘绕中心轴：$J_C = \frac{1}{2}MR^2$

由平行轴定理，绕边缘轴：

$$J = J_C + MR^2 = \frac{1}{2}MR^2 + MR^2 = \frac{3}{2}MR^2$$

质心到悬挂轴的距离：$l = R$

$$T = 2\pi\sqrt{\frac{\frac{3}{2}MR^2}{MgR}} = 2\pi\sqrt{\frac{3R}{2g}}$$

等效摆长：$l_{eq} = \frac{3R}{2}$

实验应用：复摆是实验中测量重力加速度 $g$ 的重要方法。只需测量 $T$、$J$、$m$、$l$，即可由 $g = \frac{4\pi^2 J}{mT^2 l}$ 计算 $g$。实际中常利用"可逆摆"原理提高精度。
注意：复摆周期公式仅适用于微小振动（$\theta \ll 1$），此时才能做 $\sin\theta\approx\theta$ 的近似。

5.17 子弹射入悬挂杆问题（PPT/真题经典）

问题模型

质量为 $m$ 的子弹以水平速度 $v_0$ 射入悬挂在铰链上的静止均匀杆（质量 $M$，长度 $L$）的末端并嵌入其中。杆可绕顶端的光滑水平轴自由转动。

此类问题分为两个阶段：碰撞阶段（极短时间）和碰后转动阶段。

第一阶段：碰撞瞬间 — 角动量守恒

为什么用角动量守恒而不是动量守恒？

碰撞瞬间，轴对系统有反力（冲击力），该力的冲量不可忽略，因此线动量不守恒。

但轴处的力对轴的力矩为零（力臂为零），因此对轴的角动量守恒。

碰前角动量（对轴）：$L_{初} = mv_0 L$（子弹的角动量，杆静止）

碰后角动量：$L_{末} = J_{total}\omega$

其中系统总转动惯量：

$$J_{total} = J_{杆} + mL^2 = \frac{1}{3}ML^2 + mL^2 = \frac{(M+3m)L^2}{3}$$

由角动量守恒 $mv_0L = J_{total}\omega$：

$$\boxed{\omega = \frac{mv_0L}{J_{total}} = \frac{3mv_0}{(M+3m)L}}$$

第二阶段：碰后转动 — 能量守恒

碰后杆（含子弹）从水平位置开始转动，求能摆过的最大角度 $\theta_{max}$。

系统质心上升高度（杆的质心上升 $\frac{L}{2}(1-\cos\theta)$，子弹上升 $L(1-\cos\theta)$）：

$$\frac{1}{2}J_{total}\omega^2 = \left(M\frac{L}{2}+mL\right)g(1-\cos\theta_{max})$$ $$\frac{1}{2}\cdot\frac{(M+3m)L^2}{3}\cdot\omega^2 = \frac{(M+2m)Lg}{2}(1-\cos\theta_{max})$$
例（数值计算）：$m=10\,\text{g}$，$v_0=400\,\text{m/s}$，$M=1\,\text{kg}$，$L=1\,\text{m}$。求碰后角速度和最大偏转角。
解

碰后角速度：

$$\omega = \frac{3\times0.01\times400}{(1+3\times0.01)\times1} = \frac{12}{1.03} \approx 11.65\,\text{rad/s}$$

最大偏转角：

$$\frac{1}{2}\times\frac{1.03}{3}\times1^2\times11.65^2 = \frac{(1+0.02)\times1\times9.8}{2}(1-\cos\theta_{max})$$ $$\frac{1}{2}\times0.3433\times135.7 = 4.998(1-\cos\theta_{max})$$ $$23.31 = 4.998(1-\cos\theta_{max})$$ $$1-\cos\theta_{max} = 4.664$$

$1-\cos\theta_{max} > 2$，说明杆能转过最高点（$\theta_{max}>180°$），即杆做完整圆周转动！

验证：在最高点的动能为 $\frac{1}{2}J\omega^2 - (M\frac{L}{2}+mL)g\times2 > 0$，杆可以持续转动。

重要提醒：
  • 碰撞瞬间动量不守恒（轴有反力），但角动量对轴守恒（轴力矩为零）
  • 碰撞过程中能量不守恒（完全非弹性碰撞，有能量损失）
  • 碰后转动过程中机械能守恒（轴光滑，无摩擦力矩）
  • 不能把两个阶段合并处理！碰撞和碰后转动适用不同的守恒律

5.18 绕绳/卷绳问题（真题考频高）

基本模型

绳子缠绕在可绕定轴转动的圆柱（或圆筒）上，绳的另一端悬挂质量为 $m$ 的物体。释放后，物体下落同时圆柱转动。

设圆柱转动惯量为 $J$，半径为 $R$。

方程建立

对悬挂物体（取向下为正）：

$$mg - T = ma \quad \cdots(1)$$

对圆柱转动：

$$TR = J\alpha \quad \cdots(2)$$

绳不伸长约束：

$$a = R\alpha \quad \cdots(3)$$

求解

由 (2) 和 (3)：$T = J\alpha/R = Ja/R^2$

代入 (1)：$mg - Ja/R^2 = ma$

$$\boxed{a = \frac{mg}{m + J/R^2}}, \quad \boxed{T = \frac{Jmg/R^2}{m + J/R^2}} = \frac{mg}{1+mR^2/J}$$
"等效质量"概念：可以把系统看成质量为 $m_{eff} = m + J/R^2$ 的物体在重力 $mg$ 作用下做匀加速运动。$J/R^2$ 相当于圆柱的转动等效质量。
例（实心圆柱）：质量 $M$ 的实心圆柱，$J = \frac{1}{2}MR^2$，悬挂质量 $m$。
解
$$a = \frac{mg}{m + \frac{1}{2}MR^2/R^2} = \frac{mg}{m + M/2} = \boxed{\frac{2mg}{2m+M}}$$ $$T = \frac{\frac{1}{2}M\cdot mg}{m+M/2} = \frac{Mmg}{2m+M}$$

验证极端情况：

  • $M\to 0$（无圆柱）：$a\to g$，$T\to 0$ ✓
  • $M\to\infty$（极重圆柱）：$a\to 0$，$T\to mg$ ✓
例（薄壁圆筒）：质量 $M$ 的薄壁圆筒，$J = MR^2$，悬挂质量 $m$。
解
$$a = \frac{mg}{m + MR^2/R^2} = \frac{mg}{m+M} = \boxed{\frac{mg}{m+M}}$$ $$T = \frac{Mmg}{m+M}$$

对比实心圆柱：圆筒的 $J$ 更大，加速度更小（更多能量转化为转动动能）。

例（综合：求下落 $h$ 后的速度）：在上述实心圆柱问题中，物体从静止下落高度 $h$，求此时速度。
解

方法一（运动学）：$v = \sqrt{2ah} = \sqrt{\frac{4mgh}{2m+M}}$

方法二（能量法）：

$$mgh = \frac{1}{2}mv^2 + \frac{1}{2}J\omega^2 = \frac{1}{2}mv^2 + \frac{1}{2}\cdot\frac{1}{2}MR^2\cdot\frac{v^2}{R^2} = \frac{1}{2}v^2\left(m+\frac{M}{2}\right)$$ $$v = \sqrt{\frac{4mgh}{2m+M}}$$

两种方法结果一致。能量法不需要求绳张力，更简洁。

易错点：
  • 绳的张力 $T \neq mg$！由于物体有加速度，$T < mg$
  • 若圆柱不是绕中心轴转动，需用正确的转动惯量
  • 约束条件 $a = R\alpha$ 要求绳在圆柱表面不滑动

5.19 刚体运动的力学量类比总结（真题高频选择题）

匀变速转动公式（与匀变速直线运动完全类比）
匀变速直线运动匀变速转动
$v = v_0 + at$$\omega = \omega_0 + \alpha t$
$x = v_0 t + \frac{1}{2}at^2$$\theta = \omega_0 t + \frac{1}{2}\alpha t^2$
$v^2 = v_0^2 + 2ax$$\omega^2 = \omega_0^2 + 2\alpha\theta$
$x = \frac{v_0+v}{2}t$$\theta = \frac{\omega_0+\omega}{2}t$
例（飞轮减速问题）：飞轮从 $600\,\text{rpm}$ 均匀减速到停止，共转了 $100$ 圈。求角加速度和减速时间。
解

单位换算：

$\omega_0 = 600\,\text{rpm} = 600\times\frac{2\pi}{60}\,\text{rad/s} = 20\pi\,\text{rad/s}$

$\omega = 0$（停止）

$\theta = 100\,\text{圈} = 100\times2\pi = 200\pi\,\text{rad}$

求角加速度：由 $\omega^2 = \omega_0^2 + 2\alpha\theta$

$$0 = (20\pi)^2 + 2\alpha\cdot200\pi$$ $$\alpha = -\frac{(20\pi)^2}{400\pi} = -\frac{400\pi^2}{400\pi} = -\pi\,\text{rad/s}^2$$

负号表示减速。$|\alpha| = \pi \approx 3.14\,\text{rad/s}^2$。

求时间：由 $\omega = \omega_0 + \alpha t$

$$0 = 20\pi + (-\pi)t \implies t = \frac{20\pi}{\pi} = \boxed{20\,\text{s}}$$

验证：用 $\theta = \frac{\omega_0+\omega}{2}t = \frac{20\pi+0}{2}\times20 = 200\pi\,\text{rad}$ ✓

例（电机启动问题）：电机从静止开始匀加速，$5\,\text{s}$ 后达到 $300\,\text{rpm}$。求角加速度和 $5\,\text{s}$ 内转过的圈数。
解

$\omega_0 = 0$，$\omega = 300\times\frac{2\pi}{60} = 10\pi\,\text{rad/s}$，$t = 5\,\text{s}$

角加速度：$\alpha = \frac{\omega-\omega_0}{t} = \frac{10\pi}{5} = 2\pi\,\text{rad/s}^2$

转过角度：$\theta = \frac{\omega_0+\omega}{2}t = \frac{0+10\pi}{2}\times5 = 25\pi\,\text{rad}$

转过圈数：$N = \theta/(2\pi) = 25\pi/(2\pi) = \boxed{12.5\,\text{圈}}$

选择题常考点：
  • rpm 到 rad/s 的换算：$n\,\text{rpm} = n\times\frac{2\pi}{60}\,\text{rad/s}$
  • 转过的圈数 = $\theta/(2\pi)$
  • 匀减速到停止：$\omega_0^2 = 2|\alpha|\theta$，即 $\theta = \omega_0^2/(2|\alpha|)$

5.20 旋转绳索的张力分布（PPT补充）

问题描述

一根均匀绳索（质量 $M$，长度 $L$）的一端固定在竖直轴上，绕轴在水平面内以角速度 $\omega$ 匀速转动。求绳中距轴 $r$ 处的张力 $T(r)$。

推导过程

取绳上从 $r$ 到 $r+\dd r$ 的微元，质量为 $\dd m = \frac{M}{L}\dd r$。

该微元做圆周运动，需要向心力。微元两端的张力差提供向心力：

$$T(r) - T(r+\dd r) = \dd m\cdot\omega^2 r = \frac{M}{L}\omega^2 r\,\dd r$$

即 $-\dd T = \frac{M\omega^2}{L}r\,\dd r$

边界条件：绳的自由端（$r=L$）处张力为零，$T(L)=0$。

从 $r$ 积分到 $L$：

$$T(r) = \int_r^L \frac{M\omega^2}{L}r'\,\dd r' = \frac{M\omega^2}{L}\cdot\frac{r'^2}{2}\Big|_r^L = \frac{M\omega^2}{2L}(L^2-r^2)$$ $$\boxed{T(r) = \frac{M\omega^2}{2L}(L^2-r^2)}$$
关键结论
  • 张力在轴处最大：$T(0) = \frac{1}{2}M\omega^2 L$
  • 张力在自由端为零：$T(L) = 0$
  • 张力沿绳呈抛物线分布（$T$ 与 $L^2-r^2$ 成正比）
例（末端附加质量）：若绳的自由端（$r=L$）连接一个质量为 $m$ 的小球，求张力分布。
解

边界条件改变：$T(L) = m\omega^2 L$（小球的向心力由此处张力提供）。

积分同上，但下限条件变为 $T(L) = m\omega^2 L$：

$$T(r) = \frac{M\omega^2}{2L}(L^2-r^2) + m\omega^2 L$$

轴处张力：$T(0) = \frac{1}{2}M\omega^2 L + m\omega^2 L = \left(\frac{M}{2}+m\right)\omega^2 L$

物理意义：整根绳加小球的总质量为 $M+m$，它们质心到轴的平均距离对应的向心力之和。

例（数值计算）：绳长 $L=2\,\text{m}$，质量 $M=0.5\,\text{kg}$，转速 $n=60\,\text{rpm}$。求轴处张力和中点张力。
解

$\omega = 60\times\frac{2\pi}{60} = 2\pi\,\text{rad/s}$

轴处：$T(0) = \frac{1}{2}\times0.5\times(2\pi)^2\times2 = 0.5\times4\pi^2 = 2\pi^2 \approx 19.7\,\text{N}$

中点（$r=1\,\text{m}$）：

$$T(1) = \frac{0.5\times(2\pi)^2}{2\times2}(4-1) = \frac{0.5\times4\pi^2}{4}\times3 = \frac{3}{2}\pi^2 \approx 14.8\,\text{N}$$

中点张力是轴处的 $3/4$。

推广：此模型可推广到旋转飞轮轮辋的应力分析、离心机中液柱的压力分布等工程问题。

5.21 打击中心问题（PPT例4-2，考试高频）

什么是打击中心？

均匀细棒（质量 $m$，长 $l$）可绕一端水平轴 $O$ 转动，$J=\frac{1}{3}ml^2$。在距 $O$ 为 $\lambda$ 处施加水平力 $F$。问：$O$ 处轴的水平反力 $N_x$ 何时为零？

例（打击中心推导）
解

转动定律：$F\lambda = J\beta = \frac{1}{3}ml^2\beta$，得 $\beta = \frac{3F\lambda}{ml^2}$

质心运动定理（水平方向）：$F + N_x = ma_C = m\cdot\frac{l}{2}\beta$

代入 $\beta$：

$$N_x = m\cdot\frac{l}{2}\cdot\frac{3F\lambda}{ml^2} - F = F\left(\frac{3\lambda}{2l} - 1\right)$$

令 $N_x = 0$：$\lambda = \frac{2l}{3}$

即打击点在距轴 $\frac{2}{3}l$ 处时，轴的水平反力为零——此点称为打击中心。

实际应用：
  • 用棒击打岩石时，打在 $2l/3$ 处可避免手受到冲击
  • 门上制动器应安装在距铰链 $2l/3$ 处，减小对铰链的冲击力
  • 棒球、网球击球的"甜点"（sweet spot）就是打击中心
推广：对于一般形状的刚体（转动惯量 $J$，质心距轴 $d$），打击中心距轴为 $\lambda = J/(md)$。

5.22 圆盘在粗糙面上转动的摩擦力矩（PPT例4-3）

例（均匀圆盘绕中心在粗糙面上转动）：质量 $M$、半径 $R$ 的均匀圆盘平放在摩擦系数 $\mu$ 的粗糙水平面上，以初角速度 $\omega_0$ 绕中心 $O$ 转动。求停止时间和转过的角度。
解

计算摩擦力矩：取距中心 $r$、宽 $\dd r$ 的圆环微元。

质量：$\dd m = \frac{M}{\pi R^2}\cdot 2\pi r\,\dd r = \frac{2M}{R^2}r\,\dd r$

摩擦力：$\dd f = \mu g\,\dd m = \frac{2\mu Mg}{R^2}r\,\dd r$

摩擦力矩：$\dd M_f = r\,\dd f = \frac{2\mu Mg}{R^2}r^2\,\dd r$

$$M_f = \int_0^R \frac{2\mu Mg}{R^2}r^2\,\dd r = \frac{2\mu MgR}{3}$$

转动定律：$J\frac{\dd\omega}{\dd t} = -M_f$，其中 $J=\frac{1}{2}MR^2$

$$\frac{1}{2}MR^2\frac{\dd\omega}{\dd t} = -\frac{2\mu MgR}{3}$$ $$\frac{\dd\omega}{\dd t} = -\frac{4\mu g}{3R}$$

停止时间：$\omega_0 = \frac{4\mu g}{3R}t$，$\boxed{t = \frac{3R\omega_0}{4\mu g}}$

转过角度：用 $\omega\dd\omega = \beta\dd\theta$：

$$\theta = \frac{\omega_0^2}{2|\beta|} = \frac{\omega_0^2}{2\cdot\frac{4\mu g}{3R}} = \boxed{\frac{3R\omega_0^2}{8\mu g}}$$
对比：均匀细棒（长 $l$）绕中心在粗糙面上转动，摩擦力矩为 $\frac{1}{2}\mu mgl$（PPT也有此题），方法完全类似。

5.23 电风扇启动与稳定转速问题（PPT例4-1）

例（恒功率电机 + 线性阻力矩）：电风扇恒定功率 $P$，风叶受空气阻力矩 $M_f = -k\omega$，转动惯量 $J$。求通电后 $\omega(t)$ 和稳定角速度。
解

电机力矩：$M = P/\omega$（恒功率）

转动定律：$J\frac{\dd\omega}{\dd t} = \frac{P}{\omega} - k\omega$

分离变量并积分：

$$\int_0^\omega \frac{k\,\dd\omega}{P - k\omega^2} = \frac{1}{J}\int_0^t\dd t$$ $$\boxed{\omega(t) = \sqrt{\frac{P}{k}}\left(1 - e^{-2kt/J}\right)^{1/2}}$$

更常考的简化版本：恒力矩 $M$，阻力矩 $M_f = -k\omega$：

$$J\frac{\dd\omega}{\dd t} = M - k\omega \implies \omega(t) = \frac{M}{k}\left(1-e^{-kt/J}\right)$$

稳定角速度：$\omega_\infty = M/k$（当 $t\to\infty$）

稳定转速后断电，风叶减速：$J\frac{\dd\omega}{\dd t} = -k\omega$，$\omega = \omega_\infty e^{-kt/J}$

断电后转过的角度：$\theta = \int_0^\infty \omega\,\dd t = \frac{J\omega_\infty}{k} = \frac{JM}{k^2}$

变形考法：
  • 阻力矩 $\propto \omega^2$ 时，稳定角速度为 $\omega_\infty = \sqrt{M/k}$
  • 恒功率 + 阻力矩 $\propto \omega^3$ 时，$\omega_\infty = (P/k)^{1/4}$
  • 断电后停止的时间和角度是高频考点

5.24 跷板弹射问题（PPT补充例）

例（杂技跷板）：跷板长 $l$、质量 $M$，支点 $C$ 在中点。质量 $m$ 的演员 W 从高 $h$ 处落到 A 端（完全非弹性碰撞），弹起另一端质量 $m$ 的演员 N。求 N 能弹起的高度。
解

第一阶段：W 自由落体到达 A 端的速度 $v_0 = \sqrt{2gh}$

第二阶段：碰撞瞬间（W 与板完全非弹性碰撞），以 W、N、板为系统，对支点 $C$ 角动量守恒。

碰前：$L = mv_0\cdot\frac{l}{2}$（W 的角动量，其余静止）

碰后：W 粘在 A 端，N 在 B 端，板转动。设共同角速度 $\omega$：

$$mv_0\frac{l}{2} = \left(\frac{1}{12}Ml^2 + m\frac{l^2}{4} + m\frac{l^2}{4}\right)\omega = \left(\frac{M}{12}+\frac{m}{2}\right)l^2\omega$$

N 在 B 端的线速度：$u = \omega\cdot\frac{l}{2}$

第三阶段：N 以速度 $u$ 竖直弹起：$h' = \frac{u^2}{2g}$

$$h' = \frac{m^2}{(M/6+m)^2}\cdot\frac{v_0^2}{8g} = \frac{m^2 h}{(M/6+m)^2\cdot 4} = h\cdot\frac{9m^2}{(M+6m)^2}$$

（化简：$(M/6+m)^2\cdot4 = (M+6m)^2/36\cdot4 = (M+6m)^2/9$）

当 $M=0$ 时，$h'=h\cdot9m^2/(6m)^2=h/4$（完全非弹性碰撞损失了大量能量，即使跷板无质量也无法全部传递！）

易错点：碰撞阶段不能用机械能守恒（完全非弹性碰撞有能量损失），但可以用角动量守恒（支点处外力矩为零或极短时间可忽略）。

5.25 两平行圆柱接触转动（PPT补充例）

例（两圆柱摩擦接触）：两平行圆柱可绕各自轴转动，半径 $R_1, R_2$，转动惯量 $J_1, J_2$。初始时圆柱1以 $\omega_{10}$ 转动，圆柱2静止。两圆柱接触后由摩擦力带动，直到相对滑动停止。求最终角速度。
解

无滑动条件：$\omega_1 R_1 = \omega_2 R_2$（接触点线速度相等）

分别对各自轴用角动量定理：

设接触点摩擦力为 $f$，对圆柱1：$J_1(\omega_1 - \omega_{10}) = -fR_1 \cdot t$

对圆柱2：$J_2\omega_2 = fR_2 \cdot t$

消去 $ft$：$J_1(\omega_{10}-\omega_1)R_2 = J_2\omega_2 R_1$

联立无滑动条件 $\omega_2 = \omega_1 R_1/R_2$：

$$\boxed{\omega_1 = \frac{J_1 R_2^2\,\omega_{10}}{J_1 R_2^2 + J_2 R_1^2}}, \quad \omega_2 = \frac{J_1 R_1 R_2\,\omega_{10}}{J_1 R_2^2 + J_2 R_1^2}$$
注意：此问题中系统角动量不守恒！因为两个轴对各自的力矩方向相反（对不同轴），不能合并为一个角动量守恒方程。必须对每个轴分别列角动量定理，再联立无滑动条件。这是与同轴转盘对接问题的本质区别。

5.26 角冲量与碰撞中的转动（PPT高频题型汇总）

角冲量定理

转动定律的积分形式：

$$\int_{t_1}^{t_2} M\,\dd t = J\omega_2 - J\omega_1$$

左端称为角冲量（冲量矩），右端为角动量变化。对于碰撞等短过程中外力矩可忽略的情况，角动量守恒。

例（水平冲量打击静止竖直杆）：均匀杆（$m, l$, $J=ml^2/3$）可绕一端 $O$ 轴转动，静止于竖直位置。水平冲量 $I$ 作用于距 $O$ 为 $\frac{2l}{3}$ 处。求冲量作用后杆的角速度。
解

对 $O$ 轴用角冲量定理：

$$I\cdot\frac{2l}{3} = J\omega - 0 = \frac{1}{3}ml^2\omega$$ $$\omega = \frac{2I}{ml}$$

注意此处打击在打击中心（$2l/3$），轴反力冲量为零。

例（子弹射入棒端——PPT例4-10）：长 $l$ 的轻杆两端固定 $m$ 和 $2m$ 的小球，可绕 $O$ 转动（$O$ 距 $m$ 为 $l/3$，距 $2m$ 为 $2l/3$）。杆竖直静止时，质量 $m$ 的子弹以 $v_0$ 水平射入下端 $m$ 球并嵌入，碰后以 $v_0/2$ 返回。求碰后角速度。
解

对 $O$ 轴角动量守恒（轴力矩为零）：

碰前：$L_0 = mv_0\cdot\frac{2l}{3}$（子弹对 $O$ 的角动量，选方向为正）

碰后：子弹以 $v_0/2$ 反向返回，$L_子 = -m\frac{v_0}{2}\cdot\frac{2l}{3}$

杆的角动量：$L_杆 = \left[m\left(\frac{2l}{3}\right)^2 + 2m\left(\frac{l}{3}\right)^2\right]\omega = \frac{2}{3}ml^2\omega$

$$mv_0\cdot\frac{2l}{3} = -m\frac{v_0}{2}\cdot\frac{2l}{3} + \frac{2}{3}ml^2\omega$$ $$\omega = \frac{3v_0}{2l}$$
碰撞+转动问题的固定模式：
  1. 判断碰撞瞬间用什么守恒律——有固定轴时通常用角动量守恒（对轴），因为轴力的力矩为零
  2. 碰后若需求最大偏角等，再用机械能守恒
  3. 不要把碰撞过程和碰后运动过程混在一起！

5.27 虫爬杆问题——变J转动（PPT补充例）

例（虫落在转动杆上）：质量 $m$、长 $l$ 的均匀杆绕中心 $O$ 轴可转动（$J=\frac{1}{12}ml^2$），水平静止。质量 $m$ 的小虫以速率 $v_0$ 竖直落在距 $O$ 为 $l/4$ 处并附着。要使杆以恒定角速度 $\omega$ 转动，虫应以什么速率爬向端点？
解

碰撞瞬间（角动量守恒）：

$$mv_0\cdot\frac{l}{4} = \left[\frac{1}{12}ml^2 + m\left(\frac{l}{4}\right)^2\right]\omega = \frac{7}{48}ml^2\omega$$ $$\omega = \frac{12v_0}{7l}$$

碰后恒定角速度转动：虫在距轴 $r$ 处时，系统转动惯量 $J(r) = \frac{1}{12}ml^2 + mr^2$。

角动量定理：$M_{ext} = \frac{\dd}{\dd t}[J(r)\omega]$

因为 $\omega = \text{const}$：$M_{ext} = \omega\frac{\dd J}{\dd t} = \omega\cdot 2mr\frac{\dd r}{\dd t}$

外力矩来源是重力矩（设杆转过角 $\theta = \omega t$）：$M_{ext} = mgr\cos\theta$

$$mgr\cos(\omega t) = 2m\omega r\frac{\dd r}{\dd t}$$ $$\frac{\dd r}{\dd t} = \frac{g\cos(\omega t)}{2\omega}$$

这就是虫相对杆的爬行速率（沿杆方向的分量）。

此类变 $J$ 问题的关键：当系统转动惯量随时间变化时，角动量定理写成 $M = \frac{\dd(J\omega)}{\dd t}$，不能简单地用 $M = J\beta$（$\beta$ 不定义在同一个 $J$ 上）。

5.28 齿轮啮合问题（PPT例4-9）

例（两齿轮正交啮合）：齿轮1（半径 $r_1$，转动惯量 $J_1$）初始以 $\omega_0$ 转动，齿轮2（半径 $r_2$，$J_2$）初始静止。两轮啮合后求稳定角速度。
解

两轮绕不同轴转动，分别对各轴用角动量定理。设啮合力为 $F$：

对轮1：$J_1(\omega_1 - \omega_0) = -Fr_1\cdot t$

对轮2：$J_2\omega_2 = Fr_2\cdot t$

啮合条件（无滑动）：$\omega_1 r_1 = \omega_2 r_2$

消去 $Ft$：

$$\omega_1 = \frac{J_1 r_2^2\,\omega_0}{J_1 r_2^2 + J_2 r_1^2}, \quad \omega_2 = \frac{J_1 r_1 r_2\,\omega_0}{J_1 r_2^2 + J_2 r_1^2}$$

结构与上面两圆柱接触问题完全相同。

区分三种"接合"问题：
类型守恒律约束
同轴对接（离合器）角动量守恒 $J_1\omega_1+J_2\omega_2=(J_1+J_2)\omega$$\omega$相同
齿轮啮合/圆柱接触分别对各轴列角动量定理$\omega_1 r_1=\omega_2 r_2$
皮带传动分别列角动量定理$\omega_1 r_1=\omega_2 r_2$（无滑动时）

5.29 双轮轴（不等半径）系统（PPT高频）

例（同轴双轮系统）：半径 $R_1, R_2$ 的两轮同轴固结（总转动惯量 $J$），绳端分别挂 $m_1, m_2$。求角加速度。
解

对 $m_1$（设 $m_1$ 下降）：$m_1g - T_1 = m_1 a_1 = m_1 R_1\beta$

对 $m_2$（上升）：$T_2 - m_2g = m_2 a_2 = m_2 R_2\beta$

对轮轴：$T_1 R_1 - T_2 R_2 = J\beta$

联立解得：

$$\boxed{\beta = \frac{(m_1 R_1 - m_2 R_2)g}{J + m_1 R_1^2 + m_2 R_2^2}}$$

注意此处两个物体的线加速度不同：$a_1 = R_1\beta$，$a_2 = R_2\beta$。

5.30 恒星坍缩为中子星（PPT补充例）

例（中子星角速度）：恒星内核（均匀球体）半径 $R_0 = 2\times10^7\,\text{m}$，自转周期 $T_0 = 45$ 天，坍缩为半径 $R = 6\times10^3\,\text{m}$ 的中子星。求中子星角速度。
解

角动量守恒：$J_0\omega_0 = J\omega$

均匀球 $J = \frac{2}{5}mR^2$，故 $R_0^2\omega_0 = R^2\omega$：

$$\omega = \omega_0\left(\frac{R_0}{R}\right)^2 = \frac{2\pi}{45\times86400}\times\left(\frac{2\times10^7}{6\times10^3}\right)^2 \approx 18\,\text{rad/s}$$

相当于每秒转约 $3$ 圈！这就是脉冲星快速旋转的物理原因。

物理启示：角动量守恒 $J\omega = \text{const}$ 中，$R$ 减小 $\sim 3000$ 倍，$\omega$ 增大 $\sim 10^7$ 倍。从每 $45$ 天转一圈到每秒转 $3$ 圈的巨大变化，完全来自于角动量守恒这一基本定律。
