第1章 质点运动学

1.1 位置矢量与位移

参考系与坐标系

描述运动必须选定参考系（Reference Frame）。同一运动在不同参考系中描述不同。通常选地面为参考系。

在参考系上建立坐标系来定量描述位置：直角坐标系 $(x,y,z)$、极坐标 $(r,\theta)$、自然坐标系（切向-法向）。

位置矢量（位矢）

位矢的定义

质点在空间中的位置用位置矢量描述：

$$\vv{r} = x\ii + y\jj + z\kk$$

大小：$r = |\vv{r}| = \sqrt{x^2+y^2+z^2}$

当坐标 $x=x(t),\,y=y(t),\,z=z(t)$ 为时间函数时，称为运动方程（参数方程形式）。消去 $t$ 得轨迹方程。

位移与路程

位移 vs 路程

位移（矢量）：$\Delta\vv{r} = \vv{r}_2 - \vv{r}_1 = \Delta x\,\ii + \Delta y\,\jj + \Delta z\,\kk$

位移大小：$|\Delta\vv{r}| = \sqrt{(\Delta x)^2+(\Delta y)^2+(\Delta z)^2}$

路程（标量）：$\Delta s = \int |\dd\vv{r}|$ —— 沿轨迹的弧长

易错：$|\Delta\vv{r}| \neq \Delta s$（除非直线运动且方向不变）。位移是"起点到终点的有向线段"，路程是"走过的总弧长"。

1.2 速度与加速度

速度

速度的定义

平均速度：$\bar{\vv{v}} = \frac{\Delta\vv{r}}{\Delta t}$（矢量，方向沿位移方向）

瞬时速度：$\vv{v} = \frac{\dd\vv{r}}{\dd t} = \dot{x}\,\ii + \dot{y}\,\jj + \dot{z}\,\kk$（矢量，方向沿轨迹切线）

速率（标量）：$v = |\vv{v}| = \sqrt{\dot{x}^2+\dot{y}^2+\dot{z}^2} = \frac{\dd s}{\dd t}$

加速度

加速度的定义

瞬时加速度：$\vv{a} = \frac{\dd\vv{v}}{\dd t} = \frac{\dd^2\vv{r}}{\dd t^2} = \ddot{x}\,\ii + \ddot{y}\,\jj + \ddot{z}\,\kk$

例1（运动方程→速度→加速度）：质点运动方程 $x=2t,\;y=4t-t^2$（SI），求 $t=1\,\text{s}$ 时速度和加速度。
解

$v_x = \dot{x} = 2\,\text{m/s}$，$v_y = \dot{y} = 4-2t$

$t=1$ 时：$v_x=2,\;v_y=2$，$v=\sqrt{4+4}=2\sqrt{2}\,\text{m/s}$

$a_x = \ddot{x} = 0$，$a_y = \ddot{y} = -2\,\text{m/s}^2$

加速度 $\vv{a} = -2\jj\,\text{m/s}^2$，方向沿 $y$ 轴负方向。

1.3 运动学两类基本问题

第一类：已知运动方程，求速度、加速度

方法：求导 $\vv{r}(t) \xrightarrow{\dd/\dd t} \vv{v}(t) \xrightarrow{\dd/\dd t} \vv{a}(t)$

第二类：已知加速度（力），求速度、运动方程

方法：积分（需要初始条件）

$$\vv{v}(t) = \vv{v}_0 + \int_0^t \vv{a}(t')\dd t'$$ $$\vv{r}(t) = \vv{r}_0 + \int_0^t \vv{v}(t')\dd t'$$
技巧：当加速度是速度或位置的函数时，需要换元：
  • $a = a(v)$：分离变量 $\frac{\dd v}{a(v)} = \dd t$
  • $a = a(x)$：利用 $a = v\frac{\dd v}{\dd x}$，则 $v\dd v = a(x)\dd x$
例2（积分法）：质点沿 $x$ 轴运动，加速度 $a=2+6t$（SI），$t=0$ 时 $x_0=0,\;v_0=0$，求 $v(t)$ 和 $x(t)$。
解

$v = \int_0^t(2+6t')\dd t' = 2t+3t^2$

$x = \int_0^t(2t'+3t'^2)\dd t' = t^2+t^3$

1.4 抛体运动

斜抛运动（初速 $v_0$，仰角 $\theta$）

分量方程（取发射点为原点，向上为 $y$ 正方向）：

$$x = v_0\cos\theta\cdot t,\quad y = v_0\sin\theta\cdot t - \tfrac{1}{2}gt^2$$ $$v_x = v_0\cos\theta = \text{const},\quad v_y = v_0\sin\theta - gt$$

消去 $t$ 得轨迹方程：

$$y = x\tan\theta - \frac{g}{2v_0^2\cos^2\theta}x^2$$

最大高度（令 $v_y=0$）：$H = \frac{v_0^2\sin^2\theta}{2g}$

水平射程（令 $y=0$）：$R = \frac{v_0^2\sin 2\theta}{g}$

结论：$\theta=45°$ 时射程最大 $R_{max}=v_0^2/g$；互补角 $\theta$ 和 $90°-\theta$ 射程相同。

1.5 圆周运动

角量描述

角量的定义

角位移：$\theta$（rad）

角速度：$\omega = \frac{\dd\theta}{\dd t}$（rad/s）

角加速度：$\alpha = \frac{\dd\omega}{\dd t} = \frac{\dd^2\theta}{\dd t^2}$（rad/s²）

线量与角量的关系

线量-角量关系（半径 $R$）
$$s = R\theta,\quad v = R\omega,\quad a_\tau = R\alpha,\quad a_n = R\omega^2 = \frac{v^2}{R}$$

匀变速圆周运动

当 $\alpha = \text{const}$ 时（类比匀变速直线运动）：

直线运动圆周运动
$v = v_0 + at$$\omega = \omega_0 + \alpha t$
$x = v_0 t + \frac{1}{2}at^2$$\theta = \omega_0 t + \frac{1}{2}\alpha t^2$
$v^2 = v_0^2 + 2ax$$\omega^2 = \omega_0^2 + 2\alpha\theta$
例3（圆周运动）：飞轮从静止开始以 $\alpha=2\,\text{rad/s}^2$ 匀角加速转动，半径 $R=0.5\,\text{m}$，求 $t=3\,\text{s}$ 时轮缘上点的切向和法向加速度。
解

$\omega = \alpha t = 2\times3=6\,\text{rad/s}$

$a_\tau = R\alpha = 0.5\times2 = 1\,\text{m/s}^2$

$a_n = R\omega^2 = 0.5\times36 = 18\,\text{m/s}^2$

总加速度 $a = \sqrt{a_\tau^2+a_n^2} = \sqrt{1+324} \approx 18.03\,\text{m/s}^2$

1.6 自然坐标系（切向-法向分解）

一般曲线运动的加速度分解

在轨迹上每一点建立切向 $\ee_\tau$ 和法向 $\ee_n$ 单位矢量：

$$\vv{a} = a_\tau\ee_\tau + a_n\ee_n$$

其中：

  • 切向加速度：$a_\tau = \frac{\dd v}{\dd t}$ —— 改变速度大小
  • 法向加速度：$a_n = \frac{v^2}{\rho}$ —— 改变速度方向，$\rho$ 为曲率半径

总加速度大小：$a = \sqrt{a_\tau^2 + a_n^2}$

高频考点：求曲率半径 $\rho$！
  • 若已知 $v$ 和 $a_n$：$\rho = v^2/a_n$
  • 若已知轨迹 $y=f(x)$：$\rho = \frac{[1+(y')^2]^{3/2}}{|y''|}$
  • 若已知 $v$ 和 $a$：先求 $a_\tau = \dd v/\dd t$，再由 $a_n = \sqrt{a^2-a_\tau^2}$，最后 $\rho = v^2/a_n$
例4（2022期中真题改编）：质量 $m$ 的飞机飞行曲线 $y=Ax^3$（$A$ 为常数），$x$ 方向速度 $v_x$ 恒定，求 $y$ 方向合外力。
解

$y = Ax^3$，$\dot{y} = 3Ax^2\dot{x} = 3Ax^2 v_x$

$\ddot{y} = 6Ax\dot{x}\cdot v_x = 6Axv_x^2$

$F_y = m\ddot{y} = 6mAxv_x^2$

例5（曲率半径）：抛体运动中，在最高点的曲率半径是多少？
解

最高点 $v_y=0$，故 $v = v_x = v_0\cos\theta$，法向加速度 $a_n = g$（此时速度水平，重力提供全部法向加速度）。

$$\rho = \frac{v^2}{a_n} = \frac{v_0^2\cos^2\theta}{g}$$

1.7 相对运动

经典（Galilei）速度叠加

设 $S'$ 系相对 $S$ 系以 $\vv{u}$ 运动，质点相对 $S$ 系速度为 $\vv{v}$，相对 $S'$ 系速度为 $\vv{v}'$：

$$\vv{v} = \vv{v}' + \vv{u}$$

即：绝对速度 = 相对速度 + 牵连速度

当 $\vv{u}=\text{const}$ 时，$\vv{a} = \vv{a}'$（加速度在惯性系间相同）

例6：船以 $v_1=4\,\text{m/s}$ 正东方向过河，水流 $v_2=3\,\text{m/s}$ 向正南，求船对地速度。
解

$v = \sqrt{v_1^2+v_2^2} = \sqrt{16+9} = 5\,\text{m/s}$，方向偏南 $\alpha = \arctan(3/4) \approx 36.9°$。

1.8 极坐标描述

平面极坐标 $(r,\theta)$

位矢：$\vv{r}=r\ee_r$

速度：$\vv{v}=\dot{r}\ee_r+r\dot{\theta}\ee_\theta$

  • $v_r=\dot{r}$：径向速度（远离/靠近原点）
  • $v_\theta=r\dot{\theta}=r\omega$：横向速度（绕原点旋转）

加速度：$\vv{a}=(\ddot{r}-r\dot{\theta}^2)\ee_r+(r\ddot{\theta}+2\dot{r}\dot{\theta})\ee_\theta$

圆周运动特例：$r=R=\text{const}$，则 $\dot{r}=\ddot{r}=0$，$a_r=-R\omega^2$（向心），$a_\theta=R\alpha$（切向）。

1.9 重要二级结论与技巧

匀变速直线运动公式汇总
  1. $v=v_0+at$
  2. $x=v_0t+\frac{1}{2}at^2$
  3. $v^2=v_0^2+2ax$
  4. $x=\frac{v_0+v}{2}\cdot t$（中间时刻速度 = 平均速度）
  5. $x_n-x_{n-1}=aT^2$（连续等时间间隔位移之差为恒量）
竖直平面内曲线运动分析技巧

对光滑曲面/钢丝约束的质点，使用自然坐标系：

沿曲面运动满足：$v\dfrac{\dd v}{\dd s}=a_\tau$ → 可改写为 $v\dd v=a_\tau\dd s$

对竖直面上的运动：$a_\tau=-g\sin\theta$，而 $\dd s\sin\theta=\dd y$，故：

$$v\dd v=-g\dd y \implies v^2=v_0^2-2g(y-y_0)$$

这与能量守恒一致！说明在光滑曲面上，速度只取决于高度差，与轨迹形状无关。

抛体运动中任意时刻的曲率半径

任意时刻速度方向与水平夹角 $\phi$，则：

$a_n=g\cos\phi$（加速度在法向的分量）

$$\rho=\frac{v^2}{g\cos\phi}$$

其中 $v=\sqrt{v_x^2+v_y^2}$，$\cos\phi=v_x/v=v_0\cos\theta/v$

例8（PPT拓展：最速降线问题）：两定点A、B高度不同，所有连接曲线中哪条使质点从A到B时间最短？
解

答案是摆线（旋轮线），不是直线也不是圆弧！这是变分法的经典问题（Johann Bernoulli, 1696）。

使此积分最小的曲线就是摆线。这说明"最短"和"最快"不是一回事。

1.10 运动学综合题型（PPT高频）

例9（轨迹方程求法）：质点运动方程 $x=R\cos\omega t$，$y=R\sin\omega t$，求轨迹方程。
解

$\cos\omega t=x/R$，$\sin\omega t=y/R$

$\cos^2\omega t+\sin^2\omega t=1$ → $x^2+y^2=R^2$（圆，半径 $R$）

速度大小：$v=\sqrt{\dot{x}^2+\dot{y}^2}=R\omega$（匀速圆周运动）

例10（加速度为速度函数）：质点沿直线运动，$a=kv$（$k<0$为常数），初速 $v_0$。求 $v(x)$。
解

利用 $a=v\frac{\dd v}{\dd x}$：$v\frac{\dd v}{\dd x}=kv$ → $\frac{\dd v}{\dd x}=k$

积分：$v-v_0=kx$ → $v=v_0+kx$

停止位置（$v=0$）：$x=-v_0/k$

1.11 运动学图形分析（PPT重点）

x-t 图（位移-时间图）

x-t 图的物理意义
  • 斜率 = 速度：直线的斜率为匀速运动的速度 $v = \Delta x / \Delta t$
  • 切线斜率 = 瞬时速度：曲线上某点切线的斜率即为该时刻的瞬时速度 $v = \dd x / \dd t$
  • 斜率为正 → 沿正方向运动；斜率为负 → 沿负方向运动；斜率为零 → 瞬时静止（极值点）
  • 斜率越大（越陡）→ 速度越大；曲线弯曲 → 做变速运动

常见图形：

图形形状运动类型
水平线静止（$v=0$）
过原点的直线（斜率 $k$）匀速运动（$v=k$）
抛物线（开口向上）匀加速运动（$x = x_0 + v_0 t + \frac{1}{2}at^2$）
抛物线（开口向下）匀减速运动

v-t 图（速度-时间图）

v-t 图的物理意义
  • 斜率 = 加速度：$a = \dd v / \dd t$，切线斜率即瞬时加速度
  • 图线与时间轴围成的面积 = 位移：$\Delta x = \int_{t_1}^{t_2} v\,\dd t$
  • $v>0$ 部分面积为正位移，$v<0$ 部分面积为负位移
  • 面积在时间轴上方为正，下方为负，总位移 = 代数和

常见图形：

图形形状运动类型加速度
水平线匀速运动$a=0$
倾斜直线匀加速/匀减速$a=\text{const}$
曲线（斜率递减）加速度逐渐减小$a$ 递减
区分位移与路程：v-t 图中面积的代数和是位移，面积的绝对值之和是路程。当速度改变方向（$v$ 穿越零轴）时两者不同。

a-t 图（加速度-时间图）

a-t 图的物理意义
  • 图线与时间轴围成的面积 = 速度变化量：$\Delta v = \int_{t_1}^{t_2} a\,\dd t$
  • 面积为正 → 速度增加；面积为负 → 速度减小
  • 已知 $v_0$，可由面积逐段累加得到 $v(t)$
三种图形的对应关系总结
图形斜率的意义面积的意义
x-t 图速度 $v$（无直接物理意义）
v-t 图加速度 $a$位移 $\Delta x$
a-t 图（加速度的变化率）速度变化量 $\Delta v$
例11（v-t图分析：分段线性）：质点沿直线运动，其 v-t 图如下描述：$0\sim2\,\text{s}$ 内速度从 $0$ 线性增到 $4\,\text{m/s}$；$2\sim5\,\text{s}$ 内速度保持 $4\,\text{m/s}$；$5\sim7\,\text{s}$ 内速度从 $4\,\text{m/s}$ 线性减到 $0$。求：(1) 各段加速度；(2) $0\sim7\,\text{s}$ 总位移。
解

(1) 各段加速度（v-t 图斜率）：

  • $0\sim2\,\text{s}$：$a_1 = \dfrac{4-0}{2-0} = 2\,\text{m/s}^2$（匀加速）
  • $2\sim5\,\text{s}$：$a_2 = 0$（匀速）
  • $5\sim7\,\text{s}$：$a_3 = \dfrac{0-4}{7-5} = -2\,\text{m/s}^2$（匀减速）

(2) 总位移（v-t 图下面积）：

$$x = \underbrace{\frac{1}{2}\times2\times4}_{三角形} + \underbrace{3\times4}_{矩形} + \underbrace{\frac{1}{2}\times2\times4}_{三角形} = 4 + 12 + 4 = 20\,\text{m}$$

也可以看成一个梯形：$x = \frac{1}{2}(2+7-0)\times4$...不对，用分段法更清晰。

整体看，图形为梯形（上底 $= 5-2 = 3\,\text{s}$，下底 $= 7\,\text{s}$，高 $= 4\,\text{m/s}$）：

$$x = \frac{1}{2}(3+7)\times 4 = 20\,\text{m} \checkmark$$
考试陷阱：
  • v-t 图中速度为负的部分，面积要取负值计算位移，但取绝对值计算路程
  • x-t 图不是轨迹图！x-t 图是一维运动的位置随时间变化，不表示二维空间的运动轨迹
  • v-t 图中两条曲线交点表示两物体速度相等，不一定表示相遇

1.12 追及与相遇问题

追及问题的核心思路

追及问题分析方法

追及条件：后方物体的位移 = 前方物体的位移 + 初始间距

$$x_{追} = x_{被追} + d_0$$

其中 $d_0$ 为初始距离。

关键时刻：当两者速度相等时，两者距离最小或最大：

  • 若追者速度从大于被追者到等于被追者 → 此时距离最小（之后追者变慢，距离拉大）
  • 若追者速度从小于被追者到等于被追者 → 此时距离最大（之后追者变快，开始追上）
判断能否追上的方法
  1. 令 $v_{追} = v_{被追}$，求出此临界时刻 $t^*$
  2. 计算 $t^*$ 时刻两者的距离差 $\Delta x(t^*)$
  3. 若 $\Delta x(t^*) \leq 0$（追者已到达或超过被追者位置）→ 能追上，在 $t^*$ 之前已追上
  4. 若 $\Delta x(t^*) > 0$（追者仍在后方）→ 追不上，$t^*$ 时刻距离最近

相遇问题的核心思路

相遇条件

同一时刻 $t$，两物体到达同一位置：$x_1(t) = x_2(t)$

对于相向运动的两物体，相遇条件为：

$$x_1(t) + x_2(t) = d_0$$

其中 $d_0$ 为初始距离，$x_1,x_2$ 分别为各自出发点的位移。

解题策略：
  • 方程法：分别写出两物体运动方程，令位移满足追及/相遇条件，联立求解
  • 图形法：画两物体的 x-t 图，交点即为相遇时刻和位置；画 v-t 图，面积差等于初始间距时即追上
  • 相对运动法：以一个物体为参考系，分析另一物体的相对运动
例12（经典追及问题）：A车以恒定速度 $v_0$ 做匀速直线运动。B车从A车后方距离 $d$ 处由静止出发，以加速度 $a$ 做匀加速直线运动追赶A车。求：(1) B追上A需要多长时间？(2) 追上前两车最大距离是多少？
解

设 $t=0$ 时B车出发，取B车出发点为原点，正方向为运动方向。

A车位置：$x_A = d + v_0 t$

B车位置：$x_B = \frac{1}{2}at^2$

(1) 追上条件：$x_B = x_A$

$$\frac{1}{2}at^2 = d + v_0 t$$ $$\frac{1}{2}at^2 - v_0 t - d = 0$$ $$t = \frac{v_0 + \sqrt{v_0^2 + 2ad}}{a}$$

（取正根）

(2) 最大距离：两车速度相等时距离最大

$$v_B = v_A \implies at^* = v_0 \implies t^* = \frac{v_0}{a}$$

此时距离差：

$$\Delta x = x_A(t^*) - x_B(t^*) = \left(d + v_0\cdot\frac{v_0}{a}\right) - \frac{1}{2}a\left(\frac{v_0}{a}\right)^2 = d + \frac{v_0^2}{2a}$$
例13（相向运动相遇）：甲、乙两人相距 $L=100\,\text{m}$，甲以 $v_1=3\,\text{m/s}$ 匀速向乙运动，同时乙从静止以 $a=2\,\text{m/s}^2$ 匀加速向甲运动。求何时相遇？
解

取甲的出发位置为原点，向乙方向为正。

甲的位置：$x_1 = v_1 t = 3t$

乙的位置：$x_2 = L - \frac{1}{2}at^2 = 100 - t^2$

相遇条件：$x_1 = x_2$

$$3t = 100 - t^2 \implies t^2 + 3t - 100 = 0$$ $$t = \frac{-3 + \sqrt{9+400}}{2} = \frac{-3+\sqrt{409}}{2} \approx \frac{-3+20.22}{2} \approx 8.6\,\text{s}$$

相遇位置：$x = 3\times8.6 \approx 25.8\,\text{m}$（距甲出发点）。

例14（追及临界问题）：警车从静止以 $a=2.5\,\text{m/s}^2$ 加速追赶前方 $d=50\,\text{m}$ 处以 $v_0=10\,\text{m/s}$ 匀速行驶的汽车。但警车最高速度为 $v_m=20\,\text{m/s}$。求追上所需时间。
解

阶段分析：警车先匀加速到 $v_m$，然后匀速。

加速阶段持续时间：$t_1 = v_m/a = 20/2.5 = 8\,\text{s}$

$t_1$ 时刻位移：警车 $x_B = \frac{1}{2}\times2.5\times64 = 80\,\text{m}$；汽车 $x_A = 50 + 10\times8 = 130\,\text{m}$

距离差：$130 - 80 = 50\,\text{m}$（仍未追上）

$t > t_1$ 后，警车以 $v_m = 20\,\text{m/s}$ 匀速，汽车以 $v_0 = 10\,\text{m/s}$ 匀速，相对速度 $\Delta v = 10\,\text{m/s}$。

剩余距离追赶时间：$t_2 = 50/10 = 5\,\text{s}$

总时间：$t = t_1 + t_2 = 8 + 5 = 13\,\text{s}$

追及问题常见陷阱：
  • 有最大速度限制时，不能一直用匀加速公式，需分段讨论
  • 若 $v_0^2 + 2ad < 0$（判别式为负），说明追不上（但本题中 $d > 0$ 时判别式总为正）
  • 二次方程两个根：较小的正根为第一次追上（从后面追上），较大正根为第二次相遇（追过后减速再被追上的情况）

1.13 经典力学时空观与Galileo变换（PPT补充）

Newton的绝对时空观
  • 绝对时间：时间流逝与参考系无关，所有观察者测量的时间间隔相同
  • 绝对空间：空间是独立于物质的"容器"，长度测量与参考系无关

这两个假设在经典力学中完全适用（$v\ll c$），但在高速运动中被相对论推翻。

Galileo变换

$S'$ 系相对 $S$ 系沿 $x$ 方向以速度 $u$ 运动：

$$x' = x - ut,\quad y' = y,\quad z' = z,\quad t' = t$$

速度变换：$\vv{v}' = \vv{v} - \vv{u}$

加速度变换：$\vv{a}' = \vv{a}$（加速度在不同惯性系中相同！）

Galileo相对性原理

力学规律在一切惯性参考系中具有相同的形式。$F=ma$ 在任何惯性系中都成立（因为 $\vv{a}'=\vv{a}$，$\vv{F}'=\vv{F}$）。

局限：Galileo变换不适用于电磁现象（Maxwell方程在Galileo变换下不协变），这正是Einstein建立狭义相对论的动机之一。

相对运动问题：
  • 船在流水中：$v_{地}=v_{船对水}+v_{水对地}$（矢量加法）
  • 雨伞问题：在移动参考系中分析雨滴的相对运动方向
  • 风中行走：顺风和逆风中雨滴打在人身上的角度不同

1.14 曲率半径的计算方法（MOOC/PPT补充）

曲率半径的一般公式

显函数形式 $y = f(x)$

若轨迹可以表示为 $y = f(x)$，则曲率半径为：

$$\rho = \frac{\left[1+\left(\dfrac{\dd y}{\dd x}\right)^2\right]^{3/2}}{\left|\dfrac{\dd^2 y}{\dd x^2}\right|}$$

其中 $y' = \dd y/\dd x$ 为一阶导数，$y'' = \dd^2 y/\dd x^2$ 为二阶导数。

参数方程形式 $x = x(t),\;y = y(t)$

若轨迹以参数 $t$ 表示，则曲率半径为：

$$\rho = \frac{\left[\left(\dfrac{\dd x}{\dd t}\right)^2+\left(\dfrac{\dd y}{\dd t}\right)^2\right]^{3/2}}{\left|\dfrac{\dd x}{\dd t}\cdot\dfrac{\dd^2 y}{\dd t^2}-\dfrac{\dd^2 x}{\dd t^2}\cdot\dfrac{\dd y}{\dd t}\right|}$$

简记为：$\rho = \dfrac{(\dot{x}^2+\dot{y}^2)^{3/2}}{|\dot{x}\ddot{y}-\ddot{x}\dot{y}|}$

物理方法（利用法向加速度）

在任意曲线运动中，若已知速率 $v$ 和法向加速度 $a_n$：

$$\rho = \frac{v^2}{a_n}$$

这是最常用的物理方法，尤其适合已知运动学量的情况。

例15（抛体运动最高点的曲率半径）：以初速度 $v_0$、仰角 $\alpha$ 抛出一个物体（忽略空气阻力），求最高点处的曲率半径。
解法1：物理方法

在最高点，$v_y = 0$，速度完全沿水平方向：$v = v_0\cos\alpha$

此时速度方向水平，而加速度 $g$ 竖直向下，完全充当法向加速度：$a_n = g$

$$\rho = \frac{v^2}{a_n} = \frac{v_0^2\cos^2\alpha}{g}$$
解法2：公式法

轨迹方程：$y = x\tan\alpha - \dfrac{g}{2v_0^2\cos^2\alpha}x^2$

$y' = \tan\alpha - \dfrac{g}{v_0^2\cos^2\alpha}x$

$y'' = -\dfrac{g}{v_0^2\cos^2\alpha}$

最高点处 $y' = 0$（切线水平），代入公式：

$$\rho = \frac{[1+0^2]^{3/2}}{|y''|} = \frac{1}{g/(v_0^2\cos^2\alpha)} = \frac{v_0^2\cos^2\alpha}{g}$$

两种方法结果一致。

例16（抛物线顶点的曲率半径）：曲线 $y = Ax^2$（$A > 0$），求顶点 $x = 0$ 处的曲率半径。
解

$y' = 2Ax$，$y'' = 2A$

在 $x = 0$ 处：$y' = 0$，$y'' = 2A$

$$\rho = \frac{[1+0^2]^{3/2}}{|2A|} = \frac{1}{2A}$$

所以抛物线 $y = Ax^2$ 顶点处的曲率半径为 $\rho = 1/(2A)$。

对于抛体运动轨迹 $y = x\tan\alpha - \frac{g}{2v_0^2\cos^2\alpha}x^2$，在最高点处等效系数 $A = \frac{g}{2v_0^2\cos^2\alpha}$，于是 $\rho = 1/(2A) = v_0^2\cos^2\alpha/g$，与前例一致。

例17（参数方程求曲率半径）：质点运动方程 $x = R\cos\omega t$，$y = R\sin\omega t$，求曲率半径。
解

$\dot{x} = -R\omega\sin\omega t$，$\ddot{x} = -R\omega^2\cos\omega t$

$\dot{y} = R\omega\cos\omega t$，$\ddot{y} = -R\omega^2\sin\omega t$

$\dot{x}^2 + \dot{y}^2 = R^2\omega^2$

$\dot{x}\ddot{y} - \ddot{x}\dot{y} = R^2\omega^3\sin^2\omega t + R^2\omega^3\cos^2\omega t = R^2\omega^3$

$$\rho = \frac{(R^2\omega^2)^{3/2}}{R^2\omega^3} = \frac{R^3\omega^3}{R^2\omega^3} = R$$

匀速圆周运动的曲率半径处处等于圆的半径，符合预期。

理解与技巧：在抛体运动最高点，速度完全沿水平方向，重力加速度 $g$ 完全充当法向加速度（因为 $a_\tau = 0$）。此时 $g = v^2/\rho$，由此可以直接得到 $\rho = v^2/g$。这是"法向加速度法"的典型应用。
易错：使用公式法时，$y'$ 和 $y''$ 是对 $x$ 的导数，不是对 $t$ 的导数。若给的是参数方程，必须用参数方程版公式。

1.15 弧长的计算（真题常考）

弧长公式

弧长的积分公式

质点从 $t_1$ 到 $t_2$ 时间内走过的路程（弧长）：

$$s = \int_{t_1}^{t_2}|\vv{v}|\,\dd t = \int_{t_1}^{t_2}\sqrt{v_x^2 + v_y^2 + v_z^2}\,\dd t$$

对于平面运动（二维）：

$$s = \int_{t_1}^{t_2}\sqrt{\dot{x}^2 + \dot{y}^2}\,\dd t$$

若轨迹为显函数 $y = f(x)$：

$$s = \int_{x_1}^{x_2}\sqrt{1 + \left(\frac{\dd y}{\dd x}\right)^2}\,\dd x$$
例18（2022期中真题改编）：质点运动方程为 $\vv{r} = 2t\,\ii + t^2\,\jj$（SI），求从 $t = 1\,\text{s}$ 到 $t = 3\,\text{s}$ 经过的路程。
解

速度分量：$v_x = \dot{x} = 2$，$v_y = \dot{y} = 2t$

速率：$v = \sqrt{v_x^2 + v_y^2} = \sqrt{4 + 4t^2} = 2\sqrt{1 + t^2}$

$$s = \int_1^3 2\sqrt{1+t^2}\,\dd t$$

令 $t = \sinh u$，则 $\dd t = \cosh u\,\dd u$，$\sqrt{1+t^2} = \cosh u$：

$$s = 2\int \cosh^2 u\,\dd u = 2\int\frac{1+\cosh 2u}{2}\dd u = u + \frac{\sinh 2u}{2} + C$$

回代 $u = \text{arsinh}\,t = \ln(t+\sqrt{1+t^2})$，$\sinh 2u = 2t\sqrt{1+t^2}$：

$$s = \left[\ln(t+\sqrt{1+t^2}) + t\sqrt{1+t^2}\right]_1^3$$ $$= \left[\ln(3+\sqrt{10}) + 3\sqrt{10}\right] - \left[\ln(1+\sqrt{2}) + \sqrt{2}\right]$$ $$\approx (1.818 + 9.487) - (0.881 + 1.414) \approx 9.01\,\text{m}$$
例19（简单弧长计算）：质点做匀速圆周运动 $x = R\cos\omega t$，$y = R\sin\omega t$，求一个周期 $T = 2\pi/\omega$ 内的路程。
解

$v = \sqrt{\dot{x}^2+\dot{y}^2} = R\omega$（常数）

$$s = \int_0^T R\omega\,\dd t = R\omega\cdot\frac{2\pi}{\omega} = 2\pi R$$

正好是圆的周长，验证了公式的正确性。

路程 $\neq$ 位移大小：
  • 路程是标量，等于速率对时间的积分：$s = \int |\vv{v}|\,\dd t$，恒为正值
  • 位移大小是矢量差的模：$|\Delta\vv{r}| = |\vv{r}(t_2) - \vv{r}(t_1)|$
  • 只有在直线运动且方向不变时两者才相等
  • 上例中：位移 $\Delta\vv{r} = (6-2)\ii + (9-1)\jj = 4\ii + 8\jj$，$|\Delta\vv{r}| = \sqrt{16+64} = \sqrt{80} \approx 8.94\,\text{m}$，而路程 $s \approx 9.01\,\text{m} > |\Delta\vv{r}|$
考试技巧：如果弧长积分不好算（如含有 $\sqrt{1+t^2}$ 等），可以尝试查积分表或用双曲代换 $t = \sinh u$。有些题目会设计为弧长积分恰好可以简化（如速率为常数，或被积函数为完全平方）。

1.16 运动学两类问题详解（PPT重点）

第一类问题：已知运动方程，求速度和加速度

方法：逐次求导
$$\vv{r}(t) \xrightarrow{\dd/\dd t} \vv{v}(t) \xrightarrow{\dd/\dd t} \vv{a}(t)$$

这类问题较为直接，只要会求导即可。关键在于正确计算矢量各分量的导数。

第二类问题：已知加速度或力，求速度和运动方程

方法：积分 + 初始条件
$$\vv{a}(t) \xrightarrow{\int\dd t} \vv{v}(t) = \vv{v}_0 + \int_0^t\vv{a}(t')\,\dd t' \xrightarrow{\int\dd t} \vv{r}(t) = \vv{r}_0 + \int_0^t\vv{v}(t')\,\dd t'$$

积分会产生常数，必须利用初始条件（$t = 0$ 时的 $\vv{v}_0$ 和 $\vv{r}_0$）来确定。

例20（经典第二类问题：空气阻力下的运动）：质点沿直线运动，受到与速度成正比的阻力 $a = -kv$（$k > 0$ 为常数），初始速度 $v_0$，初始位置 $x_0 = 0$。求 $v(t)$ 和 $x(t)$。
解

求 $v(t)$：$\dfrac{\dd v}{\dd t} = -kv$

分离变量：$\dfrac{\dd v}{v} = -k\,\dd t$

两边积分：$\ln v - \ln v_0 = -kt$

$$\boxed{v(t) = v_0\,e^{-kt}}$$

速度随时间指数衰减，永远不会精确为零，但实际上很快趋近于零。

求 $x(t)$：

$$x = \int_0^t v\,\dd t' = \int_0^t v_0\,e^{-kt'}\,\dd t' = v_0\left[-\frac{1}{k}e^{-kt'}\right]_0^t$$ $$\boxed{x(t) = \frac{v_0}{k}\left(1 - e^{-kt}\right)}$$

讨论：当 $t \to \infty$ 时：

$$x_{\max} = \frac{v_0}{k}$$

物体最终停在有限距离处！虽然速度永远不为零，但行程却是有限的（收敛的无穷级数）。这是指数衰减运动的重要特征。

例21（加速度为位置函数）：弹簧振子，$a = -\omega_0^2 x$（$\omega_0$ 为常数），初始条件 $x(0) = A$，$v(0) = 0$。利用运动学方法求 $v(x)$。
解

利用换元 $a = v\dfrac{\dd v}{\dd x}$：

$$v\,\dd v = -\omega_0^2 x\,\dd x$$

积分（从 $x = A, v = 0$ 到 $x, v$）：

$$\frac{v^2}{2} - 0 = -\omega_0^2\left(\frac{x^2}{2} - \frac{A^2}{2}\right)$$ $$v^2 = \omega_0^2(A^2 - x^2)$$ $$v = \pm\omega_0\sqrt{A^2 - x^2}$$

这就是简谐运动中速度与位移的关系，当 $x = 0$ 时速度最大 $v_{max} = \omega_0 A$，当 $x = \pm A$ 时速度为零。

积分时必须代入初始条件：
  • 不定积分法：积出含任意常数 $C$ 的通解，再用初始条件定 $C$
  • 定积分法（推荐）：直接用初始条件作为积分下限，结果不含待定常数
  • 忘记代入初始条件是考试中最常见的扣分原因之一
第二类问题的分类处理：
  • $a = f(t)$：直接对 $t$ 积分
  • $a = f(v)$：分离变量 $\dd v/f(v) = \dd t$ 求 $v(t)$；或用 $v\,\dd v/f(v) = \dd x$ 求 $v(x)$
  • $a = f(x)$：用 $v\,\dd v = f(x)\,\dd x$ 求 $v(x)$，再由 $\dd x/v = \dd t$ 求 $x(t)$

1.17 极坐标系中的运动描述（PPT补充）

极坐标基本量

极坐标中的位矢、速度、加速度

在平面极坐标 $(r, \theta)$ 中，定义径向单位矢量 $\hat{e}_r$（指向远离原点方向）和横向单位矢量 $\hat{e}_\theta$（与 $\hat{e}_r$ 垂直，沿 $\theta$ 增大方向）。

位矢：

$$\vv{r} = r\hat{e}_r$$

速度：

$$\vv{v} = \dot{r}\hat{e}_r + r\dot{\theta}\hat{e}_\theta$$
  • $v_r = \dot{r}$：径向速度——描述质点远离或靠近原点的快慢
  • $v_\theta = r\dot{\theta} = r\omega$：横向速度——描述质点绕原点旋转的快慢

加速度：

$$\vv{a} = (\ddot{r} - r\dot{\theta}^2)\hat{e}_r + (r\ddot{\theta} + 2\dot{r}\dot{\theta})\hat{e}_\theta$$
  • $a_r = \ddot{r} - r\omega^2$：径向加速度。第一项 $\ddot{r}$ 是径向距离的变化率，第二项 $-r\omega^2$ 是向心加速度（指向原点）
  • $a_\theta = r\ddot{\theta} + 2\dot{r}\dot{\theta} = r\alpha + 2\dot{r}\omega$：横向加速度。第一项 $r\alpha$ 是角加速带来的切向加速度，第二项 $2\dot{r}\omega$ 是Coriolis项

圆周运动特例

$r = R = \text{const}$ 时的简化

当质点做圆周运动时，$r = R$，$\dot{r} = 0$，$\ddot{r} = 0$：

$$a_r = -R\omega^2 \quad\text{（向心加速度，指向圆心）}$$ $$a_\theta = R\alpha \quad\text{（切向加速度）}$$

这与自然坐标系中的结果完全一致：$a_n = R\omega^2 = v^2/R$（法向），$a_\tau = R\alpha$（切向）。

例22（极坐标运动分析）：质点在平面内运动，极坐标描述为 $r = a e^{b\theta}$（等角螺线），其中 $a, b$ 为正常数，$\dot{\theta} = \omega_0 = \text{const}$。求速度大小和加速度的径向分量。
解

$r = ae^{b\theta}$，$\dot{r} = ab\dot{\theta}e^{b\theta} = b\omega_0 r$，$\ddot{r} = b\omega_0\dot{r} = b^2\omega_0^2 r$

速度：

$v_r = \dot{r} = b\omega_0 r$，$v_\theta = r\omega_0$

$$v = \sqrt{v_r^2 + v_\theta^2} = r\omega_0\sqrt{1+b^2}$$

径向加速度：

$$a_r = \ddot{r} - r\dot{\theta}^2 = b^2\omega_0^2 r - r\omega_0^2 = (b^2 - 1)\omega_0^2 r$$

当 $b = 1$ 时 $a_r = 0$（径向加速度为零），质点沿螺线向外运动。

例23（行星运动中的极坐标应用）：在中心力场中，质点受力方向沿径向（$F_\theta = 0$）。证明角动量守恒。
解

由 $F_\theta = ma_\theta = 0$，得：

$$r\ddot{\theta} + 2\dot{r}\dot{\theta} = 0$$

注意到 $\dfrac{1}{r}\dfrac{\dd}{\dd t}(r^2\dot{\theta}) = r\ddot{\theta} + 2\dot{r}\dot{\theta}$，所以：

$$\frac{\dd}{\dd t}(r^2\dot{\theta}) = 0 \implies r^2\dot{\theta} = \text{const} = L/m$$

其中 $L = mr^2\omega$ 就是角动量。这说明在中心力场中角动量守恒（Kepler第二定律的本质）。

应用场景：极坐标在以下问题中特别有用：
  • 中心力问题（万有引力、库仑力）：力沿径向，$F_\theta = 0$
  • 螺线运动：$r = f(\theta)$ 形式简洁
  • 行星轨道、卫星运动：椭圆/双曲线的极坐标方程更自然
  • 旋转参考系问题：极坐标中的Coriolis项 $2\dot{r}\omega$ 自然出现
注意：极坐标的 $\hat{e}_r$ 和 $\hat{e}_\theta$ 方向随质点位置而变化（不是固定方向），这是极坐标加速度表达式复杂的根本原因。求导时 $\dot{\hat{e}}_r = \dot{\theta}\hat{e}_\theta$，$\dot{\hat{e}}_\theta = -\dot{\theta}\hat{e}_r$。

1.18 铁锤敲钉问题（PPT经典 — $a = f(x)$ 综合）

例24（铁锤敲钉子）：用铁锤把钉子敲入木板，木板对钉子的阻力与钉子进入木板的深度成正比（$f = kx$）。铁锤第一次敲打时把钉子敲入 $h_1$。若第二次敲打速度与第一次完全相同，求第二次敲入的深度 $h_2$。
解

设敲打瞬间钉子获得初速 $v_0$，阻力 $f = kx$（$x$ 为从钉尖算起的深度）。

第一次敲打：从 $x = 0$ 到 $x = h_1$，$v$ 从 $v_0$ 降为 $0$。

利用 $v\dd v = -\frac{k}{m}x\dd x$，积分：

$$\int_{v_0}^{0}v\dd v = -\frac{k}{m}\int_0^{h_1}x\dd x$$ $$-\frac{v_0^2}{2} = -\frac{k}{2m}h_1^2 \implies v_0^2 = \frac{k}{m}h_1^2$$

第二次敲打：钉子已在 $x = h_1$ 处，从 $h_1$ 到 $h_1 + h_2$，$v$ 从 $v_0$ 降为 $0$。

$$\int_{v_0}^{0}v\dd v = -\frac{k}{m}\int_{h_1}^{h_1+h_2}x\dd x$$ $$-\frac{v_0^2}{2} = -\frac{k}{2m}\left[(h_1+h_2)^2 - h_1^2\right]$$

利用第一次的结果 $v_0^2 = kh_1^2/m$：

$$h_1^2 = (h_1+h_2)^2 - h_1^2 = 2h_1 h_2 + h_2^2$$ $$h_2^2 + 2h_1 h_2 - h_1^2 = 0$$ $$h_2 = h_1(\sqrt{2}-1) \approx 0.414\,h_1$$

物理意义：第二次敲入深度明显比第一次少，因为钉子已进入较深处，阻力更大。

方法总结：当阻力 $f = f(x)$ 且题目涉及速度-位移关系时，直接用 $v\dd v = a(x)\dd x$ 换元法，不必求 $v(t)$ 再求 $x(t)$。

1.19 绳拉物体的几何约束与加速度分析（PPT重点）

通过定滑轮匀速收绳的问题

这类问题的核心是：收绳速率 $v_0 = \text{const}$，但物体的实际速度和加速度并非常数。需通过几何约束关系求导得出。

例25（两人拉物体）：两人通过定滑轮对称地将质量 $m$ 的物体向上拉起。已知 $BO = OC = b$（两滑轮间距的一半），拉绳速率均为常数 $v_0$。当两绳夹角为 $2\theta$ 时，绳中张力为多少？
解

设物体在 $O$ 正下方高度 $y$ 处，绳长 $l = \sqrt{y^2 + b^2}$。

由 $l^2 = y^2 + b^2$，对时间求导：$l\dot{l} = y\dot{y}$

物体上升速度：$v = \dot{y} = \frac{l}{y}\dot{l} = \frac{v_0}{\cos\theta}$（因为 $\cos\theta = y/l$，$\dot{l} = -v_0$）

再求导得加速度：

$$a = \frac{\dd v}{\dd t} = \frac{v_0^2 b^2}{l^2 y} \cdot \frac{l}{y} = \frac{v_0^2 \tan^2\theta}{y}$$

更简洁地写为 $a = \dfrac{v_0^2 b^2}{y^3}$。

由牛顿第二定律（向上）：$2T\cos\theta - mg = ma$

$$T = \frac{m(g + a)}{2\cos\theta} = \frac{mg + mv_0^2\tan^2\theta/y}{2\cos\theta}$$
例26（绳拉船问题）：湖中小船质量 $m$，有人在岸上高 $h$ 处以匀速率 $v_0$ 收绳（绕过定滑轮）。求船在水平位置 $x$ 时的水平加速度及绳中张力。
解

绳长 $l = \sqrt{x^2 + h^2}$，$l\dot{l} = x\dot{x}$。收绳速率 $\dot{l} = -v_0$。

船的水平速度：$\dot{x} = -\frac{l}{x}v_0 = -v_0\sqrt{1 + h^2/x^2}$

对水平速度再求导，经过计算得水平加速度（大小）：

$$a_x = \frac{v_0^2 h^2}{x^3}\cdot\frac{x}{\sqrt{x^2+h^2}} = \frac{v_0^2 h^2}{x^2(x^2+h^2)^{1/2}} \cdot x$$

简化：$a_x = \dfrac{v_0^2 h^2}{x^3}$（当仅取水平分量时）。

水平方向：$T\dfrac{x}{\sqrt{x^2+h^2}} = ma_x$

竖直方向：$N + T\dfrac{h}{\sqrt{x^2+h^2}} = mg$

$$T = \frac{mv_0^2 h^2}{x^4}\cdot\sqrt{x^2+h^2}$$
易错点：收绳速度 $v_0$ 不等于物体的运动速度！物体的速度是绳速度在运动方向上的分量。即使收绳速率恒定，物体也在做变加速运动（加速度随位置变化）。

1.20 旋转容器中液面和曲面形状（PPT补充）

例27（旋转碗内表面形状）：一光滑瓷碗绕竖直轴以角速度 $\omega$ 匀速转动。若碗内表面上任意位置放置的小球都能保持平衡，求碗的内表面形状。
解

在碗的参考系中（或用惯性系分析），小球在距轴 $x$ 处平衡：

竖直方向：$N\cos\alpha = mg$

水平方向（提供向心力）：$N\sin\alpha = m\omega^2 x$

两式相除：$\tan\alpha = \dfrac{\omega^2 x}{g}$

而曲面切线斜率 $\tan\alpha = \dfrac{\dd y}{\dd x}$，所以：

$$\frac{\dd y}{\dd x} = \frac{\omega^2}{g}x$$

积分：$y = \dfrac{\omega^2}{2g}x^2 + C$

取底部为原点，$C = 0$：

$$\boxed{y = \frac{\omega^2}{2g}x^2}$$

碗的内表面为旋转抛物面。

例28（加速容器中液面形状）：装有不可压缩液体的容器沿水平方向做匀加速直线运动，加速度为 $a$。求液面形状。
解

在容器参考系中，液面上每个质点受重力 $mg$（向下）和惯性力 $ma$（水平，与加速度方向相反）。

液面与这两个力的合力垂直，因此液面斜率为：

$$\frac{\dd y}{\dd x} = -\frac{a}{g}$$

液面为一倾斜的平面，倾斜角 $\theta = \arctan(a/g)$。

例29（U型管加速度计）：U型管加速度计装在匀加速运动的物体上，两管相距 $L$，测得两管液面差 $h$。求加速度。
解

液面斜率 $= a/g$，而两管间液面差 $h$ 对应水平距离 $L$：

$$\frac{h}{L} = \frac{a}{g} \implies \boxed{a = \frac{gh}{L}}$$

1.21 PPT补充例题精选（力学综合）

最速降线问题（Brachistochrone）——变分法的起源

例30（PPT：等时降线 / 最速降线）：1696年 Johann Bernoulli 在《教师学报》上提出公开挑战：设两定点 A、B 高度不同且不在同一铅垂线上，无摩擦和空气阻力，在所有连接 A、B 的曲线中，求使初速度为零的质点在重力作用下从 A 沿曲线下滑到 B 的时间最短的那条曲线。
推导

取 A 为原点，$y$ 轴竖直向下。质点从 A 静止出发，下滑到高度 $y$ 处时速率由能量守恒得：

$$v = \sqrt{2gy}$$

沿曲线微元 $\dd s = \sqrt{1+y'^2}\,\dd x$，所需时间微元：

$$\dd t = \frac{\dd s}{v} = \frac{\sqrt{1+y'^2}}{\sqrt{2gy}}\,\dd x$$

总时间为泛函：

$$t[y] = \int_0^a \frac{\sqrt{1+y'^2}}{\sqrt{2gy}}\,\dd x$$

使此泛函取极值的曲线满足 Euler-Lagrange 方程，解为摆线（旋轮线）：

$$x = \frac{R}{2}(\theta - \sin\theta),\quad y = \frac{R}{2}(1 - \cos\theta)$$

其中 $R$ 由 B 点坐标确定。

历史趣闻：1697年《教师学报》同时发表了 Leibniz、Newton 以及 Bernoulli 兄弟的答案。Newton 当时已是皇家造币厂厂长，据说仅用一个晚上就给出正确答案并匿名发表。Johann Bernoulli 看到后惊呼："我从这锋利的爪痕上认出了这头雄狮！"
核心结论："最短"（直线）和"最快"（摆线）不是一回事！摆线虽然路径更长，但质点先快速下降获得较大速度，总时间反而最短。

长方体浸入液体的运动分析（PPT完整系列）

模型设定

密度 $\rho$ 等于水的长方体（高 $h$、底面积 $S$），平衡时全部浸入水中。取 $y$ 轴竖直向上，原点为长方体刚好全部浸入的位置。设顶面距水面为 $y$（$0 < y < h$）时，露出水面部分的重力即为合力：

$$\rho S y g = \rho S h \cdot a \implies a = \frac{g}{h}y$$

利用 $v\dd v = \dfrac{g}{h}y\dd y$ 进行积分。

例31（$y_0 > h$：顶面提出水面后再自由下落）：将顶面提到 $y_0 > h$（即长方体完全出水）后静止释放，求刚好全部浸入时的速度。
解

阶段一（$y_0 \to h$）：长方体完全在水面上方，只受重力（浮力为零），做自由落体。

$$v_1^2 = 2g(y_0 - h) \implies v_1 = \sqrt{2g(y_0 - h)}$$

阶段二（$h \to 0$）：长方体逐渐浸入水中，$v\dd v = \dfrac{g}{h}y\dd y$：

$$\int_{v_1}^{v} v\dd v = \int_h^0 \frac{g}{h}y\dd y \implies \frac{v^2 - v_1^2}{2} = -\frac{g}{2h}h^2$$ $$v = \sqrt{v_1^2 + gh} = \sqrt{2g(y_0 - h) + gh} = \sqrt{g(2y_0 - h)}$$
例32（给初速度向上：求底面刚好能离开水面的条件）：初始时长方体刚好全部浸入水中（$y = 0$），给予向上初速度 $v_0$。若底面刚好能离开水面（即 $y = h$ 时 $v = 0$），求 $v_0$。
解

在 $0 \leq y \leq h$ 范围内（向上运动），合力向下为 $\rho Syg$，加速度 $a = -gy/h$（取向上为正）。

$$v\dd v = -\frac{g}{h}y\dd y$$ $$\int_{v_0}^{0} v\dd v = -\frac{g}{h}\int_0^h y\dd y$$ $$-\frac{v_0^2}{2} = -\frac{gh}{2} \implies \boxed{v_0 = \sqrt{gh}}$$
例33（初速度过大：底面离开水面后继续上升）：若初速度 $v_0 > \sqrt{gh}$，求底面离开水面后还能上升的高度。
解

在 $y = h$ 时速度 $v_h = \sqrt{v_0^2 - gh}$。此后长方体完全脱离水面，做竖直上抛：

$$\Delta h = \frac{v_h^2}{2g} = \frac{v_0^2 - gh}{2g}$$

底面离开水面后还能上升 $\dfrac{v_0^2 - gh}{2g}$。

方法要点：当力是位置的函数时，用 $v\dd v = a(y)\dd y$ 换元法最方便。分段运动问题要注意各阶段的初速度衔接。

绳绕圆柱的张力分布——Capstan 方程

例34（PPT 例1-15：缆绑方程）：轻质绳索绕在固定圆柱上，绳绕圆柱的张角为 $\theta_0$，绳与圆柱间静摩擦系数为 $\mu$。求绳处于滑动临界点时两端张力 $F_A$ 和 $F_B$ 的关系。
解

取绕在圆柱上的一段微元 $\dd s$，其张角为 $\dd\theta$，两端张力分别为 $F$ 和 $F + \dd F$。

法向方向（指向圆心，平衡）：

$$\dd N = F\sin\frac{\dd\theta}{2} + (F+\dd F)\sin\frac{\dd\theta}{2} \approx F\dd\theta$$

（小角近似 $\sin(\dd\theta/2) \approx \dd\theta/2$，忽略高阶无穷小）

切向方向（沿绳，平衡）：

$$\dd F = \dd f = \mu\dd N = \mu F\dd\theta$$

分离变量并积分：

$$\int_{F_B}^{F_A}\frac{\dd F}{F} = \mu\int_0^{\theta_0}\dd\theta$$ $$\boxed{F_A = F_B\,e^{\mu\theta_0}}$$
物理意义：张力随绕角指数增长！以 $\mu = 0.25$ 为例：
$\theta_0$$F_A/F_B$
$\pi$（半圈）$\approx 2.2$
$2\pi$（一圈）$\approx 4.8$
$10\pi$（五圈）$\approx 2\,500$

这解释了船员只需将缆绳在缆桩上缠绕数圈，就能以一人之力拉住巨大船舶。

关键思想：广延物体 → 取微元 → 对微元用牛顿第二定律 → 得到微分方程 → 积分求解。这是处理连续体力学问题的标准方法。

球在光滑球面上滑下——脱离点分析

例35（PPT补充例：球面脱离问题）：质量为 $m$ 的小球置于固定光滑圆柱体（半径 $R$）顶端，从静止开始下滑。求：(1) 小球滑到角度 $\theta$ 处时的速度和法向支持力；(2) 小球飞离圆柱体的位置。
解

在角度 $\theta$ 处，用自然坐标系：

切向：$mg\sin\theta = m\dfrac{\dd v}{\dd t}$

利用 $\dfrac{\dd v}{\dd t} = \dfrac{\dd v}{\dd\theta}\cdot\dfrac{\dd\theta}{\dd t} = \dfrac{v}{R}\dfrac{\dd v}{\dd\theta}$，得 $v\dd v = gR\sin\theta\,\dd\theta$。

积分（从 $\theta = 0,\, v = 0$）：

$$\frac{v^2}{2} = gR(1 - \cos\theta) \implies v^2 = 2gR(1 - \cos\theta)$$

法向（指向圆心）：$mg\cos\theta - N = \dfrac{mv^2}{R}$

$$N = mg\cos\theta - \frac{mv^2}{R} = mg(3\cos\theta - 2)$$

脱离条件：$N = 0$

$$3\cos\theta - 2 = 0 \implies \boxed{\cos\theta_0 = \frac{2}{3},\quad \theta_0 = \arccos\frac{2}{3} \approx 48.2°}$$

脱离时速度：$v_0 = \sqrt{\dfrac{2gR}{3}}$

考试要点：
  • 脱离条件是 $N = 0$（不是 $N < 0$），因为曲面只能施加推力不能拉
  • 脱离角 $\theta_0 = \arccos(2/3) \approx 48.2°$ 与质量 $m$ 和半径 $R$ 无关！
  • 脱离后做斜抛运动，初速 $v_0 = \sqrt{2gR/3}$，方向沿 $\theta_0$ 处的切线

锥面/漏斗壁上的匀速圆周运动——转速临界条件

例36（PPT补充例：漏斗壁上小球的临界转速）：一小物体放在绕竖直轴匀速转动的漏斗内壁上，漏斗壁与水平面成角 $\beta$，物体与壁间静摩擦系数为 $\mu_s$，物体中心距轴 $r$，每秒转 $n$ 圈。求物体不滑动的条件。
解

物体受重力 $mg$、法向支持力 $N$、静摩擦力 $f$。合力提供向心力 $m\omega^2 r$，其中 $\omega = 2\pi n$。

情况一：转速较低，物体有下滑趋势（摩擦力沿壁面向上）

水平方向：$N\sin\beta - f\cos\beta = m\omega^2 r$

竖直方向：$N\cos\beta + f\sin\beta = mg$

临界条件 $f = \mu_s N$，联立解得：

$$n_{\min} = \frac{1}{2\pi}\sqrt{\frac{g(\sin\beta - \mu_s\cos\beta)}{r(\cos\beta + \mu_s\sin\beta)}}$$

情况二：转速较高，物体有上滑趋势（摩擦力沿壁面向下）

水平方向：$N\sin\beta + f\cos\beta = m\omega^2 r$

竖直方向：$N\cos\beta - f\sin\beta = mg$

临界条件 $f = \mu_s N$，解得：

$$n_{\max} = \frac{1}{2\pi}\sqrt{\frac{g(\sin\beta + \mu_s\cos\beta)}{r(\cos\beta - \mu_s\sin\beta)}}$$

物体不滑动的条件：$n_{\min} \leq n \leq n_{\max}$

例37（PPT练习：锥面内壁匀速圆周运动）：半角为 $\alpha$ 的光滑锥面内壁，距顶点高 $h$ 处有一小球，以水平初速 $v_0$ 沿内壁射出并做匀速圆周运动。求 $v_0$。
解

小球做半径 $r = h\tan\alpha$ 的水平匀速圆周运动，受重力 $mg$ 和锥面法向力 $N$。

竖直方向：$N\cos\alpha = mg$

水平方向（向心）：$N\sin\alpha = \dfrac{mv_0^2}{r} = \dfrac{mv_0^2}{h\tan\alpha}$

两式相除：$\tan\alpha = \dfrac{v_0^2}{gh\tan\alpha}$

$$\boxed{v_0 = \sqrt{gh}\,\tan\alpha = \sqrt{gh\tan^2\alpha}}$$
拓展：若初速 $v_1 = 2v_0$（大于匀速圆周所需），小球将做螺旋上升运动。由机械能守恒与角动量分量守恒（合力矩沿轴线方向为零），可求得最大高度 $x = 3h$。这是角动量守恒的典型应用。

旋转碗内表面形状——含摩擦的推广

例38（PPT练习：有摩擦的旋转碗）：碗绕竖直轴以角速度 $\omega$ 匀速转动，小球在碗内任意位置平衡且处于下滑临界状态（$f = \mu_s N$，摩擦力向上）。求碗的内表面形状。
解

设曲面上距轴 $x$ 处的切线与水平面成角 $\alpha$，小球受重力、法向力 $N$、摩擦力 $f$。

竖直方向：$N\cos\alpha + f\sin\alpha = mg$

水平方向：$N\sin\alpha - f\cos\alpha = m\omega^2 x$

令 $f = \mu_s N$，两式相除：

$$\frac{\sin\alpha - \mu_s\cos\alpha}{\cos\alpha + \mu_s\sin\alpha} = \frac{\omega^2 x}{g}$$

而 $\tan\alpha = \dfrac{\dd y}{\dd x}$，记 $y' = \tan\alpha$，则 $\sin\alpha = \dfrac{y'}{\sqrt{1+y'^2}}$，$\cos\alpha = \dfrac{1}{\sqrt{1+y'^2}}$，代入化简得：

$$\frac{y' - \mu_s}{1 + \mu_s y'} = \frac{\omega^2 x}{g}$$

这是一个一阶 ODE，一般不能初等积分。当 $\mu_s = 0$ 时退化为 $y' = \omega^2 x/g$，积分得 $y = \dfrac{\omega^2}{2g}x^2$（旋转抛物面），与无摩擦情况一致。

对比：无摩擦时碗面为抛物面；有摩擦时碗面形状由含 $\mu_s$ 的微分方程决定，不再是简单抛物面。考试一般只考无摩擦版本，但需要知道摩擦如何改变平衡条件。

连接体约束条件的系统方法

解题步骤
  1. 隔离对象：对每个物体分别进行受力分析
  2. 列方程：对每个物体用牛顿第二定律写出分量方程
  3. 约束条件：由绳不可伸长、接触不脱离等条件建立加速度之间的关系
  4. 补充条件：摩擦力 $f = \mu N$、轻绳张力处处相等等
  5. 联立求解
例39（PPT：斜面上的连接体）：质量为 $m_1$ 的楔形块 A 放在光滑水平面上，斜面倾角 $\alpha$，上面放质量为 $m_2$ 的滑块 B。所有面光滑，求 A 的加速度 $a_1$。
解（地面系+分量法）

设 A 向左加速度为 $a_1$，B 相对 A 沿斜面加速度为 $a_{rel}$。

B 在地面系中的加速度分量（取向左、向下为正）：

  • 水平：$a_{Bx} = a_1 - a_{rel}\cos\alpha$（B随A向左，同时沿斜面分量向右）
  • 竖直：$a_{By} = a_{rel}\sin\alpha$（B沿斜面下滑的竖直分量）

A 的水平方程：$N_2\sin\alpha = m_1 a_1$ ...(1)

B 的水平方程：$-N_2\sin\alpha = m_2(a_1-a_{rel}\cos\alpha)$ ...(2)

B 的竖直方程：$m_2g - N_2\cos\alpha = m_2 a_{rel}\sin\alpha$ ...(3)

由 (1)+(2)：$0 = (m_1+m_2)a_1 - m_2 a_{rel}\cos\alpha$，即 $a_{rel} = \frac{(m_1+m_2)a_1}{m_2\cos\alpha}$ ...(4)

由 (1)：$N_2 = m_1 a_1/\sin\alpha$。代入 (3)：

$$m_2 g - \frac{m_1 a_1\cos\alpha}{\sin\alpha} = m_2\sin\alpha\cdot\frac{(m_1+m_2)a_1}{m_2\cos\alpha}$$ $$m_2 g = \frac{m_1 a_1\cos\alpha}{\sin\alpha} + \frac{(m_1+m_2)a_1\sin\alpha}{\cos\alpha}$$ $$m_2 g = a_1\left[\frac{m_1\cos^2\alpha + (m_1+m_2)\sin^2\alpha}{\sin\alpha\cos\alpha}\right] = a_1\cdot\frac{m_1 + m_2\sin^2\alpha}{\sin\alpha\cos\alpha}$$ $$\boxed{a_1 = \frac{m_2 g\sin\alpha\cos\alpha}{m_1 + m_2\sin^2\alpha}}$$

验证：$\alpha\to 0$（水平面）：$a_1\to 0$ ✓；$\alpha\to\pi/2$（竖直面）：$a_1 = 0$（B自由落体，不推A）✓；$m_1\to\infty$：$a_1\to 0$（A太重不动）✓。

例40（PPT：竖直弹簧振子）：竖直弹簧振子（劲度系数 $k$、质量 $m$），自然平衡后用恒力 $F$ 向下拉 $S$ 后撤去力，以撤力瞬间为 $t = 0$。已知 $k = 200\,\text{N/m}$，$m = 4.0\,\text{kg}$，$F = 100\,\text{N}$，$S = 0.5\,\text{m}$。求振动方程。
解

以平衡位置为原点，向下为正。圆频率：

$$\omega = \sqrt{\frac{k}{m}} = \sqrt{\frac{200}{4}} = 7.07\,\text{rad/s}$$

$t = 0$ 时刻：$x_0 = S - F/k = 0.5 - 0.5 = 0\,\text{m}$（恰好在平衡位置）。

向下拉的过程中，由功能原理求振动振幅：

$$FS = \frac{1}{2}kA^2 \implies A = \sqrt{\frac{2FS}{k}} = \sqrt{\frac{2\times100\times0.5}{200}} = \frac{\sqrt{2}}{2} \approx 0.707\,\text{m}$$

$t = 0$ 时 $x_0 = 0$ 且速度向上（$v_0 < 0$）。由 $x = A\cos(\omega t + \varphi_0)$：$x_0 = A\cos\varphi_0 = 0$ 给出 $\varphi_0 = \pm\pi/2$；$v_0 = -A\omega\sin\varphi_0 < 0$ 要求 $\sin\varphi_0 > 0$，故 $\varphi_0 = \pi/2$。

$$\boxed{x = 0.707\cos\!\left(7.07t + \frac{\pi}{2}\right)\;\text{m}}$$
竖直弹簧振子要点：
  • 以平衡位置为原点，不是弹簧原长位置。这样重力与弹簧弹力的静态部分抵消，运动方程与水平弹簧振子形式完全相同
  • 恒力作用下静态位移 $\Delta = F/k$，撤力后从新的位置开始谐振动
  • $t = 0$ 的位移和速度共同确定振幅 $A$ 和初相 $\varphi_0$
