第1章 质点运动学

1.1 位置矢量与位移

参考系与坐标系

描述运动必须选定参考系（Reference Frame）。同一运动在不同参考系中描述不同。通常选地面为参考系。

在参考系上建立坐标系来定量描述位置：直角坐标系 $(x,y,z)$、极坐标 $(r,\theta)$、自然坐标系（切向-法向）。

位置矢量（位矢）

位矢的定义

质点在空间中的位置用位置矢量描述：

$$\vv{r} = x\ii + y\jj + z\kk$$

大小：$r = |\vv{r}| = \sqrt{x^2+y^2+z^2}$

当坐标 $x=x(t),\,y=y(t),\,z=z(t)$ 为时间函数时，称为运动方程（参数方程形式）。消去 $t$ 得轨迹方程。

位移与路程

位移 vs 路程

位移（矢量）：$\Delta\vv{r} = \vv{r}_2 - \vv{r}_1 = \Delta x\,\ii + \Delta y\,\jj + \Delta z\,\kk$

位移大小：$|\Delta\vv{r}| = \sqrt{(\Delta x)^2+(\Delta y)^2+(\Delta z)^2}$

路程（标量）：$\Delta s = \int |\dd\vv{r}|$ —— 沿轨迹的弧长

易错：$|\Delta\vv{r}| \neq \Delta s$（除非直线运动且方向不变）。位移是"起点到终点的有向线段"，路程是"走过的总弧长"。

1.2 速度与加速度

速度

速度的定义

平均速度：$\bar{\vv{v}} = \frac{\Delta\vv{r}}{\Delta t}$（矢量，方向沿位移方向）

瞬时速度：$\vv{v} = \frac{\dd\vv{r}}{\dd t} = \dot{x}\,\ii + \dot{y}\,\jj + \dot{z}\,\kk$（矢量，方向沿轨迹切线）

速率（标量）：$v = |\vv{v}| = \sqrt{\dot{x}^2+\dot{y}^2+\dot{z}^2} = \frac{\dd s}{\dd t}$

加速度

加速度的定义

瞬时加速度：$\vv{a} = \frac{\dd\vv{v}}{\dd t} = \frac{\dd^2\vv{r}}{\dd t^2} = \ddot{x}\,\ii + \ddot{y}\,\jj + \ddot{z}\,\kk$

例1（运动方程→速度→加速度）：质点运动方程 $x=2t,\;y=4t-t^2$（SI），求 $t=1\,\text{s}$ 时速度和加速度。
解

$v_x = \dot{x} = 2\,\text{m/s}$，$v_y = \dot{y} = 4-2t$

$t=1$ 时：$v_x=2,\;v_y=2$，$v=\sqrt{4+4}=2\sqrt{2}\,\text{m/s}$

$a_x = \ddot{x} = 0$，$a_y = \ddot{y} = -2\,\text{m/s}^2$

加速度 $\vv{a} = -2\jj\,\text{m/s}^2$，方向沿 $y$ 轴负方向。

1.3 运动学两类基本问题

第一类：已知运动方程，求速度、加速度

方法：求导 $\vv{r}(t) \xrightarrow{\dd/\dd t} \vv{v}(t) \xrightarrow{\dd/\dd t} \vv{a}(t)$

第二类：已知加速度（力），求速度、运动方程

方法：积分（需要初始条件）

$$\vv{v}(t) = \vv{v}_0 + \int_0^t \vv{a}(t')\dd t'$$ $$\vv{r}(t) = \vv{r}_0 + \int_0^t \vv{v}(t')\dd t'$$
技巧：当加速度是速度或位置的函数时，需要换元：
  • $a = a(v)$：分离变量 $\frac{\dd v}{a(v)} = \dd t$
  • $a = a(x)$：利用 $a = v\frac{\dd v}{\dd x}$，则 $v\dd v = a(x)\dd x$
例2（积分法）：质点沿 $x$ 轴运动，加速度 $a=2+6t$（SI），$t=0$ 时 $x_0=0,\;v_0=0$，求 $v(t)$ 和 $x(t)$。
解

$v = \int_0^t(2+6t')\dd t' = 2t+3t^2$

$x = \int_0^t(2t'+3t'^2)\dd t' = t^2+t^3$

1.4 抛体运动

斜抛运动（初速 $v_0$，仰角 $\theta$）

分量方程（取发射点为原点，向上为 $y$ 正方向）：

$$x = v_0\cos\theta\cdot t,\quad y = v_0\sin\theta\cdot t - \tfrac{1}{2}gt^2$$ $$v_x = v_0\cos\theta = \text{const},\quad v_y = v_0\sin\theta - gt$$

消去 $t$ 得轨迹方程：

$$y = x\tan\theta - \frac{g}{2v_0^2\cos^2\theta}x^2$$

最大高度（令 $v_y=0$）：$H = \frac{v_0^2\sin^2\theta}{2g}$

水平射程（令 $y=0$）：$R = \frac{v_0^2\sin 2\theta}{g}$

结论：$\theta=45°$ 时射程最大 $R_{max}=v_0^2/g$；互补角 $\theta$ 和 $90°-\theta$ 射程相同。

1.5 圆周运动

角量描述

角量的定义

角位移：$\theta$（rad）

角速度：$\omega = \frac{\dd\theta}{\dd t}$（rad/s）

角加速度：$\alpha = \frac{\dd\omega}{\dd t} = \frac{\dd^2\theta}{\dd t^2}$（rad/s²）

线量与角量的关系

线量-角量关系（半径 $R$）
$$s = R\theta,\quad v = R\omega,\quad a_\tau = R\alpha,\quad a_n = R\omega^2 = \frac{v^2}{R}$$

匀变速圆周运动

当 $\alpha = \text{const}$ 时（类比匀变速直线运动）：

直线运动圆周运动
$v = v_0 + at$$\omega = \omega_0 + \alpha t$
$x = v_0 t + \frac{1}{2}at^2$$\theta = \omega_0 t + \frac{1}{2}\alpha t^2$
$v^2 = v_0^2 + 2ax$$\omega^2 = \omega_0^2 + 2\alpha\theta$
例3（圆周运动）：飞轮从静止开始以 $\alpha=2\,\text{rad/s}^2$ 匀角加速转动，半径 $R=0.5\,\text{m}$，求 $t=3\,\text{s}$ 时轮缘上点的切向和法向加速度。
解

$\omega = \alpha t = 2\times3=6\,\text{rad/s}$

$a_\tau = R\alpha = 0.5\times2 = 1\,\text{m/s}^2$

$a_n = R\omega^2 = 0.5\times36 = 18\,\text{m/s}^2$

总加速度 $a = \sqrt{a_\tau^2+a_n^2} = \sqrt{1+324} \approx 18.03\,\text{m/s}^2$

1.6 自然坐标系（切向-法向分解）

一般曲线运动的加速度分解

在轨迹上每一点建立切向 $\ee_\tau$ 和法向 $\ee_n$ 单位矢量：

$$\vv{a} = a_\tau\ee_\tau + a_n\ee_n$$

其中：

  • 切向加速度：$a_\tau = \frac{\dd v}{\dd t}$ —— 改变速度大小
  • 法向加速度：$a_n = \frac{v^2}{\rho}$ —— 改变速度方向，$\rho$ 为曲率半径

总加速度大小：$a = \sqrt{a_\tau^2 + a_n^2}$

高频考点：求曲率半径 $\rho$！
  • 若已知 $v$ 和 $a_n$：$\rho = v^2/a_n$
  • 若已知轨迹 $y=f(x)$：$\rho = \frac{[1+(y')^2]^{3/2}}{|y''|}$
  • 若已知 $v$ 和 $a$：先求 $a_\tau = \dd v/\dd t$，再由 $a_n = \sqrt{a^2-a_\tau^2}$，最后 $\rho = v^2/a_n$
例4（2022期中真题改编）：质量 $m$ 的飞机飞行曲线 $y=Ax^3$（$A$ 为常数），$x$ 方向速度 $v_x$ 恒定，求 $y$ 方向合外力。
解

$y = Ax^3$，$\dot{y} = 3Ax^2\dot{x} = 3Ax^2 v_x$

$\ddot{y} = 6Ax\dot{x}\cdot v_x = 6Axv_x^2$

$F_y = m\ddot{y} = 6mAxv_x^2$

例5（曲率半径）：抛体运动中，在最高点的曲率半径是多少？
解

最高点 $v_y=0$，故 $v = v_x = v_0\cos\theta$，法向加速度 $a_n = g$（此时速度水平，重力提供全部法向加速度）。

$$\rho = \frac{v^2}{a_n} = \frac{v_0^2\cos^2\theta}{g}$$

1.7 相对运动

经典（Galilei）速度叠加

设 $S'$ 系相对 $S$ 系以 $\vv{u}$ 运动，质点相对 $S$ 系速度为 $\vv{v}$，相对 $S'$ 系速度为 $\vv{v}'$：

$$\vv{v} = \vv{v}' + \vv{u}$$

即：绝对速度 = 相对速度 + 牵连速度

当 $\vv{u}=\text{const}$ 时，$\vv{a} = \vv{a}'$（加速度在惯性系间相同）

例6：船以 $v_1=4\,\text{m/s}$ 正东方向过河，水流 $v_2=3\,\text{m/s}$ 向正南，求船对地速度。
解

$v = \sqrt{v_1^2+v_2^2} = \sqrt{16+9} = 5\,\text{m/s}$，方向偏南 $\alpha = \arctan(3/4) \approx 36.9°$。

1.8 极坐标描述

平面极坐标 $(r,\theta)$

位矢：$\vv{r}=r\ee_r$

速度：$\vv{v}=\dot{r}\ee_r+r\dot{\theta}\ee_\theta$

  • $v_r=\dot{r}$：径向速度（远离/靠近原点）
  • $v_\theta=r\dot{\theta}=r\omega$：横向速度（绕原点旋转）

加速度：$\vv{a}=(\ddot{r}-r\dot{\theta}^2)\ee_r+(r\ddot{\theta}+2\dot{r}\dot{\theta})\ee_\theta$

圆周运动特例：$r=R=\text{const}$，则 $\dot{r}=\ddot{r}=0$，$a_r=-R\omega^2$（向心），$a_\theta=R\alpha$（切向）。

1.9 重要二级结论与技巧

匀变速直线运动公式汇总
  1. $v=v_0+at$
  2. $x=v_0t+\frac{1}{2}at^2$
  3. $v^2=v_0^2+2ax$
  4. $x=\frac{v_0+v}{2}\cdot t$（中间时刻速度 = 平均速度）
  5. $x_n-x_{n-1}=aT^2$（连续等时间间隔位移之差为恒量）
竖直平面内曲线运动分析技巧

对光滑曲面/钢丝约束的质点，使用自然坐标系：

沿曲面运动满足：$v\dfrac{\dd v}{\dd s}=a_\tau$ → 可改写为 $v\dd v=a_\tau\dd s$

对竖直面上的运动：$a_\tau=-g\sin\theta$，而 $\dd s\sin\theta=\dd y$，故：

$$v\dd v=-g\dd y \implies v^2=v_0^2-2g(y-y_0)$$

这与能量守恒一致！说明在光滑曲面上，速度只取决于高度差，与轨迹形状无关。

抛体运动中任意时刻的曲率半径

任意时刻速度方向与水平夹角 $\phi$，则：

$a_n=g\cos\phi$（加速度在法向的分量）

$$\rho=\frac{v^2}{g\cos\phi}$$

其中 $v=\sqrt{v_x^2+v_y^2}$，$\cos\phi=v_x/v=v_0\cos\theta/v$

解

设顶面距水面为 $y$（$0

例8（PPT拓展：最速降线问题）：两定点A、B高度不同，所有连接曲线中哪条使质点从A到B时间最短？
解

答案是摆线（旋轮线），不是直线也不是圆弧！这是变分法的经典问题（Johann Bernoulli, 1696）。

下落到高度 $y$ 时速率 $v=\sqrt{2gy}$，沿曲线元 $\dd s$的时间 $\dd t=\dd s/v$。

$$t=\int_A^B\frac{\sqrt{1+y'^2}}{\sqrt{2gy}}\dd x$$

使此积分最小的曲线就是摆线。这说明"最短"和"最快"不是一回事。

1.10 运动学综合题型（PPT高频）

例9（轨迹方程求法）：质点运动方程 $x=R\cos\omega t$，$y=R\sin\omega t$，求轨迹方程。
解

$\cos\omega t=x/R$，$\sin\omega t=y/R$

$\cos^2\omega t+\sin^2\omega t=1$ → $x^2+y^2=R^2$（圆，半径 $R$）

速度大小：$v=\sqrt{\dot{x}^2+\dot{y}^2}=R\omega$（匀速圆周运动）

例10（加速度为速度函数）：质点沿直线运动，$a=kv$（$k<0$为常数），初速 $v_0$。求 $v(x)$。
解

利用 $a=v\frac{\dd v}{\dd x}$：$v\frac{\dd v}{\dd x}=kv$ → $\frac{\dd v}{\dd x}=k$

积分：$v-v_0=kx$ → $v=v_0+kx$

停止位置（$v=0$）：$x=-v_0/k$

运动学常见换元技巧总结
已知关系换元方法
$a=f(t)$$v=v_0+\int_0^t a\dd t'$，$x=x_0+\int_0^t v\dd t'$
$a=f(v)$$\frac{\dd v}{f(v)}=\dd t$ 或 $v\frac{\dd v}{f(v)}=\dd x$
$a=f(x)$$v\dd v=f(x)\dd x$
期中/期末高频考法：
  • 给运动方程求 $t$ 时刻的速度方向角、加速度方向角
  • 已知加速度分量（含 $v$ 或 $x$），列微分方程求解
  • 圆周运动求某时刻 $a_n$、$a_t$、$a$ 和 $\rho$

1.11 运动学图形分析（PPT重点）

x-t 图（位移-时间图）

x-t 图的物理意义
  • 斜率 = 速度：直线的斜率为匀速运动的速度 $v = \Delta x / \Delta t$
  • 切线斜率 = 瞬时速度：曲线上某点切线的斜率即为该时刻的瞬时速度 $v = \dd x / \dd t$
  • 斜率为正 → 沿正方向运动；斜率为负 → 沿负方向运动；斜率为零 → 瞬时静止（极值点）
  • 斜率越大（越陡）→ 速度越大；曲线弯曲 → 做变速运动

常见图形：

图形形状运动类型
水平线静止（$v=0$）
过原点的直线（斜率 $k$）匀速运动（$v=k$）
抛物线（开口向上）匀加速运动（$x = x_0 + v_0 t + \frac{1}{2}at^2$）
抛物线（开口向下）匀减速运动

v-t 图（速度-时间图）

v-t 图的物理意义
  • 斜率 = 加速度：$a = \dd v / \dd t$，切线斜率即瞬时加速度
  • 图线与时间轴围成的面积 = 位移：$\Delta x = \int_{t_1}^{t_2} v\,\dd t$
  • $v>0$ 部分面积为正位移，$v<0$ 部分面积为负位移
  • 面积在时间轴上方为正，下方为负，总位移 = 代数和

常见图形：

图形形状运动类型加速度
水平线匀速运动$a=0$
倾斜直线匀加速/匀减速$a=\text{const}$
曲线（斜率递减）加速度逐渐减小$a$ 递减
区分位移与路程：v-t 图中面积的代数和是位移，面积的绝对值之和是路程。当速度改变方向（$v$ 穿越零轴）时两者不同。

a-t 图（加速度-时间图）

a-t 图的物理意义
  • 图线与时间轴围成的面积 = 速度变化量：$\Delta v = \int_{t_1}^{t_2} a\,\dd t$
  • 面积为正 → 速度增加；面积为负 → 速度减小
  • 已知 $v_0$，可由面积逐段累加得到 $v(t)$
三种图形的对应关系总结
图形斜率的意义面积的意义
x-t 图速度 $v$（无直接物理意义）
v-t 图加速度 $a$位移 $\Delta x$
a-t 图（加速度的变化率）速度变化量 $\Delta v$
例11（v-t图分析：分段线性）：质点沿直线运动，其 v-t 图如下描述：$0\sim2\,\text{s}$ 内速度从 $0$ 线性增到 $4\,\text{m/s}$；$2\sim5\,\text{s}$ 内速度保持 $4\,\text{m/s}$；$5\sim7\,\text{s}$ 内速度从 $4\,\text{m/s}$ 线性减到 $0$。求：(1) 各段加速度；(2) $0\sim7\,\text{s}$ 总位移。
解

(1) 各段加速度（v-t 图斜率）：

  • $0\sim2\,\text{s}$：$a_1 = \dfrac{4-0}{2-0} = 2\,\text{m/s}^2$（匀加速）
  • $2\sim5\,\text{s}$：$a_2 = 0$（匀速）
  • $5\sim7\,\text{s}$：$a_3 = \dfrac{0-4}{7-5} = -2\,\text{m/s}^2$（匀减速）

(2) 总位移（v-t 图下面积）：

$$x = \underbrace{\frac{1}{2}\times2\times4}_{三角形} + \underbrace{3\times4}_{矩形} + \underbrace{\frac{1}{2}\times2\times4}_{三角形} = 4 + 12 + 4 = 20\,\text{m}$$

也可以看成一个梯形：$x = \frac{1}{2}(2+7-0)\times4$...不对，用分段法更清晰。

整体看，图形为梯形（上底 $= 5-2 = 3\,\text{s}$，下底 $= 7\,\text{s}$，高 $= 4\,\text{m/s}$）：

$$x = \frac{1}{2}(3+7)\times 4 = 20\,\text{m} \checkmark$$
考试陷阱：
  • v-t 图中速度为负的部分，面积要取负值计算位移，但取绝对值计算路程
  • x-t 图不是轨迹图！x-t 图是一维运动的位置随时间变化，不表示二维空间的运动轨迹
  • v-t 图中两条曲线交点表示两物体速度相等，不一定表示相遇

1.12 追及与相遇问题

追及问题的核心思路

追及问题分析方法

追及条件：后方物体的位移 = 前方物体的位移 + 初始间距

$$x_{追} = x_{被追} + d_0$$

其中 $d_0$ 为初始距离。

关键时刻：当两者速度相等时，两者距离最小或最大：

  • 若追者速度从大于被追者到等于被追者 → 此时距离最小（之后追者变慢，距离拉大）
  • 若追者速度从小于被追者到等于被追者 → 此时距离最大（之后追者变快，开始追上）
判断能否追上的方法
  1. 令 $v_{追} = v_{被追}$，求出此临界时刻 $t^*$
  2. 计算 $t^*$ 时刻两者的距离差 $\Delta x(t^*)$
  3. 若 $\Delta x(t^*) \leq 0$（追者已到达或超过被追者位置）→ 能追上，在 $t^*$ 之前已追上
  4. 若 $\Delta x(t^*) > 0$（追者仍在后方）→ 追不上，$t^*$ 时刻距离最近

相遇问题的核心思路

相遇条件

同一时刻 $t$，两物体到达同一位置：$x_1(t) = x_2(t)$

对于相向运动的两物体，相遇条件为：

$$x_1(t) + x_2(t) = d_0$$

其中 $d_0$ 为初始距离，$x_1,x_2$ 分别为各自出发点的位移。

解题策略：
  • 方程法：分别写出两物体运动方程，令位移满足追及/相遇条件，联立求解
  • 图形法：画两物体的 x-t 图，交点即为相遇时刻和位置；画 v-t 图，面积差等于初始间距时即追上
  • 相对运动法：以一个物体为参考系，分析另一物体的相对运动
例12（经典追及问题）：A车以恒定速度 $v_0$ 做匀速直线运动。B车从A车后方距离 $d$ 处由静止出发，以加速度 $a$ 做匀加速直线运动追赶A车。求：(1) B追上A需要多长时间？(2) 追上前两车最大距离是多少？
解

设 $t=0$ 时B车出发，取B车出发点为原点，正方向为运动方向。

A车位置：$x_A = d + v_0 t$

B车位置：$x_B = \frac{1}{2}at^2$

(1) 追上条件：$x_B = x_A$

$$\frac{1}{2}at^2 = d + v_0 t$$ $$\frac{1}{2}at^2 - v_0 t - d = 0$$ $$t = \frac{v_0 + \sqrt{v_0^2 + 2ad}}{a}$$

（取正根）

(2) 最大距离：两车速度相等时距离最大

$$v_B = v_A \implies at^* = v_0 \implies t^* = \frac{v_0}{a}$$

此时距离差：

$$\Delta x = x_A(t^*) - x_B(t^*) = \left(d + v_0\cdot\frac{v_0}{a}\right) - \frac{1}{2}a\left(\frac{v_0}{a}\right)^2 = d + \frac{v_0^2}{2a}$$
例13（相向运动相遇）：甲、乙两人相距 $L=100\,\text{m}$，甲以 $v_1=3\,\text{m/s}$ 匀速向乙运动，同时乙从静止以 $a=2\,\text{m/s}^2$ 匀加速向甲运动。求何时相遇？
解

取甲的出发位置为原点，向乙方向为正。

甲的位置：$x_1 = v_1 t = 3t$

乙的位置：$x_2 = L - \frac{1}{2}at^2 = 100 - t^2$

相遇条件：$x_1 = x_2$

$$3t = 100 - t^2 \implies t^2 + 3t - 100 = 0$$ $$t = \frac{-3 + \sqrt{9+400}}{2} = \frac{-3+\sqrt{409}}{2} \approx \frac{-3+20.22}{2} \approx 8.6\,\text{s}$$

相遇位置：$x = 3\times8.6 \approx 25.8\,\text{m}$（距甲出发点）。

例14（追及临界问题）：警车从静止以 $a=2.5\,\text{m/s}^2$ 加速追赶前方 $d=50\,\text{m}$ 处以 $v_0=10\,\text{m/s}$ 匀速行驶的汽车。但警车最高速度为 $v_m=20\,\text{m/s}$。求追上所需时间。
解

阶段分析：警车先匀加速到 $v_m$，然后匀速。

加速阶段持续时间：$t_1 = v_m/a = 20/2.5 = 8\,\text{s}$

$t_1$ 时刻位移：警车 $x_B = \frac{1}{2}\times2.5\times64 = 80\,\text{m}$；汽车 $x_A = 50 + 10\times8 = 130\,\text{m}$

距离差：$130 - 80 = 50\,\text{m}$（仍未追上）

$t > t_1$ 后，警车以 $v_m = 20\,\text{m/s}$ 匀速，汽车以 $v_0 = 10\,\text{m/s}$ 匀速，相对速度 $\Delta v = 10\,\text{m/s}$。

剩余距离追赶时间：$t_2 = 50/10 = 5\,\text{s}$

总时间：$t = t_1 + t_2 = 8 + 5 = 13\,\text{s}$

追及问题常见陷阱：
  • 有最大速度限制时，不能一直用匀加速公式，需分段讨论
  • 若 $v_0^2 + 2ad < 0$（判别式为负），说明追不上（但本题中 $d > 0$ 时判别式总为正）
  • 二次方程两个根：较小的正根为第一次追上（从后面追上），较大正根为第二次相遇（追过后减速再被追上的情况）
