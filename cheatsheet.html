📖 目录公式总表备考指南

考前抢分：二级结论 & 易错清单

一、力学速算二级结论

1. 匀变速运动的位移-速度公式

$v^2 - v_0^2 = 2a\Delta x$（不含时间 $t$，快速求解）

中间时刻速度 = 平均速度：$v_{t/2} = \bar{v} = \frac{v_0+v}{2}$

中间位置速度：$v_{x/2} = \sqrt{\frac{v_0^2+v^2}{2}} > v_{t/2}$

2. 抛体运动速算

斜抛最高点：$v_y=0$，$v=v_0\cos\theta$

斜抛射程：$R = \frac{v_0^2\sin2\theta}{g}$，$\theta=45°$ 时最大

任意点曲率半径：$\rho = \frac{v^2}{a_n}$，最高点 $\rho = \frac{v_0^2\cos^2\theta}{g}$

3. 圆周运动速算

匀速圆周：$a_n = \frac{v^2}{R} = \omega^2 R$，$a_t=0$

变速圆周：$a = \sqrt{a_t^2+a_n^2}$

竖直圆周最高点不脱离条件：$v_{top}^2 \geq gR$

4. 弹性碰撞公式（直接用！）
$$v_1' = \frac{m_1-m_2}{m_1+m_2}v_1,\quad v_2' = \frac{2m_1}{m_1+m_2}v_1$$

特殊情况：

  • $m_1=m_2$：交换速度（$v_1'=0,\,v_2'=v_1$）
  • $m_1\gg m_2$：$v_1'\approx v_1,\,v_2'\approx 2v_1$
  • $m_1\ll m_2$：$v_1'\approx -v_1$（原速弹回）
5. 完全非弹性碰撞能量损失
$$\Delta E_k = \frac{1}{2}\frac{m_1m_2}{m_1+m_2}(v_1-v_2)^2 = \frac{1}{2}\mu v_{rel}^2$$

$\mu = \frac{m_1m_2}{m_1+m_2}$ 为约化质量。

6. 万有引力与轨道速度

第一宇宙速度：$v_1 = \sqrt{gR} \approx 7.9\,\text{km/s}$

圆轨道：$\frac{1}{2}mv^2 = \frac{1}{2}\frac{GMm}{r}$（动能=势能绝对值的一半）

二、刚体力学速算

7. 滚动速度比较（同一斜面同一高度）

$v = \sqrt{\frac{2gh}{1+J/(MR^2)}}$

速度排序（快→慢）：滑动 > 实心球($\frac{2}{5}$) > 圆盘($\frac{1}{2}$) > 球壳($\frac{2}{3}$) > 圆环(1)

括号内为 $J/(MR^2)$，越大越慢。

8. 杆的释放问题

水平均匀杆从水平释放（绕端点）转过 $\theta$ 时：

$\omega = \sqrt{\frac{3g\sin\theta}{L}}$，初始角加速度 $\alpha_0 = \frac{3g}{2L}$

端点加速度 $a_{端} = L\alpha_0 = \frac{3}{2}g > g$！（比自由落体快）

三、相对论速算

9. 动能为静能 $n$ 倍时的公式

$E_k = nm_0c^2$ → $\gamma = n+1$ → $v = c\sqrt{1-\frac{1}{(n+1)^2}}$

$p = m_0c\sqrt{n^2+2n}$

10. 低速近似与极端相对论

低速（$v\ll c$）：$\gamma\approx 1+\frac{\beta^2}{2}$，$E_k\approx\frac{1}{2}m_0v^2$

极端相对论（$v\to c$）：$E\approx pc$，$\gamma\gg1$

四、振动与波速算

11. 弹簧振子周期（不受重力/浮力影响）

$T = 2\pi\sqrt{m/k}$，与 $g$、平衡位置无关

弹簧串联：$1/k = 1/k_1 + 1/k_2$；并联：$k = k_1+k_2$

12. 各种摆的周期

单摆：$T = 2\pi\sqrt{L/g}$

复摆：$T = 2\pi\sqrt{J/(mgh)}$（$h$ 为悬点到质心距离）

13. 初相位快速判断（旋转矢量法）

$x_0>0$ 且 $v_0>0$ → $\varphi_0$ 在第四象限（$-\pi/2<\varphi_0<0$）

$x_0>0$ 且 $v_0<0$ → $\varphi_0$ 在第一象限（$0<\varphi_0<\pi/2$）

$x_0<0$ 且 $v_0>0$ → $\varphi_0$ 在第三象限（$-\pi<\varphi_0<-\pi/2$）

$x_0<0$ 且 $v_0<0$ → $\varphi_0$ 在第二象限（$\pi/2<\varphi_0<\pi$）

口诀：$x$ 看 $\cos$ 正负定左右，$v$ 看 $\sin$ 正负定上下（$v=-A\omega\sin$，$v>0$ 时 $\sin<0$ → 下半）

14. 波动方程速写

已知原点处 $y(0,t) = A\cos(\omega t+\varphi_0)$，波沿 $+x$ 传播：

$$y(x,t) = A\cos\left[\omega\left(t-\frac{x}{u}\right)+\varphi_0\right]$$

沿 $-x$ 传播：将 $-x/u$ 改为 $+x/u$

15. 驻波与半波损失

固定端反射：半波损失（相位突变 $\pi$）→ 反射点为节点

自由端反射：无半波损失 → 反射点为波腹

驻波相邻节点间距 = $\lambda/2$，节点与相邻波腹间距 = $\lambda/4$

五、热学速算

16. 三种速率的系数记忆

$v_p:\bar{v}:v_{rms} = \sqrt{2}:\sqrt{8/\pi}:\sqrt{3} = 1.41:1.60:1.73$

口诀："141、160、173"，依次增大。

17. 四种过程 $W$、$Q$、$\Delta E$ 速查
过程$W$$Q$$\Delta E=\nu C_V\Delta T$
等容0$\nu C_V\Delta T$$=Q$
等压$p\Delta V=\nu R\Delta T$$\nu C_p\Delta T$$Q-W$
等温$\nu RT\ln\frac{V_2}{V_1}$$=W$0
绝热$-\Delta E=\frac{p_1V_1-p_2V_2}{\gamma-1}$0$-W$
18. 熵变万能公式
$$\Delta S = \nu C_V\ln\frac{T_2}{T_1} + \nu R\ln\frac{V_2}{V_1}$$

或等价地：$\Delta S = \nu C_p\ln\frac{T_2}{T_1} - \nu R\ln\frac{p_2}{p_1}$

只与初末状态有关，任何过程（包括不可逆过程）都适用！

19. 卡诺循环效率

$\eta_{Carnot} = 1-T_2/T_1$（$T_1$ 高温，$T_2$ 低温，用绝对温度！）

制冷系数：$w = T_2/(T_1-T_2)$

卡诺定理：一切工作于 $T_1,T_2$ 间的热机 $\eta\leq\eta_{Carnot}$

六、致命易错清单

1. 角量与线量混用
$v=R\omega$、$a_t=R\alpha$、$a_n=R\omega^2$ 中的 $R$ 是到转轴的距离，不是物体半径！
2. 有质量滑轮两侧张力不等
无质量滑轮 $T_1=T_2$；有质量滑轮 $T_1\neq T_2$，差值提供转动力矩。
3. 纯滚动的摩擦力是静摩擦
纯滚动接触点无相对滑动，摩擦力为静摩擦力（大小由运动方程确定，不用 $\mu N$）。
4. 平行轴定理只能从质心轴出发
$J = J_C + Md^2$ 中 $J_C$ 必须是过质心的轴！不能从任意轴推另一个轴。
5. 固有量的判断
固有时间（原时）：两事件在同一地点发生的参考系。
固有长度（原长）：物体静止的参考系。
原时最短，原长最长！
6. 相对论动量用 $\gamma m_0 v$，不是 $m_0v$
动能 $E_k=(\gamma-1)m_0c^2$，不是 $\frac{1}{2}m_0v^2$！只有低速才近似。
7. 等温过程 $\Delta E=0$（理想气体）
理想气体内能只是温度的函数，等温→内能不变→$Q=W$。
8. 绝热自由膨胀不是绝热过程！
自由膨胀：$W=0$，$Q=0$→$\Delta E=0$→$\Delta T=0$。
不能用 $pV^\gamma=C$ 或绝热方程！因为这是不可逆过程。
9. 熵变计算必须用可逆路径
不可逆过程的熵变 = 设计一条连接相同初末态的可逆路径，沿可逆路径积分 $\int\dd Q/T$。
10. $\bar{\varepsilon}_k=\frac{3}{2}kT$ 是平动动能，不是总动能
温度只量度平动动能。分子总平均动能 $\bar{\varepsilon}=\frac{i}{2}kT$（$i$ 含转动自由度）。
11. $f(v)$ 是概率密度，不是概率
$f(v)$ 的量纲是 s/m。$f(v)\dd v$ 才是速率在 $v\sim v+\dd v$ 的分子比率。
12. 角动量守恒条件：合力矩为零
有外力不代表角动量不守恒——关键是外力对所选参考点的力矩是否为零。
13. 功是标量，但有正负
力与位移同向→正功；反向→负功。势能增加→保守力做负功。
14. 多普勒效应公式中谁是分子谁是分母
$$\nu' = \nu\frac{u\pm v_{obs}}{u\mp v_{src}}$$

口诀：接近频率升高。观察者接近→分子取 $+$；源接近→分母取 $-$。
