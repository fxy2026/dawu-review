第3章 动量与角动量

3.1 冲量与动量定理

动量与冲量

动量：$\vv{p} = m\vv{v}$

冲量：$\vv{I} = \int_{t_1}^{t_2}\vv{F}\dd t$

质点动量定理：$\vv{I} = \Delta\vv{p} = m\vv{v}_2 - m\vv{v}_1$

微分形式：$\vv{F} = \frac{\dd\vv{p}}{\dd t}$（牛顿第二定律的原始形式）

3.2 动量守恒定律

质点系动量定理与动量守恒

质点系动量定理：$\vv{F}_{外合} = \frac{\dd\vv{p}_{总}}{\dd t}$（内力不影响总动量）

动量守恒条件：$\vv{F}_{外合} = 0 \implies \vv{p}_{总} = \text{const}$

分量形式：若某方向合外力为零，则该方向动量守恒。

常见动量守恒情形：碰撞、爆炸（内力远大于外力的短过程）、光滑面上的系统、反冲。

3.3 质心与质心运动定理

质心

离散质点系：$\vv{r}_C = \frac{\sum m_i\vv{r}_i}{\sum m_i} = \frac{\sum m_i\vv{r}_i}{M}$

连续体：$\vv{r}_C = \frac{\int\vv{r}\dd m}{M}$

质心速度：$\vv{v}_C = \frac{\sum m_i\vv{v}_i}{M} = \frac{\vv{p}_{总}}{M}$

质心运动定理：$\vv{F}_{外合} = M\vv{a}_C$

例1（真题高频：弹簧-墙-双木块）：两个质量 $m$ 的木块用弹簧（$k$）连接，$A$ 靠墙，推 $B$ 压缩弹簧 $x_0$ 后释放。求：(1) 系统质心最大速度和最大加速度；(2) 弹簧最大伸长量。
解

(1) 释放瞬间弹簧弹力 $F=kx_0$，$A$ 受墙壁支撑不动，系统所受外力为墙壁支撑力 $N=kx_0$

质心加速度最大值：$a_{C,max} = \frac{kx_0}{2m}$

$A$ 与墙分离时（弹簧恢复自然长度），$B$ 速度最大为 $v_B$：$\frac{1}{2}kx_0^2 = \frac{1}{2}mv_B^2$，$v_B = x_0\sqrt{k/m}$

此后系统无外力，质心速度恒定：$v_C = \frac{mv_B}{2m} = \frac{x_0}{2}\sqrt{\frac{k}{m}}$

(2) 弹簧最大伸长时两物体共速 $v_C$，由能量守恒：$\frac{1}{2}(2m)v_C^2 = \frac{1}{2}kx_{max}^2 + \frac{1}{2}(2m)v_C^2$？不对——

正确方法：在质心系中，弹簧最大伸长时两者相对静止。弹性势能在 $A$ 离墙后守恒：$\frac{1}{2}mv_B^2 = \frac{1}{2}(2m)v_C^2 + \frac{1}{2}kx_{max}^2$

$\frac{1}{2}kx_0^2 = \frac{1}{2}(2m)\frac{v_B^2}{4} + \frac{1}{2}kx_{max}^2 = \frac{1}{4}kx_0^2 + \frac{1}{2}kx_{max}^2$

$x_{max} = \frac{x_0}{\sqrt{2}}$

3.4 角动量与角动量定理

角动量

质点角动量（对参考点 $O$）：$\vv{L} = \vv{r}\times\vv{p} = \vv{r}\times m\vv{v}$

大小：$L = rmv\sin\theta = mv\cdot d$（$d$ 为速度到 $O$ 的距离）

力矩

$\vv{M} = \vv{r}\times\vv{F}$，大小 $M = rF\sin\theta = F\cdot d$（$d$ 为力臂）

角动量定理

$\vv{M} = \frac{\dd\vv{L}}{\dd t}$

角动量守恒：若合力矩 $\vv{M}=0$，则 $\vv{L}=\text{const}$

常见情况：有心力（力过参考点）→ 对该点角动量守恒

例2（真题：质点系角动量定理推导）：由牛顿第二定律推导质点系角动量定理 $\vv{M}_{外} = \frac{\dd\vv{L}}{\dd t}$。
解

对第 $i$ 个质点：$\vv{r}_i \times \vv{F}_i^{(外)} + \sum_{j\neq i}\vv{r}_i\times\vv{F}_{ji} = \vv{r}_i\times m_i\vv{a}_i$

注意 $\frac{\dd}{\dd t}(\vv{r}_i\times m_i\vv{v}_i) = \vv{v}_i\times m_i\vv{v}_i + \vv{r}_i\times m_i\vv{a}_i = \vv{r}_i\times m_i\vv{a}_i$

对所有质点求和：$\sum\vv{r}_i\times\vv{F}_i^{(外)} + \sum_{i}\sum_{j\neq i}\vv{r}_i\times\vv{F}_{ji} = \frac{\dd}{\dd t}\sum\vv{r}_i\times m_i\vv{v}_i$

内力力矩之和为零（$\vv{F}_{ij}$ 和 $\vv{F}_{ji}$ 等大反向且共线）

故：$\vv{M}_{外} = \frac{\dd\vv{L}}{\dd t}$

3.5 变质量系统（真题高频！）

变质量问题的处理方法

核心：用动量定理处理 $\dd t$ 时间内的动量变化。

设 $t$ 时刻系统质量 $m$，速度 $v$；在 $\dd t$ 内增加质量 $\dd m$（其速度为 $u$）：

$$m\frac{\dd v}{\dd t} = F_{外} + (u-v)\frac{\dd m}{\dd t}$$
例3（真题：雨中卡车）：质量 $M$ 的卡车初速 $v_0$，雨水竖直均匀落入，每秒质量 $\mu$，忽略摩擦。车厢装满水量 $m_0$ 后溢出。求 $v(t)$。
解

水未满时（$t < m_0/\mu$）：系统质量增加，水平动量守恒（雨水竖直落下，水平方向无冲量）

$Mv_0 = (M+\mu t)v$，故 $v = \frac{Mv_0}{M+\mu t}$

水满后（$t \geq m_0/\mu$）：流出的水在车运动方向相对车速度为零，即流出水速度等于车速。

每秒进入的水（速度0水平分量）和流出的水（速度$v$）产生的动量变化：

$(M+m_0)\frac{\dd v}{\dd t} = 0\cdot\mu - v\cdot\mu \cdot 0 = -\mu v$... 实际上流入和流出质量相等，系统质量恒定 $M+m_0$。

分析：流入的雨水水平动量为 0，流出的水水平动量为 $v\dd m$。净效果：$(M+m_0)\dd v = -v\dd m_{流出} + 0\cdot\dd m_{流入}$，但流入和流出速率都是 $\mu$，所以 $(M+m_0)\dd v = 0\cdot\mu\dd t - v\cdot 0 = 0$？

关键：溢出的水流出时相对车速度为零 → 绝对速度等于车速 → 不带走额外动量。新进入的水水平动量为零。所以 $(M+m_0)\frac{\dd v}{\dd t} = -\mu v$（类似于阻力）

$$v = v_1 e^{-\mu(t-t_1)/(M+m_0)}$$

其中 $t_1 = m_0/\mu$，$v_1 = \frac{Mv_0}{M+m_0}$

3.8 动量与角动量守恒判断

守恒条件快速判断
  • 动量守恒：$\vv{F}_{外合}=0$；或某方向外力分量为零→该方向分量守恒
  • 角动量守恒：$\vv{M}_{外合}=0$（对某轴）；或外力过轴/平行于轴

碰撞/爆炸等短时间过程中：内力冲量>>外力冲量→动量近似守恒

易错：
  • 重力场中水平方向动量不一定守恒（看有没有水平外力）
  • 角动量守恒必须指明对哪个轴/点
  • 碰撞问题中不能用机械能守恒（除非完全弹性碰撞）

解

无外力矩 → 角动量守恒：$J_1\omega_1=J_2\omega_2$

$\omega_2=\frac{J_1}{J_2}\omega_1 > \omega_1$（转快了）

动能变化：$\Delta E_k=\frac{1}{2}J_2\omega_2^2-\frac{1}{2}J_1\omega_1^2=\frac{1}{2}J_1\omega_1^2(\frac{J_1}{J_2}-1)>0$

动能增加！能量来源：人收臂做的功（内力做功）。

例（子弹打木块+角动量）：质量 $m$ 的子弹以 $v_0$ 水平射入悬挂在细绳下端的质量 $M$ 的木块（绳长 $L$）下端。求：木块上升的最大高度。
解

碰撞瞬间（完全非弹性）→ 动量守恒：

$mv_0=(m+M)v$，$v=\frac{mv_0}{m+M}$

碰后机械能守恒（绳不做功）：

$\frac{1}{2}(m+M)v^2=(m+M)gh$

$$h=\frac{v^2}{2g}=\frac{m^2v_0^2}{2g(m+M)^2}$$

这是典型的弹道摆问题——两阶段分别用不同守恒律。

例（有心力+角动量守恒）：小球在光滑水平面上用细绳绕中心孔做圆周运动，初始半径 $r_1$，速度 $v_1$。从下方缓慢拉绳使半径减为 $r_2$，求拉力做的功。
解

对中心孔的角动量守恒（绳张力过中心）：$mv_1r_1=mv_2r_2$

$v_2=v_1r_1/r_2$

拉力做功 = 动能增量：

$$W=\frac{1}{2}m(v_2^2-v_1^2)=\frac{1}{2}mv_1^2\left(\frac{r_1^2}{r_2^2}-1\right)$$

3.10 冲量计算与应用

冲量计算要点

恒力：$\vv{I}=\vv{F}\cdot\Delta t$

变力：$\vv{I}=\int_{t_1}^{t_2}\vv{F}\dd t$

间接法：$\vv{I}=\Delta\vv{p}=m\vv{v}_2-m\vv{v}_1$（不需要知道力的具体表达式！）

例（重力冲量 vs 绳张力冲量）：质量 $m$ 的球用绳系住做竖直面上的圆周运动，求从最低点到最高点过程中绳的冲量。
解

向上为正。动量变化：$\Delta p_y=mv_{top}(-\hat{j})-mv_{bottom}(\hat{j})$... 实际上需要矢量分析。

用间接法更简单：取向上为正

$y$ 方向：从最低点（$v_y=v_B$向左/右切向）到最高点（$v_y=-v_T$向左/右切向）

其实用分量形式：重力冲量 $I_g=-mg\cdot\Delta t$（向下），剩余由张力提供。

更好的做法：利用 $\int T\dd t = \Delta p - \int(-mg)\dd t$，根据题目条件具体计算。

常见考法：求碰撞中的平均力——已知碰撞时间 $\Delta t$ 和动量变化 $\Delta p$，则 $\bar{F}=\Delta p/\Delta t$。

3.11 质心系与质心参考系

质心参考系（C系）的特点
  1. 在C系中，系统总动量恒为零：$\sum m_i\vv{v}_i'=0$
  2. C系通常是非惯性系（若质心加速运动）
  3. 碰撞问题在C系中分析特别简单
柯尼希定理

系统总动能 = 质心动能 + 相对质心的动能：

$$E_k = \frac{1}{2}Mv_C^2 + \sum\frac{1}{2}m_iv_i'^2$$

其中 $v_i'$ 为第 $i$ 个质点相对质心的速度。

应用：完全非弹性碰撞损失的动能 = 碰前在质心系中的总动能 = $\frac{1}{2}\mu v_{rel}^2$，其中 $\mu=m_1m_2/(m_1+m_2)$ 为约化质量，$v_{rel}$ 为相对速度。
