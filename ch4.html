第4章 动量与角动量

4.1 冲量与动量定理

动量与冲量

动量：$\vv{p} = m\vv{v}$

冲量：$\vv{I} = \int_{t_1}^{t_2}\vv{F}\dd t$

质点动量定理：$\vv{I} = \Delta\vv{p} = m\vv{v}_2 - m\vv{v}_1$

微分形式：$\vv{F} = \frac{\dd\vv{p}}{\dd t}$（牛顿第二定律的原始形式）

4.2 动量守恒定律

质点系动量定理与动量守恒

质点系动量定理：$\vv{F}_{外合} = \frac{\dd\vv{p}_{总}}{\dd t}$（内力不影响总动量）

动量守恒条件：$\vv{F}_{外合} = 0 \implies \vv{p}_{总} = \text{const}$

分量形式：若某方向合外力为零，则该方向动量守恒。

常见动量守恒情形：碰撞、爆炸（内力远大于外力的短过程）、光滑面上的系统、反冲。

4.3 质心与质心运动定理

质心

离散质点系：$\vv{r}_C = \frac{\sum m_i\vv{r}_i}{\sum m_i} = \frac{\sum m_i\vv{r}_i}{M}$

连续体：$\vv{r}_C = \frac{\int\vv{r}\dd m}{M}$

质心速度：$\vv{v}_C = \frac{\sum m_i\vv{v}_i}{M} = \frac{\vv{p}_{总}}{M}$

质心运动定理：$\vv{F}_{外合} = M\vv{a}_C$

例1（真题高频：弹簧-墙-双木块）：两个质量 $m$ 的木块用弹簧（$k$）连接，$A$ 靠墙，推 $B$ 压缩弹簧 $x_0$ 后释放。求：(1) 系统质心最大速度和最大加速度；(2) 弹簧最大伸长量。
解

(1) 释放瞬间弹簧弹力 $F=kx_0$，$A$ 受墙壁支撑不动，系统所受外力为墙壁支撑力 $N=kx_0$

质心加速度最大值：$a_{C,max} = \frac{kx_0}{2m}$

$A$ 与墙分离时（弹簧恢复自然长度），$B$ 速度最大为 $v_B$：$\frac{1}{2}kx_0^2 = \frac{1}{2}mv_B^2$，$v_B = x_0\sqrt{k/m}$

此后系统无外力，质心速度恒定：$v_C = \frac{mv_B}{2m} = \frac{x_0}{2}\sqrt{\frac{k}{m}}$

(2) 弹簧最大伸长时两物体共速 $v_C$，由能量守恒：$\frac{1}{2}(2m)v_C^2 = \frac{1}{2}kx_{max}^2 + \frac{1}{2}(2m)v_C^2$？不对——

正确方法：在质心系中，弹簧最大伸长时两者相对静止。弹性势能在 $A$ 离墙后守恒：$\frac{1}{2}mv_B^2 = \frac{1}{2}(2m)v_C^2 + \frac{1}{2}kx_{max}^2$

$\frac{1}{2}kx_0^2 = \frac{1}{2}(2m)\frac{v_B^2}{4} + \frac{1}{2}kx_{max}^2 = \frac{1}{4}kx_0^2 + \frac{1}{2}kx_{max}^2$

$x_{max} = \frac{x_0}{\sqrt{2}}$

4.4 角动量与角动量定理

角动量

质点角动量（对参考点 $O$）：$\vv{L} = \vv{r}\times\vv{p} = \vv{r}\times m\vv{v}$

大小：$L = rmv\sin\theta = mv\cdot d$（$d$ 为速度到 $O$ 的距离）

力矩

$\vv{M} = \vv{r}\times\vv{F}$，大小 $M = rF\sin\theta = F\cdot d$（$d$ 为力臂）

角动量定理

$\vv{M} = \frac{\dd\vv{L}}{\dd t}$

角动量守恒：若合力矩 $\vv{M}=0$，则 $\vv{L}=\text{const}$

常见情况：有心力（力过参考点）→ 对该点角动量守恒

例2（真题：质点系角动量定理推导）：由牛顿第二定律推导质点系角动量定理 $\vv{M}_{外} = \frac{\dd\vv{L}}{\dd t}$。
解

对第 $i$ 个质点：$\vv{r}_i \times \vv{F}_i^{(外)} + \sum_{j\neq i}\vv{r}_i\times\vv{F}_{ji} = \vv{r}_i\times m_i\vv{a}_i$

注意 $\frac{\dd}{\dd t}(\vv{r}_i\times m_i\vv{v}_i) = \vv{v}_i\times m_i\vv{v}_i + \vv{r}_i\times m_i\vv{a}_i = \vv{r}_i\times m_i\vv{a}_i$

对所有质点求和：$\sum\vv{r}_i\times\vv{F}_i^{(外)} + \sum_{i}\sum_{j\neq i}\vv{r}_i\times\vv{F}_{ji} = \frac{\dd}{\dd t}\sum\vv{r}_i\times m_i\vv{v}_i$

内力力矩之和为零（$\vv{F}_{ij}$ 和 $\vv{F}_{ji}$ 等大反向且共线）

故：$\vv{M}_{外} = \frac{\dd\vv{L}}{\dd t}$

4.5 变质量系统（真题高频！）

变质量问题的处理方法

核心：用动量定理处理 $\dd t$ 时间内的动量变化。

设 $t$ 时刻系统质量 $m$，速度 $v$；在 $\dd t$ 内增加质量 $\dd m$（其速度为 $u$）：

$$m\frac{\dd v}{\dd t} = F_{外} + (u-v)\frac{\dd m}{\dd t}$$
例3（真题：雨中卡车）：质量 $M$ 的卡车初速 $v_0$，雨水竖直均匀落入，每秒质量 $\mu$，忽略摩擦。车厢装满水量 $m_0$ 后溢出。求 $v(t)$。
解

水未满时（$t < m_0/\mu$）：系统质量增加，水平动量守恒（雨水竖直落下，水平方向无冲量）

$Mv_0 = (M+\mu t)v$，故 $v = \frac{Mv_0}{M+\mu t}$

水满后（$t \geq m_0/\mu$）：流出的水在车运动方向相对车速度为零，即流出水速度等于车速。

每秒进入的水（速度0水平分量）和流出的水（速度$v$）产生的动量变化：

$(M+m_0)\frac{\dd v}{\dd t} = 0\cdot\mu - v\cdot\mu \cdot 0 = -\mu v$... 实际上流入和流出质量相等，系统质量恒定 $M+m_0$。

分析：流入的雨水水平动量为 0，流出的水水平动量为 $v\dd m$。净效果：$(M+m_0)\dd v = -v\dd m_{流出} + 0\cdot\dd m_{流入}$，但流入和流出速率都是 $\mu$，所以 $(M+m_0)\dd v = 0\cdot\mu\dd t - v\cdot 0 = 0$？

关键：溢出的水流出时相对车速度为零 → 绝对速度等于车速 → 不带走额外动量。新进入的水水平动量为零。所以 $(M+m_0)\frac{\dd v}{\dd t} = -\mu v$（类似于阻力）

$$v = v_1 e^{-\mu(t-t_1)/(M+m_0)}$$

其中 $t_1 = m_0/\mu$，$v_1 = \frac{Mv_0}{M+m_0}$

4.8 动量与角动量守恒判断

守恒条件快速判断
  • 动量守恒：$\vv{F}_{外合}=0$；或某方向外力分量为零→该方向分量守恒
  • 角动量守恒：$\vv{M}_{外合}=0$（对某轴）；或外力过轴/平行于轴

碰撞/爆炸等短时间过程中：内力冲量>>外力冲量→动量近似守恒

易错：
  • 重力场中水平方向动量不一定守恒（看有没有水平外力）
  • 角动量守恒必须指明对哪个轴/点
  • 碰撞问题中不能用机械能守恒（除非完全弹性碰撞）

4.9 角动量守恒典型问题（PPT重点）

例（人+转台问题——真题高频）：人站在光滑转台上，手持哑铃伸臂旋转。初始状态总转动惯量 $J_1$，角速度 $\omega_1$。收臂后 $J_2$（$J_2
解

无外力矩 → 角动量守恒：$J_1\omega_1=J_2\omega_2$

$\omega_2=\frac{J_1}{J_2}\omega_1 > \omega_1$（转快了）

动能变化：$\Delta E_k=\frac{1}{2}J_2\omega_2^2-\frac{1}{2}J_1\omega_1^2=\frac{1}{2}J_1\omega_1^2(\frac{J_1}{J_2}-1)>0$

动能增加！能量来源：人收臂做的功（内力做功）。

例（子弹打木块+角动量）：质量 $m$ 的子弹以 $v_0$ 水平射入悬挂在细绳下端的质量 $M$ 的木块（绳长 $L$）下端。求：木块上升的最大高度。
解

碰撞瞬间（完全非弹性）→ 动量守恒：

$mv_0=(m+M)v$，$v=\frac{mv_0}{m+M}$

碰后机械能守恒（绳不做功）：

$\frac{1}{2}(m+M)v^2=(m+M)gh$

$$h=\frac{v^2}{2g}=\frac{m^2v_0^2}{2g(m+M)^2}$$

这是典型的弹道摆问题——两阶段分别用不同守恒律。

例（有心力+角动量守恒）：小球在光滑水平面上用细绳绕中心孔做圆周运动，初始半径 $r_1$，速度 $v_1$。从下方缓慢拉绳使半径减为 $r_2$，求拉力做的功。
解

对中心孔的角动量守恒（绳张力过中心）：$mv_1r_1=mv_2r_2$

$v_2=v_1r_1/r_2$

拉力做功 = 动能增量：

$$W=\frac{1}{2}m(v_2^2-v_1^2)=\frac{1}{2}mv_1^2\left(\frac{r_1^2}{r_2^2}-1\right)$$

4.10 冲量计算与应用

冲量计算要点

恒力：$\vv{I}=\vv{F}\cdot\Delta t$

变力：$\vv{I}=\int_{t_1}^{t_2}\vv{F}\dd t$

间接法：$\vv{I}=\Delta\vv{p}=m\vv{v}_2-m\vv{v}_1$（不需要知道力的具体表达式！）

例（重力冲量 vs 绳张力冲量）：质量 $m$ 的球用绳系住做竖直面上的圆周运动，求从最低点到最高点过程中绳的冲量。
解

向上为正。动量变化：$\Delta p_y=mv_{top}(-\hat{j})-mv_{bottom}(\hat{j})$... 实际上需要矢量分析。

用间接法更简单：取向上为正

$y$ 方向：从最低点（$v_y=v_B$向左/右切向）到最高点（$v_y=-v_T$向左/右切向）

其实用分量形式：重力冲量 $I_g=-mg\cdot\Delta t$（向下），剩余由张力提供。

更好的做法：利用 $\int T\dd t = \Delta p - \int(-mg)\dd t$，根据题目条件具体计算。

常见考法：求碰撞中的平均力——已知碰撞时间 $\Delta t$ 和动量变化 $\Delta p$，则 $\bar{F}=\Delta p/\Delta t$。

4.11 质心系与质心参考系

质心参考系（C系）的特点
  1. 在C系中，系统总动量恒为零：$\sum m_i\vv{v}_i'=0$
  2. C系通常是非惯性系（若质心加速运动）
  3. 碰撞问题在C系中分析特别简单
柯尼希定理

系统总动能 = 质心动能 + 相对质心的动能：

$$E_k = \frac{1}{2}Mv_C^2 + \sum\frac{1}{2}m_iv_i'^2$$

其中 $v_i'$ 为第 $i$ 个质点相对质心的速度。

应用：完全非弹性碰撞损失的动能 = 碰前在质心系中的总动能 = $\frac{1}{2}\mu v_{rel}^2$，其中 $\mu=m_1m_2/(m_1+m_2)$ 为约化质量，$v_{rel}$ 为相对速度。

4.12 碰撞问题专题（PPT重点）

一维弹性碰撞完整推导（一般情况）

两物体 $m_1$（初速 $v_1$）和 $m_2$（初速 $v_2$）发生一维弹性碰撞，碰后速度分别为 $v_1'$ 和 $v_2'$。

守恒方程：

动量守恒：

$$m_1 v_1 + m_2 v_2 = m_1 v_1' + m_2 v_2' \quad\cdots(1)$$

动能守恒：

$$\frac{1}{2}m_1 v_1^2 + \frac{1}{2}m_2 v_2^2 = \frac{1}{2}m_1 v_1'^2 + \frac{1}{2}m_2 v_2'^2 \quad\cdots(2)$$

推导技巧：将 (1) 改写为 $m_1(v_1 - v_1') = m_2(v_2' - v_2)$ ...(1')

将 (2) 改写为 $m_1(v_1^2 - v_1'^2) = m_2(v_2'^2 - v_2^2)$

即 $m_1(v_1 - v_1')(v_1 + v_1') = m_2(v_2' - v_2)(v_2' + v_2)$ ...(2')

(2') 除以 (1')：

$$v_1 + v_1' = v_2' + v_2$$

即 $v_1 - v_2 = -(v_1' - v_2')$，也就是相对速度反向（恢复系数 $e=1$）...(3)

联立 (1) 和 (3) 解得：

$$\boxed{v_1' = \frac{(m_1-m_2)v_1 + 2m_2 v_2}{m_1+m_2}}$$ $$\boxed{v_2' = \frac{(m_2-m_1)v_2 + 2m_1 v_1}{m_1+m_2}}$$
特殊情况速查：
  • $v_2=0$（靶粒子静止）：退化为 $v_1'=\frac{m_1-m_2}{m_1+m_2}v_1$，$v_2'=\frac{2m_1}{m_1+m_2}v_1$
  • $m_1=m_2$：$v_1'=v_2$，$v_2'=v_1$（速度交换）
  • $m_1\gg m_2$，$v_2=0$：$v_1'\approx v_1$，$v_2'\approx 2v_1$（轻粒子被弹飞）
  • $m_1\ll m_2$，$v_2=0$：$v_1'\approx -v_1$，$v_2'\approx 0$（轻粒子反弹）
例（中子与原子核弹性碰撞——能量传递率）：质量为 $m$ 的中子以速度 $v_0$ 与静止的质量为 $M$ 的原子核发生正碰（一维弹性碰撞）。求碰撞后原子核获得的动能占中子初始动能的比例（能量传递率）。
解

碰后原子核速度：$v_2' = \frac{2m}{m+M}v_0$

原子核获得的动能：

$$E_{k2} = \frac{1}{2}Mv_2'^2 = \frac{1}{2}M\cdot\frac{4m^2 v_0^2}{(m+M)^2} = \frac{2Mm^2 v_0^2}{(m+M)^2}$$

中子初始动能：$E_{k0} = \frac{1}{2}mv_0^2$

能量传递率：

$$\eta = \frac{E_{k2}}{E_{k0}} = \frac{4mM}{(m+M)^2}$$

物理讨论：

  • $m=M$ 时：$\eta = 4mM/(2m)^2 = 1$，能量全部传递！这就是为什么用氢原子核（质子）作为中子减速剂最有效——质子质量近似等于中子质量。
  • $M\gg m$ 时：$\eta \approx 4m/M \to 0$，几乎不传递能量（用重核减速效率极低）。
  • $M\ll m$ 时：$\eta \approx 4M/m \to 0$，同样效率低。

$\eta(M)$ 在 $m=M$ 时取最大值 $1$，这是一个二次函数极值问题。

二维碰撞简介（斜碰）

二维弹性碰撞基本关系

$m_1$ 以 $v_0$ 沿 $x$ 轴正方向碰静止的 $m_2$，碰后 $m_1$ 速度方向与 $x$ 轴成 $\theta_1$ 角，$m_2$ 速度方向与 $x$ 轴成 $\theta_2$ 角。

三个方程（动量 $x$、动量 $y$、动能）：

$$m_1 v_0 = m_1 v_1'\cos\theta_1 + m_2 v_2'\cos\theta_2 \quad (x\text{ 方向})$$ $$0 = m_1 v_1'\sin\theta_1 - m_2 v_2'\sin\theta_2 \quad (y\text{ 方向})$$ $$\frac{1}{2}m_1 v_0^2 = \frac{1}{2}m_1 v_1'^2 + \frac{1}{2}m_2 v_2'^2 \quad (\text{动能})$$

四个未知数（$v_1', v_2', \theta_1, \theta_2$），三个方程 → 需要一个额外条件（如给定 $\theta_1$ 或 $\theta_2$）才能求解。

等质量斜碰的重要结论：当 $m_1 = m_2$ 且 $m_2$ 初始静止时，碰后两者速度方向互相垂直，即 $\theta_1 + \theta_2 = 90°$。

证明：由动能守恒 $v_0^2 = v_1'^2 + v_2'^2$，而由动量守恒 $\vv{v}_0 = \vv{v}_1' + \vv{v}_2'$，两边平方得 $v_0^2 = v_1'^2 + v_2'^2 + 2\vv{v}_1'\cdot\vv{v}_2'$，比较可得 $\vv{v}_1'\cdot\vv{v}_2' = 0$，即两者垂直。

考试要点：
  • 二维碰撞比一维多一个未知量，必须额外给定一个条件
  • 等质量弹性碰撞中"碰后速度垂直"是常考结论，可以直接引用
  • 实际考题通常给定散射角或碰撞参数，不会出现无法求解的情况

4.13 火箭推进（变质量）

齐奥尔科夫斯基火箭方程推导

设 $t$ 时刻火箭质量 $m$，速度 $v$。在 $\dd t$ 内喷出质量 $|\dd m|$（$\dd m<0$），喷出气体相对火箭的速度为 $v_e$（向后喷出）。

动量守恒（忽略外力/重力的理想情况）：

$t$ 时刻系统动量：$mv$

$t+\dd t$ 时刻：火箭 $(m+\dd m)(v+\dd v)$ + 喷出气体 $(-\dd m)(v - v_e)$

（注意 $\dd m < 0$，所以喷出质量为 $-\dd m > 0$）

$$mv = (m+\dd m)(v+\dd v) + (-\dd m)(v - v_e)$$

展开：

$$mv = mv + m\dd v + v\dd m + \dd m\dd v - v\dd m + v_e\dd m$$

略去二阶小量 $\dd m\dd v$：

$$0 = m\dd v + v_e\dd m$$ $$m\dd v = -v_e\dd m = v_e(-\dd m)$$

分离变量并积分，设初始质量 $m_0$、初速 $v_0 = 0$，末质量 $m$：

$$\int_0^v \dd v = -v_e\int_{m_0}^{m}\frac{\dd m}{m}$$ $$\boxed{v = v_e \ln\frac{m_0}{m}}$$

这就是齐奥尔科夫斯基火箭方程（Tsiolkovsky rocket equation），是航天工程的基础公式。

物理含义：
  • $v_e$ 是喷气速度（比冲的体现），$v_e$ 越大，同样质量比下速度越高
  • $m_0/m$ 是质量比（初始质量/最终质量），包含了燃料占比信息
  • 速度与质量比的对数成正比——要获得高速度，需要指数级的质量比，这就是火箭工程的核心困难
  • 若考虑重力（竖直发射）：$v = v_e\ln(m_0/m) - gt$
例（火箭速度计算）：一火箭在太空中从静止开始加速，喷气相对火箭的速度为 $v_e = 3000\,\text{m/s}$，初始总质量 $m_0 = 5000\,\text{kg}$，燃料质量 $m_f = 4000\,\text{kg}$。求：(1) 燃料耗尽时火箭的速度；(2) 若要达到第一宇宙速度 $v_1 = 7900\,\text{m/s}$，质量比 $m_0/m$ 至少是多少？
解

(1) 燃料耗尽后火箭质量 $m = m_0 - m_f = 5000 - 4000 = 1000\,\text{kg}$

质量比 $m_0/m = 5000/1000 = 5$

$$v = v_e\ln\frac{m_0}{m} = 3000\times\ln 5 = 3000\times 1.609 = 4828\,\text{m/s}$$

(2) 令 $v_1 = v_e\ln(m_0/m)$：

$$\ln\frac{m_0}{m} = \frac{v_1}{v_e} = \frac{7900}{3000} = 2.633$$ $$\frac{m_0}{m} = e^{2.633} \approx 13.9$$

即火箭初始质量必须是最终质量的近 14 倍！燃料占初始质量的 $1-1/13.9 \approx 93\%$，这就是多级火箭被发明的原因——单级火箭的结构质量比几乎不可能达到这个要求。

变质量问题解题要点：
  • 火箭方程推导中的关键：喷出气体的速度是相对火箭的 $v_e$，不是相对地面的速度
  • $\dd m < 0$（火箭质量减少），注意正负号！
  • 实际火箭还要考虑重力和空气阻力，公式会更复杂
  • 多级火箭的优势：抛掉空壳结构后，有效质量比 $m_0/m$ 增大

4.14 对称性与守恒律（PPT补充）

Noether定理——对称性与守恒律的深层联系

物理规律的每一种对称性都对应一个守恒量（Noether, 1918）：

对称性含义对应守恒量
时间平移对称性物理定律不随时间改变能量守恒
空间平移对称性物理定律不因位置改变动量守恒
空间旋转对称性物理定律不因方向改变角动量守恒
物理理解：
  • 如果空间是均匀的（平移不变），则没有"优先位置"对粒子施力 → 动量守恒
  • 如果空间是各向同性的（旋转不变），则没有"优先方向"产生力矩 → 角动量守恒
  • 如果物理规律不随时间变化，则系统的总能量不变 → 能量守恒

这个联系在经典力学和量子力学中都成立，是物理学中最深刻的结论之一。

4.15 恢复系数与碰撞分类详解（PPT补充）

恢复系数 $e$ 的定义与意义

恢复系数描述碰撞的"弹性程度"：

$$\boxed{e = -\frac{v_2'-v_1'}{v_2-v_1} = \frac{\text{碰后相对分离速率}}{\text{碰前相对接近速率}}}$$
碰撞类型$e$ 值动能特点
完全弹性碰撞$e=1$完全守恒相对速度反向不变
非完全弹性碰撞$0部分损失实际碰撞大多属此类
完全非弹性碰撞$e=0$最大损失碰后粘合，共速运动
例（用恢复系数解碰撞）：质量 $m_1=2\,\text{kg}$ 以 $v_1=4\,\text{m/s}$ 碰静止的 $m_2=3\,\text{kg}$，恢复系数 $e=0.6$。求碰后速度和动能损失。
解

动量守恒：$2\times4+3\times0=2v_1'+3v_2'$，即 $2v_1'+3v_2'=8$ ...(1)

恢复系数：$e=\frac{v_2'-v_1'}{v_1-v_2}=\frac{v_2'-v_1'}{4}=0.6$，即 $v_2'-v_1'=2.4$ ...(2)

由 (1)(2) 解得：$v_1'=0.16\,\text{m/s}$，$v_2'=2.56\,\text{m/s}$

动能损失：$\Delta E_k=\frac{1}{2}\times2\times16-\frac{1}{2}\times2\times0.16^2-\frac{1}{2}\times3\times2.56^2=16-0.026-9.83=6.14\,\text{J}$

损失率：$6.14/16=38.4\%$

解题技巧：用恢复系数 $e$ 代替动能守恒方程，可以统一处理所有类型的碰撞。动量守恒+恢复系数关系 = 两个方程解两个未知数（$v_1'$, $v_2'$），比处理二次方程简单得多。

4.16 动量定理的积分形式与平均力（PPT补充）

短时间碰撞中的平均力

碰撞时间 $\Delta t$ 极短，但力 $F(t)$ 极大。定义平均力：

$$\bar{F} = \frac{\Delta p}{\Delta t} = \frac{m(v_2-v_1)}{\Delta t}$$

这是动量定理最常见的应用之一。

例（水流冲击力）：水以速率 $v=10\,\text{m/s}$ 从截面积 $A=5\times10^{-4}\,\text{m}^2$ 的管口射出，垂直打在墙上后沿墙面流走（不反弹）。求水流对墙的平均力。
解

单位时间内打到墙上的水的质量：$\dot{m}=\rho Av=1000\times5\times10^{-4}\times10=5\,\text{kg/s}$

水流方向的动量变化率（水从速度 $v$ 变为 0）：

$$F=\dot{m}\cdot v=5\times10=50\,\text{N}$$

若水流被墙弹回（反向以 $v$ 离开），则 $F=\dot{m}\cdot2v=100\,\text{N}$。

连续流体的动量定理：对于持续的质量流（水流、气流、传送带上的沙等），用 $F=\dot{m}\cdot\Delta v$（$\dot{m}=\dd m/\dd t$ 为质量流率）比逐个分析方便得多。
