第6&7章习题 振动与波

题 6-2 弹簧振子振动方程

题目：一弹簧振子，弹簧劲度系数 $k=50\;\text{N/m}$，物体质量 $m=2\;\text{kg}$。$t=0$ 时 $x_0=0.05\;\text{m}$，$v_0=0$。写出振动方程。
解题过程

第一步：求角频率 $\omega$

$$\omega = \sqrt{\frac{k}{m}} = \sqrt{\frac{50}{2}} = \sqrt{25} = 5\;\text{rad/s}$$

第二步：求振幅 $A$

利用初始条件公式：

$$A = \sqrt{x_0^2 + \left(\frac{v_0}{\omega}\right)^2} = \sqrt{(0.05)^2 + 0} = 0.05\;\text{m}$$

第三步：求初相 $\varphi_0$

由初相判据：

$$\tan\varphi_0 = -\frac{v_0}{\omega x_0} = -\frac{0}{5 \times 0.05} = 0$$

$\tan\varphi_0 = 0$ 有两个解：$\varphi_0 = 0$ 或 $\varphi_0 = \pi$。

因为 $x_0 = 0.05 > 0$，即 $A\cos\varphi_0 > 0$，所以 $\cos\varphi_0 > 0$，取 $\varphi_0 = 0$。

第四步：写出振动方程

$$\boxed{x = 0.05\cos(5t)\;\text{m}}$$
提示：确定初相 $\varphi_0$ 时，仅靠 $\tan\varphi_0$ 的值不够，还需结合 $x_0$ 和 $v_0$ 的正负号来确定 $\varphi_0$ 所在象限。规则：$x_0 = A\cos\varphi_0$ 定余弦符号，$v_0 = -A\omega\sin\varphi_0$ 定正弦符号。

题 6-7 SHM 的运动学量

题目：一简谐振动 $x = 0.04\cos(2\pi t + \pi/3)$（m）。求：
(1) 振幅、周期、初相；
(2) $t=0$ 时的位移、速度、加速度；
(3) 最大速度和最大加速度。
解题过程

(1) 基本参量

与标准形式 $x = A\cos(\omega t + \varphi_0)$ 对比可得：

  • 振幅 $A = 0.04\;\text{m}$
  • 角频率 $\omega = 2\pi\;\text{rad/s}$，周期 $T = \dfrac{2\pi}{\omega} = 1\;\text{s}$
  • 初相 $\varphi_0 = \dfrac{\pi}{3}$

(2) $t = 0$ 时刻的运动学量

位移：

$$x(0) = 0.04\cos\frac{\pi}{3} = 0.04 \times 0.5 = 0.02\;\text{m}$$

速度（对位移求导）：

$$v = \frac{\dd x}{\dd t} = -A\omega\sin(\omega t + \varphi_0)$$ $$v(0) = -0.04 \times 2\pi \times \sin\frac{\pi}{3} = -0.04 \times 2\pi \times \frac{\sqrt{3}}{2} \approx -0.218\;\text{m/s}$$

加速度（对速度求导）：

$$a = \frac{\dd v}{\dd t} = -A\omega^2\cos(\omega t + \varphi_0)$$ $$a(0) = -0.04 \times (2\pi)^2 \times \cos\frac{\pi}{3} = -0.04 \times 4\pi^2 \times 0.5 \approx -0.789\;\text{m/s}^2$$

(3) 最大速度和最大加速度

$$v_{\max} = A\omega = 0.04 \times 2\pi \approx 0.251\;\text{m/s}$$ $$a_{\max} = A\omega^2 = 0.04 \times 4\pi^2 \approx 1.579\;\text{m/s}^2$$
易错点：速度公式带负号！$v = -A\omega\sin(\omega t + \varphi_0)$，不要漏掉负号。加速度与位移反号：$a = -\omega^2 x$，这是 SHM 的本质特征。

题 6-11 同频率 SHM 的合成

题目：两个同频率简谐振动：$x_1 = 3\cos(2t + \pi/6)\;\text{cm}$，$x_2 = 4\cos(2t + 2\pi/3)\;\text{cm}$。求合振动。
解题过程

第一步：确定已知量

  • $A_1 = 3\;\text{cm}$，$\varphi_1 = \pi/6$
  • $A_2 = 4\;\text{cm}$，$\varphi_2 = 2\pi/3$
  • 相位差 $\Delta\varphi = \varphi_2 - \varphi_1 = \dfrac{2\pi}{3} - \dfrac{\pi}{6} = \dfrac{\pi}{2}$

第二步：求合振幅

$$A = \sqrt{A_1^2 + A_2^2 + 2A_1 A_2\cos\Delta\varphi}$$ $$= \sqrt{9 + 16 + 2 \times 3 \times 4 \times \cos\frac{\pi}{2}}$$ $$= \sqrt{9 + 16 + 0} = \sqrt{25} = 5\;\text{cm}$$

第三步：求合振动初相

用旋转矢量的分量合成：

$$\tan\varphi = \frac{A_1\sin\varphi_1 + A_2\sin\varphi_2}{A_1\cos\varphi_1 + A_2\cos\varphi_2}$$

分子：

$$3\sin\frac{\pi}{6} + 4\sin\frac{2\pi}{3} = 3 \times 0.5 + 4 \times \frac{\sqrt{3}}{2} = 1.5 + 3.464 = 4.964$$

分母：

$$3\cos\frac{\pi}{6} + 4\cos\frac{2\pi}{3} = 3 \times \frac{\sqrt{3}}{2} + 4 \times (-0.5) = 2.598 - 2 = 0.598$$ $$\tan\varphi = \frac{4.964}{0.598} \approx 8.30$$

分子 $> 0$，分母 $> 0$，$\varphi$ 在第一象限：

$$\varphi = \arctan(8.30) \approx 83.1° \approx 1.45\;\text{rad}$$

第四步：写出合振动

$$\boxed{x = 5\cos(2t + 1.45)\;\text{cm}}$$
提示：当 $\Delta\varphi = \pi/2$（正交）时，合振幅简化为 $A = \sqrt{A_1^2 + A_2^2}$，即 3-4-5 直角三角形。这是一个很好的快速验算手段。
易错点：用 $\arctan$ 求初相时，必须判断分子分母的正负号来确定象限。计算器只返回 $(-\pi/2, \pi/2)$ 的值，若分母为负需加 $\pi$。

题 6-14 平面简谐波的波动方程

题目：一平面简谐波沿 $+x$ 方向传播，波速 $u = 10\;\text{m/s}$，频率 $\nu = 5\;\text{Hz}$。在 $x = 0$ 处 $t = 0$ 时 $y = 0$，$\dfrac{\partial y}{\partial t} > 0$。振幅 $A = 0.02\;\text{m}$。写出波动方程。
解题过程

第一步：求波动参量

$$\omega = 2\pi\nu = 2\pi \times 5 = 10\pi\;\text{rad/s}$$ $$k = \frac{\omega}{u} = \frac{10\pi}{10} = \pi\;\text{rad/m}$$

第二步：确定初相 $\varphi_0$

波动方程一般形式（沿 $+x$ 传播）：

$$y = A\cos(\omega t - kx + \varphi_0)$$

在 $x = 0$，$t = 0$ 处：

$$y(0,0) = A\cos\varphi_0 = 0 \implies \varphi_0 = \pm\frac{\pi}{2}$$

振动速度：

$$v_y = \frac{\partial y}{\partial t} = -A\omega\sin(\omega t - kx + \varphi_0)$$ $$v_y(0,0) = -A\omega\sin\varphi_0 > 0 \implies \sin\varphi_0 < 0$$

$\sin\varphi_0 < 0$，所以 $\varphi_0 = -\dfrac{\pi}{2}$。

第三步：写出波动方程

$$y = 0.02\cos\!\left(10\pi t - \pi x - \frac{\pi}{2}\right)\;\text{m}$$

利用 $\cos(\theta - \pi/2) = \sin\theta$，化简为：

$$\boxed{y = 0.02\sin(10\pi t - \pi x)\;\text{m}}$$
易错点：初相 $\varphi_0$ 的确定是波动方程最容易出错的环节。务必同时利用 $y$ 和 $v_y$ 的初始条件来联合判断 $\varphi_0$ 的象限。
注意：波动方程中的初相是原点（$x=0$）处质点的初相。若题目给出的是其他点的初始状态，需要先将 $x$ 值代入方程来建立约束。

题 6-16 由波动方程求各量

题目：已知波动方程 $y = 0.05\cos(10\pi t - 4\pi x)$（SI）。求：
(1) 波的振幅、频率、波长、波速；
(2) $x = 0.5\;\text{m}$ 处质点的振动方程和初相。
解题过程

(1) 各波动参量

与标准形式 $y = A\cos(\omega t - kx)$ 对比：

  • 振幅：$A = 0.05\;\text{m}$
  • 角频率：$\omega = 10\pi\;\text{rad/s}$，频率 $\nu = \dfrac{\omega}{2\pi} = 5\;\text{Hz}$
  • 波数：$k = 4\pi\;\text{rad/m}$，波长 $\lambda = \dfrac{2\pi}{k} = \dfrac{2\pi}{4\pi} = 0.5\;\text{m}$
  • 波速：$u = \dfrac{\omega}{k} = \dfrac{10\pi}{4\pi} = 2.5\;\text{m/s}$

(2) $x = 0.5\;\text{m}$ 处的振动方程

将 $x = 0.5$ 代入波动方程：

$$y = 0.05\cos(10\pi t - 4\pi \times 0.5) = 0.05\cos(10\pi t - 2\pi)$$

因为 $\cos(\theta - 2\pi) = \cos\theta$：

$$\boxed{y = 0.05\cos(10\pi t)\;\text{m}}$$

初相 $\varphi = 0$。

提示：$x = 0.5\;\text{m}$ 恰好等于一个波长 $\lambda$，所以该点与原点同相。一般地，相距整数个波长的两点总是同相振动。

题 6-18 弦线驻波

题目：一弦线固定两端相距 $L = 1\;\text{m}$，基频 $\nu_1 = 100\;\text{Hz}$。求：
(1) 波速；
(2) 第三谐波的频率和波长。
解题过程

前置知识：两端固定弦线的驻波条件

$$L = n\frac{\lambda_n}{2} \quad (n = 1, 2, 3, \ldots)$$ $$\nu_n = \frac{nu}{2L} = n\nu_1$$

(1) 求波速

基频对应 $n = 1$：

$$\nu_1 = \frac{u}{2L} \implies u = 2L\nu_1 = 2 \times 1 \times 100 = 200\;\text{m/s}$$

(2) 第三谐波（$n = 3$）

频率：

$$\nu_3 = 3\nu_1 = 3 \times 100 = 300\;\text{Hz}$$

波长：

$$\lambda_3 = \frac{u}{\nu_3} = \frac{200}{300} = \frac{2}{3}\;\text{m}$$

也可由 $\lambda_3 = \dfrac{2L}{3} = \dfrac{2}{3}\;\text{m}$ 直接得出。

提示：两端固定弦线只能产生基频的整数倍谐波。第 $n$ 次谐波有 $n$ 个波腹和 $n+1$ 个波节（含两端）。

题 6-22 多普勒效应

题目：波源频率 $\nu_0 = 1000\;\text{Hz}$，波速 $v = 340\;\text{m/s}$。
(1) 波源以 $v_s = 20\;\text{m/s}$ 向静止观测者运动，求观测频率；
(2) 观测者以 $v_o = 20\;\text{m/s}$ 向静止波源运动，求观测频率。
解题过程

多普勒效应公式：

$$\nu' = \nu_0 \frac{v + v_o}{v - v_s}$$

其中 $v_o$：观测者靠近波源取 $+$；$v_s$：波源靠近观测者取 $+$。

(1) 波源运动，观测者静止（$v_o = 0$，$v_s = 20\;\text{m/s}$）

$$\nu' = \nu_0 \frac{v}{v - v_s} = 1000 \times \frac{340}{340 - 20} = 1000 \times \frac{340}{320} = 1062.5\;\text{Hz}$$

(2) 观测者运动，波源静止（$v_o = 20\;\text{m/s}$，$v_s = 0$）

$$\nu' = \nu_0 \frac{v + v_o}{v} = 1000 \times \frac{340 + 20}{340} = 1000 \times \frac{360}{340} \approx 1058.8\;\text{Hz}$$

对比：两种情况结果不同！

  • 波源运动：$\nu' = 1062.5\;\text{Hz}$
  • 观测者运动：$\nu' \approx 1058.8\;\text{Hz}$

虽然相对速度相同（都是 $20\;\text{m/s}$ 的相互靠近），但多普勒效应与媒质参考系有关，不仅取决于相对运动。

易错点 - 符号约定：使用公式 $\nu' = \nu_0\dfrac{v + v_o}{v - v_s}$ 时，$v_o$ 和 $v_s$ 的符号取决于运动方向：
- $v_o$：观测者靠近波源取正，远离取负；
- $v_s$：波源靠近观测者取正，远离取负。
记忆口诀："近正远负，上接下离"（$v_o$ 在分子，靠近加；$v_s$ 在分母，靠近减）。
物理解释：波源运动改变的是波长（压缩或拉伸波列），观测者运动改变的是单位时间内接收的波峰数。两种机制不同，因此结果不同。只有当 $v_s, v_o \ll v$ 时，两者近似相等。

题 6-24 相干波叠加

题目：两列相干波在某点的相位差 $\Delta\varphi = 3\pi$，振幅 $A_1 = A_2 = A$。求该点的合振幅。
解题过程

合振幅公式：

$$A_\text{合} = \sqrt{A_1^2 + A_2^2 + 2A_1 A_2\cos\Delta\varphi}$$

代入数据：

$$A_\text{合} = \sqrt{A^2 + A^2 + 2A^2\cos 3\pi}$$

注意 $\cos 3\pi = \cos\pi = -1$：

$$A_\text{合} = \sqrt{2A^2 + 2A^2 \times (-1)} = \sqrt{2A^2 - 2A^2} = \sqrt{0} = 0$$ $$\boxed{A_\text{合} = 0 \quad\text{（完全相消干涉）}}$$
干涉条件速查：
- $\Delta\varphi = 2n\pi$（$n = 0, \pm1, \pm2, \ldots$）：完全加强，$A_\text{合} = A_1 + A_2$
- $\Delta\varphi = (2n+1)\pi$：完全减弱，$A_\text{合} = |A_1 - A_2|$
- 本题 $\Delta\varphi = 3\pi = (2 \times 1 + 1)\pi$，属于完全减弱；又因 $A_1 = A_2$，合振幅为零。
易错点：$\cos 3\pi \neq \cos 3 \times 180° = +1$。正确做法：$3\pi$ 是奇数倍 $\pi$，所以 $\cos 3\pi = \cos\pi = -1$。做题时先判断 $\Delta\varphi$ 是偶数倍还是奇数倍 $\pi$，再确定加强还是减弱。
